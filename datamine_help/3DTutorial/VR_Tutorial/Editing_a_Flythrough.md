# Editing a Flythrough - Exercise

This exercise outlines how you can edit the settings of an existing flythrough simulation, specifically, how to change the viewing angle of the flythrough camera.

## Prerequisites

  * You have loaded an unmodified version of the VR-Tutorial3.evr project file from C:\Database\DMTutorials\Projects\S3VRTut\Standard.

  * You have saved this file under a different name, preserving the original copy.

  * You have created a string as shown in Manually drawing a string.

  * You have smoothed your string as shown in Smoothing a string.

  * You have created a flythrough animation as shown in [Creating a Flythrough](<Creating_and_Conditioning_a_Flight_Path.md>).

## Exercise: Editing a flythrough's viewpoint

When you run the flythrough the view changes as the object moves along the string. You may want the view to look at a specific point when the object is moving. To do this; 

  1. Press Play to start the flythrough

  2. Select the Look At icon from the toolbar

  3. The icon will change shape when it is in the workspace.

  4. Place the mouse cursor on the place you want the view to point towards.

  5. Click the left mouse button to set the viewpoint.

  6. To stop the Look At mode select the icon again. The view will change as the flythrough progresses.

If you require additional information about any of the functions outlined in this Help file, please refer to your online Help for your VR application.
