# File Locations

Assuming that a default installation has been performed, all sample files referenced by this tutorial can be found under:

  * C:\Database\DMTutorials\Data\VBUG\Datamine

  * C:\Database\DMTutorials\Data\VBOP\Datamine

  * C:\Database\DMTutorials\Data\VBOP\Pics

  * C:\Database\DMTutorials\Data\VBOP\Text

  * C:\Database\DMTutorials\Projects\S3VRTut\ProjFiles\Standard

These folders contain either project files or a series of data-type-specific files including amongst others, native Datamine format files(*.dm), image and text files. Please note that these folders contain many files and that only the files listed in the table below are relevant to the 3D window tutorial.
