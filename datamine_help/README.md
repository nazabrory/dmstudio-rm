# Datamine Help Documentation Library (Markdown Edition)

Welcome to the Markdown-converted edition of the Datamine Help Documentation. This library has been compiled, cleaned, and formatted specifically to be easily digestible for AI agent context windows and LLM retrieval.

The documentation is organized into subfolders corresponding to each original `.chm` help file, each containing its own structured sitemap index (`README.md`).

---

## 📚 Available Reference Manuals & Tutorials

Below is the list of converted manuals, their sizes, and quick links to their main table of contents:

### 1. [Studio RM Reference Manual & Command Help](StudioRM/README.md)
* **Folder**: `StudioRM/`
* **Size**: 3,755 pages
* **Description**: The comprehensive reference manual for Datamine Studio RM. It includes detailed guides on command syntax, processes, plots, declination optimization, decline optimizer, uniform conditioning, and core Studio RM operations.

### 2. [Geological Modeling Tutorial](StudioRM_GeoMod_Tutorial/README.md)
* **Folder**: `StudioRM_GeoMod_Tutorial/`
* **Size**: 67 pages
* **Description**: Step-by-step tutorial on geological modeling workflows, wireframing, and block model construction within Studio RM.

### 3. [Grade Estimation Tutorial](StudioRM_GradeEst_Tutorial/README.md)
* **Folder**: `StudioRM_GradeEst_Tutorial/`
* **Size**: 63 pages
* **Description**: Comprehensive walkthrough of grade estimation, geostatistics, variography, and block model evaluation methods.

### 4. [3D View & VR Tutorial](3DTutorial/README.md)
* **Folder**: `3DTutorial/`
* **Size**: 42 pages
* **Description**: Tutorial for configuring 3D visualizations, virtual reality environments, and presentation setups inside Datamine.

### 5. [Data Presentation Tutorial](Studio_DataPresTutorial/README.md)
* **Folder**: `Studio_DataPresTutorial/`
* **Size**: 46 pages
* **Description**: Guides on generating reports, logs, plot sheets, and custom visualization presentations of Datamine data.

### 6. [Macros Tutorial](Studio_Macros_Tutorial/README.md)
* **Folder**: `Studio_Macros_Tutorial/`
* **Size**: 30 pages
* **Description**: A developer's guide on writing, recording, editing, and executing Datamine macros (`.mac` / `.men` scripts).

### 7. [Scripting Tutorial](Studio_ScriptingTutorial/README.md)
* **Folder**: `Studio_ScriptingTutorial/`
* **Size**: 27 pages
* **Description**: A developer's guide to customizing Datamine products using modern COM/ActiveX automation scripting (JavaScript/VBScript) and building custom HTML interfaces.

## 📁 Navigating the Folders

* Each subdirectory listed above contains a local `README.md` which maps the **original Table of Contents hierarchy** with markdown links to individual topic pages.
* All external media and diagrams are stored inside local `Images/` or `Resources/Images/` directories within each subfolder, maintaining relative paths so that images display correctly when markdown files are opened.
* All links between pages have been normalized to point to `.md` files rather than `.htm` files to enable smooth browsing in Markdown editors.
