# Format a 3D Overlay Group  
  
To access this screen:

  * In the **Plots** window, right-click a **3D Overlay Group** menu item and select **3D Overlay Group Properties...**

These settings control general settings for the selected [3D Overlay](<../PLOTS_LOGS/3D%20Overlay%20Concept.md>) group of a plot projection.

[See 3D overlays and rendering methods](<../PLOTS_LOGS/3D%20Overlay%20Concept.md>)

Activity Steps:

  1. Set **Pixels Per Inch** to the resolution you need, up to a maximum of 500. 

  2. Set **Use Secondary Clipping**. By default, secondary clipping is not displayed in 3D plot projections.
