# Assign Keyboard Shortcut

Use the **Assign Keyboard Shortcuts** dialog to apply a keyboard shortcut (also referred to as an 'accelerator) to a command.

Activity Steps

  1. In the [Customize](<customize.md>) dialog, select Keyboard.

  2. Select a command **Category** , or pick _All Commands_.

  3. Using the **Commands** list, pick a command.

If a shortcut key already exists for the selected command, it appears under **Current Keys**. You can assign another key if you wish, or unassign the current key first by selecting and clicking **Remove**.

  4. Click into the Press New Shortcut Key field.

  5. Press a key to become the shortcut for the selected command.

The pressed key appears in the active field.  

**Tip** : You can use combinations of keys and modifiers such as <CTRL>, <SHIFT> and <ALT> when selecting a shortcut.

  6. Click **Assign** to assign the shortcut key to the command.

  7. **Close** the dialog.

Your new keyboard shortcut can be used.
