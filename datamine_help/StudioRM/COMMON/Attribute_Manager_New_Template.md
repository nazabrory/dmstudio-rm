# Add New Attribute Template

To access this screen:

  * Using the [Attribute Manager](<Attribute_Manager.md>), click Add Template.

  * Using the [Attribute Manager](<Attribute_Manager.md>), right click the index column of an entry in the Object Table and select Save as New Template.

This screen lets you record the name of a new, empty template to be added to the Object Template Table (the lower half of the Attribute Manager where default templates are defined) or it can be used to define a description in which to store the custom attributes of a selected object in memory.
