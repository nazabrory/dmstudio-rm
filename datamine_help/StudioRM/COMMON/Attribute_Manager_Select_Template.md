# Select Template

To access this screen:

  * Using the [Attribute Manager](<Attribute_Manager.md>), right-click the index cell (representing an object name) in the **Object Table** , select Copy from Template.

  * Using the [Attribute Manager](<Attribute_Manager.md>), right-click the index cell (representing an object name) in the **Object Table** , select Copy to Template.

This screen available attribute templates. Select a template that will either be used to store custom attribute details of a selected object (Copy to Template) or used to extract custom attributes for the purpose of appending that object (Copy from Template).
