# Batch Settings for Veins

To access this screen:

  * Display the **[Create Vein Surfaces](<Create_Vein_Surface.md>)** task and expand **Value** and choose _< Batch>_, then **Batch Settings**.

This screen is used to enable or disable values to be used for modelling discrete vein or vein-like structures. It is used by the Create Vein Surfaces screen to control which values are used in batch processing.

Click OK to confirm the scope of batch processing and return to the **Create Vein Surfaces** screen.
