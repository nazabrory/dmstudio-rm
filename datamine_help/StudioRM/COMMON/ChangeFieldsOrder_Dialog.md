# Change Fields Order

To access this screen:

  * In the [Compositor](<Compositor%20Control%20Bar%20Overview.md>) control bar, right-click to select Change Fields Order.

Change or reset the order of the fields displayed in a grid table.

Fields on the left are listed in the same top-bottom order as the **Compositor** window. Pick any field to shift it up or down the list, or **Reset** the list to its default order.
