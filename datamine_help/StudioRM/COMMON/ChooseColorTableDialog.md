# Choose a Color Table

To access this screen:

  * Click Color Results on the [Table View](<Evaluation2DStringsTableViewDialog.md>) screen.

Select a loaded colour table and apply the formatting rules contained therein to the current dynamic evaluation report. See [Dynamic Evaluation Reports](<Dynamic%20Evaluation%20Report%20Introduction.md#color>).
