# Distance Contours 

To access this function:

  * **Geology** ribbon >> **Contours** >> **Distance**

  * Enter "generate-distance-contours" into the [**Command Line**](<Command_Toolbar.md>).

This function can be used to generate output contours and a grid model representing the distance of each surface point from neighbouring samples. 

You can pre-select your data to generate contours around a subset of the input data object, and you can do this whilst the Generate Contours dialog is on display if you wish. This applies to any of the input data types (points, fault strings or clipping polygons).

**Note** : This function will default to overwriting the current object (string, model or wireframe) when it outputs data. You can create your own objects as part of this procedure, or you can choose to create the output objects manually first (using the [Current Object](<Concept_Current_Object.md>) toolbar) then select them on the Output Data screen. It's up to you.

The [generate-distance-contours](<../command_help/generate-distance-contours.md>) command offers:

    * Smooth contour lines
    * Gridded data output
    * Distance-from-sample maps
