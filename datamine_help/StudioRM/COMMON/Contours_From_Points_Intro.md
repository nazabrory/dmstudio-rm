# Contours from Points

To access this screen:

  * **Geology** ribbon >> **Contours** >> **Points**

  * Enter "generate-contours-from-points" into the [**Command Line**](<Command_Toolbar.md>).

Generate elevation- and grade-based contours, using either points or string data as inputs (and fault strings, if present).

You can pre-select data to generate contours around a subset of the input data object, and you can do this whilst the Generate Contours dialog is on display if you wish. This applies to any of the input data types (points, fault strings or clipping polygons).

Data is generated within the currently loaded contour function output, if one exists. You can create your own objects as part of this procedure, or you can choose to create the output objects manually first (using the [Current Object](<Concept_Current_Object.md>) toolbar) then select them on the Output Data screen. It's up to you.

For distance contouring functionality (to create contours representing the proximity of a surface grid point to a sample), see [Distance Contours - Introduction](<Contours_Distance_Introduction.md>).
