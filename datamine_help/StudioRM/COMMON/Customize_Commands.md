# Customize Commands

To access this dialog:

  * Display the [Customize](<customize.md>) screen and activate the Commands tab.

Add and remove buttons from displayed toolbars by dragging and dropping buttons from the Commands list to the appropriate toolbar in the toolbar section of the application interface.

To add a command to an existing toolbar:

  1. Select the toolbar containing the item you wish to add to another toolbar (or select _All Menus_ to see a list of all available commands).

  2. Click (left) and drag the item from the Commands window to the appropriate toolbar.

  3. Release the mouse.

**Note** : create a new, empty toolbar to add commands to using the[Toolbars](<Customize_Toolbars.md>) tab and selecting New.
