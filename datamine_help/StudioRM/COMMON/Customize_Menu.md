# Customize Menu

To access this dialog:

  * Display the [Customize](<customize.md>) screen and activate the Menu tab.

Specify general settings for some of your application's context menus. 

**Note** : application frame menus are no longer used in Studio products as they have been replaced by more effective ribbons. Hence, these legacy and hidden menus can't be customized here. Similarly, not all context menus are available for editing as they contain essential functions.

To modify context menu contents:

  1. Display the **Customize Menu** panel.

  2. Expand the **Select context menu** list.

A list of all customizable menus displays.

  3. Select a menu to customize.

A preview of the menu appears on screen.

  4. Using the menu preview, drag and drop items to a new position in the list.

  5. Right click a menu item to reveal the following options:

     * **Reset to Default** perform a 'factory reset' on the menu. All commands return to their default positions and configuration.

     * **Copy Button Image** copy the button image (if there is one) to the clipboard. It can be pasted into other documents.

     * **Delete** remove the item from the menu. 

     * **Button Appearance** display the **Button Appearance** screen to edit the button type and label. 

**Note** : not all options are available as some menu items require specific configurations.

     * **Image, Text, Image and Text** swap between button types.

     * **Start Group** add a menu separator before the menu item.

**Tip** : click **Reset** to return the selected menu to default items. This will reinstate deleted items.

  6. Set up your **Menu animations** , using the following options (which apply to all menus, not just the selected one):

     * None no menu animations are performed.
     * Unfold: all menus will unfold from the top downwards.

     * Slide: menus will slide into place.

     * Fade: menus will gradually fade into view.

**Note** : not all menu transitional effects are supported by all operating system and graphic card combinations. If you are having problems with the menu transitions, set the above field to [none].

  7. Choose if you want to display **Menu shadows**. 

**Tip** : if you're struggling with system performance, try unchecking **Menu shadows** and **Menu animations**.
