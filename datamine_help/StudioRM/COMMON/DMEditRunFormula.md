# Run Formula

Generates values for a particular column (field) in the table according to a formula.

Formulae are not saved to the table - only the values are saved. Formulae only exist until they have been run, and the cells contain values.

Select a Column, enter a known Formula, or browse to display the **[Edit Formula](<DmEditEditFormula.md>)** screen.
