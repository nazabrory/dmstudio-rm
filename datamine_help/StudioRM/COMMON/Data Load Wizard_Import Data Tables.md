# Data Load Wizard: Import Data Tables

To access this screen:

  * Using the Data Load Wizard, complete the [Import Data Types](<Data%20Load%20Wizard_%20Import%20Data%20Types.md>) screen.

This screen is used to create a list of data tables of the type(s) selected on the [Import Data Types](<Data%20Load%20Wizard_%20Import%20Data%20Types.md>) screen. 

Initially empty, you can add file(s) to the list by clicking Add. This displays the [Data Import](<data%20import%20dialog.md>) dialog, used to select a file, data driver and (optionally) filter settings that will be used to import data into your application. 

When import settings are defined (the screens displayed during data source driver and file selection depend on the associated driver), your data file is added to the list on the Import Data Tables screen. 

Add as many files as required by repeating this process. You can remove items from the list by highlighting them and clicking Remove.

Selecting Next loads the selected files and displays the [Complete Load](<data%20load%20wizard_%20complete%20load.md>) screen.
