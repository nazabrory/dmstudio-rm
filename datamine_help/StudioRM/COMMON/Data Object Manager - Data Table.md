# Data Object Manager: Data Table

To access this screen:

  * In the [Data Object Manager](<Data%20Manager%20Dialog.md>) dialog, select the Data Table tab.

Use this panel to view the contents of the selected data object. You can't edit data table contents here.

Selecting an object from the list on the left of the panel automatically updates the contents of this panel. Once a data table is displayed, you can use the toolbar icons above:

![](../Images/Icons/DOM/1%20Import%20Data.png) | Import Data | Display the [Data Import](<data%20import%20dialog.md>) screen to import data using the a Data Source Driver (DSD).  
---|---|---  
![](../Images/Icons/DOM/2%20Export%20Data.png) | Export Data |  Display the [Data Export](<ExportTable.md>) screen to export the selected file using a DSD.  
![](../Images/Icons/DOM/3%20Refresh%20Data.png) | Refresh Data | Refresh the currently selected object. See [Data Refresh](<Data%20Refresh%20Dialog.md>)  
![](../Images/Icons/DOM/4%20Refresh%20All%20Data.png) | Refresh All Data | Refresh all loaded data objects.   
![](../Images/Icons/DOM/5%20Reload%20Data.png) | Reload Data | Reload the currently selected object. See [Data Reload](<Data%20Reload%20Dialog.md>).  
![](../Images/Icons/DOM/6%20Unload%20Data.png) | Unload Data | Remove the selected object from memory. Note that this does not remove the file from the project, or affect physical files.  
![](../Images/Icons/DOM/7%20Combine%20Objects.png) | Combine Objects | Display the [Combine Data Objects](<Data%20Objects%20Combine%20Dialog.md>) screen to join objects together.  
![](../Images/Icons/DOM/8%20Extract.png) | Extract From Object | Select a property or properties of an object to extract. Displays the [Extract Data Object](<ExtractDataObject_Dialog.md>) screen.  
![](../Images/Icons/DOM/9%20First%20Record.png) | First Record | Move to the first record of the table.  
![](../Images/Icons/DOM/10%20Previous%20Record.png) | Previous Record | Move to the previous record of the table.  
![](../Images/Icons/DOM/11%20Next%20Record.png) | Next Record | Move to the next record of the table.  
![](../Images/Icons/DOM/12%20Last%20Record.png) | Last Record | Move to the last record of the table.  
![](../Images/Icons/DOM/13%20Find.png) | Search |  Find a value in the table.  
![](../Images/Icons/DOM/14%20Find%20Next.png) | Find Next | Find the next instance of the previously searched value.  
![](../Images/Icons/DOM/15%20Find%20Previous.png) | Find Previous | Find the previous instance of the previously searched value.  
![](../Images/Icons/DOM/16%20Definition.png) | Definition Viewer | View the data definition of the selected object using the [Definition Viewer](<DmEditDefinitionEditor.md>).  
![](../Images/Icons/DOM/17%20Excel.png) | Export to Excel  | Export the loaded data to an installed Excel application. If the row count exceeds Excel capacity, it will display as many records as it can.  
![](../Images/Icons/DOM/18%20Grid%20Format.png) | Grid Format Options | Display the **[Data Table Format](<DataObjectManager_GridFormat.md>)** screen to change the appearance of the table grid.
