# Data Unload

To access this screen:

  * Right-click the top-level project icon in the Loaded Data control bar and select Unload >> Select.

Remove currently loaded project files from memory. 

To unload data:

  1. Display the **Data Unload** screen.

  2. Select one or more objects from the **Select objects to unload** list. You can use keyboard modifiers **SHIFT** and **CTRL** to select multiple items.

**Tip** : click **Select All** or **Select None** to perform a global selection or deselection of items.

  3. Click **OK**.

The selected data objects are unloaded from memory.
