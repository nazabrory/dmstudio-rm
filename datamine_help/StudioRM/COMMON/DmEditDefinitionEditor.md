# Definition Viewer

To access this screen:

  * In the [Data Object Manager](<Data%20Manager%20Dialog.md>), select the [Data Table](<Data%20Object%20Manager%20-%20Data%20Table.md>) tab, click Definition Viewer .

View the data definition of an object selected in the [**Data Object Manager**](<Data%20Manager%20Dialog.md>).

Select a data attribute on the left to see its definition on the right.

Object Name | The name of the selected data object.  
---|---  
Name | The name of the selected attribute.  
Type | Numeric or Alphanumeric.  
Length | The maximum number of characters that can be stored in an alphanumeric field.  
Default Value | The default value used in the absence of data values. Typically, this is "-"
