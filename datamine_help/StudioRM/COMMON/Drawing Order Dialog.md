# Format Drawing Order

To access this screen:

  * On the Format Display screen, select the Drawing Order tab.

  * Type 'fdd' in the Plots window followed by the Drawing Order tab.

This tab forms part of the [Format Display](<Format%20Overlays%20Dialog.md>) screen.

Define in which order objects and overlays are set down on a [plot sheet](<Window_PLOTS_Overview.md>). The first item is at the bottom (or at the back) and subsequent items are placed on top of one another. An item at the back may be completely obscured whereas one at the front will be completely visible.

Use the up and down arrows, and the First and Last buttons to modify the drawing order of the objects and overlays to give the desired display.
