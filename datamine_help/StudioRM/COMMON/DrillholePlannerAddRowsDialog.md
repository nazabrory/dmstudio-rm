# Add Multiple Deviations

To access this screen:

  * In the **[Drillhole Planner](<DrillholePlanner-Create-New.md>)** (**Deviations** table) select **Add Rows**

You use this screen to add one or more equally spaced [deviation](<DrillholePlanner-Lift-Drift.md>) records to the **Drillhole Planner** 's _Deviations_ table.

Activity steps

  1. Define the **Start depth** at which the first deviation record will apply.

  2. Enter the consistent **Spacing** between deviation records. As many deviation records as possible will be created, up to the end of the hole.

  3. Enter the **Quantity** of deviation records to create.

  4. Click **OK** to return to the **Drillhole Planner**.

  5. Set appropriate **Lift** and **Drift** values for each deviation record.

  6. Click **OK** to return to the **Drillhole Planner** and continue to [Edit Planned Drillholes](<DrillholePlanner-Edit.md>)
