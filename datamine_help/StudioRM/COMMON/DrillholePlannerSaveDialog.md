# Save Drillhole Tables

To access this screen: 

  * In the [Drillhole Planner](<DrillholePlanner-Create-New.md>), click Save to Tables.

Define component drillhole tables representing drillholes generated by the **Drillhole Planner** , to be output in CSV format (collars, surveys, samples with all being optional). 

These files will be based on the current state of the Drillhole Table on the Drillhole Planner screen and are saved to the current project folder by default. You can select an alternative location for any of the files if you wish, or edit the file paths manually. Changing the location of one of the 3 files will automatically set the other 2 to the same path.

Select the file(s) to create (by default, all component files will be created) and click OK to generate CSV output.

Optionally, you can output a Single sample per hole, which could be useful when the sample interval length may not be known (say, it is defined by a field for each drillhole within the table. If the Single sample per hole option is selected, the generated Samples records will be restricted to one per hole, where the sample **FROM** position will equal its **LENGTH**.
