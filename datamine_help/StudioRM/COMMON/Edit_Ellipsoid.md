# Edit Ellipsoid

To access this screen:

  * **Geology** ribbon **> > Ellipsoids >> New**.

  * Enter "edit-ellipsoid" into the **Command** toolbar and press <ENTER>.

  * Use the quick key combination "eel" with a 3D window active.

  * Display the **[Find Command](<findcommand.md>)** screen, locate **edit-ellipsoid** and click **Run**.

Edit ellipsoids stored within a loaded [ellipsoids object](<../VR_Help/Ellipsoids_Overview.md>). Select an ellipsoid and edit its size, location and/or orientation. 

Activity steps:

  1. Load and display ellipsoids to be edited.
  2. Run the **edit-ellipsoid** command.
  3. Select an ellipsoid with the picker tool (select any displayed ellipsoid). 

The current ellipsoid parameters replace the current screen values.

  4. Enter the new Position, Size and/or Rotation settings.

Tip: Use the **Size** and **Rotation** spin buttons to iteratively adjust a setting and see the result in the 3D window immediately.

Tip Quickly perform a 90 degree rotation about any axis using the buttons to the right of each rotation field.

  5. Optionally, Expand or Shrink the selected ellipsoid (in all directions) using the buttons provided. Lengths are scaled by 25% of the displayed value.
  6. Click Apply to update the selected ellipsoid, or OK to do the same and dismiss the screen.
