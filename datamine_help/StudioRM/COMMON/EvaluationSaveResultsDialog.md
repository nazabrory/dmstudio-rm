# Save Evaluation Results

To access this screen:

  * Run the command [dynamic-evaluation-report](<../command_help/dynamic-evaluation-report.md>), enter the relevant settings and click OK. Click Save Results on the [Table View](<Evaluation2DStringsTableViewDialog.md>) screen.

  * **Report** ribbon >> Evaluate >> Strings, then enter the relevant settings and click OK. Click Save Results in the [Table View](<Evaluation2DStringsTableViewDialog.md>) screen.

  * **Report** ribbon >> Evaluate >> Wireframes, then enter the relevant settings and click OK. Click Save Results in the [Table View](<Evaluation2DStringsTableViewDialog.md>) screen.

Store the results of a dynamic evaluation, optionally with summary totals, in a results object.

To define storage settings for evaluation results:

  1. Enter the name of a Results Object (or select an existing object, which will be replaced).

  2. Check Store Totals row to include summary evaluation totals for all categories. By default, evaluation field totals for all evaluation categories are not stored (unchecked) within the generated results object.

  3. Click **OK** to generate or update a results data object.
