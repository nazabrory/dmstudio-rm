# Export Google Earth KML: Points

To access this screen:

  1. Display the [Data Export](<ExportTable.md>) screen.

  2. Select the _Google Earth KML_ **Driver Category**.

  3. Select the _Points_ **Data Type**.

  4. Click **OK**.

Use this screen to export loaded points data to Google Earth. See [Export to Google Earth](<ExportDataGoogleEarth_Concept.md>).
