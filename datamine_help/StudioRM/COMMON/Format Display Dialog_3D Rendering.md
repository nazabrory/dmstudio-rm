# 3D Rendering

Note: These settings relate to the format of data object overlays in 2D plot projections. See [Projection Overlay Types](<../PLOTS_LOGS/Projection%20Overlay%20Types.md>).

To access this screen:

  * On the **[Format Display](<Format%20Overlays%20Dialog.md>)** screen, select the **3D Rendering** tab. 

**Note** : 3D Rendering mode is largely superseded by **[3D projection](<../PLOTS_LOGS/Projection%20Overlay%20Types.md>)** formatting options. Consider creating a 3D projection instead.

Control how the 3D rendering routines are performed in plot projections displaying legacy 2D overlays. See[Projection Overlay Types](<../PLOTS_LOGS/Projection%20Overlay%20Types.md>). If enabled, you can also choose whether to display transparencies (or just display opaque colours).

Settings made on this screen affects all overlays in the current view.

**Warning** : For [2D (static) projection types](<../PLOTS_LOGS/Projection%20Overlay%20Types.md>), consider disabling 3D rendering when not in use; this display mode is sensitive to hardware configuration and, as such, results may not always be as expected. The best way to view your data in 3D is using a 3D window, or by configuring a 3D projection containing a 3D overlay group.
