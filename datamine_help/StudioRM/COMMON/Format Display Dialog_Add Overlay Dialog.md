# Add Overlay

To access this screen using the Plot Item Library:

  1. Right click a Plot sheet overlay in the **[Sheets](<Sheets%20Control%20Bar%20Overview.md>)** or **Project Data** control bar.

  2. Select **Insert**.

The [**Plot Item Library**](<../PLOTS_LOGS/plotitemlibrary.md>) displays.

  3. Select a plot item type and click **OK**.

To access this screen using the Format Display wizard:

  1. Display the [**Formay Display**](<format%20display%20dialog_overlays.md>) wizard.

  2. Select the **Overlays** tab.

  3. In the **Overlay Objects** group, select **Show Overlays**.

  4. Click **Add**.

Add object overlays to the current sheet (in the Plots, Logs, Tables or Reports windows).

Select an object and click **OK** to generate a new plot sheet overlay with default settings.

Note: More than one overlay can be created for the same object, although this screen can only be used to add one overlay at a time.

## What is an Overlay?

An overlay represents a single view of an object in memory; this can be loaded data or data created in your application. One file (object) can have multiple overlays, meaning that you can view the same object, in different ways, on screen simultaneously if required. This can be useful if you wish to create a label overlay, and draw the underlying vector data to the screen at the same time \- in this situation you need two overlays for the same object: one for the vector data, and one for the labels, ensuring both are visible (using the **Style** tab).

See [The View Hierarchy](<View%20Hierarchy.md>) and [Windows, Sheets, Projections and Overlays](<concept_views%20sheets%20overlays.md>).
