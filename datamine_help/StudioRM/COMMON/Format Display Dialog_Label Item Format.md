# Label Item Format

To access this screen:

  1. Display the [**Format Display**](<Format%20Overlays%20Dialog.md>) screen.

  2. Select the **Labels** tab.

  3. If labels are not already available:

     1. Click Reset to display the [Reset Labels](<Format%20Display%20Dialog_Reset%20Labels.md>) screen. 

     2. Select one or more data fields to use for annotation, and their tabular orientation.

     3. Click OK to return to the **Labels** tab.

  4. Just above the label properties grid, select the **Style** tab.

This screen displays part of the label definition process when adding annotation to an plot projection overlay. Use this screen to set the [Alignment](<../PLOTS_LOGS/Format_Column_Alignment_Dialog.md>), [Border/Color](<../PLOTS_LOGS/Format_Column_Borders_Dialog.md>) and [Text](<../PLOTS_LOGS/Format_Column_Text_Dialog.md>) settings for the projection overlay labels.
