# Contours from Drillhole Intercepts - Input Data

To access this screen:

  * This is the initial screen of the **[Generate Contours from Intercepts](<Contours%20from%20Intercepts_Procedures.md>)** screen.

This screen, part of the contouring suite of functions in your product, is used to define data inputs for contouring of a unique attribute value of a loaded drillhole object.

An input drillhole object or selected drillhole data must be defined, as must an attribute and value for modelling.

Optionally, additional (supplemental) point data, fault trace data and clipping polygons can be specified to refine the contouring output.

You also use this screen to define the plane used for contouring.
