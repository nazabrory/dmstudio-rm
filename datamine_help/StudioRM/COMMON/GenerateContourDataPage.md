# Contours from Points - Input Data

To access this screen:

  * This is the initial screen of the **[Generate Contours from Points](<Contours_From_Points_Intro.md>)** tool.

  * This is the initial screen of the **[Generate Distance Contours](<Contours_Distance_Introduction.md>)** tool.

This screen, part of the contouring suite of functions in your product, is used to define data inputs for contouring. Depending on how it is accessed, some functionality may not be available. For example, if distance contouring, only a subset of options are available, when compared to point contouring.

An input point object is required, and other mandatory fields may be present (for example, if point contouring, an attribute for modelling must be specified).

If point contouring, optionally, fault trace data and clipping polygons can be specified to refine the contouring output.

You also use this screen to define the plane used for contouring.

This screen is explained in the following procedures:

  * [Generate Contours from Points](<Contours%20from%20Points%20-%20Worked%20Example1.md>)

  * [Generate Distance Contours](<Contours%20-%20Distance%20-%20Procedure.md>)
