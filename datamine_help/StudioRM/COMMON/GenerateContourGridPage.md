# Generate Contours - Gridding

To access this screen:

  * This is the second screen of the [Drillhole Intercept Contours](<Contours_From_Intercepts_Intro.md>) tool.

  * This is the second screen of the [Contours from Points](<Contours_From_Points_Intro.md>) tool.

  * This is the second screen of the [Distance Contours ](<Contours_Distance_Introduction.md>) tool.

The Gridding screen is part of the contour generation function. It is used to set up parameters required to generate the output data (grid model, contours and/or surface).

The same panel is used when generating contours from points and drillhole intercepts. If a distance-from-sample map is being created, some functions are not available.

You can choose to calculate the size of your output grid based on either your current loaded/specified data or you can choose a custom area.

This screen is explained in the following procedures:

  * [Generate Drillhole Intercept Contours](<Contours%20from%20Intercepts_Procedures.md>)

  * [Generate Contours from Points](<Contours%20from%20Points%20-%20Worked%20Example1.md>)

  * [Generate Distance Contours](<Contours%20-%20Distance%20-%20Procedure.md>)
