# Generate Contours - Output Data

To access this screen:

  * This is the final screen of the [Drillhole Intercept Contours](<Contours_From_Intercepts_Intro.md>) tool.

  * This is the final screen of the [Contours from Points](<Contours_From_Points_Intro.md>) tool.

  * This is the final screen of the [Distance Contours ](<Contours_Distance_Introduction.md>) tool.

The Output Data screen is part of the contour generation functions. It is used to define the objects that will be created during contouring and their positions. It is also used to set the scope of the contour strings and the interval used for string contouring. 

This screen is explained in the following procedures:

  * [Generate Drillhole Intercept Contours](<Contours%20from%20Intercepts_Procedures.md>)

  * [Generate Contours from Points](<Contours%20from%20Points%20-%20Worked%20Example1.md>)

  * [Generate Distance Contours](<Contours%20-%20Distance%20-%20Procedure.md>)
