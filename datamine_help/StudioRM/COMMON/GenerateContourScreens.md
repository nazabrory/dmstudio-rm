# Contouring Tools

Your product contains various tools for generating contour strings, grids and even wireframe surfaces between landmark data values. The following tools are available:

  * [Drillhole Intercept Contours](<Contours_From_Intercepts_Intro.md>): Generate strings, grid and surface data representing the uppermost or lowermost instance of a particular drillhole categorical value. You could use this tool for example, to generate a contact surface between lithologies down a drillhole.

  * [Contours from Points](<Contours_From_Points_Intro.md>): Create outputs representing smooth contours between known data point positions. This could be used, for example, to establish a smoothly undulating topographical surface based on surface drone data.

  * [Distance Contours ](<Contours_Distance_Introduction.md>): Generate output contours and a grid model representing the distance of each surface point from neighbouring samples. 

Where possible, similar functionality is provided across multiple tools. For example, the Generate Contours from Points and Generate Contours from Intercepts functions have similar (but not identical) data input pages, and identical gridding parameters and output pages.
