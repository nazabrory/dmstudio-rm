# Gradient Convention

To access this screen:

  * **Data** ribbon **> > Conventions >> Gradient**.

  * In the **Command** toolbar, run the command [set-gradient-convention](<../command_help/set-gradient-convention.md>).

Specify how gradient values are represented in your application. Define the gradient type and a dip convention (which determines if positive dip values mean an upward or downward direction).

Note: The convention is used in combination with other design commands, but some commands can override this setting (typically forcing a convention in degrees). Only help for each situation clarifies this.

Activity steps:

  1. Display the **Gradient Convention** screen.

  2. Pick a convention type using **Gradient In** options:

     * Percent %

     * Degrees (default)

     * Ratio 1:N

  3. Pick a dip convention using **Direction** options:

     * +ve UP  Positive values indicate or force increasing Z values (rise up).

     * +ve DOWN Positive values indicate or force decreasing Z values (dip down).

  4. Click **OK**.

  5. Save your project. The convention is stored with your project for next time.
