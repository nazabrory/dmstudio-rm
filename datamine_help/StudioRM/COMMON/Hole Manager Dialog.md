# Hole Set

To access this screen:

  * Display the [Format Display](<Format%20Overlays%20Dialog.md>) screen (with static drillhole data loaded) and select the Hole Set tab. Click Edit Sets.

The Hole Sets screen is used to configure the sets of drillhole data ('hole sets') available in the current session, and to configure the contents of each hole set.

To create a new hole set:

  1. Display the **Hole Sets** screen.

  2. Click **New**.

An empty hole set is created.

To edit the contents of a hole set (add holes, remove holes):

  1. Display the **Hole Sets** screen.

  2. Select an existing hole set.

  3. Click **Edit**.

  4. Use the **Hole Set Contents** screen to add or remove holes. See [Hole Set Contents](<Hole%20Set%20Dialog.md>).

  5. Click **OK**.

  6. Click **Close**.

Note: You can also add loaded static holes to a set by clicking **Add Holes**.

Note: Making changes to a hole set updates the [Holes](<Holes%20Control%20Bar%20Overview.md>) control bar.
