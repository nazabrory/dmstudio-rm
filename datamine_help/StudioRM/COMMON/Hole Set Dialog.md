# Hole Set Contents

To access this screen:

  * Display the [Hole Set](<Hole%20Manager%20Dialog.md>) screen and click **Edit**.

Configure the contents of the selected hole set. 

Use Add and Remove to swap holes from one list to another, and click OK when the list of holes looks correct.
