# Maps Missing Dates

To access this screen: 

  * On the [Import Map Data from External Files](<ImportMaps.md>) screen, click Import. If maps are found without time/date information, the Missing Map Dates dialog is displayed.

Modern Studio Mapper systems will generate map date with a date/time stamp. Older versions may not.

During the import of map data to your application, if any maps are found to be undated, they will be listed here. By default, they are unchecked, meaning they won't be imported. 

**Note** : If you choose to import a map with no date context, the map data will still be imported but without any time or date context and may be outside the range specified on the Import Map Data from External Files screen.
