# Import Multiple Files to Project

If you are importing specific data formats, you can import multiple data files at once.

Note: Import screens are often provided as part of a 3rd party driver package, but in some cases we have customized these to provide a more seamless experience when bringing data into Studio products.

When each file is imported, it is first subjected to the functions of the relevant data source driver, which prepare the file for loading. In effect, they 'translate' the incoming file into a format recognizable by your application. 

Once all driver-specific screens are completed, the final **Import Files** screen displays. See [Import and Load Data](<Concept_Importing%20and%20Exporting%20Data.md>) for activities.

To import multiple files to the project at once:

  1. **Data** ribbon **> > Import >> Add to Project**.

The **Select Source Files** browser displays.

  2. Select one or more files to import. All supported files are displayed. You can further filter the display of file types using the list in the bottom right of the screen.

  3. Click **Open**.

The **Add to Project** screen displays.

  4. Select the Datamine file types you intend to create as a result of importation. 

     * Points

     * Wireframes

     * Strings

     * Block Model

     * Table

  5. Use Add suffix to apply a standard suffix to the filename of the imported files of each type. These are the defaults:

     * **Points** PT

     * **Wireframes** WF

     * **Strings** ST

     * **Block Model** BM

     * **Table**

  6. Click **Save**.

  7. Complete the driver-specific screens. The contents of these screens are unique to each driver type. Click **Help** or press F1 for more information on these screens.

  8. Click **OK**.

Importation of all selected files completes.
