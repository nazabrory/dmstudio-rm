# Viewing Data

Your application provides many ways for viewing and interrogating geological and mining data. It does this through a variety of Windows which each concentrate on a different aspect of the data.

The **[view hierarchy](<View%20Hierarchy.md>)** is closely related the way your application handles data and formats it for display. Data is loaded into memory as 3D objects which, depending upon their nature, can be viewed through one or more of the windows.

In principle, each window contains at least one sheet and sheets present views which are made up of overlays which are graphical representations, which can be formatted, of 3D objects.
