# Load External Data

To load a single file of any supported format into memory using the Data ribbon:

  1. Expand the **Load >> External** ribbon on the **Data** ribbon.

  2. Select the type of data you are loading, for example, _Block Model_.

The [Data Import](<data%20import%20dialog.md>) screen displays.

  3. Select a driver and data type and click **OK**.

  4. Select a source file to load.

  5. Complete the driver-specific screens. These determine how data is converted to an understandable format and loaded into Studio. Press F1 or click **Help** for more information on any screen.

  6. The data file is loaded and (if possible) displayed in the primary 3D window.
