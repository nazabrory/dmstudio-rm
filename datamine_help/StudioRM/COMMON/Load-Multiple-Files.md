# Load Multiple Files

To load multiple files at once (same or different data formats) into Studio at once:

  1. Click the top-level **Load >> External** button on the **Data** ribbon.

The **Select Source Files** screen displays.

  2. Select one or more files and click **OK**.

  3. Driver-specific screens appear. Complete these. For more information on a screen, press F1 or click **Help**.

  4. Data files are loaded and display in the primary 3D window.
