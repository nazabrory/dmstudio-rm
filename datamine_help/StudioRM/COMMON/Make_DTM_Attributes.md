# Make DTM: Edit Attributes

To access this screen:

  1. Display the **[Make DTM](<Make_DTM_Dialog.md>)** wizard.

  2. On the [**General Options**](<Make_DTM_General.md>) screen, select Attributes >> User Defined.

  3. Configure and click **Next** on the [**Select Points and Strings**](<Make_DTM_Points_Strings.md>) screen.

  4. Configure and click Next on the Select Boundary Strings screen.

The [**Edit Attributes**](<edit%20attributes%20pick%20dialog.md>) screen displays. This is the standard screen used throughout Studio to edit attributes.
