# Make DTM: Select Boundary Strings

To access this wizard:

  * Click **Next** on the [**Select Points and Strings**](<Make_DTM_Points_Strings.md>) screen of the [Make DTM](<Make_DTM_Dialog.md>) wizard.

The Make DTM wizard is used to create a digital terrain model from loaded string or points data.

The Select Boundary Strings screen is shown if:

  * **Use boundary strings** is checked on the [General Options](<Make_DTM_General.md>) screen, and;

  * String data is selected on the **[Select Points and Strings](<Make_DTM_Points_Strings.md>)** screen.

Only closed strings can be used. If selected, boundary strings limit the extent of the DTM, but do not contribute to the DTM itself. Where this is required, the strings should be selected in both the Select DTM Points and String, and this screen.
