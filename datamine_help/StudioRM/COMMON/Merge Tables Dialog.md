# Merge Tables

To access this screen:

  * **Data** ribbon **> > Objects >> Combine**.

  * Run the command 'combine-data-objects'.

Merge the contents of data from at least two loaded data objects. New tables are created in memory, not as physical files. 

Activity steps:

  1. Load the data objects to merge.

  2. Display the **Merge Tables** screen.

  3. Select the **Tables to merge** ; the data objects you wish to combine.

  4. Enter the **New table name** containing merged data.

  5. Click **OK**.

The merged data object is created.
