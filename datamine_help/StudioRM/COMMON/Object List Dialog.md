# Object to Export

To access this screen:

  * **Data** ribbon **> > Export >> External**.

Choose which data objects to export to an external file.

Activity steps:

  1. Select a loaded data object to export.

  2. Click **OK**.

The [Data Export](<ExportTable.md>) screen displays.

  3. Choose the export format for your file and click **OK**. 

  4. Complete the remaining screens displayed. These depend on the data category and type being exported. Press F1 on any screen for more help.
