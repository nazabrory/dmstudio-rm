# Translate Object

To access this screen:

  * **Digitize** ribbon **> > Transform >> Translate**.
  * **Explicit** ribbon **> > Edit >> Translate**.
  * Using the **[command line](<Command_Toolbar.md>)** , enter "translate-point".

  * Using the **[command line](<Command_Toolbar.md>)** , enter "translate-string".

  * Using the **[command line](<Command_Toolbar.md>)** , enter "translate-string-opt".

  * Using the **[command line](<Command_Toolbar.md>)** , enter "translate-wireframe".

  * Use the quick key combination "tra".

  * Use the quick key combination "trw".

  * Display the **[Find Command](<findcommand.md>)** screen, locate **translate-point** and click **Run**.

  * Display the **[Find Command](<findcommand.md>)** screen, locate **translate-string** and click **Run**.

  * Display the **[Find Command](<findcommand.md>)** screen, locate **translate-string-opt** and click **Run**.

  * Display the **[Find Command](<findcommand.md>)** screen, locate **translate-wireframe** and click **Run**.

Translates a points, string or wireframe data a set distance in X, Y and Z directions.

You can translation selected data only. This can be either all or part of an object, and selected data can be from multiple objects and object types. For example, you can select drive centreline string and wireframe solids data at the same time.

**Note** : This command supports [**flexible wireframe selection**](<Wireframe_Selection_Concept.md>).
