# System Options

To access this screen:

  * **Home** ribbon **> > Settings >> System**.

  * Using the **[command line](<Command_Toolbar.md>)** , enter "show-system-options"

  * On the **[Find Command](<findcommand.md>)** screen, highlight **show-system-options** and click **Run**.

The Options screen is where the behavior and appearance of your application and its plug ins are defined. 

System options affect all projects.

The menu on the left offers the following option categories:

### Environment Options

Options to control how your application appears. See [Options: Environment](<Options_Environment.md>).

### Data Options

General data handling options are found here. See [Data Options](<data%20options.md>).

### Plots Options

Options that determine how things are done in the Plots window. See [Plots Options](<Options_Plots.md>).

### Project Options

Project options are managed here. See [Project Options](<Options_Project.htm.md>).

### 3D Options

These options determine how 3D windows behave in your projects. See [Options: 3D ](<Options_InTouch.md>).

### Commands Options

General settings for handling undo behaviour for commands. See [Options: Commands](<Options_Commands.md>).
