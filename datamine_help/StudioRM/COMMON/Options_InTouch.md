# Options: 3D 

To access these options:

  * On the [Options](<Options.md>) screen, expand the 3D folder.

Configure settings that apply to displayed 3D windows. Select a link below for more information:

  * **[General](<Options_InTouch-General.md>)** configure settings relating to general 3D viewing, such as the appearance of wireframes, how cylinders are rendered, the colour of selected data and other settings. See [Options: 3D General](<Options_InTouch-General.md>).

  * **[Stereo](<Options_InTouch-Stereo.md>)** define settings tocontrol the display of stereo 3D views. See [3D Options: Stereo](<Options_InTouch-Stereo.md>).

  * **[Initial States](<Options_InTouch-Initial-States.md>)** control how 3D windows appear each time you open them. See [Options: 3D Initial States](<Options_InTouch-Initial-States.md>).

  * **[Printing](<Options_InTouch-Printing.md>)** choose how 3D window snapshots are scaled when printing. See [Options: 3D Printing](<Options_InTouch-Printing.md>).
