# Overlay Selection

To access this screen:

  * Activate the Home ribbon and select the summary button in the Select group of commands.

The Overlay Selection screen is used to set individual object overlay selection settings for all 3D windows.

To choose overlay data that can be selected:

  1. **Check** or **uncheck** object overlays.

  2. Click **OK** to update the selection roster.

Subsequent data selection in 3D windows honours the updated settings.
