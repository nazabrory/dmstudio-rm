# Project Settings: Legends

To access this screen:

  * On the [Project Settings](<ProjectSettings.md>) screen, select the Legends tab.

Specify if a legend is created automatically when data of a particular type is created.

An automatic legend is based on the first numeric legend found (on a left-right basis) for the object in question, however, if the data contains a COLOUR field, it is used instead. 

By default, automatic legends are generated for all visual data types.

These settings are specific to the current project.
