# Character Size Settings

To access this screen:

  * On the [Project Settings: Data Display](<Project%20Settings_Data%20Display.md>) screen, click Grid Character Size.

Set the properties for grid text displayed in your application.

To define character size settings:

  1. Display the **Character Size Settings** screen.

  2. Enter a **Character Size** in proportional font sizing terms.

  3. Enter an **Aspect Ratio**. This is a value between 0.01 and 1, and represents the 'gap' between the grid's line markers and the text. For example, a setting of _0.5_ displays:
