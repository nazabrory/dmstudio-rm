# Project Properties: Summary

To access this screen:

  * In the [Loaded Data](<Loaded%20Data%20Control%20Bar.md>) or [Sheets](<Sheets%20Control%20Bar%20Overview.md>) control bars, double-click the top-level project icon. Select the Summary tab.

This screen displays summary information about the currently loaded project.

**Note** : all fields on this screen are optional.

To define summary properties for your project:

  1. Display the **Summary** tab.

  2. Choose a **Name** for your project. This will not change the project file name, and can be anything you like.

  3. Add a **Project Code**. This should uniquely identify your project.

  4. Enter a **Short Title**. This can be useful if the project has a particular purpose.

  5. Enter an **Extended Title**. This should be a fuller description of the project contents, and any other potentially useful supporting information.

  6. If the project relates to a particular customer, enter their name in the **Client** field.
