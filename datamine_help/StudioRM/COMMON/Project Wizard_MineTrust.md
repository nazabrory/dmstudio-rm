# Project Wizard: MineTrust Settings

Note: This Project Settings screen is only available for MineTrust-enabled Studio products and where [**MineTrust Connector**](<../MineTrust/MineTrust-Aware-Projects.md>) is already installed and the MineTrust Connector service is running. This can be installed whilst your product is running. See [MineTrust-Enabled Projects](<../MineTrust/MineTrust-Aware-Projects.md>)

To display this screen:

  1. Install **MineTrust Connector** and ensure a MineTrust endpoint is configured. See [MineTrust-Enabled Projects](<../MineTrust/MineTrust-Aware-Projects.md>)

  2. Launch the [Project Wizard](<ProjectWizard.md>).

  3. Define your project's initial settings and progress the wizard until the **MineTrust Connector Settings** screen displays.

Note: All MineTrust-enabled projects are hosted securely by MineTrust Online. Datamine takes your data security and privacy and are proactive in eliminating global data security threats.
