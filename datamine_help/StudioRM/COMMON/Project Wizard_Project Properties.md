# Project Wizard: Project Properties

To access this screen:

  * The **Project Properties** screen is displayed as part of the [Project Wizard](<ProjectWizard.md>).

The Project Properties screen is used to record general properties relating to the project you are creating. It also determines what the system does when Next is clicked.

To define project properties:

  1. Open the **Project Wizard**.

  2. Enter a project **Name**. Consider a project naming convention if you are likely to work with multiple projects, or plan to share your project with others.

  3. Enter a **Location** for your project file. Either type in the system path to the location you wish to store your project file in, or use the ellipsis button to browse for a directory.

  4. To add all files currently in the specified Location automatically to the project when it is created, **check** **Automatically Add Files**. If you do not wish to add files in the specified location (you can add them manually later, or on the next screen of this wizard), leave it **unchecked**.

  5. To define the various settings for your project, click **Project Settings**. 

The **Project Settings** screen displayed. See [Project Settings](<Project%20Settings_General.md>). 

**Note** : You can change project settings at any time after creation.

  6. Click **Next** to move to the next screen of the wizard.

     * If [**MineTrust Connector**](<../MineTrust/MineTrust-Aware-Projects.md>) is installed, the next screen shown is **MineTrust Settings**.

     * If **MineTrust Connector** is not installed, the next screen shown is [Project Wizard: Project Files](<Project%20Wizard_Project%20Files.md>)
