# Project Settings

To access this dialog:

  * This dialog is opened at the relevant properties tab as part of several application functions.

  * Enter [show-project-settings](<../command_help/show-project-settings.md>) into the command line and press <ENTER>

  * **Home** ribbon **> > Settings >> Project**.

The Project Settings screen is used to configure system-wide properties for your application. The options contained within this dialog will differ depending on how you accessed the dialog:

  * If you accessed this screen from the [Project Wizard](<ProjectWizard.md>), you will only see a General tab.

  * If you accessed this screen from the File menu, with an active project, you will see multiple tabs.
