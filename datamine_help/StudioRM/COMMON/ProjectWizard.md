# Project Wizard 

To access this screen:

  * Using the Start window, select New Project or **New MineTrust Project**.

  * Select the New Project icon on the Quick Access ribbon.

  * Click the File button and select New Project or **New MineTrust Project**.

Create a new [project](<Concept_The_Studio_3_Project_File.md>) using a step-by-step wizard. 

The three main functions of the Project Wizard are:

  * to set up general project properties such as name and location, for example. 

See [Project Wizard: Project Properties](<Project%20Wizard_Project%20Properties.md>).

  * to connect to, or disconnect from, the current Datamine MineTrust endpoint, effectively starting or stopping data sharing. 

See [Project Wizard: MineTrust Settings](<Project%20Wizard_MineTrust.md>).

  * to manage file references in the project database.

See [Project Wizard: Project Files](<Project%20Wizard_Project%20Files.md>).

  * to display a project summary and create the project database.

See [Project Wizard: Create Project](<project%20wizard_create.md>).

**Note** : You can display the **Project Wizard** when your application launches using system [Options](<Options_Project.htm.md>).
