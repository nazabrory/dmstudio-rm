# Projection Face Angle

To access this screen:

  *   *   * **Digitize** ribbon **> > Transform >> Project >> Set Face Angle**. 

  *   * Using the **[command line](<Command_Toolbar.md>)** , enter "set-face-angle"

  *   * Use the quick key combination "fng".
  *   * On the **[Find Command](<findcommand.md>)** screen, highlight **set-face-angle** and click **Run**.

See [set-face-angle ("fng")](<../command_help/set-face-angle.md>).
