# Item Properties

To access this screen:

  * In the Loaded Data control bar, select the Properties context menu option for an object or data column .

  * In the Holes control bar, select the Properties context menu option.

  * In the Sheets control bar, **Plots** folder double-click a projection or in the **Plots Overlays** folder, a _Hull_ or _Section Line_ item.

This screen displays properties specific to the selected folder, item or object. Property values can be edited by using the boxes and drop-downs in the displayed grid, unless they are read-only.

The properties displayed here are the same as shown in the [Properties control bar](<properties%20control%20bar%20overview.md>) for the selected item.
