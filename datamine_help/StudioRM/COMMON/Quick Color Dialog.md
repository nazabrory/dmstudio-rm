# Quick Color

To access this screen:

  * Run the [quick-color-all](<../command_help/quick-color-all.md>) command.

  * Run the [quick-color-drillholes](<../command_help/quick-color-drillholes.md>) command.

  * Run the [quick-color-models](<../command_help/quick-color-models.md>) command.

  * Run the [quick-color-planes](<../command_help/quick-color-planes.md>) command.

  * Run the [quick-color-points](<../command_help/quick-color-points.md>) command.

  * Run the [quick-color-strings](<../command_help/quick-color-strings.md>) command.

  * Run the [quick-color-wireframes](<../command_help/quick-color-wireframes.md>) command.

  * **Format** ribbon **> > Colour >> Quick**.

Quickly apply a specified colour or legend to selected overlays of a particular type.

Activity steps:

  1. Load and display the data to colour.

  2. Display the **Quick Color** screen.

  3. In the **Overlays to Color** menu, check the overlay(s) to colour.

  4. Choose the colour to apply to the selected overlay(s):

     * Select **Color** to apply a fixed, solid colour.

     * Select Legend to pick an existing legend and attribute **Column** containing colouring values.

**Note** : You can also reuse a previous colouring instruction using the **History** list to display the 10 most recent colouring events. If a range or value legend was applied, the entry appears in the format Legend - Attribute, or simply Legend if a filter legend was applied.
