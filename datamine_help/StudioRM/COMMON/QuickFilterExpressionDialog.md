# Saved Filter

To access this screen: 

  * In the [**Quick Filter**](<Quick%20Filter%20Dialog.md>) control bar, **Save** a new filter definition.
  * In the **Quick Filter** control bar, **Edit** an existing saved filter.

This screen supports the **Quick Filter** control bar's filter saving functions.

If you have opened the **Saved Filter** screen by saving a new filter, you will only be able to edit the _Name_ (which will default to the saved filter expression).

If you are editing an existing saved filter, you will be able to edit the _Name_ and also overwrite the current filter definition with the currently defined one using **Use Current Filter**.
