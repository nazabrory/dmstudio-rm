# Read ESRI data

To access this screen:

  * This screen is displayed during the data import process.

Define which data columns are imported and how validation of incoming ESRI data is to be performed. The top section of the screen reports summary information about the previously selected file (file name, type of data, number of entities and data limit coordinates).

Otherwise, using the following settings to control the import of data:

  * Data Fields Check columns you wish to import, or clear check boxes to filter data from the import process.

  * Constant To specify a numeric value for validation purposes, enter it into this field. 

  * Use Field If the incoming data file has a dedicated data column containing validation values, use the drop-down list to select the necessary data column.
