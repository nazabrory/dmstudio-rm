# Manage Ribbon   

Your application uses [ribbons](<Ribbons-overview.md>) to present commands.

The Manage ribbon is specific to the Plots data window, and will only appear when that window is active. This ribbon contains functions to allow you to construct a report-ready plot, including projections, sheets and plot items. It also allows you to insert table views into your project.
