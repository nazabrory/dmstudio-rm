# Quick Access Toolbar

Located at the top-left of your screen, and always available, the Quick Access toolbar can be used to create a group of commonly used commands.

To configure the Quick Access Toolbar:

  1. Display the **Customization** screen by expanding the drop down menu in the top left of the screen:

![](../Images_STUDIORM_ONLY/Ribbon_QuickAccess2.jpg)

  2. Select More Commands...

The **Quick Access Toolbar** customization screen displays.

  3. Expand the Choose commands from list to pick a command category.
  4. Use the **Commands** list to select a command to add to the **Quick Access** toolbar.
  5. Click **Add**.

The selected command is transferred to the right-hand panel, indicating it is now present on the toolbar.

  6. Choose where to position the **Quick Access Toolbar**.

     * Check **Show Quick Access Toolbar** below the ribbon to move the menu below ribbon contents.

     * Uncheck **Show Quick Access Toolbar** to show the menu above the ribbon contents (default setting).

  7. To reset the toolbar to default contents, click **Reset**.

  8. Click OK to update the **Quick Access Toolbar**.
