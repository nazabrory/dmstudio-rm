# Sample Analysis Ribbon   

Your application uses [ribbons](<Ribbons-overview.md>) to present commands.

The Sample Analysis ribbon contains commands related to the investigation of sample data; a wealth of statistical and geochemical analysis functions, charting options, drillhole data building and compositing plus a range of drillhole data editing commands.
