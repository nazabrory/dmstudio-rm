# User Tools Ribbon   

Your application uses [ribbons](<Ribbons-overview.md>) to present commands.

The User Tools ribbon contains the [Customize Ribbon](<Customize_Tools.md>) tool. This can be used to customize what appears in your ribbon interface. With it, you can edit existing ribbons or create new ones. You can also add external tools (anything that can be launched from the Windows Command Line) and also your own scripts or macros.
