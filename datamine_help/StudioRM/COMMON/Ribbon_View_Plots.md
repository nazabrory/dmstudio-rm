# View Ribbon (Plots Window)  

Your application uses [ribbons](<Ribbons-overview.md>) to present commands.

The View ribbon is associated with the [Plots](<Window_PLOTS_Overview.md>) window. This ribbon is only displayed when a Plots sheet is displayed, and should not be confused with the **[View ribbon for the 3D window](<Ribbon_View_VR.md>)**. This ribbon contains all the tools you need to format the view of your plot sheet, log or table display, including tools to define your sectional view, scale, zoom and pan settings.
