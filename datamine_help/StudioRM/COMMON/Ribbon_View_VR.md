# View Ribbon (3D Window)  

Your application uses [ribbons](<Ribbons-overview.md>) to present commands.

This topic covers the contents of the View ribbon that is displayed when a 3D window is active. Note that the contents of this ribbon are dependent on the data window type that is active. This ribbon contains a variety of commands related to the formatting of the 3D viewing window, including definition of views, sections, window arrangements and clipping. If you're into stereoscopic viewing, this is also the place to enable it.
