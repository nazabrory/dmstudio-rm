# Rotate Object

To access this screen:

  * **Explicit** ribbon **> > Edit >> Rotate**.

  *   *   * Enter "rotate-wireframe" into the **Command** toolbar and press <ENTER>

  *   * Use the quick key combination "row" with a 3D window active.

  * Display the **[Find Command](<findcommand.md>)** screen, locate **rotate-wireframe** and click **Run**.

Interactive command to rotate wireframe data around one or more axes. This is the screen displayed when running the [rotate-wireframe ("row")](<../command_help/rotate-wireframe.md>) design command.

This command rotates data around a point which can be the wireframe centre or its centroid, or any user-defined point.

This command can be used to rotate entire wireframe objects or partial wireframe data including selected triangles. Where part of a wireframe object is rotated, your application will attempt to retain links, preserving the integrity of surrounding data. For example, if part of a volume is rotated, the result will remain as a closed volume.

**Note** : This command supports [**flexible wireframe selection**](<Wireframe_Selection_Concept.md>).

Command steps:

  1. Select the wireframe that your wish to rotate. See [Wireframe Data Selection](<selecting_wireframes.md>).

  2. Run the command. 

The Rotate Object screen displays.

  3. Enter the Rotate By values that you wish to apply to the wireframe in X, Y and Z directions.

  4. Select the point around which the wireframe is rotated or use the picking tool to specify a point in any 3D window.

  5. Click Rotate. This rotates the selected wireframe data.
