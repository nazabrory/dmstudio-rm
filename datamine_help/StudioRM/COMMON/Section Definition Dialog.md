# View Settings

To access this screen:

  * Right-click a plot projection and select **View Settings**.

  * This screen is also displayed in response to other commands that require a view direction to be set.

Configure the view and section plane settings that are used to display Plots window projections. See [Sections and Projections](<../PLOTS_LOGS/alignviewwithsection.md>).

Settings are categorized using the following tabs:

  * **Section Definition** Define the planar section for the projection. This may be used to align the view, clip data or simply display as an indicator of a position of interest of loaded 3D data. 

See [Section Definition](<../PLOTS_LOGS/CustomSections.md>).

  * **View Direction** Orient the viewing direction along which data is viewed. This can be regarded as 'setting the camera' for a plot projection. 

See [View Direction](<../PLOTS_LOGS/RotateView.md>).

  * **Rotate** Rotate the view of data in the target projection around the view mid point. 

See [View Rotation Settings](<../PLOTS_LOGS/Plots-ViewSettings-Rotate.md>).

  * **Size/Scale** Choose the **[zoom](<../PLOTS_LOGS/Zooming.md>)** factor for the current projection.

See [Size/Scale](<view%20settings%20scale%20dialog.md>).

  * **Section Width** Dynamically manage the section corridor for **[clipping data](<../PLOTS_LOGS/ClipView.md>)**.

See [Section Width Settings](<../PLOTS_LOGS/Plots-ViewSettings-SectionWidth.md>).

  * **Exaggeration** Increase or decrease the view scale along any world axis.

See [Projection Exaggeration Settings](<../PLOTS_LOGS/Plots-ViewSettings-Exaggeration.md>).
