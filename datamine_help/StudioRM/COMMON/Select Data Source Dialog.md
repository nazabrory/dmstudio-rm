# Select Data Source

To access this screen:

  1. [Data Import](<data%20import%20dialog.md>) screen **> > ODBC** and any **Data Type**. Click **OK**

  2. If a block model type was specified, in the **Specify Block Model Parameters** screen, click **From Database**.

Displayed when an ODBC data source connection is attempted, this screen is used to configure settings required for data export or import.

This screen contains the following tabs:

  * File Data Source: select the file data source that describes the driver that you wish to connect to. You can use any file data source that refers to an ODBC driver which is installed on your machine. See [File Data Source](<Select%20Data%20Source_File%20Data%20Source.md>).

  * Machine Data Source: a machine data source is specific to your local machine, and cannot be shared. "User" data sources are specific to a user on your machine. "System" data sources can be used by all users of your machine, or by a system-wide service. See [Machine Data Source](<Select%20Data%20Source_Machine%20Data%20Source.md>).
