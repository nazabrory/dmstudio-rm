# Machine Data Source

To access this screen:

  * [Select Data Source](<Select%20Data%20Source%20Dialog.md>) screen >> **File Data Source** tab.

Connect with a data source that has a user data source name (DSN) or a system DSN. Machine data sources are specific to this machine and cannot be shared. Typically, this type of data source is useful where the connection details are unlikely to change and portability isn't important.

Note: You can also specify a [File Data Source](<Select%20Data%20Source_File%20Data%20Source.md>).

To connect to an ODBC database via a machine data source:

  1. Display the **Machine Data Source** screen.

  2. Select an existing data source in the list provided.

  3. Click **OK** to display the first (and possibly only) data-source-specific import screen. For example, if an _Excel Files_ data source was selected, the first screen is **Select Workbook**.

Note: These screens are provided by your operating system and managed by Microsoft. Datamine can't adjust these screens.

  4. Complete the driver-specific screens to connect to the machine data source.
