# Delete Overlay

To access this screen:

  * Select a 3D object overlay in the [Sheets](<Sheets%20Control%20Bar%20Overview.md>) control bar and press <Delete>.

This screen displays whenever you attempt to delete a 3D overlay for a loaded 3D object, using the [Sheets](<Sheets%20Control%20Bar%20Overview.md>) control bar.

This screen is only shown when an overlay is deleted using the <Delete> keyboard button; the right-click **Delete** option found for a 3D data object in the 3D window will always delete the selected overlay only, and not the corresponding data object.

As an object can be represented by more than one overlay, or by no overlays at all, the Delete Overlay screen offers a choice when you attempt to delete an overlay;

  * Just delete overlay You can elect to delete the selected overlay only, and leave the associated data object in memory (even if there are no remaining overlays for that object - it will still remain accessible via the Project Data control bar).
  * Unload data Selecting this object will remove the selected over and all other 3D overlays associated with the same data object, and then unload the data object from memory - it will no longer be accessible via the **Sheets** or Project Data control bars.
