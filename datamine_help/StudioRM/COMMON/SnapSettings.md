# Snap Mode

To access this screen:

  * **Home** ribbon **> > Snapping >> Settings**.

  * **Digitize** ribbon, **Snap** group, click the expander button.

Choose how snapping is performed when digitizing. You can select a data type and one or more overlays that can be snapped.

These data snapping settings apply to the currently active 3D window only. Snap settings are independently set for each window. 

**Note** : If you 'snap' to data, you will automatically enforce selection of that data, regardless of what data selection options were previously set on the [Overlay Selection](<OverlaySelectionDialog.md>) screen.

Activity steps:

  1. Choose the type of data to be a snap target, using **Snap to** options:

     * _Off_ Disable snapping to any data, regardless of other settings.

     * _Points_ Snap only to point object data, string edge midpoints, vertices of string data, wireframe vertices or centre points.

     * _Lines_ Snap to the nearest line i.e. including any position along a string, drillhole or wireframe triangle edge.

     * _Grid_ Snap to the intersection of the defined **[grid](<grid_format.md>)** , whether visible or not.

     * _Surface_ Snap onto a wireframe surface.

  2. Decide which **Overlays** can be used as a target for snapping. All loaded object overlays are listed and all are enabled by default. Check or uncheck overlays.

  3. Click **OK** to update snap settings.
