# Snapping Grid Parameters

To access this screen:

  * **Home** ribbon **> > Snap >> Snap >> Grid Snapping Parameters**.

  * **Digitize** ribbon **> > Snap >> Snap Grid Parameters**.

  * Use the quick key combination "gs".

  * Using the **[command line](<Command_Toolbar.md>)** , enter "set-snap-grid-parameters".

  * On the **[Find Command](<findcommand.md>)** screen, highlight **set-snap-grid-parameters** and click **Run**.

The Snapping Grid screen is used to define the snapping grid's origin and interval parameters. Define the XYZ increment values and origin. These settings are used when digitizing new or, editing existing data, e.g. points, strings and wireframes.

The **[Snap to Grid](<../command_help/set-snap-grid-switch.md>)** mode must be active for these settings to be of any use.

Note: 3D window grids aren't a target for snapping. The snapping grid is separate and formed using the Snapping Grid Parameters screen.
