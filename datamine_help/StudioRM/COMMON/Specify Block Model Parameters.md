# Specify Block Model Parameters

To access this screen:

  1. [Data Import](<data%20import%20dialog.md>) screen **> > ODBC**.
  2. Select the _Block Model_ **Data Type**. 
  3. Click **OK**

Use this screen to define block model prototype parameters prior to importing block model data from an ODBC data source. This prototype is used to construct the in-memory model for visualization and downstream analysis.

Settings can either be provided manually or can be derived from the target database.

Activity steps:

  1. Display the **Specify Block Model Parameters** screen.

  2. Define **XYZ** values for:

     * Model Origin (world coordinates)

     * Block Size

     * **Number of Cells**

Alternatively, click **From Database** to display the [Select Data Source](<Select%20Data%20Source%20Dialog.md>) screen and connect to the target ODBC database. Prototype information is extracted and posted back to this screen.
