# Editing String Data

Strings are useful constructs for defining and interpreting orebodies, mine planning, and mine design. They can define specific regions and wireframes may be generated from these to calculate volumes plus tonnes and weighted grades. A string is termed "closed" if its start and finish points are coincident; this is also called a perimeter.

Your application has a comprehensive collection of commands for creating and editing strings in the 3D window.

You can access most design commands using your ribbon system or directly by entering a command name into the **Command** toolbar. There are hundreds to choose from, all of them listed on the [Find Command](<findcommand.md>) screen (which lists all commands, including those related to string data editing).
