# Quick Keys

Many Studio [commands](<Studio%203%20Commands%20and%20Processes.md>) have quick keys of two to 4 characters. If the shortcut key is typed when a 3D window is active, the corresponding command will run provided it is available in your application. 

For example, "ns" launches the **new-string** command, which permits strings to be digitizing within a 3D window.

The quick keys are listed in the tables of commands for each menu and on each command help page.

You can also assign quick keys to ribbon commands, using the **[Customize](<Customize_Keyboard.md>)** screen. These are either single key or key+modifier combinations (such as SHIFT and F2, for example).
