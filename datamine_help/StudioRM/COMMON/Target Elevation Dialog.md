# Target Elevation

To access this screen:

  1. Run the [Project String at Angle](<../command_help/project-string-at-angle.md>) command.

  2. Specify "B" on the Road String Projection screen.

The **Target Elevation** screen displays.

Activity steps:

  1. Display the **Target Elevation** screen.

  2. Choose a **Target Elevation** to which a new ramp string will be extended. 

  3. Click OK to display the Road Gradient screen.
