# Text Wizard

To access this screen:

  * Display the Data Import screen.

  * Select the _Text_ **Driver Category** and a **Data Type**.

  * Click **OK**.

The **Text Wizard** displays.

The Text Wizard is used to import and export ASCII text data, after the selection of a text file using a standard browser (and setting additional options in the case of text-based block model files).

Depending on whether you are importing or exporting data, different options will be displayed. Select an option from below:

  * [Text Data Importing](<text wizard_import.md>)

  * [Text Data Exporting](<Text Wizard_Export.md>)
