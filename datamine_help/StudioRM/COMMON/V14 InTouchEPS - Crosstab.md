# The Crosstab Control bar

To show or hide this control bar:

  * The **DTS Crosstab** control bar displays automatically when the current project has connected to an active DTS schedule.

The Crosstab control bar displays schedule summary information from the last time your application was synchronized with DTS. Summary values are displayed for a specific date range - for example, average grade value, fuel cost, and so on.

Presuming crosstab information has been set up in the connected DTS project (consult your DTS help for information on how to do this), during playback of an animation (using the **Sequence Control** toolbar), the data columns of the control bar update to show a consolidated or compound value for the given data range and phase in the animation.
