# Grid Properties: More Line Formatting

Note: A Datamine [eLearning course](<https://datamine.learnupon.com/>) is available that covers functions described in this topic. Contact your local Datamine office for more details.

To access this screen:

  * Display the [**Grid Properties**](<grid%20properties%20dialog.md>) screen and select the **More Line Formatting** tab.

Grids can be fine-tuned using the settings on this tab, allowing precise control over their graphical format. 

Note: Some of the following options are dependant on other settings grid settings. For example, you can only set the properties for a particular spatial axis if it is enabled on the [Options](<VR_Grids_Options.md>) screen.

The following options are available. All settings are independently configurable for each major axis and the grid border:

Option | Description  
---|---  
Line Color | The color of the lines used for each grid axis or border. Click to display a colour picker.  
Major line style | Choose the line style for each grid axis and border.  
Major line weight | Define the thickness of the grid lines for each grid element.  
Major line intensity % | Choose the opacity of the lines for each element, ranging from 0 (invisible) to 100 (fully opaque).  
Minor line style | The line style for minor grid lines.  
Minor line weight | The line weight for minor grid lines.  
Minor line intensity % | The line opacity for minor grid lines.
