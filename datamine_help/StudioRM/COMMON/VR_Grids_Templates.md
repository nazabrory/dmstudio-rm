# Grid Properties: Templates

Note: A Datamine [eLearning course](<https://datamine.learnupon.com/>) is available that covers functions described in this topic. Contact your local Datamine office for more details.

To access this screen:

  * Display the [**Grid Properties**](<grid%20properties%20dialog.md>) screen and select the **Templates** tab.

Create, modify or load 3D display templates containing grid information. This makes it simple to quickly format your display grids to a standard format. 

Grids in 3D windows are shown as overlays, just like other 3D data. As such, this screen and the [3D Display Templates](<../VR_Help/3D_Templates.md>) screen are identical, so use that help file for guidance for this screen.

Tip: Once a grid template has been defined, select it using the **Sheets** or **Project Data** control bar's context-sensitive menu (top-level **Grids** folder) and automatically create and configure a grid at the same time.
