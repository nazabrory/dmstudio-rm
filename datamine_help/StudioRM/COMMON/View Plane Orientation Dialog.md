# Orientation

To access this screen

  * Run the [plane-by-one-point ("1")](<../command_help/plane-by-one-point.md>) command (quick key "1") and digitize a point in a 3D window.

  * Run the [plane-by-two-points ("2")](<../command_help/plane-by-two-points.md>)command (quick key "2") and digitize two points.

  * Run the [view-by-one-point ("v1")](<../command_help/view-by-one-point.md>) command (quick key "1") and digitize a point.

  * Run the [view-by-two-points ("v2")](<../command_help/view-by-two-points.md>) command (quick key "2") and digitize two points.

  * **View** ribbon **> > Sections >> 1 Point** and digitize a point.

  * **View** ribbon **> > Sections >> 2 Points** and digitize two points.

Note: Plane commands adjust the current 3D window [section definition](<../VR_Help/Sections.md>) for digitizing and design, whilst view commands simply orient the camera in the active 3D window. These commands are also available in other tasks and on other screens, such as the [Create Vein Surface](<Create_Vein_Surface.md>) screen, for example.

Define either the orientation of the currently active 3D section definition, or adjust the view direction of the 3D camera. The same screen is used for both, but contents vary depending on the command used.
