# Add a Display Template

To display this screen:

  * Show the [Format Display](<Format%20Overlays%20Dialog.md>) screen. In the **Overlay Objects** section select Templates, then click Add. 

Define a new [overlay](<concept_views%20sheets%20overlays.md>) template to store visual formatting properties. A template can be applied to other loaded data **[objects](<../VR_Help/Objects_Introduction.md>)**. You can also define a default template so that data of the selected type is formatted consistently when first loaded.

Activity Steps

  1. Open **[Format Display](<Format%20Overlays%20Dialog.md>)**. 
  2. In the **Overlay Objects** group, click **Show Templates.**
  3. Click **Add**.
  4. Enter the template **Name**.
  5. Select the data **Type** from _Numeric_ , _Alphanumeric_ or a calculated field.
  6. Click **OK**.

The new template appears in the **Overlay Objects** list.

  7. Choose when the template is applied:
     * If the template is **checked** , the template is the default template for all new object overlays of the selected type.  

**Note** : A default overlay is created when 3D data is loaded. By enabling a template, you are changing the formatting of subsequent default overlays, as opposed to using system defaults.

     * If **unchecked** , the template is available for other objects, but will not be applied when a default overlay is created.
  8. **Save** your project.
