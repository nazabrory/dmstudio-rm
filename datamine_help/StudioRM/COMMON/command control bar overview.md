# Command Control Bar

The [Command](<Command_Toolbar.md>) tool bar, commonly referred to as the "command line" lets you launch process or design commands by entering their name and pressing <ENTER>. 

Like other control bars, you can position this one wherever you want on the screen, or even have it floating above other screen items. See [Customizing Control Bars](<Customizing.md>).

The Command control bar typically shows status messages during processing of either a [design command or a process](<Studio%203%20Commands%20and%20Processes.md>). 

It also displays useful startup information, such as the current product name and version, and if an external graphics card is detected (and if so, which type).

Right-clicking this control bar reveals the following menu options:

  * Copy: Copies the currently highlighted text to the clipboard.

  * Clear: Clears the contents of the control bar, ready for new content.

  * Select All: Selects and highlights all text in the control bar, even if it is currently scrolled out of view.

  * Save As: Save the contents of the control bar in a Rich Text Format (.rtf) file.

  * Print: Access connected printers to create a hardcopy of the current control bar contents.

  * Font: Pick a different font (this change is retained between application sessions.

  * Word Wrap: Choose if text wraps at the end of the control bar or if a scroll bar is displayed instead.

  * Hide: Hides the control bar. It can be redisplayed using the **Show** menu.
