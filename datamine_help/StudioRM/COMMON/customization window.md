# Customization Control Bar

**Note** : For security reasons, you cannot load content into the **Customization** window that is outside of your local PC domain. This includes Internet content.

The Customization control bar is a web page through which browser-supported scripts can be run. 

Load any local script (.htm, .html) into the Customization window by dragging and dropping it.

You can also automatically launch a script when your project opens by specifying it in your [Project Settings](<ProjectSettings.md>).

The default **Customization** page is stored in your application's release directory, but you can choose to display your own page using the Options screen's Web Browser tab. See [Options: Environment](<Options_Environment.md>).
