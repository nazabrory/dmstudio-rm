# Customize

To access this screen:

  * Right-click a toolbar icon and select Customize.

Set up general system behaviour and configure the content of toolbars.

This screen displays the following tabs. 

  * [Commands](<Customize_Commands.md>)Add and remove buttons from displayed toolbars.

  * [Toolbars](<Customize_Toolbars.md>)Display or hide toolbars, and other options.

  * [Tools](<Customize_Tools.md>)Set up and configure external applications.

  * [Keyboard](<Customize_Keyboard.md>)Set up accelerator keys for using your application.

  * [Menu](<Customize_Menu.md>)Customize menu items.

  * [Mouse](<Customize_Mouse.md>)Set up mouse behaviour and sensitivity.

  * [Options](<Customize_Options.md>)General system options.
