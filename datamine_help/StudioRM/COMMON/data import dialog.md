# Data Import

To access this screen:

  * **Data** ribbon **> > Load >> External**. Select a file to import.

  * **Data** ribbon **> > Load >> External >> any data type**.

Define the driver category and data type settings so that the correct data source driver is used to import or load a non-Datamine file.

Activity steps:

  1. Select a **Driver Category**. 

  2. Select a Data Type within the selected category.

  3. Click OK to launch the screens appropriate to the selected **Driver Category** and **Data Type**.

**Tip** : find out more about a driver and the screens it displays by clicking **Driver Help**.
