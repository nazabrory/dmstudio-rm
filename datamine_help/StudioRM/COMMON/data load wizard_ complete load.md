# Data Load Wizard: Complete Load

To access this screen:

  * Using the Data Load Wizard, complete the [Import Data Tables](<Data%20Load%20Wizard_Import%20Data%20Tables.md>) and [Import Data Types](<Data%20Load%20Wizard_%20Import%20Data%20Types.md>) screens.

This is the final screen of the Data Load Wizard, and provides further options relating to your data import. Click Finish to close the wizard.

This screen asks you two questions:

  * Would you like to create 3D hole traces from the collar and survey data?: the wizard will automatically create 3D Drillhole strings from collar and survey data, if this option is selected.

  * Would you like an intersections table created for you?: if appropriate data is loaded, the wizard will give the option to create an intersections table. See [Create an Intersections Table](<../PLOTS_LOGS/CreateIntersections.md>) and [Selecting, naming and saving intersections](<../PLOTS_LOGS/SaveIntersections.md>).
