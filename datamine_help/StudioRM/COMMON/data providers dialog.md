# Data Providers

To access this screen:

  * On the [Data Import](<data%20import%20dialog.md>) screen, select the [ODBC v2] data source type and click OK.

This dialog is shown as part of the data import process, and is used to select an ODBC data provider prior to configuring the data source, and finally loading the required tables. This dialog will display all registered ODBC Data Providers on your system. 

Select a provider and click OK to move to the next step of the import process.
