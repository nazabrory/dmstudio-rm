# Default Hole Set  
  
To access this screen:

  * With static drillhole data in memory (see [Defining and Building Drillholes](<LoadingDynamicDrillholeData.md>)), select the Holes control bar and right-click the top-level project icon to select Default.

This screen is used to set the default hole set. The default set will be the base set from which data will be copied during specific Holes control bar menu options.

Select a hole set description from the supplied list and click OK.
