# Project Properties: Contents

To access this screen:

  * In the [Loaded Data](<Loaded%20Data%20Control%20Bar.md>) or [Sheets](<Sheets%20Control%20Bar%20Overview.md>) control bars, double-click the top-level project icon. Select the Contents tab.

The Contents tab is one of three available tabs in the Project Properites screen. For for the current project, it displays information relating to the number of holes, log sheets, report sheets, section sheets and table sheets the project contains. These fields are read-only.
