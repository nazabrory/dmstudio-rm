# Format Display: Overlays

To access this screen:

  * On the [Format Display](<Format%20Overlays%20Dialog.md>) screen, select the Overlays tab.

Manage [plot projection overlays](<../PLOTS_LOGS/Plots-overlays.md>) and formatting templates and define their associated formatting parameters. 

The Overlays screen may contain one or more of the following tabs in the **Overlay Format** section. The contents of this screen depend on the data type of the overlay that has been selected.

Select a link for more information:

  * [Style](<Format%20Display%20Dialog_Overlays_Style.md>)

  * [Drillholes](<Format%20Drillholes.md>)

  * [Color](<Format%20Display%20Dialog_Overlays_Color.md>)

  * [Symbols](<format%20display%20dialog_overlays_symbols.md>)

  * [Feature](<Format%20Display%20Dialog_Feature.md>)

  * [Labels](<Format%20Display%20Dialog_Overlays_Labels.md>)

  * [Advanced](<Format%20Display%20Dialog_Overlays_Advanced.md>)

**Note** : This screen manages the formatting of legacy 2D plot projection contents. 3D projection data is formatted using the corresponding 3D properties screen. See [Projection Overlay Types](<../PLOTS_LOGS/Projection%20Overlay%20Types.md>)

There are two sections to this screen; **Overlay Objects** and **Overlay Format**.

## Overlay Objects

Add, edit, delete and select object overlays and use templates.

This section of the screen displays either **Overlays** or **Templates** (depending on the selection at the top).

  * **Overlays** Displays the current 2D projection overlays (that relate to loaded 3D data) of the active plot sheet. Pick an overlay to format it using the **Overlay Format** section on the right. See "Overlay Format", below.

  * **Templates** List any 2D projection overlay templates associated with the current project. See [Format Display: Templates](<Data%20Display%20Templates.md>).

## Overlay Format

Specify the formatting parameters for the selected overlay or template.

This screen is only populated if **Overlays** is selected on the left, and an overlay has been picked from the list. 

Note: If **Templates** is selected on the left, this screen is empty.

See above for links to data-type-specific information.
