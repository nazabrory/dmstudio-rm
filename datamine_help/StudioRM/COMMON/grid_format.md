# Table Grid Format

To access this screen: 

  * Right-click a control bar containing tabular results and choose **Grid Format**.

Configure the contents of a tabular control bar.

Activity steps:

  1. Select a table column in the column list. This is the column you wish to format.

  2. To change the left-right order of the selected column, use **Move up** and **Move Down**.

  3. To hide the selected column, **check** **Hide column**. This does not prevent table data from being calculated. Only the display of that data is blocked.

  4. For numeric data, choose the Digits after decimal point.

  5. To change the alignment of data values within table cells, **check** Right justify.
