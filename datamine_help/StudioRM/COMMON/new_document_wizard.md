# Data Load Wizard

To access this wizard:

  * Activate the Data ribbon and select Load >> Hole Wizard.

Load drillhole data into your project.

To import drillhole data using the Data Load wizard:

  1. Unless you have chosen not to display it during a previous session, you will see the Welcome screen. Optionally, you can Skip this page in future.

  2. Click **Next**. 

The **Import Data Types** screen displays. 

  3. Specify the type(s) of data to import. See [Import Data Types](<Data%20Load%20Wizard_%20Import%20Data%20Types.md>).

  4. Click **Next**.

The **Import Tables** screen displays.

  5. Create a list of data tables to import. See [Import Tables](<Data%20Load%20Wizard_Import%20Data%20Tables.md>).

  6. Click **Next**.

The **Complete Loading** screen displays.

  7. Specify final options and dismiss the wizard. See [Complete Loading](<data%20load%20wizard_%20complete%20load.md>).

Dynamic drillhole data is loaded and, if you chose for 3D hole traces to be created, drillhole data are represented in the **3D** view(s) using default settings.
