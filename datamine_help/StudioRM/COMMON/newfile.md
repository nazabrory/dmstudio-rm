# Add New File

To access this screen:

  * Click the **Add New File to Project** icon on the Project Files toolbar.

  * **Home** ribbon **> > Project >> Add Files >> Create File**.

The Create File screen is used to create a file representing a specified type (table or macro) and add it to your currently active project file.

To create a new, empty file and add it to your project:

  1. Select which **Table Type** to create; either _Table_ or _Macro_.

Note: The Datamine binary table format will be either .dm or .dmx depending on your [**system options**](<data%20options.md>).

  2. Enter the **File Name** of the file to create.

  3. Specify the location to store the new file, or browse for one.
