# Project Wizard: Create Project

To access this screen:

  * Display the [Project Wizard - Project Files](<Project%20Wizard_Project%20Properties.md>) screen and, after adding and removing the relevant files, click Next.

The Project Wizard - Create Project screen is the final screen of the [Project Wizard](<ProjectWizard.md>) and is used to confirm project details and create the project file. If the listed Project Summary details are correct, click Finish to create and open your project file.

**Note** : If files were added to your project in a previous step, they will not be loaded into memory when the project is created. You will need to do this separately, although when you save your project, you are offered the chance to automatically reload data in memory when the project is next reopened. See **[Loading Data](<Concept_Loading%20Data.md>)**.

**Note** : If files in a non-Datamine format have been added, they are automatically converted to the Datamine default format (either .dm or .dmx) as part of project creation. See [Datamine File Formats](<Datamine-File-Format.md>).
