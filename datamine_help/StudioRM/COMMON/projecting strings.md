# Projecting String Data

Your application provides a number of commands for projecting strings.  

## project-string-onto-wf

Command |  Quick Key  
---|---  
[project-string-onto-wf](<../command_help/project-string-onto-wf-limit.md>) |  ptw  
  
Projects a string vertically up or down onto the current wireframe to create a new string in which all points lie on the wireframe.

## project-string-at-angle

Command |  Quick Key  
---|---  
[project-string-at-angle](<../command_help/project-string-at-angle.md>) |  pro  
  
Creates a new string by projecting an existing string either up or down a specified distance. The projection is vertical in the Z direction. This command gives choices for whether the projection should be Up, Down, Both or Relative. It also provides appropriate prompts.

Related commands are: [define-new-rosette](<../command_help/define-new-rosette.md>), use-rosettes-switch, [set-face-angle](<../command_help/set-face-angle.md>), [use-modelfile-switch](<../command_help/use-modelfile-switch.md>).

## project-to-view-plane 

Command |  Quick Key  
---|---  
[project-to-view-plane](<../command_help/project-to-view-plane.md>) |  ptv  
  
Creates a new string by projecting an existing string perpendicularly onto the current view plane.

## project-string-onto-wf-limit

Command |  Quick Key  
---|---  
[project-string-onto-wf-limit](<../command_help/project-string-onto-wf-limit.md>) |  -  
  
Enables a string to be projected onto a wireframe using minimum and maximum distances to control up and down projections.
