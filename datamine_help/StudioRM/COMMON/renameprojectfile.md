# Rename Project File

To access this screen:

  * Right click a file reference in the **[Project Files](<Concept_Project%20Files%20Control%20Bar%20Overview.md>)** control bar and click **Rename**.

Rename a project file reference by entering a new file name into the **To** field (the original name is shown as **From**).

Toggle the display of the file path using Show Full File Paths; if checked, the full system file path for both original and new project files displays, otherwise only the file name is shown.
