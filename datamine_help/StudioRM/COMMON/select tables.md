# Select Tables

To access this screen:

  * Display the [Data Providers](<data%20providers%20dialog.md>) screen.

    1. For Datamine **Fusion** database connections, select Remote or Local and click **OK**.

    2. For legacy **Earthworks** database connections, Click OK and select a [File Data Source](<Select%20Data%20Source_File%20Data%20Source.md>) or [Machine Data Source](<Select%20Data%20Source_Machine%20Data%20Source.md>).

This screen displays as part of the data import process, and allows you to select the information you wish to derive from the connected ODBC data source.

The tree menu displayed shows a list of data types associated with the data you are connecting to. 

Expand each area to see a list of components underneath. You can either 'drill down' to component level and select or clear the appropriate check boxes, or select a folder item to ensure all items within it are included in the import.

Alternatively, click Add Connection to see a list of available data providers and add another ODBC data source.
