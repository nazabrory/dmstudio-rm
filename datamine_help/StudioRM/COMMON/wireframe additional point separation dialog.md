# Maximum Point Separation

To access this screen:

  * Using the **[command line](<Command_Toolbar.md>)** , enter "dtm-new-point-separation"

  * Use the quick key combination "nps".

  * Display the **[Find Command](<findcommand.md>)** screen, locate **dtm-new-point-separation** and click **Run**.

This screen is used to specify the **Maximum distance between additional points** in a wireframe (the maximum length of a wireframe mesh segment). It is displayed by running the [dtm-new-point-separation ("nps")](<../command_help/dtm-new-point-separation.md>) command.

This setting will have a direct affect on the resolution of data within created wireframe bodies. A lower maximum distance will increase the overall resolution of the data, but will also increase the file size exponentially.
