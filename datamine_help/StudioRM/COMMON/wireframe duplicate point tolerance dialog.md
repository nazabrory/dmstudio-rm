# Duplicate Point Tolerance

This screen is used to determine the minimum distance that is permitted between wireframe points, before the group of points is treated as a single entity. For example, if a 3 meter tolerance is specified, all points falling within a 3 meter radius of the original point are merged, to create a single unit of data. In effect, this setting will determine the overall maximum resolution of the wireframe.

**Note** : you can also change this setting via the **Project Settings** screen (Wireframes tab). See [Project Settings: Wireframing](<Project%20Settings_Wireframing.md>).
