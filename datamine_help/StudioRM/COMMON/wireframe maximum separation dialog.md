# Maximum Point Separation

To access this screen:

  * **Explicit** ribbon **> > Create >> Point Separation**.
  * Using the **[command line](<Command_Toolbar.md>)** , enter "dtm-new-point-separation"

  * Use the quick key combination "nps".

  * Display the **[Find Command](<findcommand.md>)** screen, locate **dtm-new-point-separation** and click **Run**.

This screen is used to specify the maximum distance between wireframe points (the maximum length of a wireframe mesh segment) when linking strings, including end linking and DTM creation. Measurement is applied using the currently specified units for the session (metric or imperial, depending on your [data options](<data%20options.md>)).

**Note** : Adjusting the point separation via the **dtm-new-point-separation** command also updates the **Maximum Segment Length** value on the **Project Settings** screen (**Wireframe Linking** tab). See [Project Settings: Wireframe Linking](<Project%20Settings_%20Wireframe%20Linking.md>).a
