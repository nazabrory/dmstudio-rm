# Allowable Offsets

To access these controls:

  * Display the [Decline Optimizer](<DeclineOptimizerDialog.md>) screen and select the **Path Control** tab.

The **Allowable Offsets** tools are used to define tolerance cuboids which define the tolerance within which the an underground decline is allowed to deviate from the points on the input control string.

The cuboids are defined with X, Y and Z **Minimum Offset** and **Maximum Offset** values which allow the tolerance cuboid to be off centre from the point on the control string.

You can also specify how (and if) specified offsets are applied, using **Allow Offsets In** options.
