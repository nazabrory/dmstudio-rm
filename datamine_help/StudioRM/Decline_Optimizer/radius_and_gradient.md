# Radius and Gradient

To access these controls:

  * Display the [Decline Optimizer](<DeclineOptimizerDialog.md>) screen and select the **Path Control** tab.

Define the radius and gradient of incline models generated with the optimize-decline command.

  * **Radius Control** Define a Target Radius for the curved sections of the decline or alternatively, if the **Variable** option is selected, a minimum and maximum radius within which the radius of curved sections should lie.

  * **Gradient Control** Define a Maximum Gradient (in degrees, 1:N or percent) for inclined sections of the decline. Alternatively, if the Variable option is selected, a minimum and maximum gradient can be specified.
