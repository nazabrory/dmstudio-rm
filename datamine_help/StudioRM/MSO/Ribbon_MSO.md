# MSO Ribbon   

![](../Images_STUDIORM_ONLY/ribbon_MSO.jpg)   

Your application uses [ribbons](<../COMMON/Ribbons-overview.md>) to present commands.

The MSO ribbon contains a series of functions that allow you to define optimal stope shapes based on your selection of model, mining method and economics. Specifically, it provides access to the MSO "v3" engine, a major update to MSO functionality available in earlier versions of Studio products.

![note.gif \(1017 bytes\)](../Images/note.gif) |  MSO Licensing The MSO ribbon contains functions that are licensed separately from your core system, and independent licensing exists for the legacy (v2) version of MSO and the current implementation (v3). To use the MSO functions provided by the MSO ribbon, you will need to register a standalone MSOV3 license and make it available to the host machine. An MSO license (for version 3) is available from your local Datamine representative. If you are a previous user of MSO functionality, contact your Datamine office to discuss how to upgrade your existing license to allow the latest version. Please note that MSO ribbon functions cannot be enabled with existing MSO (version 2) licenses.
