# NOHOLD Macro Command

Allows exit from a macro or menu when a fatal error is encountered or if ! is entered into the command line.

The current command in the menu or macro is terminated, and control returns to the keyboard with the systems prompt:
    
    
    >>> Press return to invoke command processor <<<

Once the `!NOHOLD` command has been given, it remains in force until the ![HOLD](<hold.md>) command is executed.
