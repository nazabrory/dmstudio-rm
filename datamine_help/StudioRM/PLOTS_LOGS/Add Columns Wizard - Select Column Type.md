# Select Column Type

### To access this dialog:

From the Tables window

  * In the Tables window, right-click a table and select Format.

  * In theColumnstab, clickAdd.

From the Logs window

  * In the Logs window, click a table to select it.

  * Right-click and select Properties.

  * In the Log View Properties dialog, select theColumnstab.

  * By theColumns in Viewwindow, clickAdd.

This is the first screen in the Column Wizard, and defines the type of data column that you wish to add to a table.

Field Details:

The following options are available:

Data Column: select this option to display all downhole sample data from all data tables \- for example, grades and lithology codes. You could use this column type, for example, to display any of the original data fields from imported tables.

System Field: select this option to display computed field values such as downhole depth, inclination, dip and coordinates of the desurveyed trace. Use this column type to graph, say, how the azimuth of the hole trace varies downhole.

Axis Column: where available, select this option to display downhole depth or the coordinates of hole traces. Use this column type, for example, to display a depth or elevation axis.
