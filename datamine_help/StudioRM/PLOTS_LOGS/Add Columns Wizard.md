# Column Wizard

### To access this dialog:

  * In the Tables window, right-click a table and select Format.

  * In theColumnstab, clickAddby theColumns in Viewwindow.

The ColumnWizard can be activated when adding columns to Tables and Logs. 

The wizard uses the following dialogs:

  * [Add Columns Wizard - Select Column Type](<Add%20Columns%20Wizard%20-%20Select%20Column%20Type.md>): select the type of data column to be added.

  * [Column wizard - Select Table](<add%20Columns%20Wizard%20-%20Select%20Table.md>): select the table.

  * [Add Columns Wizard - Select Fields](<Add%20Columns%20Wizard%20-%20Data%20Columns.md>): select a field or fields for the new data column(s).

  * [Add Columns Wizard - Composited Fields](<Add%20Columns%20Wizard%20-%20Compositing%20Parameters.md>): if a compositable field was chosen during the Select Fields stage, this screen is used to define the compositing method.

  * [Add Columns Wizard - Select Style](<add%20columns%20wizard%20-%20column%20styles.md>): select the display style for the new column(s).

Tables and logs may display all or some of the fields defined in an imported data file. You can also add composited fields from other tables, and fields which are computed by the program like sample length or desurveyed sample coordinates.

Choose the Help button on individual dialogs for more information.
