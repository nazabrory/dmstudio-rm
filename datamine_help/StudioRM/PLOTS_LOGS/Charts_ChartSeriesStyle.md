# Chart Series Style

### To access this dialog:

  * In the [Histogram](<Chart_Histogram_DataSelection.md>) or [Scatter Plot](<Chart_ScatterPlot.md>) dialog, ensure that the Show Toolbar checkbox (bottom left corner of dialog) is ticked.

  * Select a chart from the chart thumbnails pane, click the Chart Legend icon ![](../Icon_Popups/icChartLegend.gif) (i.e. toggle on), double-click a colored box next to a legend item.

The Chart Series Style dialog is used to format the display of the histogram or scatter plot data series.

![](../Images/Chart_ChartSeriesStyle%201.gif)

This dialog contains the following tabs:

  * Interior: set the basic visual formatting for each bar or line, including foreground and background color etc.

  * Border: edit the border of the chart items, including color and thickness.

  * Text: for all displayed text, this tab is used to control all font formatting, including alignment options.

  * Shadow: define bar shadow settings.

  * Symbol: the symbol used to denote the bin values can be configured here.

  * Fancy Tool Tip: this area is reserved for future use.

![note.gif \(1017 bytes\)](../Images/note.gif) | 

  * click the '+' to the left of each item in order to display additional parameters
  * If multiple charts have been generated according to the initial data selection, the scope of this operation will be applied to the entire chart- however - if a compound chart has been created, you can edit each of the data series that comprise that chart independently.
