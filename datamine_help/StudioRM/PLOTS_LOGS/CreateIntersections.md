# Create an Intersections Table  
  
If you use the [Data Load Wizard](<../COMMON/new_document_wizard.md>) to load drillhole data, you can create a default intersections table. 

The default table definition can be changed, or you may create a new, empty intersections table at any time. A wide range of field types are available for inclusion in the intersections file including an intersection label, averaged hole properties, length weighted average grades, intercept coordinates, dominant text values and more.

To create a default intersections table:

  1. Display the [Select Table Type](<../COMMON/Select%20Table%20Type%20Dialog.md>) screen.

  2. Select the Intersections table type.

  3. Click OK.

An _Intersections_ table object is created and listed in the Loaded Data control bar.
