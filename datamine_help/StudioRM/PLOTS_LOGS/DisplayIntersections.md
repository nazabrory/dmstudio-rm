# Displaying intersections  
  
The contents of the intersections table can be displayed, like any other hole table, in plots and logs windows. 

Note: The following procedure assumes that either a Plots or Logs data window is visible.

You might, for example, wish to add the **Zone** field and one or more length weighted average grade fields to section and log plots:

To display an intersections table:

  1. Select the section or log view in which you wish to display the intersection field.

  2. **Manage** ribbon >> Sheet Layout >> Format Display.

  3. Select a drillhole overlay on the left.

  4. Activate the **Drillholes** tab.

  5. Check **Display downhole columns**.

  6. Click **Insert**.

The **Column Wizard** displays.

  7. Select the Data Column, type and click **Next**.
  8. Select Intersections and click **Next**.
  9. Choose the data attribute that represents the intervals, for example, ZONE. 
  10. Click Next to display the Format screen. See [Format Overlay or Column](<../VR_Help/DH_PropDialog_Columns_Format.md>).
  11. Click OK twice to dismiss both screens.

  12. **Manage** ribbon **> > Plot Items >> Table**.

  13. Select Intersections and click **OK**.

An Intersections table plot item is added to the current sheet.
