# Format Hole Set Dialog

To access this screen:

  * Display the [Log View Properties](<Log-View-Properties.md>) screen and select the **Hole Set** tab.

To see the changes made by formatting hole sets, select the Holes tab from the Workspace window. Here you will see the different hole sets added to the project and the individual holes.

Choose |  To Do This  
---|---  
Default set |  Bring in the default set of holes.  
All holes |  Bring in all the holes in the project.  
Active hole |  Bring in the hole that is currently selected.  
Selected set of holes |  Bring in a set of holes.  
Selected set box |  Displays the different sets of holes.  
Edit Sets |  Open the Hole sets dialog  
Hole Sets dialog |  Display the different hole sets in the display box.  
New |  Display the New hole set dialog.  
Add Holes |  Display the New hole set dialog.  
Edit |  Display the Hole set contents dialog.  
Rename |  Display the Hole set name dialog.  
New hole set dialog |  There are 4 options. To add active hole, add all holes, holes from a table and holes from another set. Once the selection is made click Next.  
Select drillhole table dialog |  The box displays all the tables present in the project. Select a table and move on.  
Hole set contents dialog |  This dialog displays the holes within a set and the holes not in a set. You can move holes in and out of a set using the Add and Remove buttons.  
Hole set name dialog |  Enter the name that best describes the particular set of holes.
