# Changing Log Column Formats

Log columns can be displayed as text or in a variety of graphical formats.

![](../Images/V2NewLogViewS.gif)+

##  ![](../Images/StepByStep2.gif)

## To change the display style of a column

  1. Right click a column in the Log View and selectFormat Display....

  2. In the Properties dialog, select the Columns tab.

  3. Select the column name in the **Columns in View** box.

  4. Select the **Style Templates** tab to display the current display style. The **Customized style** check box indicates whether the default template style has been modified. Clear the **Customized style** check box to reset the style to the original template settings.

  5. Select the column display style required from the **gallery** of pre-defined styles. Settings on the other style tabs will be modified according to the selected template.

  6. View and modify the style settings on the other tabs as required.

  7. Choose **Apply** to view the changes or **OK** to close the dialog.
