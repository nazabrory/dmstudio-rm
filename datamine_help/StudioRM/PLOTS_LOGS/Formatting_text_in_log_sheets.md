# Format Text Dialog

Log sheets consist of header, column and footer sections, the layout and contents of each are fully customizable. The default layout created by the program when a new log view is created contains a useful selection of information about the project and the holes defined in the document. You may, however, wish to emphasize certain text that is of particular importance in the context such as a specific lithology. 

  1. In Log view, select **Format | Text Highlights** form the main menu to open the **Format Text** dialog.

  2. Type the word you wish to format into the edit box and select one or more of the following to be applied:

## Note:

The feature is case sensitive and will only pick out single and complete words. 

  
**Choose** |  **To do this**  
---|---  
Bold |  Embolden the word.  
Italic |  Italicise the word.  
Underline |  underline the word.  
Strikeout |  Strike through the word.  
  
**Choose** |  **To do this**  
---|---  
Add |  Add a formatted word to the Created Formats list. Type the word, select the formatting required then click Add.  
Remove |  Remove a formatted word from the Created Formats list. Select the format then click Remove.  
Update |  Alter an existing Created Format. Select the text format you wish to alter, make the change then click Update.  
  
  1. Click **OK** to apply the formatting to your log plot. **Cancel** will abort the formatting.
