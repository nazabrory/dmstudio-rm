# Import an Intersections File

When an intersections file is imported:

  * The intersection table is added to the drillhole table definition.

  * A table view sheet is created using default format settings.

To import an intersections file

  1. Activate the Data ribbon and select Load >> External >> Other.

The [Data Import](<../COMMON/data%20import%20dialog.md>) screen displays.

  2. Select a Driver Category and Data Type.

  3. Select the file to import and complete the driver screens. Press F1 for more information on each screen.  

  4. The Define Drillhole Data Table screen displays. 

Expand the Table Type list and select _Intersections_.

  5. Map the file's data fields to the properties of an intersection file.
