# Insert Line

This dialog is displayed when a new 3D line is being added to a section in the Plots window using theManageribbon'sInsert | Linecommand (click Apply when line design is complete).

This dialog is used to define the type of line that is to be added to the section view.

When you have selected a line type, click Next to display the [Create Line](<createlinedialog.md>) dialog. The new line will be stored within your application as an object, and can either be created as a standalone object, or part of another existing one.

Field Details:

Ordinary 3d line: add a standard data string to the current project.

Fault: define the new line as a fault indicator.

Contact: define the new line as a contact line..

other linear geological feature: if your new line represents some other geological feature, select this option.
