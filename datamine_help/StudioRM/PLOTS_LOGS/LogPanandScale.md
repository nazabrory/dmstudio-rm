# Hole Position and Scale  
  
The data inside a [plot sheet projection](<alignviewwithsection.md>) or [log sheet](<../COMMON/Window%20Overview_%20Logs.md>) can be positioned and scaled.

These commands are accessed in different ways for plot projections and log sheets:

  * **Plot projections** Clicking into a plot projection reveals a **View** ribbon, containing a range of functions to adjust the position and scale of data within it.

  * **Log sheets** Clicking a log sheet displays a context-sensitive Logs ribbon. The **Scale** group contains commands for scaling the presentation of log sheet contents.
