# Merging tables

Interval log tables with different sampling intervals (Depth_From and Depth_To values) can be merged into a single table which preserves all the fields and sampling intervals from both tables.

## To merge two or more interval log tables

  1. Using theDataribbon, selectObjects | Combine.

  2. Select the tables you wish to merge.

  3. Type the name of the new merged table in the New Object Name box.

  4. Choose OK. A new table view is created.
