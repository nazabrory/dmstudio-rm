# Opening documents

By default, your application will start up to display the Start page. Regardless of how the application layout was last saved, you will have access to the following functions:

  * create a new project using the Project Wizard.

  * open an existing project using the[Project](<../COMMON/Ribbon_File_Button.md>)button

  * open a blank (empty) project.

## When a document is opened

  * all external data files referenced by the document are imported

  * data files are validated

  * all table, section and log views defined by the document are restored.

In this way, any changes made to the external data files, including the addition of new holes, surveys, assays and logs, are automatically incorporated in the existing tables and plots. You can also update the tables and plots while the document is open by activating theDataribbon and selectingRefresh | Refresh All.
