# View Rotation Settings

To access this screen:

  * Display the [**View Settings**](<../COMMON/Section%20Definition%20Dialog.md>) screen and click the **Rotate** tab.

Use this screen to rotate the view direction of the target plot projection using sliders. 

By default, the projection updates automatically as the sliders move, but you can turn this behaviour off by unchecking Dynamic.

Set the view using the **Dip** and **Azimuth** sliders, or set the absolute value you want with the corresponding text field.

**Note** : You can also set the view dip and azimuth using the **[View Direction](<RotateView.md>)** screen.
