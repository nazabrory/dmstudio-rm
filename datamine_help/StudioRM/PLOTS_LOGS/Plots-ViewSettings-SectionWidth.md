# Section Width Settings

Use the slider on this screen to adjust the width of the currently selected section corridor. Alternatively type in a value (of the total width shared by the front and back **[clipping](<ClipView.md>)** distances.

Enable Dynamic to update the projection in real time. 

**Note** : Adjusting these settings only affects the view of 3D data if clipping is applied.
