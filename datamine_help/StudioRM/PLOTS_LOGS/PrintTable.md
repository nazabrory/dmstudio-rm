# Printing tables

  * Use the Page Setup command to change printing options including margins, titles, grid lines, page order and alignment.

  * Use the Print Setup command to select the printer device, print page size and page orientation.

  * Use the Print Preview command to view pages prior to printing.

  * Use the Print command to print the selected table view using the current page and printer settings.
