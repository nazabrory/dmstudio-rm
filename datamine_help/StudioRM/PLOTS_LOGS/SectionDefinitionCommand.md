# Choose a New Section Orientation

The section width and section coordinates of the current section view are displayed.

Enter a new section width and coordinate value or select OK to use the defaults.

After selecting OK you will be asked if you want the view direction to be perpendicular to the new section. select Yes if you do, or No if you want the view direction to be unchanged.
