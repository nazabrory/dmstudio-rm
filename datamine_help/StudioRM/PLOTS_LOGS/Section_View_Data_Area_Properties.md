# Section View Properties - Data Area

Choose |  To do this  
---|---  
Frame |   
Height |  Change the data area frame height  
Width |  Change the data area frame width  
Reset |  Reset frame to page margins  
Page Setup |  Select page size, orientation and margins  
Hull |   
Do not display |  Hide the data limits hull  
Automatic |  Display the data limits  
Style |  Select a line style for the data limits hull  
(Color) |  Select a color for the data limits hull  
Hull |   
Do not display |  Hide the section boundary  
Automatic |  Display the section boundary  
Style |  Select a line style for the section boundary  
(Color) |  Select a color for the section boundary
