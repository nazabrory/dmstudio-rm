# Symbol 

To access this screen:

  * Insert a new symbol plot item into a plot sheet. See [Symbol Plot Items](<Symbols.md>).

  * Double-click a symbol plot item on a plot sheet.

  * Right-click a symbol plot item and select **Properties**.

The Symbol screen is used to define the symbol type, size and colour of a **[symbol plot item](<Symbols.md>)**.

The following tabs are shown:

  * Symbol PropertiesEditable symbol properties are shown here.
  * Drawing OrderUsed to determine at which point in the screen drawing process the current item is drawn to the screen. See [Drawing Order](<Format_Drawing_Order_Dialog.md>).

### Symbol Properties

The following settings can be applied to text boxes:

Symbol | Choose a symbol to display.  
---|---  
Group | Choose the category of symbols to choose from. This automatically updates the **Symbol** list (see above).  
Size | Set the size of the symbol, in points.  
Colour | Choose the colour of the symbol.  
  
**Note** : A complete list of symbol plot item properties is accessible using the **[Properties](<Symbol-properties.md>)** control bar.
