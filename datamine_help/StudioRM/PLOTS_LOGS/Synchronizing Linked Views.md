# Synchronizing linked views

Holes or intervals selected in one view can be displayed in any other view using thePlot Viewribbon (Plotswindow active) |View | Synchronizecommand from the menu. The action of other views when a selection is synchronized is determined by the status of the **![](../Icon_Popups/icviewlinked.gif)Linked** and ![](../Icon_Popups/icviewlive.gif)**Live** settings  

**When view is** |  **Synchronizing a selection in another view will**  
---|---  
Linked only |  Select and display the same selection. The data selected will not always be visible.  
Linked and Live |  Select and display the same selection and modify the view so that the selected data is visible.  
Live only |  Modify the view so that the selected data is visible but not selected.  
  
## You can synchronize views in many ways

![](../Images/synchronize.gif)

Choose the method which best suits the way you work:

  * ThePlot Viewribbon (Plotswindow active) |View | Synchronizecommand, or
  * Select a hole interval then right-click to display the context menu. Select Synchronize.

##

Data can be selected in any table, section, 3D or log view and synchronized with all other linked table, section, 3D and log views.
