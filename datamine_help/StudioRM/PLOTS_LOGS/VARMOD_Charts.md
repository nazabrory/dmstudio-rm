# Variograms: Charts  
  
To access this screen:

  * Display the [Variogram](<VARMOD_Introduction.md>) screen and select the Charts tab.

Control the display of loaded variograms using the folder tree.

Activity steps

  1. Display the **Variogram** screen.

  2. Select the **Charts** tab.

  3. Use the charts tree menu to hide or display chart data.

Note: Charts are enabled or disabled using the [Variograms: Data Selection](<VARMOD_Data_Selection.md>) tab. The **Charts** tab controls an enabled chart's visibility.
