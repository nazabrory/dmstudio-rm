# Variogram Properties

To display this screen:

  * On the [Variogram](<VARMOD_Introduction.md>) screen, right-click a variogram in the **Variograms** table and select **Edit Properties**.

Manage the display of variogram chart point formatting. 

Select a Colour and Symbol, the click **OK** to update the preview of the target variogram's data point(s).
