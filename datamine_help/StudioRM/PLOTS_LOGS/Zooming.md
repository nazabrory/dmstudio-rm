# Zoom, Scale & Pan

There are a few ways to zoom into or out of a plot sheet, scale the contents of a projection or pan data in a projection to locate an area of interest.

When you change the position and scale of a projection, this will automatically update any plot items which are related to the data view including:

  * Grid lines and labels.

  * [A scale bar](<ScaleBar.md>).

  * Scale fields in **[title boxes](<TitleBlock.md>)**.

  * Surface profiles in profile boxes.

To scale the contents of a projection:

  * Right-click a projection (with [Page Layout](<PageLayoutMode.md>) mode active) and expand the **Projection >> Scale** menu. This scales the contents of the projection.

  * Activate the **(Plots) View** ribbon (regardless of the Page Layout mode) and use one of the options in the **Scale** group.

To zoom into or out of a plot sheet:

  * Right-click a projection (with **Page Layout** mode inactive) and expand the Zoom menu. This scales the view of the entire plot sheet, including all plot items.

  * Activate the **(Plots) View** ribbon (regardless of the Page Layout mode) and use one of the options in the **Zoom** group.

To pan the contents of a projection:

  * Right-click a projection (with [Page Layout](<PageLayoutMode.md>) mode active) and expand the **Projection >> Pan** menu. This pans the view of data within the target projection.

  * Activate the **(Plots) View** ribbon (regardless of the Page Layout mode) and use one of the options in the **Pan** group. 

**Note** : If the current plot sheet extends beyond your application frame, scroll bars appear automatically. Data is reduced to outlines whilst a scroll bar is used to improve performance.
