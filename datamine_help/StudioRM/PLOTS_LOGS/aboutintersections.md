# Creating Intersections

Sample intervals selected in table, section, 3D or log views can be labelled and saved to a user defined intersections table. These intersection intervals can be included as columns in section and hole log plots. The intersections table can also be exported for use in reports and spreadsheets, or as input to geostatistics and geological modelling programs. You can alternatively import the intersections from an external data source.
