# Printing Plots, Logs, Tables and Reports

The page settings are handled differently for each view type:

## Reports

  * Use the Projectbutton to select the print device, and select Page Setup to change the page size and orientation.

You can also copy and paste text from the report view to text editors via the Windows clipboard.

## Tables

  * You can change the table printing options including page size, orientation, margins, titles, grid lines, page order and alignment individually for each table view. For more information about changing the page settings for tables see:

[Printing tables](<PrintTable.md>)

## Sections and Logs

  * Page size, orientation and margins can be set individually for each section and log view and should be determined during the process of designing each plot. The page settings can be changed prior to printing, however this may necessitate adjustments to the plot scale, data area limits and plot items inserted into the plot page. For more information about changing the page settings for section and log plots see:

[Changing page size, orientation and margins](<PageSetup.md>)

## Printer settings

  * The current page settings for the printer need not necessarily match those of a particular view. Before printing a selected view, the printer settings should be reviewed and adjusted where necessary. For more information about previewing and changing printer settings see:

[Print setup and printing](<Printing.md>)
