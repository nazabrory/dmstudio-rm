# Column Wizard - Select Table

### To access this dialog:

  * In the Logs window, click a table to select it.

  * Right-click and select Properties.

  * In the Log View Properties dialog, select theColumnstab.

  * By theColumns in Viewwindow, clickAdd.

  * In theType of column(s) to add:screen of theColumn Wizard, select eitherData ColumnorSystem Field, and clickNext>>.

Select the table by which you wish to define the sample interval (for example, Depth From, Depth To). Only one table can be selected at a time.

![note.gif \(1017 bytes\)](../Images/note.gif) |  Note that the table selected does not have to contain the fields you wish to display: for example, by choosing the lithology table here, and a grade field from the assays table in the next step, a composited field is created. This means that the selected grade field is composited over the sample intervals defined by the selected table.
