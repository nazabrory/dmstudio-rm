# Column Wizard - Axis Columns

### To access this dialog:

  * In the Logs window, click on a table to highlight it.

  * Right-click, and select Properties.

  * In theLog View Propertiesdialog, select theColumnstab.

  * In theColumns in Viewwindow, clickAdd.

  * In theColumn Wizarddialog, selectAxis Column (vertical x, y, z or depth axis), and clickNext.

![](../Images/StepByStep2.gif)

  1. In the Column Wizarddialog, select one or more fields from the Select the axis columns to add: window.
  2. Specify the depth interval as either Automatic or Fixed:

  1.      * Automatic: drillhole depth intervals are added automatically along the length of the borehole.

     * Fixed: drillhole depth intervals can be specified using the spin buttons.

  3. In the Column Wizarddialog, clickEnd.
  4. In theLog View Propertiesdialog, clickOk.
