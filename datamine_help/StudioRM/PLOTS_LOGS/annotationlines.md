# Geological Features - Strings

### To access this group of commands:

  * Right-click a projection whilst Page Layout mode is inactive and select Insert. 

  * Choose the type of feature string you want to digitize.

These commands are used to annotate plot sheets with polylines (a strings object) which can be use to represent linear rock contacts, fault lines etc.. 

The program automatically defines objects with default legends when one of these object types is defined, and saves them in the project. It is not necessary to exort the data to save these features, although, if required, the data can be saved to file either on saving the project for the first time, or via the Loaded Data control bar, using the context menu for a specific object.

These features are drawn on the current section plane, using left-click or right-click (snapping) mouse actions.

The following linear geological features are available and can be used in plan or section:

  * Linear Feature- define a polyline, with an automatically generated Dip Dir and Dip attributes. Saved to the Linear Features object.

  * Fault- define a polyline, with an automatically generated Feature attribute; the Feature attribute is set to 'Fault'. Saved to the Linear Features object.

  * Contact - define a polyline, with an automatically generated Feature attribute; the Feature attribute is set to 'Contact'. Saved to the Linear Features object.
