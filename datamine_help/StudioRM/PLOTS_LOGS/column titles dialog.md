# Column Titles

To access this screen:

  * Display the [Log View Properties](<Log-View-Properties.md>) screen and select the **Column Titles** tab.

This tab is part of the [Log View Properties](<Log%20View%20Hole%20Properties.md>) dialog, and is used to format the column titles for log views. To access this command, you will need to be in the Logs window, with an active log sheet (to add a Log sheet to your project, activate theManageribbon and selectInsert | Sheet | Log.

Field Details:

Table Names Row: select the Include check box to ensure each row is indexed with a title cell. If you elect to include a Table Names cell, you can click Format... to access the [Cell Format](<cell%20format%20dialog.md>) dialog. This dialog is used to configure the formatting of text within a cell.

Column Names Row: select the Include check box to ensure each row is indexed with a title cell. If you elect to include a Column Names cell, you can click Format... to access the [Cell Format](<cell%20format%20dialog.md>) dialog. This dialog is used to configure the formatting of text within a cell.
