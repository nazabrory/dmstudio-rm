# Configuring Logs for Printing

Choose |  To Do This  
---|---  
Hole Range |   
Current hole |  Print the currently displayed hole  
Hole Set |  Display the selected set of holes  
Edit Set |  Add, rename, delete and edit a set of holes  
Hole Extents |   
Entire hole |  Print the entire length of the hole  
Automatic |  Print the section of hole that is currently displayed in the log  
Current |  Print the section of hole that is currently displayed in the log (active when current hole is selected)  
Scale |   
Fixed |  Set the scale of the log  
Fit to |  Make the log fit to a selected number of pages
