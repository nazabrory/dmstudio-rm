# Data Properties

This dialog displays information about the selected data. The categories change according to the type of data selected such that information about cells and subcells are displayed when block model data is selected and information about points is displayed when points data is selected. 

Category |  Displays  
---|---  
Contents |   
|  Information about faces, lines, points, cells, subcells.  
Data Table Properties |   
No. Columns |  Number of columns in the data table.  
No. Records |  Number of records in the data table.  
General |   
3D object name |  Name given to the object.  
Description |  Usually the file name.   
Object type |  Type of data, i.e. table, wireframe etc.  
Source |   
Store in project file |  Whether data is archived or not - Yes or No.  
Limits |  The maximum and minimum coordinates in the X,Y and Z directions.
