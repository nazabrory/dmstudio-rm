# Default Font Dialog

Click **Change** button to edit the default font.

Choose |  To Do  
---|---  
Font |  Change the font, i.e. Times new roman or Arial etc.  
Font style |  Make the font bold or italic etc.  
Max\Min size |  Change the size of the text.
