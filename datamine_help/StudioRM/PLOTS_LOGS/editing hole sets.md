# Editing Hole Sets

Hole sets can be edited from the Hole Set tab of the Format Display dialog (opened by activating the Manage ribbon and selecting Format | Overlays.

This tab has a window which lists the hole sets and enables various selections to be made.

![](../Images/StepByStep2.gif)

  1. Click Edit Sets... and the Hole Sets Manager will open.

  2. Select the set you wish to edit and click Edit. The Hole Set Contents dialog will open.

  3. Select the drillholes (using <CTRL> or <SHIFT> with the cursor to select multiple items) you wish to exclude and choose Remove.  
  
Holes can also be included by selecting and choosing Add.
