# Profile Box - Properties

Choose |  To Do This  
---|---  
Border |   
Display Border |  Show and hide the border.  
(Color) |  Change the border line color.  
Fill |   
Opaque |  Make the profile box opaque or transparent.  
Size |   
Height |  Set the box width in mm.  
Width |  Set the box height in mm.  
Font |  Change the font style, color and size of text labels.
