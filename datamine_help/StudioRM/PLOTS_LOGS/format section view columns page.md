# Format Columns

## Toggles

Toggle |  Function  
---|---  
Enable static drillholes and Downhole Columns |  Toggle the display of both the static drillholes and any downhole columns.  
Display Downhole Columns |  If "Enable Drillhole Traces and Downhole Columns" is "on", toggle the display of downhole columns.  
  
The Columns in View box lists the fields currently displayed with the hole traces in the selected section view. Select a field name to display the format settings for:

[Style Templates](<Format_Column_Style_Dialog.md>) |  [Graph/Color](<Format_Column_Graph_Dialog.md>) |  [Text](<Format_Column_Text_Dialog.md>)  
---|---|---  
[Alignment](<Format_Column_Alignment_Dialog.md>) |  [Border](<Format_Column_Borders_Dialog.md>) |  [Width/Margins](<format_column_margins_dialog.md>)  
|  [Filter](<Format_Column_Filter_Dialog.md>) |   
  
Choose |  To Do This  
---|---  
Columns in View |  Select a field name to remove, copy, change position or format.  
Up |  Move the column selected in the Columns in View box to the left.  
Down |  Move the column selected in the Columns in View box to the right.  
Delete |  Remove the column selected in the Columns in View box.  
Add |  Add a column.  
Copy |  Add a copy of the column selected in the Columns in View box.  
Reset |  Restore default columns.  
Font |  Change the font style, size and color for all text style columns.
