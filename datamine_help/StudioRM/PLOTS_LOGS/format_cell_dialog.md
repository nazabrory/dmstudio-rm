# Cell Format Dialog

Use for formatting the cells in a title box.

Choose |  To Do This  
---|---  
Include |   
Border |  Show and hide the cell border  
Font |   
Use Default font |  Use the displayed default font style  
Modify |  Choose this button to display the Font dialog.  
Horizontal |   
Left |  Left justify cell contents  
Right |  Right justify cell contents  
Center |  Center cell contents  
Wrap |  Wrap text inside cell.  
Vertical |   
Top |  Place cell contents at top  
Middle |  Place cell contents in the middle  
Bottom |  Place the cell contents at the base  
Label |  Show and hide the cell field name
