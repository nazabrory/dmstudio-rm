# Hole / All Hole Properties

This dialog is used to set global properties for loaded hole data.

Field Details:

Default set: whether the selected set or hole is part of the default set

Name: name of the set of holes.

No. Holes: number of holes in the set.

From To XYZ: minimum and maximum values for X Y and Z coordinates

Mid Point XYZ: section center coordinates

Span XYZ: cifference in coordinates for the X Y and Z minimum and maximum
