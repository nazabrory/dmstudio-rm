# Insert Outline

When adding an outline plot feature (using theManageribbon'sInsert | Outlinecommand), click Apply displays this dialog, which you use to define the type of outline data you are adding. After an outline type has been defined, you will need to specify how this new data is to be held within the current project. This is done using the [Create Line](<createlinedialog.md>) dialog, displayed when Next is selected.

Field Details:

Features to create: select an outline type from the displayed list and click Next to proceed. The available options are:

  * Ordinary polygon

  * Lithology boundary

  * Stratum

  * Ore Boundary

  * Vein

  * Reef

  * Other geological outline
