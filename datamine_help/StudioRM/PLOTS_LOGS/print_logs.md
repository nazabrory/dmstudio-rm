# Print Logs

Hole extents, scales and print ranges are all editable and selectable from the Properties window.

Choose |  To Do This  
---|---  
Initial Extents |   
Entire hole |  Print entire hole  
Automatic |  Use default settings  
Scale |   
Fixed Scale 1:n |  Specify scale of downhole columns  
Print Range |   
Current hole |  Print current hole only  
All holes |  Print all holes  
Use table |  Print holes in table selected from Table drop-down Select whether or not to Use table for extents
