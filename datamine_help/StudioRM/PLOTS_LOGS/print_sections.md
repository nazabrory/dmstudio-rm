# Print Sections

Sections can be printed using the Project button's Print command.

The Print dialog can be used to define which of the current sections are to be printed (assuming a one-page-per-section rule). You can also select Properties to access your printer-specific settings (refer to your hardware supplier's documentation for more information on how to set printer properties).
