# Profile Plot Properties

Choose |  To Do This  
---|---  
Label |   
Minimum Value |  Display the minimum value of the profile.  
Maximum Value |  Display the maximum value of the profile.  
Automatic Label |  Display a profile label generated by the program.  
Text |  Display a custom profile label. Enter label in Text box.  
Trace |   
Color |  Change the profile line color.  
Line Style |  Change the profile line style.  
Limits |   
Automatic Limits |  Auto scale the profile within the box limits.  
Minimum Value |  Set the profile value corresponding to the top of the box.  
Maximum Value |  Set the profile value corresponding to the bottom of the box.  
Display horizontal axis at |  Display a horizontal axis at the specified value.
