# Report Type

This dialog is displayed when a new Report sheet is added to the current project, using Insert | Sheet | Report from the Plots, Logs, Tables or Reports windows.

There are 3 report options available:

  * Table Validation: drillhole data files are automatically validated when they are first imported. Validation information will be reported in the Validation column of the report for each imported table.

  * Hole Summary: use this option to create a Hole Summary Report indicating the types of tables available and the status of the drillholes. This report summarizes the desurveying of each hole, its coordinates and horizontal and vertical thickness.

  * Desurvey Report: generate a report showing, on a hole-by-hole basis, the associated desurvey status, method, tolerance, desurveyed length and other desurvey-specific items.
