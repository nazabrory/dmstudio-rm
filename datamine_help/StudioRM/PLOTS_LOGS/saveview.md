# Saving the View to an Image

3D window views can be saved to an image using theDataribbon'sExport | Imageoption

Logs, Plots, Reports and Table views can also be saved to a metafile using theManageribbon'sSave | Vector | MetafileorEnhanced Metafileoptions

## You can also...

Copy the image in the 3D window to the Windows clipboard using theDataribbon'sExport | Image | Copy Imageoption
