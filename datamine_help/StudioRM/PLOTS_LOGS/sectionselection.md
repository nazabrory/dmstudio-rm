# View Settings Select Section

This tab is only visible when a Section Definition file is loaded and being used. The section definition file holds a number of pre-set section views which may not all be parallel or have the same clipping settings fore and aft.

Select the view you want to be displayed. With thePlot Viewribbon'sSection | Table - Usetoggle enabled, the section navigation arrows will take you between the sections defined in the file.
