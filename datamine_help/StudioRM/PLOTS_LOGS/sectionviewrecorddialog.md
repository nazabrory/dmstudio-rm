# Set Exaggeration

Sets the extent by which the data will be stretched in one or more axis directions. A value of 1.0 will retain the selected scale, whereas values greater or less than one will stretch or contract the data proportionately on the chosen axis.

Choose |  To Do this  
---|---  
General |   
X |  Set exaggeration factor for X axis  
Y |  Set exaggeration factor for Y axis  
Z |  Set exaggeration factor for Z axis
