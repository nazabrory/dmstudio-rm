# Data Conversions Dialog

Applies conversions to table data according the definition set in this dialog.

Choose |  To Do This  
---|---  
Column |   
Table |  Select a table.  
Column |  Select the column field name to which the conversion is to be applied.  
Apply to all columns |  Apply conversions to all columns in selected table.  
Value |   
Data Value |  Type the data value to be converted e.g. -9.  
Converted To |  Type the value the Data Value is to be converted to, or click the down arrow button to select from the list of special values e.g. absent, trace, dl, +, -.  
Add |  Add the data conversion specified in the Data Value and Converted To boxes to the selected field / table.  
Current Conversions |  Displays the data conversions currently set for the selected field / table.  
Delete |  Select a data conversion in the Current Conversions box and choose the Delete button to remove.
