# Text Annotation

Type the required annotation text into the field supplied. Click OK to add this text to the current sheet.

![](../Images/StepByStep2.gif)

To insert a text annotation:

(in the Plots window)

  1. Select a position (left-click) on your plot where text is to be inserted.
  2. Enter the required text.
  3. Select OK.

To move a text annotation

  1. Select the Text Annotation object on your plot view.
  2. Right-click in the same plot view.
  3. Select Edit Points.
  4. Select the Text Annotation object to be moved.
  5. Select a new position.
