# Plot Item Frame Properties

To access this screen:

  * Display the [Log View Properties](<Log-View-Properties.md>) screen and select the **Frame** tab.

Field Details:

Size: set the Height and/or Width of the frame. The current size of the frame is reported in these fields, but can be amended as required.

![](../Images/Warning.gif) | Note that by manually setting these fields, it is possible to offset the frame beyond the limits of the current document, which will cause data to be clipped when printed.  
---|---  
  
Border: if this check box is selected (the default setting) you can specify the line width using the Line Width field.

Color: select a frame color from the color picker displayed when this drop-down list is accessed.

Font...: displays a font picker dialog, allowing you to format the text associated with all cells within the current frame. Note that this will override the current cell format settings for individual cells.

Opaque: make the contents of the current frame opaque (i.e. zero transparency)

Draw Grid: selecting this option (the default setting) forces the individual cell borders to be drawn. If cleared, cell contents will be visible without a surrounding framework.

Cell Spacing: determines the space (in pixels) between cell borders.
