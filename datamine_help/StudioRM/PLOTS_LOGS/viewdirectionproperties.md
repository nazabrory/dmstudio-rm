# View Direction

To access this screen:

  * With **[Page Layout](<PageLayoutMode.md>)** mode active, right-click a plot **[projection](<alignviewwithsection.md>)** and select (3D) **Projection >> Direction >> Custom**.

Define the current view direction for a projection by setting a custom **Dip** and **Azimuth** value. Click **OK** to update the projection properties.
