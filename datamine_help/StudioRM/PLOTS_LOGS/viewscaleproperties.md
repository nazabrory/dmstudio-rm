# View Scale Properties

Accessed using Log | Scale | Custom..., this dialog is used to set the scale of a log view. 

Two tabs are available, the first - Scale \- provides an interface for setting the scale using a drop-down list, and the second - Properties \- provides summary information about the scale settings.
