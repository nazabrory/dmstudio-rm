# DMFD Process

To access this process:

  * View the **[Find Command](<../COMMON/findcommand.md>)** screen, select **DMFD** and click **Run**.
  * Enter "DMFD" into the [Command Line](<../COMMON/Command_Toolbar.md>) and press <ENTER>.

See this process in the [Command Table](<../command_help/COMMAND%20TABLE_D.md#DMFD>).

## Process Overview

Output a block model in a format suitable for input to Whittle FOUR-D.

## Input Files

Name |  Description |  I/O Status |  Required |  Type  
---|---|---|---|---  
IN |  Input model file. |  Input |  Yes |  Block Model
