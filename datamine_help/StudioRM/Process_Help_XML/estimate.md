# ESTIMATE Process

To access this process:

  * **Estimate** ribbon **> > Interpolate >> Standard**.

  * Enter "ESTIMATE" into the [Command Line](<../COMMON/Command_Toolbar.md>) and press <ENTER>.
  * ESTIMATE is the process that powers the ESTIMATE function.

See this process in the [[Command Table](<../command_help/COMMAND%20TABLE_E.md#ESTIMATE>)](<../command_help/COMMAND%20TABLE_E.md#ESTIMATE>).

## Process Overview

The process provides a comprehensive choice of methods and controls for estimating grades.  See [ESTIMATE Wizard](<../STUDIO_RM/EstimateDialog.md>).
