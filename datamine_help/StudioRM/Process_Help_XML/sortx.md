# SORTX Process

To access this process:

  * **Data** ribbon **> > Data Tools >> Tables >> Sort**.
  * Enter "SORTX" into the [Command Line](<../COMMON/Command_Toolbar.md>) and press <ENTER>.
  * View the **[Find Command](<../COMMON/findcommand.md>)** screen, select **SORTX** and click **Run**.

See this process in the [Command Table](<../command_help/COMMAND%20TABLE_S.md#SORTX>).

## Process Overview

Note: **SORTX** and [MGSORT](<mgsort.md>) have identical functionality. **SORTX** is retained for legacy macro support. **MGSORT** offers a performance improvement over **SORTX**.

See [MGSORT Process](<mgsort.md>).
