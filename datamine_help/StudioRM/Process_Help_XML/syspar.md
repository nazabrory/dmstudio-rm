# SYSPAR Process  
  
To access this process:

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, select **SYSPAR** and click **Run**.
  * Enter "SYSPAR" into the [Command Line](<../COMMON/Command_Toolbar.md>) and press <ENTER>.

See this process in the [Command Table](<../command_help/COMMAND%20TABLE_S.md#SYSPAR>).

## Process Overview

Displays application system parameters. No parameters are required. Information is output to the **Command** window.
