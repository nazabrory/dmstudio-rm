# UNLINK Process

Note: This command is maintained for macro backward compatibility only and is intended to only be run from macros.

To access this process:

  * Enter "UNLINK" into the [Command Line](<../COMMON/Command_Toolbar.md>) and press <ENTER>.
  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **UNLINK** and click **Run**.

See this process in the [Command Table](<../command_help/COMMAND%20TABLE_U.md#UNLINK>).

## Process Overview

Unlink a Datamine binary file from the current project.

The file name is attached to symbolic file &**IN**. The name of the external binary file is requested interactively.

## Input Files

Name |  Description |  I/O Status |  Required |  Type  
---|---|---|---|---  
IN |  File to be unlinked. |  Input |  Yes |  Undefined
