# ESTIMATE Wizard

To access this screen:

  * **Estimate** ribbon **> > Interpolate >> Standard**.

Estimate is an interactive version of the [ESTIMA](<../Process_Help_XML/estima.md>) process, with additional functions provided by **[INDEST](<../Process_Help_XML/indest.md>)**.

A series of tabs follow the process of grade estimation from left to right. Select a link below for more information:

  * **Files** define input and output files associated with grade estimation processes. 

See [ESTIMATE: Files](<Estimate_Files.md>).

  * **Unfolding** edit the fields and parameters associated with unfolding data. 

See [ESTIMATE: Unfolding](<Estimate_Unfolding.md>).

  * **Search Volumes** define search volume parameters. You can also import and export a Search Volume Parameters file. 

See [ESTIMATE: Search Volumes](<Estimate_Search.md>).

  * **Variogram Models** set up variogram models. Import or export a Variogram Model Parameter file. 

See [ESTIMATE: Variogram Models](<Estimate_Variogram.md>).

  * **Estimation Types** create new estimations, edit existing estimations, or import or export an Estimation Parameter file. 

See [ESTIMATE: Estimation](<Estimate_Estimation.md>).

  * **Controls** define zonal control parameters. 

See [ESTIMATE: Controls](<estimate_controls.md>)

  * **Preview** view a summary of the defined files, fields and parameters. 

See [ESTIMATE: Preview](<Estimate_Preview.md>).
