# Model Parameters

To access this screen:

  * Display the **[Fit Models](<Multivariate_Fit_Models.md>)** screen and select the **Model Parameters** tab.

Define manual model fitting parameters for fitting a model to the currently selected variogram.

The Model Parameters tab is split into two areas. Expand each area by clicking its title bar:

  * Variogram Model Set PropertiesDefine variogram models within the univariate or multivariate estimation case. See [Variogram Model Set Properties](<Multivariate_VariogramModelSetProperties.md>).

  * ParametersDisplays information about the variogram set for the current estimation scenario. See [Model Parameters](<Multivariate_VariogramParameters.md>).

**Note** : Where multivariate variograms are selected, if the nugget and/or sill matrices are not positive definite, a warning is issued indicating the variogram is invalid and that sill and nugget matrices should be reviewed.
