# Define Search Volumes

To access this screen:

  * **[Advanced Estimation](<Multivariate_Introduction.md>)** wizard >> **Define Search Vol.**.

Create a set of search parameters for each estimation that has an associated variogram model.

The dimensions of the search volume are set equal to the maximum range in each directions and the search ellipsoid is assigned the model variogram rotation angles and axes. 

The same default set of sample numbers and sector parameters is assigned to all search volumes. You can edit the default values for each search volume.

If you have [imported data from Datamine Supervisor](<Multivariate_Scenario_Setup.md>), search volumes derived from the imported project appear here automatically.

There are three parts to this panel to allow you to choose and edit the required search parameters for each estimation. Select a link for more information:

  * Estimation Area: Each Estimation, as chosen in the [Define Estimations](<Multivariate_Define_Estimations.md>) panel, is shown in the area on the left of this panel. [Define Search Volumes: Select an Estimation](<Multivariate_Select_SearchVol_%20Estimation.md>).

  * Available Search Volumes: The initial set of search volumes will be generated from the variogram models associated with the **Estimations** in the left area of the panel. See [Available Search Volumes](<Multivariate_Select_SearchVol_%20Available.md>).

  * **Search Volume Parameters** : The parameters are divided into three sub-panels **Shape** , **Sample Number Limits** , **Sectors** and **Outliers**. See [Search Volume Parameters](<Multivariate_Select_SearchVol_Params.md>).
