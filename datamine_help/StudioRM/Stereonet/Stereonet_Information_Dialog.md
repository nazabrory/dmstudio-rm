# Stereonet - Information Dialog

### To access this dialog:

  * In the [Stereonet](<Stereonet_Dialog.md>) dialog, select the Information tab.

The Stereonet - Information dialog is used to view a summary of current settings, loaded and selected data statistics.

  
Field Details:

Appearance:

Hemisphere: hemisphere type i.e. upper or lower.

Projection: projection type i.e. [equal area](<projection_schmidt%20net.md>) or [equal angle](<Projection_Wulff%20Net.md>).

Loaded Data Statistics:

Total Number of Poles: total number of poles listed in the structure points object or data file.

Average Dip: average dip of all values.

Average Dip Direction: average of all dip direction values.

Set 1 Statistics: summary statistics for the named set.

Number of Poles: total number of poles within the defined set window.

Average Dip: average of all dip values within the window.

Average Dip Direction: average of all dip direction values within the window.

![note.gif \(1017 bytes\)](../Images/note.gif) |  The information listed in this dialog will change depending on whether or not sets have been defined. it will as a minimum contain the Appearance and Loaded Data Statistics information categories.
