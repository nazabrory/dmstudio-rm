# Stereonet - Settings Dialog

### To access this dialog:

  * In the [Stereonet](<Stereonet_Dialog.md>) dialog, select the Settings tab.

The Stereonet - Settings dialog is used to define distribution type, counting circle size and default color settings.

  
Field Details:

Distribution: select this option to display all the listed cones (default).

Counting Circle Size:

Size (% hemisphere area): select a value from the drop-down.

Default Colors: these color settings apply to the defaults shown in the respective sets, planes and cones tabs.

Sets: select the required sets default color from the drop-down (default blue').

Fixed Planes: select the required fixed planes default color from the drop-down (default 'orange').

Cones: select the required cones default color from the drop-down (default 'red').

![note.gif \(1017 bytes\)](../Images/note.gif) |  The Stereonet dialog is modal. This means that it can be left open while other commands, e.g. in the Design or VR windows, are run. This allows it to be used for the interactive analysis of structural data across various windows and dialogs.
