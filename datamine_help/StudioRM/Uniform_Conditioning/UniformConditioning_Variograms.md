# Uniform Conditioning: Variograms

To access this screen:

  1. Display the **[Uniform Conditioning](<UniformConditioning_Introduction.md>)** wizard.

  2. Select the **Variograms** tab.

The Variograms screen, part of the [Uniform Conditioning](<UniformConditioning_Introduction.md>) wizard provides access to Studio's existing variography tools to create a weighted variogram model file based on the contents of the declustered samples information generated in the [previous step](<UniformConditioning_Decluster.md>) of the wizard. This stage assumes that a variogram model file has already been created (for example, created by **VGRAM** or the powerful suite of [Variogram modelling functions](<../PLOTS_LOGS/VARMOD_Introduction.md>) on offer).

Select a Variogram model file using the browse button and a Variogram ID to model the weighted multivariate variograms which will feed into the following processes of Uniform Conditioning.
