# Create 3D Objects

To access this screen:

  * In the **Sheets** control bar, right click the **3D** folder and select Create from Loaded Data.

  * In the **Sheets** control bar's **3D** folder , right-click a data type folder and select Create from Loaded Data.

Choose one or more loaded data objects from which to create another data overlay in the 3D window. 

Click **OK** to create an overlay with default formatting.

**Tip** : to create a copy of an existing overlay, right-click it and choose copy. This can be a useful way to format 3D objects from the same visual starting point.
