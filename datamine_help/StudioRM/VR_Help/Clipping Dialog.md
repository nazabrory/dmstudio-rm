# Clipping Planes

To access this tool: 

  * In an [external 3D window](<../COMMON/External_3D_Windows.md>), select View >> Clip.

The Clipping Planes screen lets you set the front and back clipping planes for the current external 3D view.

See [External 3D Views](<../COMMON/External_3D_Windows.md>).

If your external window is split vertically, horizontally or both, you can independently control the clipping in each window segment by clicking inside the relevant window segment then running the View >> Clip command.

Clipping is performed on the active view only - every external 3D window can be clipped independently. 

To adjust front and back clipping planes in an external 3D view: 

  1. Display an external 3D view. 

  2. Display the **Clipping Planes** tool.

  3. Use the Front and Back sliders to set the clipping planes. 

  4. Optionally, select the Global Limits button to display the [Environmental Settings](<EnvironmentalSettings_Dialog.md>) screen to control clipping planes across all views.
