# Ellipsoids Properties

To access this screen:

  * Right-click an ellipsoid overlay in the **Sheets** or **Project Data** control bar and select Properties. 

  * Double-click an ellipsoid overlay in the **Sheets** or **Project Data** control bar.

  * Double-click an ellipsoid data object in any 3D window.

The Ellipsoids Properties screen has the following tabs:

  * General: set general visual formatting options for ellipsoid data type object overlays. See [Ellipsoids Properties: General](<Ellipsoids%20Properties%20Dialog.md>).
  * Labels: label your displayed ellipsoid data using a range of text and formatting options. See [Ellipsoids Properties: Labels](<Ellipsoids_PropDialog_Labels.md>).
  * Associated files: review and launch any external data files associated with the current ellipsoid object. See [Associated Files](<Associated%20Files%20Dialog.md>).
  * Info Mode List: control the attributes that are displayed for the ellipsoid data object when using Dynamic Information Mode. See [Info Mode List](<Traces%20Properties%20Dialog%20\(Info%20Mode%20List\).md>).
  * Templates: apply, edit, define and delete 3D window formatting templates here. See [3D Display Templates](<3D_Templates.md>).
