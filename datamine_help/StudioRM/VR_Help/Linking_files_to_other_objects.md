# Linking objects to other files 

When you link a file to a 3D object, you can open that file from the 3D window, by right-clicking any overlay of that object.

See [Associating Files with 3D Objects](<linking%20files%20list.md>).

To link a file to an object: 

  1. In either the **Sheets** or **Project Data** control bar, right-click any overlay of an object and select **Properties**.

Tip: You can also double-click any part of the displayed data in a 3D window.

  2. Select the **Associated Files** tab.

  3. Browse for a File, or type its full path.

  4. Add a **Description**. This is displayed when you right-click the associated object in the 3D window.

  5. If you want to open the associated file with a particular command line syntax (for example, you may want to disable macros in an Excel file, or open a document in read-only mode), enter the appropriate Parameters.

Note: These should be added in the same way as entering them in the Windows command window or Powershell.

  6. Click Test to check that the link works then press the down arrow to link the file to the object.

  7. Add as many files as you need. There's no limit.

  8. Click **OK**. 

To edit an associated file:

  1. Select the file to be edited in the table.

  2. Click the up arrow.

  3. Edit your **File** , **Description** or **Parameters**.

  4. Click Test to check that the link works then press the down arrow to link the file to the object.

  5. Click **OK**.

To delete an associated file:

  1. Select the file to be deleted in the table.

  2. Click the red cross.
