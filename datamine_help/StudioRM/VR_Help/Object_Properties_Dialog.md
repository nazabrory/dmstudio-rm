# VR Object Properties

To access this screen:

  * In the Sheets control bar, 3D folder, right-click the VR Objects sub-folder, select New.

  * In the Sheets control bar, 3D folder, VR Objects sub-folder, right-click an object type, select Properties.

  * Double click an existing **VR Object** item in a **Sheets** or **Project Data** control bar.

**Note** : before a VR object can be added, a VR Object Type must be created. See [[VR Object Types](<Object_Adding_an_object_type.md>)](<VR-Objects-and-Types.md>).

VR objects are ideal for simulating both static and dynamic scene elements. The object's shape and other simulation properties such as movement capabilities, light sourcing, sound and orientation are inherited from its [**type**](<VR-Objects-and-Types.md>).
