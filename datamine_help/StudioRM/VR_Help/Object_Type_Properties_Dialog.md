# VR Object Type Properties

To access this screen:

  * In the Sheets control bar, 3D folder, right-click the VR Object Types sub-folder, select New.

  * In the Sheets control bar, 3D folder, VR Object Types sub-folder, right-click an object, select Properties.

Create and define settings for a category of objects i.e. an object type.

See [VR Object Types](<Object_Adding_an_object_type.md>).

**Note** : before a VR object can be added to a **3D** view, a VR object type must be created.
