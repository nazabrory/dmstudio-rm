# Place a Group of VR Objects

A group of objects of the same object type can be placed in the world view by digitizing a string with a Vertex model defined. The Edge model can be left undefined showing no visible link between the objects (e.g. a cluster of trees) or a suitable edge model can selected (e.g. a fence defined by a "post" vertex model and a "tape" edge model).

![](../Images/VR_PlaceGroup.gif)

To display VR objects at each vertex position of a string:

  1. Double-click the string object in any 3D view to display the **String Properties** screen.

  2. Activate the **Symbols** tab.

  3. Set the symbol **Type** to **3D**.

  4. In the Style group, select **Imported model**.

  5. Browse for a suitable DirectX (.x) model to display at each vertex position of the target string.

**Note** : several DirectX objects are installed with your application. You can find them in your installation folder's **VR** sub-folder.

  6. Define other symbol properties, such as the **Position** and **Rotation**. See [Strings Properties: Symbols](<String_Properties_Dialog_VertexVisualTab.md>).

  7. To hide the original string and only show the 3D symbols:

     1. Select the **Lines** tab.

     2. Uncheck **Display lines**.

  8. Click **Apply**.
