# Stationary VR Object types

A stationary VR object type is just a type with a **Max. speed** of zero. Stationary objects are useful for static light sources or elements of 3D scenery (trees, cabin, survey station and so on).

See [VR Object Types](<Object_Adding_an_object_type.md>).

**Note** : VR object types must be added to the VR Object Type folder before VR objects can be placed in 3D scenes.
