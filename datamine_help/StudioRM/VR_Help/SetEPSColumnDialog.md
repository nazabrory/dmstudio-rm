# Change Column

To access this screen:

  * Right-click a 3D wireframe overlay in the Sheets or **Project Data** control bar and select Link to DTS.

Connect your sequenced 3D wireframe design data with your generated **Datamine Task Scheduler** (DTS) schedule and visualize 3D window data segments and how they relate to individual activities. Data selection can be performed in either Studio and DTS to force an update in the partner product.

There are a few things you need to put in place before you can start to make use of this facility:

  * An existing schedule must be generated from your current data set. This means you will have completed your mine design up to and including setting dependencies and sequencing.
  * The sequenced data (say, sequenced wireframes representing underground workings and stope development) must be loaded and displayed in a 3D window.
  * You have 'mapped' a numeric data column using the **Change Column** screen.
  * DTS must be running on the local system.

Select a numeric data columns that can be mapped (potentially) to an active schedule activity and click **OK**.
