# Project a String 

Some sources of strings objects (such as CAD drawings) define strings or polylines with coordinates only in the horizontal plane (X,Y). The vertical (Z) coordinate is constant or unspecified. In order to use this imported string data in the 3D window, it will first need to be projected onto an appropriate reference plane or surface, which could be a section or a wireframe. 

The method below outlines projecting a string to a wireframe surface. 

To project a string to a wireframe:

  1. Import and format the string.

  2. Import and format the wireframe.

  3. Hide other wireframes already imported.

  4. In the Sheets control bar, 3D folder, Strings sub-folder, right-click the string object, select Project to Wireframe/Section.

The [Project Strings/Points](<../COMMON/ProjectStringsPoints_Dialog.md>) screen displays.

  5. Define your projection settings.

  6. Click OK.

The target string data is projected to the wireframe surface in the projection direction specified.
