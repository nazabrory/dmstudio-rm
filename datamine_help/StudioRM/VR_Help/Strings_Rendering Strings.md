# Formatting Strings

Strings consists of a sequence of points (vertex) connected by lines (edge). String vertices and edges can be formatted independently.

You can:

  * Change the size or width of vertices and edges.

  * Color vertices and edges using different color legends.

  * Vary the color or texture of vertices and edges using trace attribute fields.

  * Define vertex and edge models.

  * Hide either vertices, edges or both.

  * Annotate strings with a string attribute field.

When a string is initially loaded digitized, formatting is applied according to the default 3D template for that data type. See [3D Display Templates](<3D_Templates.md>).
