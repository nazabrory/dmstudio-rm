# Trace Properties (Vertices)

To display this screen:

  * In the **Sheets** or **Project Data** control bar, right click a string trace and select **Properties**.

Adjust a string trace by inserting or deleting vertices into any position.

The current string length is displayed, and is adjusted as vertices are added or removed.

To insert a new vertex into a trace:

  1. Display the **Trace Properties** screen, **Vertices** tab.

  2. Select either the vertex before or after the position at which you will insert a vertex.

  3. Enter **X** , Y and Z coordinates of the new vertex, using **Edit** controls.

  4. To insert the new vertex above the selected vertex, click the upward facing arrow. To insert the vertex below the selection, choose the downward facing arrow.

  5. Click **OK** or **Apply**.

To delete a vertex from a trace:

There are two ways to do this:

  * Double-click an existing vertex record to remove it.

  * Select a vertex and click X.

In either case, click **OK** or **Apply** to update the trace.
