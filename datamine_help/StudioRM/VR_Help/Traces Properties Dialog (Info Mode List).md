# Info Mode List

To access this screen:

  * Display any 3D properties screen and select the Info Mode List tab.

Note: A Datamine [eLearning course](<https://datamine.learnupon.com/>) is available that covers functions described in this topic. Contact your local Datamine office for more details.

Define which, and in what order, data columns are to be displayed when [Information Mode](<vr_navigation_information.md>) is used. 

Information mode is used to interactively discover the values of data attributes and other custom data (as described here) in a 3D view. You can activate this mode in any 3D window using the **SHIFT** and **?** keys, or a ribbon equivalent button (which varies depending on your product).

By default, all object attributes and values display when the target overlay is clicked when Information Mode is active. This happens when the **Info Mode List** on this screen is empty. If it contains anything, only those attributes and values display. As such, you can constrain the information displayed by transferring attributes from the **Columns** list to the **Info Mode** List.

To constrain the list of attributes and values display in Information Mode:

  1. Load and display the data you wish to interrogate in Information Mode.

  2. Open the 3D properties screen for the target overlay. There are various ways to do this, such as double-clicking the overlay in any 3D view, for example.

  3. Select the **Info Mode List** tab.

  4. Select an attribute you wish to appear on screen from the **Columns** list.

  5. Use the down arrow to transfer it to the **Info Mode List**.

  6. Add whichever attributes you want to see and click **OK**.

  7. Click into a 3D view and press **SHIFT** and **?** on the keyboard. 

The cursor changes to a question mark.

  8. Optionally, reposition the items in the **Info Mode List** using the + and - buttons. The list order shown is honoured during Information Mode data display.

  9. Left click the modified overlay.

The list of attributes displayed (along with their values) mimics those of the **Info Mode List** earlier.
