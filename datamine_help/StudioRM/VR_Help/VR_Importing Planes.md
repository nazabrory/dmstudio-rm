# Import Plane Data

Planes are just points with a dip and dip direction. They can be useful to indicate the orientation of structures in a 3D view, for example, to highlight fracture planes for stereoscope analysis and domain failure review.

Planes can be imported as any other data. See [Importing and Exporting Data](<../COMMON/Concept_Importing%20and%20Exporting%20Data.md>).
