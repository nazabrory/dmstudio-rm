# Importing 3D Sections

As with all types of 3D Data (points, strings, wireframes etc.), there are two main ways to import or load section data. A file or loaded data object containing section parameters is called a "section definition file".

To import a 3D sections file:

To import a 3D section file into the 3D window, you will need to either;

  * Activate the **Sheets** control bar, then right-click the **Sections** folder and click **Import**. 

This displays the Data Source Drivers menu from which you can import a section definition file from a variety of sources, including Datamine formats.  

  * If your project already has a reference to a section definition table, right-click the Sections folder in the [Sheets](<../COMMON/Sheets%20Control%20Bar%20Overview.md>) control bar, **3D** folder, and select Import from Project.

Note: Each Datamine object can be represented any number of times in a 3D view. Each instance is known as an overlay. See [The View Hierarchy](<../COMMON/View%20Hierarchy.md>).

In this situation, the source object (the name of the physical file) is visible in the [Sections Properties](<Section%20Properties%20Dialog.md>).

  * You can also drag a section definition file from your PC aor Project Files control bar directly into the window to initiate the import process automatically.
