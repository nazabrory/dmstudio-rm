# View From Inside

In **View From Inside** mode the viewing position is fixed at, or offset from, the origin of the selected 3D Object or fixed viewpoint. As such, the actions of the navigation controls differ from other modes and are restricted to changing the view direction only.

The view movements available after View From Inside has been selected, are listed in the Fixed Viewpoint + View From Inside section of the table on the [navigation controls](<VR_Navigational_Controls.md>) page.

You can view a VR Object from inside it by doing any of the following:

  * **Sheets** control bar **> > VR Objects** and right click a VR object to select **Inside View**.

  * Activate the **3D View** ribbon and select a VR Object or fixed viewpoint from the View list.

  * When [Outside View](<VR_Navigation_Outside_View.md>) mode is active, choose the **View From Inside** option using the 3D folder's right-click option.

Note: This view mode is cancelled when **Floating View** mode is selected, or another viewpoint is selected from the View list, or when View From Inside is chosen for an object in the Sheets control bar.
