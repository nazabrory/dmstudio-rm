# View From Outside

In the View From Outside mode the viewing movement are made relative to the selected [VR Object](<sheets_vrobjects.md>). As such, the actions of the navigation controls differ from **Floating** mode and some actions, like panning, are not available.

You can view a VR Object from above and behind by doing any of the following:

  * **Sheets** control bar **> > VR Objects** and right click a VR object to select **Outside View**.

  * Activate the **3D View** ribbon and select a VR Object or fixed viewpoint from the View list.

  * When Inside View mode is active, choose the View From Outside option using the 3D folder's right-click option.

Note: This view mode is cancelled when **Floating View** mode is selected, or another viewpoint is selected from the View list, or when View From Inside is chosen for an object in the Sheets control bar.
