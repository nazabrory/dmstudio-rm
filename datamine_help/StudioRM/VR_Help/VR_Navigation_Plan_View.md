# Zoom Plan

**Note** : This command is available in all Studio products apart from Studio Mapper.

To access this command:

  * Activate the **3D View** ribbon and select View >> Plan.

This control is used to fit the view to the extents of the data in a plan view orientation.

When selected, the following occurs:

  * The _Floating_ viewpoint is selected,

  * The view position is moved upwards, vertically above the data,

  * The view extents are fitted to all data,

  * The view direction is rotated to look vertically down.

Note: The view [navigation controls](<VR_Navigational_Controls.md>) available after Zoom Plan has been clicked, are the same as those available when the Floating Viewpoint and Look At Mode are selected.
