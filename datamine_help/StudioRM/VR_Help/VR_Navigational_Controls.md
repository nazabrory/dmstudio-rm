# 3D Window Navigation  
  
The view position and orientation in the 3D window can be changed using standard views or by interactive navigation. 

Regularly used views can also be saved as [viewpoints](<VR_Viewpoints.md>).

The effect of the navigational controls on the view depends on the selected view mode selected at the time of operation. For more detailed information on the various view modes, click the link in the first column. For a general overview of view modes and view controls, click [here](<VR_Navigation_Modes.md>).

## Preset Views

A number of standard views can be used to view the full extents of the data in the active 3D window. You can access these view commands using the View ribbon (View commands group).

These commands include:

Plan ![](../Icon_Popups/icvr_zoomplan.gif)View the data in plan, zoomed to its full extents (quick key SHIFT + Z).

North ![](../Icon_Popups/icvr_zoomnorth.gif)View the data looking north, zoomed to its full extents (quick key SHIFT + Y).

East ![](../Icon_Popups/icvr_zoomeast.gif)View the data looking east, zoomed to its full extents (quick key SHIFT + Z).

South ![](../Icon_Popups/icvr_zoomsouth.gif)View the data looking south, zoomed to its full extents.

West ![](../Icon_Popups/icvr_zoomwest.gif)View the data looking west, zoomed to its full extents.
