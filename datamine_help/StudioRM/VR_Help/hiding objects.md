# Hiding Data

When you hide data, you are making one or more of its overlays invisible (an overlay is simply a representation of a 3D data object).

Hiding visible data from the screen will not unload the object memory or remove it from the project. 

There are many ways to hide visible data in 3D windows, including:

  * Using the [Sheets](<../COMMON/Sheets%20Control%20Bar%20Overview.md>) or **Project Data** control bar's context menu options. These allow you to hide individual overlays as well as hiding (and showing) all overlays of a particular data type.

  * Using one of the **Format** ribbon's **Show/Hide** or **Filter** options to hide data of a particular type.

  * Using data filtering to filter out displayed 3D data. See [Filtering Data](<../COMMON/Filtering_Data.md>).

  * Clipping data in a 3D scene. See [Clipping 3D Data](<Clipping-Data.md>).

  * Placing other visible data "in front" according to the 3D window's line of sight, although this is technically more "obscuring" than "hiding".

You can also [unload](<../COMMON/Unloading%20Data.md>) the data object to which the displayed overlays belong, but this is more "removing data" than "hiding data". Unloaded data stays unloaded until you choose to load it again.
