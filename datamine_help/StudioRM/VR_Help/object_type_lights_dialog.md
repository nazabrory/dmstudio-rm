# Light Properties Dialog

### To access this dialog:

  * In the [Object Type Properties](<Object_Type_Properties_Dialog.md>) dialog, Lights pane, double-click **< Add new light>**.

Configure lighting parameters for a **VR Object**. Lights can be attached to both [Mobile Simulation Objects](<Objects_Mobile%20objects.md>) and [Stationary VR Object types](<Objects_Stationary%20objects.md>).

See [Add a VR Object Light](<Objects_Object%20lights.md>).
