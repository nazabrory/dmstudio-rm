# abort-string-tracing ("ast")

See this command in the [**command table**.](<_COMMAND%20TABLE_A.md#abort-string-tracing>)

To access this command:

  * Edit ribbon >> Tools >> More >> Abort Trace

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter `abort-string-tracing`

  * Use the quick key combination "ast".

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate _abort-string-tracing_ and click **Run**.

## Command Overview

Delete the string currently being digitized and exits string tracing mode (only available in the legacy Design window).

This command is only effective in string tracing mode, activated by the **[string-trace](<string-trace.md>)** command. This command is effectively the same as clicking **Cancel** in that command.

Command Steps:

  1. Run the command whilst in string tracing mode (Cancel button is shown).

  2. The trace is deleted and the tracing operation halted.
