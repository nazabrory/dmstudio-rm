# adjust-to-gradient ("atg")

See this command in the [**command table**.](<_COMMAND%20TABLE_A.md#adjust-to-gradient>)

To access this command:

  * **Digitize** ribbon **> > Condition >> Condition >> Adjust to Gradient**

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter `adjust-to-gradient`

  * Use the quick key combination "atg".

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate _adjust-to-gradient_ and click **Run**.

## Command Overview

Adjusts a string to fit a gradient. This command modifies the elevations of points on a selected string. Segments after the selected point based on string direction (that is, the increasing PTN values) will match a specified gradient.

Command Steps:

  1. Run the command. If there is no current string, you are prompted to select one.

  2. Select a reference point on the selected string.

  3. Select the gradient you need.  
  
The string is vertically adjusted so the selected point is on the correct elevation, with all other elevations adjusted to the new gradient.
