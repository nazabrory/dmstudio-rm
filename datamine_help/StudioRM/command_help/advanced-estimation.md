# advanced-estimation

See this command in the [**command table**.](<_COMMAND%20TABLE_A.md#advanced-estimation>)

To access this command:

  * **Estimation** ribbon **> >** **Multivariate**

  *   *   * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter `advanced-estimation`

## Command Overview

Advanced Estimation provides a range of methods for estimating grades into a block model including both univariate and multivariate kriging methods. See **[Advanced Estimation Overview](<../STUDIO_RM/Multivariate_Introduction.md>)**.
