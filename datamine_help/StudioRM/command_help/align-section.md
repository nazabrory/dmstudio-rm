# align-section ("asc")

See this command in the [**command table**.](<_COMMAND%20TABLE_A.md#align-section>)

To access this command:

  * View ribbon **> >** Sections >> Align Section

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "align-section"

  * Use the quick key combination "asc".

## Command Overview

Modify the active section definition so that it is orthogonal to the current view, with the section passing through the current "Look at" point.

Command Steps:

  1. Activate a 3D window. If multiple windows are shown (due to horizontal or vertical splitting) highlight one by left-clicking it.
  2. Run [plane-by-one-point](<plane-by-one-point.md>) command and click anywhere within the 3D window
  3. Run **align-section**. The currently active section definition is modified to match the current view direction.
