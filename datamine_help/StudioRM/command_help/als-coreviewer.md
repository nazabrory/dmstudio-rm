# als-coreviewer

See this command in the [**command table**.](<_COMMAND%20TABLE_A.md#als-coreviewer>)

To access this command:

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "als-coreviewer"

  * Use the quick key combination "als"

## Command Overview

Opens an ALS CoreViewer portal to show detailed core information for the selected sample.

**Note** : This system is not directly supported or maintained by Datamine. If you do not currently have access to a CoreViewer portal, please contact ALS.
