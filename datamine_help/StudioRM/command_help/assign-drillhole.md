# assign-drillhole ("asd")

See this command in the [**command table**.](<_COMMAND%20TABLE_A.md#assign-drillhole>)

To access this command:

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "assign-lithology-assign".

  * Use the quick key combination "asd".
  * On the **[Find Command](<../COMMON/findcommand.md>)** screen, highlight **assign-drillhole** and click **Run**.

Command Overview

This command (and more commonly, the associated quick key combination) is used to apply the currently active attribute of the **[Assign Lithology](<../COMMON/Assign_Sample_Lithologies.md>)** screen to selected drillhole data.

Command steps: 

  1. Set **Default** data selection settings to _Drillholes_. See [Selecting 3D Data Interactively](<../COMMON/Selecting3DDataInteractively.md>).

  2. Load drillholes to be coded.

  3. Run the [assign-lithology](<assign-lithology.md>) command.

The **Assign Lithology** screen displays. See [Assign Lithology](<../COMMON/Assign_Sample_Lithologies.md>).

  4. Select a **Drillhole Table**.

  5. Select a Lithology Field.

  6. Select a Legend for display purposes.

  7. Enable Select Mode.

  8. Pick 3D drillhole data to be coded. 

Selected drillhole data is highlighted.

  9. Type "ala" to assign the active **Lithology Field** to the selected drillhole data.

Note: You can also click **Assign** on the **Assign Lithology** screen.
