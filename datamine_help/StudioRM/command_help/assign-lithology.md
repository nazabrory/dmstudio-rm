# assign-lithology

See this command in the [**command table**.](<_COMMAND%20TABLE_A.md#assign-lithology>)

To access this command:

  * **Implicit** ribbon **> > Domain >> Assign**.

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "assign-lithology" and press ENTER.

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **assign-lithology** and click **Run**.

## Command Overview

Select drillhole samples based on a lithology key field and assign lithological values (grouped or ungrouped). This command is commonly used in conjunction with [group-lithology](<group-lithology.md>).

The input to this command is a desurveyed file containing a key field to be edited. The output is an updated file.

This command uses the [Assign Lithology](<../COMMON/Assign_Sample_Lithologies.md>) screen to provide interactive functionality.
