# link-sel-strings-plane-attrib

See this command in the [**command table**.](<COMMAND%20TABLE_L.md#link-sel-strings-plane-attrib>)

To access this command:

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "link-sel-strings-plane-attrib"

## Command Overview

Links selected strings to form a solid wireframe, after specifying the attribute which is used to group the selected strings. 

A number of strings are grouped together. If fewer than two strings are selected when the command is run, then you are prompted to select more.

Command steps

  1. In any3D window, select multiple strings.

  2. Run the command.

  3. Specify the attribute which used to group the selected strings using Attribute for grouping. 

  4. Specify the **Number of strings** to be grouped together (must be more than 1).

  5. Click OK.

Strings are linked to form a wireframe.
