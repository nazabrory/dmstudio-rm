# blast-layout-interactive ("bli")

See this command in the [command table](<commandtable_B.md#blast-layout-interactive>).

To access this command:

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "blast-layout-interactive".

  * Use the quick key combination "bli".

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **blast-layout-interactive** and click **Run**.

## Command Overview

Display the Blast hole layout screen in the **Customization** window.

The process for blast layout design is:

  1. Create a **Hole pattern** and define other blast parameters such as fragmentation settings, elevation and SG.

  2. Create a **Blast outline** and, within it, **Blast rows** (either by manual placement or offsetting rows).

  3. Place holes and, optionally view circles of influence.

  4. Run Proximity checks.

  5. Generate a report.

For more information click **?** on the **Blast hole layout** screen to show help.
