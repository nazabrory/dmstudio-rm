# borehole-warning-report ("bhw")

See this command in the [**command table**.](<commandtable_B.md#borehole-warning-report>)

To access this command:

  * **Sample Analysis** ribbon **> > Warning**.
  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "borehole-warning-report"

  * Use the quick key combination "bhw".

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **borehole-warning-report** and click **Run**.

## Command Overview

Check proximity and intersections of drillholes with wireframe surfaces, using the [Drillhole Break Through and Proximity Warnings](<../COMMON/Borehole_Warning_Report.md>) screen.
