# box-all-points-inside-switch ("sib")

See this command in the [**command table**.](<commandtable_B.md#box-all-points-inside-switch>)

To access this command:

  * Open the **[Project Settings](<../COMMON/ProjectSettings.md>)** screen and select **Points and Strings >> Select string when all points in box**.

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "box-all-points-inside-switch"

  * Use the quick key combination "sib".

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **box-all-points-inside-switch** and click **Run**.

## Command Overview

Control whether string is selected if all points lie within selection box. If this switch is enabled, a string is selected if all points lie within the selection box. This only applies to the box selection method, not other selection methods. See [Selecting 3D Data Interactively](<../COMMON/Selecting3DDataInteractively.md>).
