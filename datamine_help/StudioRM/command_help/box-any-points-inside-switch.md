# box-any-points-inside-switch ("scb")

See this command in the [**command table**.](<commandtable_B.md#box-any-points-inside-switch>)

To access this command:

  * Open the **[Project Settings](<../COMMON/ProjectSettings.md>)** screen and select **Points and Strings >> Select string when crosses box**.

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "box-any-points-inside-switch"

  * Use the quick key combination "scb".

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **box-any-points-inside-switch** and click **Run**.

## Command Overview

Control whether string is selected if any point lies within selection box. If this switch is active, a string is selected if any part of it falls within the selection zone.
