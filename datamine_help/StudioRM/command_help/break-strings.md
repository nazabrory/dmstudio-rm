# break-strings ("bki")

See this command in the [**command table**.](<commandtable_B.md#break-strings>)

To access this command:

  * **Digitize** ribbon **> > Tools >> Break >> At Intersections**.

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "break-strings"

  * Use the quick key combination "bki".

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **break-strings** and click **Run**.

## Command Overview

Break a string where it is crossed by any other string.

Command steps:

  1. Orient the view so that appropriate string intersections are visible.

  2. Run the command.

  3. Select the string to be broken at points of intersection with other strings along the line of sight. 

  4. Complete the command by double-clicking or tapping anywhere in a **3D** window.
