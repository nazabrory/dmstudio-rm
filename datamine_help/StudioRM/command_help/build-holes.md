# build-holes

See this command in the [**command table**.](<commandtable_B.md#build-holes>)

To access this command:

  * **Sample Analysis** ribbon **> > Desurvey >> Dynamic >> Build Dynamic**.

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "build-holes".

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **build-holes** and click **Run**.

## Command Overview

Build a set of desurveyed drillhole traces from the loaded raw drillhole data tables, using the [Build Holes](<../COMMON/build%20holes%20dialog.md>) screen. 

A collars and survey data table must be loaded to use this command, although additional tables can be specified, such as lithology, survey and zone data.

Command steps:

  1. Load collars and survey data tables, plus any supporting tables (zone, lithology etc.) if required.

  2. Run the command.

  3. Define collars, survey and other tables, plus other options, using the [Define Hole Tables](<../COMMON/Define%20Holes%20Dialog.md>) screen and generate dynamic drillhole data.
