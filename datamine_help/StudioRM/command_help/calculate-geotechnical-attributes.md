# calculate-geotechnical-attributes ("cga")

See this command in the [**command table**.](<_COMMAND%20TABLE_C.md#calculate-geotechnical-attributes>)

To access this command:

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "calculate-geotechnical-attributes"

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **calculate-geotechnical-attributes** and click **Run**.

## Command Overview

Calculate the average joint spacing, fracture frequency and RQD for a set of selected planes. Optionally output results to a histogram and/or a points object.

Command Steps:

  1. Optionally select the required planes in the current data window (first applying filters if required).

  2. Run the command.

The [Calculate Geotechnical Attributes](<../Studio_3_General/Calculate_Geotechnical_Attributes_Dialog.md>) screen displays.

  3. In the **Selected Objects** group, choose one or more plane objects.

  4. In the **Group** section, select a common key field for differentiating groups of planes, or select _None_ for an average calculation across all planes.

  5. In the **Output** group, select the required outputs and data columns.

  6. Click Run.

  7. If displayed, define the necessary parameters on the [Histogram Properties](<../PLOTS_LOGS/Chart_Histogram_ChartProperties.md>) screen.
