# cancel-command ("can")

See this command in the [**command table**.](<_COMMAND%20TABLE_C.md#cancel-command>)

To access this command:

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "calculate-geotechnical-attributes"

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **calculate-geotechnical-attributes** and click **Run**.

## Command Overview

Cancels the current command. This is typically used in a script to ensure a modal command (such as insert-points, new-string or similar) is cancelled in order for another command to be run.

In practice, it has the same effect as clicking the on-screen **Done** button when running interactive commands (or double-clicking for supported commands). Pressing <ESC> emulates the same effect.
