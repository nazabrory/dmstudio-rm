# check-for-crossovers ("cxx")

See this command in the [**command table**.](<_COMMAND%20TABLE_C.md#check-for-crossovers>)

To access this command:

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "check-for-crossovers".

  * Use the quick key combination "cxx".

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **check-for-crossovers** and click **Run**.

## Command Overview

Check selected wireframe data for crossover areas. Wireframe data must be selected before running this command.

Command steps:

  1. Select the wireframe data you wish to check for crossover data. See [Selecting Wireframe Data](<../COMMON/Wireframe_Selection_Concept.md>).

  2. Run the command.

The command checks for faces that intersect, but are not adjoining. When found, a new object is created containing strings made up from the edges formed by the intersections. This helps you see where crossovers have occurred.
