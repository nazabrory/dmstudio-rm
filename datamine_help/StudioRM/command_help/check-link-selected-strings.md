# check-link-selected-strings ("clss")

See this command in the [**command table**.](<_COMMAND%20TABLE_C.md#check-link-selected-strings>)

To access this command:

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "check-link-selected-strings".

  * Use the quick key combination "clss".

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **check-link-selected-strings** and click **Run**.

## Command Overview

Check link crossovers involving 2 or more string traces. 2 or more strings must be selected before running this command.

## How to Use

  1. Select the string data that you wish to check.

  2. Run the command.

Link crossover checking is performed on the selected strings.
