# clip-strings-to-perimeters ("cltp")

See this command in the [**command table**.](<_COMMAND%20TABLE_C.md#clip-strings-to-perimeters>)

To access this command:

  * **Digitize** ribbon **> > Tools >> Clip Strings**.

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "clip-strings-to-perimeters"

  * Use the quick key combination "cltp".

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **clip-strings-to-perimeters** and click **Run**.

## Command Overview

Clip one or more (open or closed) strings to one or more perimeters (closed strings) using the [Clip Strings to Perimeters](<../COMMON/ClipStringsWithPerimetersDialog.md>) screen.

Data can be clipped so that the data that is within the clipping scope is retained, or the data outside is retained.

Command Steps:

  1. Ensure at least two strings are visible in a 3D window with at least one of them closed (and overlapping other string data).

  2. Fill out the various fields of the [Clip Strings to Perimeters](<../COMMON/ClipStringsWithPerimetersDialog.md>) screen.
  3. Click **Clip** to clip data.
