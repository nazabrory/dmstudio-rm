# clip-strings-to-wireframe ("cltw")

See this command in the [**command table**.](<_COMMAND%20TABLE_C.md#clip-strings-to-wireframe>)

To access this command:

  * **Digitize** ribbon **> > Tools >> Clip to Perimeter >> Clip to Surface**.

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "clip-strings-to-wireframe".

  * Use the quick key combination "cltw".

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **clip-strings-to-wireframe** and click **Run**.

## Command Overview

Clip selected string data to loaded wireframe surface or volume data.

Running this command displays the **[Clip Strings to Wireframe](<../COMMON/ClipStringsToWireframeDialog.md>)** screen.
