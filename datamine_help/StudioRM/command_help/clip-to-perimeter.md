# clip-to-perimeter ("ctp")

See this command in the [**command table**.](<_COMMAND%20TABLE_C.md#clip-to-perimeter>)

To access this command:

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "clip-to-perimeter".

  * Use the quick key combination "ctp".

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **clip-to-perimeter** and click **Run**.

## Command Overview

Deletes all string and point data from either inside or outside of a perimeter.

**Note** : If string or point data is loaded but not displayed, it is not affected by this command.

Command steps:

  1. Ensure data to be clipped, and the clipping perimeter, is visible in a 3D window.

  2. Run the command.

  3. Select the clipping perimeter.

  4. Select a point either inside or outside of the perimeter to choose which data is deleted. 

Strings and points are clipped to the edge of the perimeter.
