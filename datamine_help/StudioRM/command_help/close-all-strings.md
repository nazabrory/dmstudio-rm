# close-all-strings ("cla")

See this command in the [**command table**.](<_COMMAND%20TABLE_C.md#close-all-strings>)

To access this command:

  * **Digitize** ribbon **> > Condition>> Close >> Close All**.

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "close-all-strings".

  * Use the quick key combination "cla".

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **close-all-strings** and click **Run**.

## Command Overview

Change all open strings into closed perimeters by inserting a segment in each to bridge the gap between the end point and the first.

Running this command automatically cancels any other running command.

Command steps:

  1. Load and display the string data you want to close.

  2. Run the command.

  3. All open strings are closed. Closed strings are not affected.
