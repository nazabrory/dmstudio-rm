# combine-data-objects ("cdo")

See this command in the [**command table**.](<_COMMAND%20TABLE_C.md#combine-data-objects>)

To access this command:

  * **Data** ribbon **> > Objects >> Combine**.

  * Display the **Data Object Manager** and select **Combine Objects**.

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "combine-data-objects".

  * Use the quick key combination "cdo".

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **combine-data-objects** and click **Run**.

## Command Overview

Combine multiple objects into a single new object (e.g. for archiving in a dm file) using the [Combine Data Objects](<../COMMON/Data%20Objects%20Combine%20Dialog.md>) screen. 

All objects to be combined must be the same type (e.g. block models, wireframes or strings and so on). Data is output to memory and, if a physical file is required containing combined object data, it must be saved afterwards.
