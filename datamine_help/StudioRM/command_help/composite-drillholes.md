# composite-drillholes ("cmdh")  
  
See this command in the [**command table**.](<_COMMAND%20TABLE_C.md#composite-drillholes>)

To access this command:

  * **Sample Analysis** ribbon **> > Composite >> Composite**.

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "composite-drillholes"

  * Use the quick key combination "cmdh".

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **composite-drillholes** and click **Run**.

## Command Overview

Turns on the drillhole compositor mode for 3D windows. This mode enables either dynamic or static drillhole segments, holes or composites to be selected and analyzed in the Compositor control bar.

The first 50 columns of a drillhole are composited. If more than 50 columns are detected, you are shown the following message during composition:
    
    
    Warning: The picked drillhole object (name) has exceeded the maximum number of columns for the composite-drillholes command. Only the first 50 columns will be processed.

Command Steps:

  1. Load and display the drillhole data to be composited, in any **3D** window.

  2. Run the command.

  3. Select a drillhole segment (click) or composite (click-and-drag over multiple drillhole sections) in the3D window.

Note: other drillhole selection options are available. See [Compositor Control Bar](<../COMMON/Compositor%20Control%20Bar%20Overview.md>).

  4. In the [Compositor](<../COMMON/Compositor%20Control%20Bar%20Overview.md>) control bar, view the displayed results.

  5. Click **Cancel** when you have finishing compositing.
