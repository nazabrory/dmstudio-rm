# condition-string ("cond")

See this command in the [**command table**.](<_COMMAND%20TABLE_C.md#condition-string>)

To access this command:

  *   * **Digitize** ribbon **> > Condition >> Condition**.

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "condition-string"

  * Use the quick key combination "cond".

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **condition-string** and click **Run**.

## Command Overview

Uses the parameters of maximum and minimum chord lengths, and a minimum angle to modify a string. After string selected, this command displays the **[Enter Condition String Parameters](<../COMMON/Condition%20String%20Dialog.md>)** screen. 

This command is similar to the [simplify-string ("sps")](<simplify-string.md>) command.

If the command has been used previously, the most recent conditioning parameters are reinstated.

**Tip** : preselect string data to automatically display the parameters screen. You can preselect one or more strings.

Command steps:

  1. Run the command.

  2. The **[Enter Condition String Parameters](<../COMMON/Condition%20String%20Dialog.md>)** screen displays.

  3. If not already selected, select a string to condition.

  4. Enter the relevant parameters for conditioning.
