# connect-strings-attrib ("cona")

See this command in the [**command table**.](<_COMMAND%20TABLE_C.md#connect-strings-attrib>)

To access this command:

  * **Digitize** ribbon **> > Tools >> Connect >> Connect (Original Attributes)**.

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "connect-strings-attrib"

  * Use the quick key combination "cona".

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **connect-strings-attrib** and click **Run**.

## Command Overview

Connects two strings, from the selected end-point of the first to the selected end-point of the second. 

This command differs from **[connect-strings](<connect-strings.md>)** in that string point attributes are retained for both strings. To copy string attributes from the first selected string to the merged entity, use **connect-strings**.

If the two strings started as separate string objects, the second string is incorporated into the first string object and removed from the second.

Command steps:

  1. Run the command. 

If a string is already selected, you are prompted, from the message area of the status bar, to "Select selected string at the required end", otherwise the prompt "Select first string at the required end" is displayed.

  2. You are asked to "Select other string at the required end". 

The two strings are joined between the selected end points.

  3. Complete the command by clicking **Done** , or double-clicking (or tapping) in a **3D** window.
