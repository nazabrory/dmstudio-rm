# convert-wf-hull ("hts")

See this command in the [**command table**](<_COMMAND%20TABLE_C.md#convert-wf-hull>).

To access this command:

  * **Wireframe** ribbon **> > Plane >> Hull to Strings**.

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "convert-wf-hull"

  * Use the quick key combination "hts".

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **convert-wf-hull** and click **Run**.

Create string data from the hull (outline) of an existing wireframe object or partial object. The orientation of the plane to be used to create the cross-sectional string object is configurable.

**Note** : This command supports [**flexible wireframe selection**](<../COMMON/Wireframe_Selection_Concept.md>).

This command displays the [Hull to Strings](<../COMMON/hull%20to%20strings%20dialog.md>) screen.

This command can be applied to either a selected wireframe object, or to preselected wireframe triangles (in which case a hull string is generated for the selection).

Note: this command is also available using the [BOOLEAN](<../Process_Help_XML/boolean.md>) process (@METHOD=6).
