# copy-string-attributes ("coa")

See this command in the [**command table**.](<_COMMAND%20TABLE_C.md#copy-string-attributes>)

To access this command:

  * **Edit** ribbon **> > Attributes >> Copy**.

  * **Digitize** ribbon **> > Attributes >> Copy**.

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "copy-string-attributes"

  * Use the quick key combination "coa".

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **copy-string-attributes** and click **Run**.

## Command Overview

Copy the attributes of one string to another string, or from a string to a group of strings.

Command Steps:

  1. Run the command.

  2. Select the string which is the source of the attributes, as prompted.

  3. Select a string to which you wish to apply the selected string attributes.

**Note** : If a string or group of strings is selected before the command is run, are will automatically treated as strings to be changed and will receive the attributes of the next string selected.

  4. The command will remain active until cancelled or superseded by another command.
