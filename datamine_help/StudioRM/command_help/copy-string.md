# copy-string("cps")

See this command in the [**command table**](<_COMMAND%20TABLE_C.md#copy-string>).

To access this command:

  *   * **Digitize** ribbon **> > Transform >> Copy String**.

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "copy-string"

  * Use the quick key combination "cps".

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **copy-string** and click **Run**.

## Command Overview

Make a copy of a selected string in another location, leaving the original string as it is.

**Note** : data is always copied into the current string object, regardless of the target object. See [The Current Object ](<../COMMON/Concept_Current_Object.md>).

Command steps:

  1. Ensure string data is loaded and displayed.

  2. Run the command.

  3. Select a point on the string you wish to copy.

  4. Select the position of the same point on the new, copied string.

The string is copied to the new position, aligning the digitized points. Movement is performed across the currently active section.

  5. Click **Done** (or double-click) to stop editing.
