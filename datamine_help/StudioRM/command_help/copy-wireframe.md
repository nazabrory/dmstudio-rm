# copy-wireframe ("cpw")

See this command in the [**command table**.](<_COMMAND%20TABLE_C.md#copy-wireframe>)

To access this command:

  * **Explicit** ribbon **> > Edit >> Copy**.

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "copy-wireframe"

  * Use the quick key combination "cpw".

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **copy-wireframe** and click **Run**.

## Command Overview

Copy a loaded wireframe to a new location.

A new wireframe (a copy of the selected wireframe) is drawn at the selected location and saved to a new object. 

**Note** : This command supports [**flexible wireframe selection**](<../COMMON/Wireframe_Selection_Concept.md>).

Command steps:

  1. Run the Command.

  2. Click-and-drag the required wireframe to a new position. You can do this multiple times if required.

  3. Click Done to exit the command.
