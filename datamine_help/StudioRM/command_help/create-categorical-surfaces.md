# create-categorical-surfaces

See this command in the [**command table**.](<_COMMAND%20TABLE_C.md#create-categorical-surfaces>)

To access this command:

  * **Implicit** ribbon **> > Surface >> Categorical**.

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "create-categorical-surfaces"

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **create-categorical-surfaces** and click **Run**.

## Command Overview

Model rock type deposits based on drillhole attribute value.

This command uses the [Create Categorical Surfaces](<../STUDIO_RM/Implicit_Surface_From_Drillholes_Categorical.md>) screen, which is used to create one or more enclosed volumes representing a rock type deposit implied by input static drillhole data. The input to this command is a loaded desurveyed drillholes file, containing at least one attribute and corresponding value to be modelled. A common use of this command, for example, would be to model a volume representing a particular lithology.

For more information on this command see [Categorical Modelling](<../STUDIO_RM/Implicit_Surface_From_Drillholes_Categorical.md>).

**Note** : this command can be run [using an automation script](<../STUDIO_RM/Implicit_Modelling_Command_Automation.md>).
