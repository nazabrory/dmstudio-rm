# create-centerlines-from-outlines ("cen")

See this command in the [**command table**](<_COMMAND%20TABLE_C.md#create-centrelines-from-outlines>).

To access this command:

  *   * **Digitize** ribbon **> > Tools >> More >> Create Centerlines**.

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "create-centerlines-from-outlines"

  * Use the quick key combination "cen".

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **create-centerlines-from-outlines** and click **Run**.

## Command Overview

Generate centerlines for selected string data, treating the selected strings as perimeters. One or more closed strings must be pre-selected to use this command (you are unable to Generate data unless this is done).

Centrelines are either generated on a flat plane, or on the theoretical 'surface' the outline represents, if it were filled.

Optionally, you can transfer selected string attribute data to the generated centerline(s). Centerline data is added to the current string object.

Displays the [Generate Centerlines from Selected Outlines](<../COMMON/Create_Centrelines.md>) screen.
