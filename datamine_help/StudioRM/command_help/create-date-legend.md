# create-date-legend ("cdl")

See this command in the [**command table**](<_COMMAND%20TABLE_C.md#create-date-legend>).

To access this command: 

  * **Format** ribbon **> > Colours >> Date Legend**.

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "create-date-legend"

  * Use the quick key combination "cdl".

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **create-date-legend** and click **Run**.

## Command Overview

The tool is designed to generate legends based on date values contained within [Datamine Task Scheduler](<../COMMON/V14_CONCEPT_EPS_Overview.md>) (DTS) schedules. 

Displays the [Create Date Legend](<../COMMON/Create_Date_Legend.md>) screen.
