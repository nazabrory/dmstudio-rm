# create-drive ("cdr")

See this command in the [**command table**](<_COMMAND%20TABLE_C.md#create-drive>).

To access this command: 

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "create-drive"

  * Use the quick key combination "cdr".

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **create-drive** and click **Run**.

Extrude a designed profile shape along one or more design strings. You can either use a preset shape or set up a custom outline by defining the positions of the outline vertices. 

Displays the **[Create Drive](<../COMMON/create-drive.md>)** screen. Follow this link for more information.
