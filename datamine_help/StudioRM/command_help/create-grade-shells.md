# create-grade-shells

See this command in the [**command table**](<_COMMAND%20TABLE_C.md#create-grade-shells>).

To access this command: 

  * **Implicit** ribbon **> > Surface >> Grade Shells**.
  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "create-grade-shells"

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **create-grade-shells** and click **Run**.

## Command Overview

Model rocktype deposits based on drillhole attribute value.

This command displays the [Create Grade Shells](<../STUDIO_RM/Implicit_Surface_From_Drillholes_Continuous.md>) screen to create one or more enclosed volumes representing a rock type deposit implied by input static drillhole data.

The input to this command is a loaded desurveyed drillholes file, containing at least one attribute and corresponding numeric value to be modelled. A common use of this command, for example, would be to model grade shells.

**Note** : this command can be run [using an automation script](<../STUDIO_RM/Implicit_Modelling_Command_Automation.md>).
