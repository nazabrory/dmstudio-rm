# create-isosurfaces ("iso")

See this command in the [**command table**.](<_COMMAND%20TABLE_C.md#create-isosurfaces>)

To access this command:

  * **Model** ribbon **> > Data From Model >> Isosurfaces**.

  * **Explicit** ribbon **> > Automatic >> Isosurfaces**.

  * **Solids** ribbon **> > Create >> Isosurfaces**.

  * **Surfaces** ribbon **> > Create >> Isosurfaces**.

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "create-isosurfaces"

  * Use the quick key combination "iso"

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **create-isosurfaces** and click **Run**.

## Command Overview

Create a wireframe isosurface(s) from defined values within a block model, using the **[Generate Isosurfaces](<../COMMON/GenerateIsoSurfacesDialog.md>)** screen.

Related topics and activities

  * [Generate Isosurfaces](<../COMMON/GenerateIsoSurfacesDialog.md>)
