# create-model-prototype ("cmp")

See this command in the [**command table**.](<_COMMAND%20TABLE_C.md#create-model-prototype>)

To access this command:

  * **Model** ribbon **> > Create >> Auto Prototype**.

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "create-model-prototype"

  * Use the quick key combination "cmp"

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **create-model-prototype** and click **Run**.

## Command Overview

Generate and preview (non)rotated block model prototypes, using defined parameters, an existing file or loaded data objects.

Displays the [Create Model Prototype](<../COMMON/CreateModelPrototype_Dialog.md>) screen which is powered primarily by the **[PROTOM](<../Process_Help_XML/protom.md>)** process.
