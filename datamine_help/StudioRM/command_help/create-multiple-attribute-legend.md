# create-multiple-attribute-legend ("cml")

See this command in the [**command table**.](<_COMMAND%20TABLE_C.md#create-multiple-attribute-legend>)

To access this command:

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "create-multiple-attribute-legend"

  * Use the quick keys 'cml'

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **create-multiple-attribute-legend** and click **Run**.

## Command Overview

Generate a filter legend based on loaded data object attribute values or, for numeric values, nominated value ranges. 

For more information see [Multiple Attribute Legend](<../COMMON/MultipleAttributeLegend.md>).
