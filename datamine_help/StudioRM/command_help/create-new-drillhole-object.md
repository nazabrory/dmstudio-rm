# create-new-drillhole-object ("cnd")

See this command in the [**command table**.](<_COMMAND%20TABLE_C.md#create-new-drillhole-object>)

To access this command:

  * **Data** ribbon **> > Objects >> New >> Drillholes**.

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "create-new-drillhole-object"

  * Use the quick keys 'cnd'.

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **create-new-drillhole-object** and click **Run**.

## Command Overview

Creates a new drillhole object in memory. Run the command to create a new, empty object in memory.
