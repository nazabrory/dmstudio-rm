# create-new-legend ("cnl")

See this command in the [**command table**.](<_COMMAND%20TABLE_C.md#create-new-legend>)

To access this command:

  * **Format** ribbon **> > Colour >> New Legend**.

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "create-new-legend"

  * Use the quick keys 'cnl'.

  * Attempt to create a new default legend in any 3D properties screen.

  * Various other system commands that make use of legends for display or reporting.

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **create-new-legend** and click **Run**.

## Command Overview

Displays the **Create New Legend** wizard for the creation of value, range and filter legends. For more information, press **F1** when the screen displays.
