# create-new-planes-object ("cnn")

See this command in the [**command table**.](<_COMMAND%20TABLE_C.md#create-new-planes-object>)

To access this command:

  * **Data** ribbon **> > Objects >> New >> Planes**.

  * **Digitize** ribbon **> > Edit >> New Object >> Planes**.

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "create-new-planes-object"

  * Use the quick keys 'cnn'.

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **create-new-planes-object** and click **Run**.

## Command Overview

Creates a new planes object in memory. Run the command to create a new, empty object in memory.
