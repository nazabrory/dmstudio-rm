# create-new-points-object ("cnp")

See this command in the [**command table**.](<_COMMAND%20TABLE_C.md#create-new-points-object>)

To access this command:

  * **Data** ribbon **> > Objects >> New >> Points**.

  * **Digitize** ribbon **> > Edit >> New Object >> Points**.

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "create-new-points-object"

  * Use the quick keys 'cnp'.

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **create-new-points-object** and click **Run**.

## Command Overview

Creates a new points object in memory. Run the command to create a new, empty object in memory.
