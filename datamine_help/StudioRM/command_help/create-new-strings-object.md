# create-new-strings-object ("cns")

See this command in the [**command table**.](<_COMMAND%20TABLE_C.md#create-new-strings-object>)

To access this command:

  * **Data** ribbon **> > Objects >> New >> Strings**.

  * **Digitize** ribbon **> > Edit >> New Object >> Strings**.

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "create-new-strings-object"

  * Use the quick keys 'cns'.

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **create-new-strings-object** and click **Run**.

## Command Overview

Creates a new strings object in memory. Run the command to create a new, empty object in memory.
