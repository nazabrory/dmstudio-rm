# create-new-wireframe-object ("cnw")

See this command in the [**command table**.](<_COMMAND%20TABLE_C.md#create-new-wireframe-object>)

To access this command:

  * **Data** ribbon **> > Objects >> New >> Wireframe**.

  * **Digitize** ribbon **> > Edit >> New Object >> Wireframe**.

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "create-new-wireframe-object"

  * Use the quick keys 'cnw'.

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **create-new-wireframe-object** and click **Run**.

## Command Overview

Creates a new wireframe object in memory. Run the command to create a new, empty object in memory.
