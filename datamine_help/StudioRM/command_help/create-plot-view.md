# create-plot-view ("cpv")

See this command in the [**command table**.](<_COMMAND%20TABLE_C.md#create-plot-view>)

To access this command:

  * **3D View** ribbon **> > Save To >> Plot View**.

  * **Report** ribbon **> > Present >> Plot View**.

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "create-plot-view".

  * Use the quick key combination "cpv".

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **create-plot-view** and click **Run**.

## Command Overview

Create a new plot sheet containing a single projection representing the current view direction in any [3D Window](<../VR_Help/VR_Introduction.md>). The current view direction and scale settings is transformed to a [plot projection](<../COMMON/View%20Hierarchy.md>), ready for use in reporting. This command displays the [Plots from 3D Data](<../COMMON/Plot%20Overlays%20From%20Type.md>)screen.

Projections are created either as a 2D or 3D projection. See [Plot Projection Overlay Types](<../PLOTS_LOGS/Projection%20Overlay%20Types.md>).
