# create-shells ("csh")

See this command in the [**command table**.](<_COMMAND%20TABLE_C.md#create-shells>)

To access this command:

  * **Explicit** ribbon **> > Automatic >> IsoShells**.

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "create-shells"

  * Use the quick key combination "csh".

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **create-shells** and click **Run**.

## Command Overview

Create isoshell wireframes from drillhole or point sample data. See [Create Isoshells](<../COMMON/Create_Isoshells.md>).

**Note** : Your application can also use implicit modelling techniques to generate **[categorical](<../STUDIO_RM/Implicit_Surface_From_Drillholes_Categorical.md>)** and **[grade](<../STUDIO_RM/Implicit_Surface_From_Drillholes_Continuous.md>)** shells.
