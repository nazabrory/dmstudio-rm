# create-tag-string ("tgs")

See this command in the [**command table**.](<_COMMAND%20TABLE_C.md#create-tag-string>)

To access this command:

  * **Explicit** ribbon **> > Create>> Tag String**.

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "create-tag-string"

  * Use the quick key combination "tgs".

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **create-tag-string** and click **Run**.

## Command Overview

A [tag string](<../COMMON/Wireframe_Tag_Strings.md>) tags specific points to be joined when wireframing strings. This command is used to convert an existing string to a tag string.

A tag string controls how strings are linked together to form wireframes, say, when using the [link-strings ("ls")](<link-strings.md>) and other string linking commands.

Tip: Whilst digitizing, you can complete the digitizing operation by double-clicking or double-tapping the final string vertex position. You can also complete the digitizing operation by closing the string.

See [Wireframe Tag Strings](<../COMMON/Wireframe_Tag_Strings.md>).

Command steps:

  1. Digitize a new string in any 3D window, for example, using **[new-string](<new-string.md>)** , or load string data.

  2. Run the command.

  3. Select the string to become a tag string.
