# crosshair-switch ("tch")

See this command in the [**command table**.](<_COMMAND%20TABLE_C.md#crosshair-switch>)

To access this command:

  * **Project Settings** screen **> > Data Display >> Display Cursor Crosshairs**. 

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "crosshair-switch"

  * Use the quick key combination "tch".

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **crosshair-switch** and click **Run**.

## Command Overview

Toggle between standard cursor and full-size crosshairs in 3D windows in the legacy **Graphics** window. 

This does not affect the 3D view.
