# cut-and-fill-volumes

See this command in the [**command table**.](<_COMMAND%20TABLE_C.md#cut-and-fill-volumes>)

To access this command:

  * **Wireframe** ribbon **> > Process >> Cut and Fill Volumes**.

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "cut-and-fill-volumes"

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **cut-and-fill-volumes** and click **Run**.

## Command Overview

Generate volumes between surfaces representing cut and fill operations, using the [Cut and Fill Volumes](<../COMMON/Cut%20And%20Fill%20Volumes%20Dialog.md>) screen.
