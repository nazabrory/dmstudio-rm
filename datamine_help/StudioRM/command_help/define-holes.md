# define-holes

See this command in the [**command table**.](<COMMAND%20TABLE_D.md#define-holes>)

To access this command:

  * **Sample Analysis** ribbon **> > Prepare Samples >> Define Dynamic**.

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "define-holes"

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **define-holes** and click **Run**.

## Command Overview

Define the field assignment (field mapping) and where relevant, also the dip and angle conventions for the selected dynamic drillhole table. Displays the [Define Holes](<../COMMON/Define%20Holes%20Dialog.md>) screen.
