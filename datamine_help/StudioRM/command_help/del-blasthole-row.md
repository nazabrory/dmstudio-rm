# del-blasthole-row ("dbr")

See this command in the [**command table**.](<COMMAND%20TABLE_D.md#del-blasthole-row>)

To access this command:

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "del-blasthole-row".

  * Use the quick key combination "dbr".

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **del-blasthole-row** and click **Run**.

## Command Overview

Delete a row of drillholes created with the blast-layout command.

This command deletes a row of drillholes by using the **BENCH** , **BLAST** and **ROWNUM** values of the holes. You are asked to select a hole. 

All holes containing identical values for **BENCH** , **BLAST** and **ROWNUM** are then deleted.
