# delete-file ("dfi")

See this command in the [**command table**.](<COMMAND%20TABLE_D.md#delete-file>)

To access this command:

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "delete-file".

  * Use the quick key combination "dfi".

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **delete-file** and click **Run**.

## Command Overview

Deletes the selected Datamine files permanently from the project. See [Delete File(s) from Project](<../COMMON/deletefile.md>).
