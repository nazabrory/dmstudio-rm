# delete-points-mode ("dpo")

See this command in the [**command table**.](<COMMAND%20TABLE_D.md#delete-points-mode>)

To access this command:

  * **Digitize** ribbon **> > Tools >> Delete Segment**.

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "delete-points-mode".

  * Use the quick key combination "dpo".

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **delete-points-mode** and click **Run**.

## Command Overview

Deletes vertices (points) in any selected or non-selected string(s) or point(s) nearest to the cursor.

You can undo and redo command actions.

![](../Images/Tip.gif) |  This command can be completed by double-clicking (or tapping) anywhere in a 3D window.  
---|---  
  
To delete points on previously selected string(s) or point(s):

  1. Select the string(s) or point(s) to be modified.

  2. Run the command.

  3. If a string(s) has been selected, individually select (left-click or right-click) the required string point(s). 

  4. Click **Done** to complete the command.

To delete points where no string(s) or point(s) data is preselected:

  1. Run the command.

  2. For string data, individually select (left-click or right-click) the required point(s). For points data, pick the points to delete.

  3. Click **Done** to complete the command.
