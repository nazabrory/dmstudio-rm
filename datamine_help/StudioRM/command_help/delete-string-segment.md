# delete-string-segment ("dss")

See this command in the [**command table**.](<COMMAND%20TABLE_D.md#delete-string-segment>)

To access this command:

  * **Digitize** ribbon **> > Tools >> Delete Segment**.

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "delete-string-segment".

  * Use the quick key combination "dss".

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **delete-string-segment** and click **Run**.

## Command Overview

Interactively remove string segments from a loaded and displayed string data object.

  * If no strings are selected, any segment can be removed from any string. 

  * If strings are selected, removal is restricted to segments of selected strings.

Command steps:

  1. Load and display string data to modify.

  2. Run the command.

  3. Select one or more string segments to remove.

  4. Click **Done** to complete the command.
