# delete-trace-switch ("tet")

See this command in the [**command table**.](<COMMAND%20TABLE_D.md#delete-trace-switch>)

To access this command:

  * **Edit** ribbon **> > Tools >> More >> Delete when Tracing**.

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "delete-trace-switch".

  * Use the quick key combination "tet".

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **delete-trace-switch** and click **Run**.

## Command Overview

Toggles whether strings that make up a trace are retained.

If this toggle is ON then strings which were selected to make up a trace string are erased when the trace is ended.

If this toggle is OFF, original strings are not erased when the trace is ended.
