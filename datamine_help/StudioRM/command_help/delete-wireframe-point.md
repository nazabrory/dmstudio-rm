# delete-wireframe-point

See this command in the [**command table**.](<COMMAND%20TABLE_D.md#delete-wireframe-point>)

To access this command:

  * **Explicit** ribbon **> > Current Wireframe >> Delete Point**.
  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "delete-wireframe-point".

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate delete-schedule**delete-wireframe-point** and click **Run**.

## Command Overview

Deletes selected wireframe points.

**Note** : You can undo and redo this operation, but it is wise to save your wireframe before you start deleting points, just in case you need to reinstate its original shape.

Activity steps:

  1. Run the command.

  2. Select point data to be deleted. Each click deletes a point.

  3. Click **Done** to complete point editing mode.
