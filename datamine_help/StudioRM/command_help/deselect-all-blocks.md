# deselect-all-blocks ("dab")

See this command in the [**command table**.](<COMMAND%20TABLE_D.md#deselect-all-blocks>)

To access this command:

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "deselect-all-blocks".

  * Use the quick key combination "dab".

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **deselect-all-blocks** and click **Run**.

## Description

This command deselects all blocks which are currently selected for scheduling. In order to resume scheduling, blocks must be reselected using select-mining-blocks or select-all-blocks.
