# deselect-all-drillholes ("dad")

See this command in the [**command table**.](<COMMAND%20TABLE_D.md#deselect-all-drillholes>)

  * **Home** ribbon **> > Select >> Select >> None>> All Drillholes** (toggle).

  * Use the quick key combination "dad".

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "deselect-all-drillholes".

  * On the **[Find Command](<../COMMON/findcommand.md>)** screen, highlight **deselect-all-drillholes** and click **Run**.

## Command Overview

Deselects all selected static drillholes objects data.

Command steps:

  1. Run the command.

  2. Any selected static drillholes are deselected and no longer highlighted.
