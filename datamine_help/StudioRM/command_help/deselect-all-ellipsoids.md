# deselect-all-ellipsoids("dael")

See this command in the [**command table**.](<COMMAND%20TABLE_D.md#deselect-all-ellipsoids>)

  * **Home** ribbon **> > Select >> Select >> None>> All Ellipsoids** (toggle).

  * Use the quick key combination "dael".

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "deselect-all-ellipsoids".

  * On the **[Find Command](<../COMMON/findcommand.md>)** screen, highlight **deselect-all-ellipsoids** and click **Run**.

## Command Overview

Deselect all selected ellipsoid data.
