# deselect-all-limits ("dal")

See this command in the [**command table**.](<COMMAND%20TABLE_D.md#deselect-all-limits>)

To access this command:

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "deselect-all-limits".

  * Use the quick key combination "dal".

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **deselect-all-limits** and click **Run**.

## Command Overview

Deselect perimeters that have been selected to define inner or outer boundaries when creating DTMs.
