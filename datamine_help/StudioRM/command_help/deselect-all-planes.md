# deselect-all-planes ("dan")

See this command in the [**command table**.](<COMMAND%20TABLE_D.md#deselect-all-planes>)

  * **Home** ribbon **> > Select >> Select >> None>> All Planes** (toggle).

  * Use the quick key combination "dan".

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "deselect-all-planes".

  * On the **[Find Command](<../COMMON/findcommand.md>)** screen, highlight **deselect-all-planes** and click **Run**.

## Command Overview

Deselects all selected planes objects data.

Command steps:

  1. Run the command.

  2. All selected planes are deselected and no longer highlighted.
