# deselect-all-points ("dap")

See this command in the [**command table**.](<COMMAND%20TABLE_D.md#deselect-all-points>)

  * **Home** ribbon **> > Select >> Select >> None>> All Points** (toggle).

  * Use the quick key combination "dap".

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "deselect-all-points".

  * On the **[Find Command](<../COMMON/findcommand.md>)** screen, highlight **deselect-all-points** and click **Run**.

## Command Overview

Deselects all selected points objects data.

Command steps:

  1. Run the command.

  2. All selected points are deselected and no longer highlighted.
