# deselect-all-strings ("das")

See this command in the [**command table**.](<COMMAND%20TABLE_D.md#deselect-all-strings>)

  * **Home** ribbon **> > Select >> Select >> None>> All Strings** (toggle).

  * Use the quick key combination "das".

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "deselect-all-strings".

  * On the **[Find Command](<../COMMON/findcommand.md>)** screen, highlight **deselect-all-strings** and click **Run**.

## Command Overview

Deselects all selected strings objects data.

Command steps:

  1. Run the command.

  2. All selected strings are deselected and no longer highlighted.
