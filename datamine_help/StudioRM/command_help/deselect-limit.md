# deselect-limit ("dli")

See this command in the [**command table**.](<COMMAND%20TABLE_D.md#deselect-limit>)

To access this command:

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "deselect-limit".

  * Use the quick key combination "dli".

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **deselect-limit** and click **Run**.

## Command Overview

Deselect a DTM limit string previously been selected using the [pick-inner-limit](<pick-inner-limit.md>) or [pick-outer-limit](<pick-outer-limit.md>) command.
