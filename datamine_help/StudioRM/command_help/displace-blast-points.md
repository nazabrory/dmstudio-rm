# displace-blast-points

See this command in the [**command table**.](<COMMAND%20TABLE_D.md#displace-blast-points>)

To access this command:

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "displace-blast-points".

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **displace-blast-points** and click **Run**.

## Command Overview

Calculate point data displacement using an alternative algorithm to the curve method for calculating a blast.

For more information on this command, please contact your local Datamine representative.
