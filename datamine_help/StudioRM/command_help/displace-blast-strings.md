# displace-blast-strings

See this command in the [**command table**.](<COMMAND%20TABLE_D.md#displace-blast-strings>)

To access this command:

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "displace-blast-strings"

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **displace-blast-strings** and click **Run**.

## Command Overview

Calculate string data displacement using an alternative algorithm to the curve method for calculating a blast.

For more information on this command, please contact your local Datamine representative.
