# display-eval-results-switch ("drs")

See this command in the [**command table**.](<COMMAND%20TABLE_D.md#display-eval-results-switch>)

To access this command:

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "display-eval-results-switch".

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **display-eval-results-switch** and click **Run**.

## Command Overview

Toggles the display of evaluation results following evaluation processes.

If this switch is ON, evaluation results are displayed.

If this switch is OFF, evaluation results are not displayed.
