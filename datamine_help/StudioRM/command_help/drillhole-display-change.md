# drillhole-display-change ("tdhd")

See this command in the [**command table**.](<COMMAND%20TABLE_D.md#drillhole-display-change>)

To access this command:

  * **Format** ribbon **> > Show/Hide >> Holes**.

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "drillhole-display-change".

  * Use the quick key combination "tdhd".

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **drillhole-display-change** and click **Run**.

## Command Overview

Show or hide all drillhole data in all 3D windows. Running this command toggles between showing and hiding data.

**Note** : When redisplaying data, previous object filters are reapplied.
