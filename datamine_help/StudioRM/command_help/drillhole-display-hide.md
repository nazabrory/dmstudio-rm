# drillhole-display-hide

See this command in the [**command table**.](<COMMAND%20TABLE_D.md#drillhole-display-hide>)

To access this command:

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "drillhole-display-hide".

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **drillhole-display-hide** and click **Run**.

## Command Overview

Hide all drillhole data in all 3D windows.
