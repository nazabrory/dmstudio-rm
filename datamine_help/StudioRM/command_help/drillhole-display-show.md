# drillhole-display-show

See this command in the [**command table**.](<COMMAND%20TABLE_D.md#drillhole-display-show>)

To access this command:

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "drillhole-display-show".

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **drillhole-display-show** and click **Run**.

## Command Overview

Show all hidden drillhole data in all 3D windows.
