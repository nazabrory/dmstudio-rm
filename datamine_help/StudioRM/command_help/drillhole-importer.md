# drillhole-importer ("idh")

See this command in the [**command table**.](<COMMAND%20TABLE_D.md#drillhole-importer>)

To access this command:

  * **Data** ribbon **> > Import >> Drillholes**.

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "drillhole-importer"

  * Open the **[Find Command](<../COMMON/findcommand.md>)** screen, select "drillhole-importer" and click **Run**.

  * Use the quick key combination "idh".

## Command Overview

Import, validate and desurvey component drillhole tables using the Drillhole Importer console. For more information on using this facility, see [Drillhole Importer](<../COMMON/DrillholeImporter-screen.md>).
