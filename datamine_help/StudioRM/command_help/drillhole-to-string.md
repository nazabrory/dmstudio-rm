# drillhole-to-string ("d2s")

See this command in the [**command table**.](<COMMAND%20TABLE_D.md#drillhole-to-string>)

To access this command:

  * **Sample Analysis** ribbon **> > Edit Drillholes >> Hole to String**.

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "drillhole-to-string".

  * Use the quick key combination "d2s".

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **drillhole-to-string** and click **Run**.

## Command Overview

Convert static drillholes to strings, and adds them to the current strings object. Run the command and pick the drillhole data to become string data.

**Tip** : Preselect data before running the command to convert multiple drillholes to strings at once.
