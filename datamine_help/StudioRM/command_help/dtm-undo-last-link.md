# dtm-undo-last-link ("uld")

See this command in the [**command table**.](<COMMAND%20TABLE_D.md#dtm-undo-last-link>)

To access this command:

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "dtm-undo-last-link"

  * Use the quick key combination "uld".

  * Launch the **[Find Command](<../COMMON/findcommand.md>)** screen and locate "dtm-undo-last-link", then click **Run**.

## Command Overview

Erases selected wireframe data.

Command steps:

  1. Load and select wireframe data.

  2. Run the command.

  3. The prompt 'Are you sure you want to delete the selected items from ....' is shown. 

Select Yes to continue.

  4. The selected wireframe triangles are deleted.
