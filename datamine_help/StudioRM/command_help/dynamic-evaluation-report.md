# dynamic-evaluation-report

See this command in the [**command table**.](<COMMAND%20TABLE_D.md#dynamic-evaluation-report>)

To access this command:

  * **Report** ribbon **> > Evaluate Dynamic >> Strings**.

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "dynamic-evaluation-report".

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **dynamic-evaluation-report** and click **Run**.

## Command Overview

Evaluate a set of perimeters using dynamic evaluation methods (either using block model or drillhole data for calculations).

Evaluation is performed by projecting the closed string(s) either in a vertical direction or orthogonally to the currently active section.

Command steps:

  1. Run the command.

The [Evaluation 2D Strings Properties](<../COMMON/Evaluation2DStringsPropertiesDialog.md>) screen displays.

  2. Define object and evaluation settings.
  3. Click OK.
  4. Review the results displayed on the [Table View](<../COMMON/Evaluation2DStringsTableViewDialog.md>) screen.

**Note** : If absent grade values are found, then they are replaced by the average value for the grade, and a message is displayed in the Notes section.

  5. Click Save Results.

The [Save Evaluation Results](<../COMMON/EvaluationSaveResultsDialog.md>) screen displays.

  6. Define data save settings, and click OK.
