# edge-editor

See this command in the [**command table**.](<COMMAND%20TABLE_E.md#edge-editor>)

To access this command:

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "edge-editor"

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **edge-editor** and click **Run**.

## Command Overview

The Edge Editor lets you modify string design data in a **3D** window.

See [Edge Editor](<../COMMON/EdgeEditor.md>).
