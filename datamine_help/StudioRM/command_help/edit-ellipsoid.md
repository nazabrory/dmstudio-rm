# edit-ellipsoid ("eel")

See this command in the [command table](<COMMAND%20TABLE_E.md#edit-ellipsoid>).

To access this command:

  * **Geology** ribbon **> > Ellipsoids >> Edit**.

  * Enter "edit-ellipsoid" into the **Command** toolbar and press <ENTER>

  * Use the quick key combination "eel" with a 3D window active.

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **edit-ellipsoid** and click **Run**.

## Command Overview

Select and edit an ellipsoid within a loaded ellipsoid data object. 

Displays the [Edit Ellipsoid](<../COMMON/Edit_Ellipsoid.md>) dialog.
