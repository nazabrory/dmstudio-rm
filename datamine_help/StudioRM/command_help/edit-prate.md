# edit-prate ("epr")

See this command in the [**command table**](<COMMAND%20TABLE_E.md#edit-prate>).

To access this command:

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "edit-prate"
  * Use the quick key combination "epr".
  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **edit-prate** and click **Run**.

## Command Overview

Changes the production rate on faces and tonnage zones.

Each time you manually mine a unit you are given the opportunity to confirm or change the production rate. This command is useful when you wish to change production rates while using the AUTO PROGRESS facility.

Command steps:

  1. Run the command.

  2. Select a face or tonnage zone and then enter a new production rate.
