# end-schedule-session ("qsc")

See this command in the [**command table**.](<COMMAND%20TABLE_E.md#end-schedule-session>)

To access this command:

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "end-schedule-session"

  * Use the quick key combination "qsc".

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **end-schedule-session** and click **Run**.

## Command Overview

Exits from the current Mine Production Sequencing session. If mine scheduling has not been initialized this command has no effect

Command steps:

  1. Run the command.

  2. The screen layout is reset to as it was before mine scheduling was started.
