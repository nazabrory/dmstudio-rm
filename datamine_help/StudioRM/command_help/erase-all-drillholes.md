# erase-all-drillholes ("ead")

See this command in the [**command table**.](<COMMAND%20TABLE_E.md#erase-all-drillholes>)

To access this command:

  * **Data** ribbon **> > Unload >> All Static Drillholes**.

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "erase-all-drillholes"

  * Use the quick key combination "ead".

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **erase-all-drillholes** and click **Run**.

## Command Overview

Unload all static drillhole objects from memory.
