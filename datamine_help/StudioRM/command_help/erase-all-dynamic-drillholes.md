# erase-all-dynamic-drillholes

See this command in the [**command table**.](<COMMAND%20TABLE_E.md#erase-all-dynamic-drillholes>)

To access this command:

  * **Data** ribbon **> > Unload >> All Dynamic Drillholes**.

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "erase-all-dynamic-drillholes"

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **erase-all-dynamic-drillholes** and click **Run**.

## Command Overview

Unload all dynamic drillhole objects from memory.
