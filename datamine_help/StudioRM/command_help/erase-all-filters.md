# erase-all-filters ("eaf")

See this command in the [**command table**.](<COMMAND%20TABLE_E.md#erase-all-filters>)

To access this command:

  * Format ribbon >> Filter >> Clear.

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "erase-all-drillholes"

  * Use the quick key combination "eaf".

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **erase-all-drillholes** and click **Run**.

## Command Overview

Remove any previously applied in-memory filters for all loaded data objects.

Once run, use **[redraw-display](<redraw-display.md>)** to update 3D windows.
