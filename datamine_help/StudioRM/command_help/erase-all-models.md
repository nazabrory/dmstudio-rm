# erase-all-models ("eam")

See this command in the [**command table**.](<COMMAND%20TABLE_E.md#erase-all-models>)

To access this command:

  * **Data** ribbon **> > Unload >> All Models**.

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "erase-all-models"

  * Use the quick key combination "eam".

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **erase-all-models** and click **Run**.

## Command Overview

Unloads all block model objects from memory.
