# erase-all-planes ("ean")

See this command in the [**command table**.](<COMMAND%20TABLE_E.md#erase-all-planes>)

To access this command:

  * **Data** ribbon **> > Unload >> All Planes**.

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "erase-all-planes"

  * Use the quick key combination "ean".

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **erase-all-planes** and click **Run**.

## Command Overview

Unloads all planes objects from memory.
