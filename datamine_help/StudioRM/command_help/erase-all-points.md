# erase-all-points ("eap")

See this command in the [**command table**.](<COMMAND%20TABLE_E.md#erase-all-points>)

To access this command:

  * **Data** ribbon **> > Unload >> All Points**.

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "erase-all-points"

  * Use the quick key combination "eap".

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **erase-all-points** and click **Run**.

## Command Overview

Unloads all points objects from memory.
