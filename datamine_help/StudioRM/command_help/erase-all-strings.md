# erase-all-strings ("eas")

See this command in the [**command table**.](<COMMAND%20TABLE_E.md#erase-all-strings>)

To access this command:

  * **Data** ribbon **> > Unload >> All Strings**.

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "erase-all-strings"

  * Use the quick key combination "eas".

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **erase-all-strings** and click **Run**.

## Command Overview

Unloads all strings objects from memory.
