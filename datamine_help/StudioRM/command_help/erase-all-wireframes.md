# erase-all-wireframes ("eaw")

See this command in the [**command table**.](<COMMAND%20TABLE_E.md#erase-all-wireframes>)

To access this command:

  * **Data** ribbon **> > Unload >> All Wireframes**.

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "erase-all-strwireframesings"

  * Use the quick key combination "eaw".

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **erase-all-wireframes** and click **Run**.

## Command Overview

Unloads all wireframe objects from memory.
