# erase-all ("eal")

See this command in the [**command table**.](<COMMAND%20TABLE_E.md#erase-all>)

To access this command:

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "erase-all"

  * Use the quick key combination "eal".

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **erase-all** and click **Run**.

## Command Overview

Unload all unfiltered string objects currently in memory. This differs from the **[erase-all-strings](<erase-all-strings.md>)** command in that only unfiltered (visible) strings are unloaded.
