# erase-drillhole ("esd")

See this command in the [**command table**.](<COMMAND%20TABLE_E.md#erase-drillhole>)

To access this command:

  * **Data** ribbon **> > Objects >> Unload > Delete Static Drillholes**.

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "erase-drillhole"

  * Use the quick key combination "erd".

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **erase-drillhole** and click **Run**.

## Command Overview

Erase selected static drillhole(s) from memory. For information on deleting dynamic drillhole data, see [[erase-all-dynamic-drillholes](<erase-all-dynamic-drillholes.md>).](<erase-all-dynamic-drillholes.md>)

Command steps:

  1. Optionally, preselect visible static drillhole data in a 3D window.

  2. Run the command.

  3. If not already selected, select the required static drillhole(s) individually or with a selection box.

  4. Confirm the deletion of data.

  5. Click Done.
