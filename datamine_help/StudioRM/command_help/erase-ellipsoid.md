# erase-ellipsoid ("ere")

See this command in the [**command table**.](<COMMAND%20TABLE_E.md#erase-ellipsoid>)

To access this command:

  * **Data** ribbon **> > Objects >> Unload > Delete Ellipsoids**.

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "erase-ellipsoid"

  * Use the quick key combination "ere".

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **erase-ellipsoid** and click **Run**.

## Command Overview

Delete pre-selected ellipsoid data, or enters picking mode to interactively select ellipsoids for deletion (requires ellipsoid data selection to be [active](<../COMMON/Selecting3DDataInteractively.md>)).

**Note** : you can also delete ellipsoid data by selecting one or more ellipsoids and pressing DELETE.

Command steps:

  1. Optionally, preselect visible data in a 3D window.

  2. Run the command.

  3. If not already selected, select the required data items individually or with a selection box.

  4. Confirm the deletion of data.

  5. Click Done.
