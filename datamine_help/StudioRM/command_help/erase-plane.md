# erase-plane ("ern")

See this command in the [**command table**.](<COMMAND%20TABLE_E.md#erase-plane>)

To access this command:

  * **Data** ribbon **> > Objects >> Unload >> Delete Planes**.

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "erase-plane"

  * Use the quick key combination "ern".

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **erase-plane** and click **Run**.

## Command Overview

Erase selected plane(s) from memory.

  * Only visible planes can be selected and erased. Hidden overlays and data that are filtered out at the time of selection, cannot be selected even if they fall within a selection box .

Command steps:

  1. Run the command.

  2. If not already selected, select the required planes (any method). See [Selecting 3D Data Interactively](<../COMMON/Selecting3DDataInteractively.md>).

  3. Click Yes.

  4. Click Cancel.
