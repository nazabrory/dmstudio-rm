# erase-point ("dpo")

See this command in the [**command table**.](<COMMAND%20TABLE_E.md#erase-point>)

To access this command:

  * **Data** ribbon **> > Objects >> Unload >> Points**.

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "erase-point"

  * Use the quick key combination "dpo".

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **erase-point** and click **Run**.

## Command Overview

Delete vertices (points) in any selected or non-selected string(s) or point(s) nearest to the cursor.

This command is supported by undo and redo operations.

To delete points on selected string or point data:

  1. Select string(s) or point(s). See [Selecting 3D Data Interactively](<../COMMON/Selecting3DDataInteractively.md>).

  2. Run the command.

  3. If a string(s) has been selected, individually select (left-click or right-click) the required string point(s).

Note: When selecting the string points for deletion, only one point can be selected at a time. Clicking anywhere within a 3D window will delete the point, on the selected string(s), which is nearest to the cursor. The user does not need to click adjacent to each point, but just closer to the required point than any other point on the selected string.

  4. Press ESC or click **Done** to complete the command.

To delete points on unselected string or point data:

  1. Run the command.

  2. Individually select (left-click or right-click) the required point(s).

  3. Press ESC or click **Done** to complete the command.
