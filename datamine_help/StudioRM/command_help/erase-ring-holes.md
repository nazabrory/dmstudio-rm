# erase-ring-holes

See this command in the [**command table**.](<COMMAND%20TABLE_E.md#erase-ring-holes>)

To access this command:

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "erase-ring-holes"

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **erase-ring-holes** and click **Run**.

## Command Overview

Erase selected ring holes from memory. For more information on this command, please contact your local Datamine representative.
