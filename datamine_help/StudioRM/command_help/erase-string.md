# erase-string ("ers")

See this command in the [**command table**.](<COMMAND%20TABLE_E.md#erase-string>)

To access this command:

  * **Home** ribbon **> > Unload >> Delete Strings**.

  * **Data** ribbon **> > Unload >> Delete Strings**.

  * Draw (Sketches) ribbon >> Edit >> Delete Strings.

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "erase-string"

  * Use the quick key combination "ers".

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **erase-string** and click **Run**.

## Command Overview

Erase (delete) string(s) from memory. If string data is preselected, it is deleted when the command is run (after confirmation), otherwise, string data can be picked interactively after running the command.

**Note** : physical string file data is not affected by the command until the data object is saved.

Command steps:

  1. Optionally, preselect string data in a **3D** window.

  2. Run the command.

  3. If no string was selected before running the command, select individual (left or right-click) or drag a selection box over multiple strings.

  4. Click **Done** to complete the command.
