# erase-wireframe ("erw")

See this command in the [**command table**.](<COMMAND%20TABLE_E.md#erase-wireframe>)

To access this command:

  * **Explicit** ribbon **> > Edit >> Erase**.

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "erase-wireframe"

  * Use the quick key combination "erw".

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **erase-wireframe** and click **Run**.

## Command Overview

Erase currently selected wireframe data. This command requires wireframe data to be selected before running.

**Note** : This command supports [**flexible wireframe selection**](<../COMMON/Wireframe_Selection_Concept.md>).

Command steps:

  1. Select some or all triangles in the wireframe data object(s) you wish to modify.

  2. Run the command.

  3. The selected wireframe data are deleted.

  4. Click OK.

**Tip** : use the DELETE key to delete selected wireframe data, without running this command.
