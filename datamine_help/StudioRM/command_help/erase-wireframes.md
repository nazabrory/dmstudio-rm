# erase-wireframes ("eaw")

See this command in the [**command table**.](<COMMAND%20TABLE_E.md#erase-wireframes>)

To access this command:

  * **Home** ribbon **> > Unload >> All Wireframes**.

  * **Data** ribbon **> > Unload >> All Wireframes**.

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "erase-wireframes"

  * Use the quick key combination "eaw".

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **erase-wireframes** and click **Run**.

## Command Overview

Unload all wireframe data from memory.
