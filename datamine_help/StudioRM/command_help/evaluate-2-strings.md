# evaluate-2-strings ("ev2")

See this command in the [**command table**.](<COMMAND%20TABLE_E.md#evaluate-2-strings>)

To access this command:

  * **Report** ribbon **> > Evaluate >> Strings >> Two Strings**.

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "evaluate-2-strings"

  * Use the quick key combination "ev2".

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **evaluate-2-strings** and click **Run**.

## Command Overview

Evaluate two selected strings, against either a block model or drillholes, produced by linking the selected to create a wireframe.

Evaluation results can be saved to a new object using the Save Results option on the **[Evaluation Results](<../COMMON/EvaluationDynamicTableViewDialog.md>)** screen. Data is saved as a results object (visible in the Loaded Data and Project Data control bars) and can be saved to a physical file. Any existing evaluation results of the same name are overwritten.

This command displays the [Evaluation Settings](<../COMMON/EvaluationPropertiesDialog.md>) screen, followed by the [Evaluation Results](<../COMMON/EvaluationDynamicTableViewDialog.md>) screen.

**Note** : you can evaluate against either model or drillhole data. This is set using the **[Project Settings](<../COMMON/Project%20Settings_Mine%20Design.md>)** screen (**Mine Design >> Evaluation Control**).
