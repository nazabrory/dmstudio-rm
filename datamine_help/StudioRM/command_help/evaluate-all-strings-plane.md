# evaluate-all-strings-plane

See this command in the [**command table**.](<COMMAND%20TABLE_E.md#evaluate-all-strings-plane>)

To access this command:

  * **Report** ribbon **> > Evaluate >> Strings >> All Strings by Plane**.

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "evaluate-all-strings-plane"

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **evaluate-all-strings-plane** and click **Run**.

## Command Overview

Evaluate all visible string overlay objects, against either a block model or drillholes, by projecting them perpendicular to the mean plane formed from all loaded string data.

In all other ways, this command is the same as [evaluate-all-strings ("eva")](<evaluate-all-strings.md>).
