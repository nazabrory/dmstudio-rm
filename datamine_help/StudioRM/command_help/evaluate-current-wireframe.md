# evaluate-current-wireframe

See this command in the [**command table**.](<COMMAND%20TABLE_E.md#evaluate-current-wireframe>)

To access this command:

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "evaluate-current-wireframe"

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **evaluate-current-wireframe** and click **Run**.

## Command Overview

Evaluate the current wireframe, either using an enclosed volume or a volume above or below a DTM (optionally constrained to an elevation limit). Loaded wireframe data should be [verified](<../COMMON/Wireframe%20Verify%20Dialog.md>) beforehand.

This command is the same as [evaluate-wireframe](<evaluate-wireframe.md>) other than the **[current](<../COMMON/Concept_Current_Object.md>)** wireframe object is used, regardless of prior wireframe data selection.

This command displays the [Evaluate Wireframe](<../COMMON/Evaluate%20Wireframe%20Dialog.md>) screen followed by the [Evaluation Results](<../COMMON/EvaluationDynamicTableViewDialog.md>) screen.
