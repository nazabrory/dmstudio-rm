# evaluate-display-legend-switch

See this command in the [**command table**.](<COMMAND%20TABLE_E.md#evaluate-display-legend-switch>)

To access this command:

  * **Report** ribbon **> > Evaluate >> Display Legend**.

  * Display the [Project Settings: Mine Design](<../COMMON/Project%20Settings_Mine%20Design.md>) screen and toggle **Use Display Legend**.

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "evaluate-display-legend-switch"

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **evaluate-display-legend-switch** and click **Run**.

### Command Overview

Categorize evaluation results using the legend items in the current block model or drillhole display legend.

As a toggle command, this alternates your system between the following states when performing evaluations:

  * **Use the current display legend** evaluation results are categorized against the legend 'bins' associated with the display legend for the associated block model.
  * **Use the evaluation legend** regardless of the display legend in use, evaluations are completed according to the legend specified in your [Project Settings](<../COMMON/Project%20Settings_Mine%20Design.md>).
