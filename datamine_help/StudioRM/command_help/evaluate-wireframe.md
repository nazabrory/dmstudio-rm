# evaluate-wireframe ("evw")

See this command in the [**command table**.](<COMMAND%20TABLE_E.md#evaluate-wireframe>)

To access this command:

  * **Report** ribbon **> > Evaluate >> Wireframes >> Wireframe**.

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "evaluate-wireframe"

  * Use the quick key combination "evw".

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **evaluate-wireframe** and click **Run**.

## Command Overview

Evaluate the current wireframe, either using an enclosed volume or a volume above or below a DTM (optionally constrained to an elevation limit). Loaded wireframe data should be [verified](<../COMMON/Wireframe%20Verify%20Dialog.md>) beforehand.

This command is the same as [evaluate-current-wireframe](<evaluate-current-wireframe.md>) other than selected wireframe data is used to form an evaluation volume, regardless of which wireframe object is [current](<../COMMON/Concept_Current_Object.md>).

This command displays the [Evaluate Wireframe](<../COMMON/Evaluate%20Wireframe%20Dialog.md>) screen followed by the [Evaluation Results](<../COMMON/EvaluationDynamicTableViewDialog.md>) screen.

**Note** : when using this command, wireframe selection settings are temporarily disabled if they are set to any setting other than _select by object_. When this command completes, original wireframe selection settings are reinstated.
