# evaluation-control ("evp")

See this command in the [**command table**.](<COMMAND%20TABLE_E.md#evaluation-control>)

To access this command:

  * Use the quick key combination "evp".

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "evaluation-control"

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **evaluation-control** and click **Run**.

## Command Overview

Set controls for automatic evaluation during mining face advance.

You are prompted to enter the near and far projection distances that are applied to the single 'mined' perimeters during face advance, produced when using the [mine-face](<mine-face.md>) or [multiple-mine](<multiple-mine.md>) commands. If a results file is not currently open, you are prompted for one.
