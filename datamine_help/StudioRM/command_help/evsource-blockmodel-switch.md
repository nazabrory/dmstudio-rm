# evsource-blockmodel-switch ("umd")

See this command in the [**command table**.](<COMMAND%20TABLE_E.md#evsource-blockmodel-switch>)

To access this command:

  * **Report** ribbon **> > Evaluation >> Use Model** (toggle on).

  * Use the quick key combination "umd".

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "evsource-blockmodel-switch"

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **evsource-blockmodel-switch** and click **Run**.

## Command Overview

Turn on the use of the cell model in evaluations.

If turned on, subsequent evaluation commands will use the open cell model as the data source. The alternative is to evaluate from drillhole data using [evsource-drillhole-switch](<evsource-drillhole-switch.md>).

The commands affected by this switch are [evaluate-1-string](<evaluate-1-string.md>), [evaluate-2-strings](<evaluate-2-strings.md>), [evaluate-wireframe](<evaluate-wireframe.md>) and [drive-evaluation-switch](<drive-evaluation-switch.md>).
