# evsource-drillhole-switch ("udd")

See this command in the [**command table**.](<COMMAND%20TABLE_E.md#evsource-drillhole-switch>)

To access this command:

  * **Report** ribbon **> > Evaluation >> Use Model** (toggle off).

  * Use the quick key combination "udd".

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , evsource-drillhole-switch"

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **evsource-drillhole-switch** and click **Run**.

## Command Overview

Turn on the use of drillholes in evaluations.

## How to use

If [block model evaluation is turned on](<evsource-blockmodel-switch.md>), subsequent evaluation commands will use (filtered) drillholes as the data source.

The commands affected by this switch are [evaluate-1-string](<evaluate-1-string.md>), [evaluate-2-strings](<evaluate-2-strings.md>), [evaluate-wireframe](<evaluate-wireframe.md>) and [drive-evaluation-switch](<drive-evaluation-switch.md>).
