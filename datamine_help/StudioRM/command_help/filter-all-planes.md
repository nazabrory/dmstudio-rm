# filter-all-planes

See this command in the [**command table**.](<COMMAND%20TABLE_F.md#filter-all-planes>)

To access this command:

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "ilter-all-planes"

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **ilter-all-planes** and click **Run**.

## Command Overview

Filter all loaded plane data objects. See [Planes Data](<../COMMON/Studio%203%20Planes%20Overview.md>).

This filter is applied to all loaded objects and so will effect data:

  * not used in overlays

  * in visible overlays

  * in hidden overlays.

Command steps:

  1. Run the command.

  2. Using the [Expression Builder](<../COMMON/Expression%20Builder%20Dialog.md>) screen, define and check the filter Expression.

  3. Click OK.
