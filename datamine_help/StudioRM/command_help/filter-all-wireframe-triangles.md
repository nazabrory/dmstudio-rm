# filter-all-wireframe-triangles

See this command in the [**command table**.](<COMMAND%20TABLE_F.md#filter-all-wireframe-triangles>)

To access this command:

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "filter-all-wireframe-triangles"

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **filter-all-wireframe-triangles** and click **Run**.

## Command Overview

Filter all loaded wireframe data objects.

This filter is applied to all loaded objects and so will effect data:

  * not used in overlays

  * in visible overlays

  * in hidden overlays.

Command steps:

  1. Run the command.

  2. Using the [Expression Builder](<../COMMON/Expression%20Builder%20Dialog.md>) screen, define and check the filter Expression.

  3. Click OK.
