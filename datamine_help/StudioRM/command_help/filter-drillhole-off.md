# filter-drillhole-off

See this command in the [**command table**.](<COMMAND%20TABLE_F.md#filter-drillhole-off>)

To access this command:

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "filter-drillhole-off"

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **filter-drillhole-off** and click **Run**.

## Command Overview

Interactively hide drillholes individually in any 3D window.

Command steps:

  1. Run the command.

  2. Select one or more drillholes to be filtered.

Selected drillholes are temporarily hidden from view.

  3. Click Done to exit selection mode.

Tip: Use Erase All filters to reinstate the filtered holes.
