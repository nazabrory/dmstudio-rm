# filter-drillholes ("fdh")

See this command in the [**command table**.](<COMMAND%20TABLE_F.md#filter-drillholes>)

To access this command:

  * **Format** ribbon **> > Filter >> Drillholes**.

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "filter-drillholes"

  * Use the quick key combination "fdh".

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **filter-drillholes** and click **Run**.

## Command Overview

Filter all visible drillholes overlays.

This command makes it possible to display and work with different drillhole samples using a [filter expression](<../COMMON/logical%20expressions.md>). The filter is used whenever the drillholes samples are used.

Existing filters can be removed by selecting this command but not specifying any filters. Then use [redraw-display](<redraw-display.md>) to display all data.

Command steps:

  1. Run the command.

The [Expression Builder](<../COMMON/Expression%20Builder%20Dialog.md>) displays.

  2. Enter or construct a valid filter expression. See [Filtering Data](<../COMMON/Filtering_Data.md>).

  3. Click OK.
