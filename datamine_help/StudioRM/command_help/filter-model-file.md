# filter-model-file ("fm")

See this command in the [**command table**.](<COMMAND%20TABLE_F.md#filter-model-file>)

To access this command:

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "filter-model-file"

  * Use the quick key combination "fm".

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **filter-model-file** and click **Run**.

## Command Overview

Filter all visible block model overlays.

This command makes it possible to display and work with different parts of the block model using a [filter expression](<../COMMON/logical%20expressions.md>). The filter is used whenever the block model is used.

Existing filters can be removed by selecting this command but not specifying any filters. Then use the [redraw-display](<redraw-display.md>) command to display all the data.

A block model file must be open before filters may be applied. For more information on filtering data, click [here](<../COMMON/Filtering_Data.md>).

Command steps:

  1. Run the command.

The [Expression Builder](<../COMMON/Expression%20Builder%20Dialog.md>) displays.

  2. Enter or construct a valid filter expression. See [Filtering Data](<../COMMON/Filtering_Data.md>).

  3. Click OK.
