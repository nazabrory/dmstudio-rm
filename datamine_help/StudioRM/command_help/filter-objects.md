# filter-objects ("fob")

See this command in the [**command table**.](<COMMAND%20TABLE_F.md#filter-objects>)

To access this command:

  * **Format** ribbon **> > Filter >> All**.

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "filter-objects"

  * Use the quick key combination "fob".

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **filter-objects** and click **Run**.

## Command Overview

Apply a filter expression to all overlay objects, regardless of type.

This command makes it possible to selectively display and work with overlay objects sharing a common attribute which can be used to filter across all objects such as zone field called **ZONE** present in strings, drillholes and block model objects.

See [Logical Expressions](<../COMMON/logical%20expressions.md>).

Command steps:

  1. Run the command.

The [Expression Builder](<../COMMON/Expression%20Builder%20Dialog.md>) displays.

  2. Enter or construct a valid filter expression. See [Filtering Data](<../COMMON/Filtering_Data.md>).

  3. Click OK.
