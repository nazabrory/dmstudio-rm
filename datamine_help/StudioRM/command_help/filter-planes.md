# filter-planes ("fln")

See this command in the [**command table**.](<COMMAND%20TABLE_F.md#filter-planes>)

To access this command:

  * **Format** ribbon **> > Filter >> Planes**.

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "filter-planes"

  * Use the quick key combination "fln".

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **filter-planes** and click **Run**.

## Command Overview

Filter all visible planes overlays.

This command makes it possible to selectively display and work with planes overlay objects using a filter expression. Coordinate data may be used in the expressions as well as attribute fields. The standard coordinate field names for planes data are **XPT** ,**YPT** and **ZPT**.

For more information on filtering data, click [here](<../COMMON/Filtering_Data.md>).

Command steps:

  1. Run the command.

The [Expression Builder](<../COMMON/Expression%20Builder%20Dialog.md>) displays.

  2. Enter or construct a valid filter expression. See [Filtering Data](<../COMMON/Filtering_Data.md>).

  3. Click OK.
