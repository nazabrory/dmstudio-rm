# filter-point-off ("nfp")

See this command in the [**command table**.](<COMMAND%20TABLE_F.md#filter-point-off>)

To access this command:

  * **Format** ribbon **> > Filter >> Hide Selected**.

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "filter-point-off"

  * Use the quick key combination "nfp".

  * In the **[Find Command](<../COMMON/findcommand.md>)** dialog, locate **filter-point-off** and click **Run**.

## Command Overview

Prevent the display of point data without removing it from memory.

Command steps:

  1. Run the command.

If point data is already selected, it is hidden from view. If no point data is selected, you are prompted to select a string to be rendered invisible.

  2. If you wish, select more strings to hide.

**Tip** : Data can be re-displayed by using the **[filter-points](<filter-points.md>)** (quick key "fp") command and entering an empty filter expression, or simply running the **[erase-all-filters](<erase-all-filters.md>)** command.
