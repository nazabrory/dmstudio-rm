# filter-points ("fp")

See this command in the [**command table**.](<COMMAND%20TABLE_F.md#filter-points>)

To access this command:

  * **Format** ribbon **> > Filter >> Points**.

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "filter-points"

  * Use the quick key combination "fp".

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **filter-points** and click **Run**.

## Command Overview

Filter all visible points overlays.

This command makes it possible to selectively display and work with points using filter expressions. As soon as the command is invoked any previously existing filters are deleted. Coordinate data may be used in the expressions as well as attribute fields. The standard coordinate field names for point data are **XPT** ,**YPT** and **ZPT**.

Command steps:

  1. Run the command.

The [Expression Builder](<../COMMON/Expression%20Builder%20Dialog.md>) displays.

  2. Enter or construct a valid filter expression. See [Filtering Data](<../COMMON/Filtering_Data.md>).

  3. Click OK.
