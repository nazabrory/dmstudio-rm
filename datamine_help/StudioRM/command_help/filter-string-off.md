# filter-string-off ("nfs")

See this command in the [**command table**.](<COMMAND%20TABLE_F.md#filter-string-off>)

To access this command:

  * **Format** ribbon **> > Filter >> Hide Selected**.

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "filter-string-off"

  * Use the quick key combination "nfs".

  * In the **[Find Command](<../COMMON/findcommand.md>)** dialog, locate **filter-string-off** and click **Run**.

## Command Overview

Prevent the display of a string without removing it from memory.

Command steps:

  1. Run the command.

If a string is already selected, it is hidden from view. If no string is selected, you are prompted to select a string to be rendered invisible.

  2. If you wish, select more strings to hide.

**Tip** : Data can be re-displayed by using the **[filter-strings](<filter-strings.md>)** (quick key "fs") command and entering an empty filter expression, or simply running the **[erase-all-filters](<erase-all-filters.md>)** command.
