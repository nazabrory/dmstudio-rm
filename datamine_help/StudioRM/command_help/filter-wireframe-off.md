# filter-wireframe-off ("nfw")

See this command in the [**command table**.](<COMMAND%20TABLE_F.md#filter-wireframe-off>)

To access this command:

  * **Format** ribbon **> > Filter >> Hide Selected**.

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "filter-wireframe-off"

  * Use the quick key combination "nfw".

  * In the **[Find Command](<../COMMON/findcommand.md>)** dialog, locate **filter-wireframe-off** and click **Run**.

## Command Overview

Hide selected wireframe data without removing it from memory. If no wireframe data is selected when the command is run, you are asked to select a wireframe face. In this way, faces can be successively removed.

Command steps:

  1. Run the command.

If wireframe data is already selected, it is hidden from view. If no data is selected, you are prompted to select a face to be rendered invisible.

  2. If you wish, select more data to hide.

**Tip** : Data can be re-displayed by using the **[filter-wireframes](<filter-strings.md>)** (quick key "fww") command and entering an empty filter expression, or simply running the **[erase-all-filters](<erase-all-filters.md>)** command.
