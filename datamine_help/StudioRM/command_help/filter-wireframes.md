# filter-wireframes ("fww")

See this command in the [**command table**.](<COMMAND%20TABLE_F.md#filter-wireframes>)

To access this command:

  * **Format** ribbon **> > Filter >> Wireframes**.

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "filter-wireframes"

  * Use the quick key combination "fw".

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **filter-wireframes** and click **Run**.

## Command Overview

Filter all visible wireframe overlays.

This command makes it possible to selectively display and work with different wireframe triangles from the wireframes using a filter expression. The expression must be based on attributes which are associated with the wireframe triangles such as **TRIANGLE** , **GROUP** , **SURFACE** , **LINK** and so on.

Existing filters can be removed by selecting this command but not specifying any filters. After filters have been defined use the [redraw-display](<redraw-display.md>) command to display the selected data.

For more information on filtering data, click [here](<../COMMON/Filtering_Data.md>).

Command steps:

  1. Run the command.

The [Expression Builder](<../COMMON/Expression%20Builder%20Dialog.md>) displays.

  2. Enter or construct a valid filter expression. See [Filtering Data](<../COMMON/Filtering_Data.md>).

  3. Click OK.
