# generate-contours-from-holes-intercepts

See this command in the [**command table**.](<COMMAND%20TABLE_G.md#generate-contours-from-holes-intercepts>)

To access this command:

  * **Implicit** ribbon >> **Contour** >> **Attribute**.

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **generate-contours-from-holes-intercepts** and click **Run**.

  * Enter "generate-contours-from-holes-intercepts" into the [**Command Line.**](<../COMMON/Command_Toolbar.md>)

## Command Overview

Generate contours from drillhole intercepts. Wireframes can also be generated.

Displays the **[Generate Controls from Intercepts](<../COMMON/GenerateContourDHInterceptsDataPage.md>)** screen.
