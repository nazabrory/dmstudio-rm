# generate-contours-from-points

See this command in the [**command table**.](<COMMAND%20TABLE_G.md#generate-contours-from-points>)

To access this command:

  * **Implicit** ribbon >> **Contour** >> **Points**.

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **generate-contours-from-points** and click **Run**.

  * Enter "generate-contours-from-points" into the [**Command Line.**](<../COMMON/Command_Toolbar.md>)

## Command Overview

Generate contours from loaded point data. 

Displays the **[Generate Controls from Points](<../COMMON/GenerateContourDataPage.md>)** screen.
