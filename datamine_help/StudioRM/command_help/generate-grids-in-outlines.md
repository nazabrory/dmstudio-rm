# generate-grids-in-outlines ("gio")

See this command in the [**command table**.](<COMMAND%20TABLE_G.md#generate-grids-in-outlines>)

To access this command:

  * **Edit** ribbon **> > Shapes >> Outlines >> Generate Grids in Outlines**.
  * **Digitize** ribbon **> > Create >> Outlines >> Generate Grids in Outlines**.

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "generate-grids-in-outlines"

  * Use the quick key combination "gio".

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **generate-grids-in-outlines** and click **Run**.

## Command Overview

Generate a regular grid of strings within selected outline string data using the [Generate Grids in Outlines](<../COMMON/Generate_Grids_in_Outlines.md>) screen.
