# generate-outlines-attrib

See this command in the [**command table**.](<COMMAND%20TABLE_G.md#generate-outlines-attrib>)

To access this command:

  * **Edit** ribbon **> > Shapes >> Outlines >> Generate outlines without attributes**.

  * **Digitize** ribbon **> > Create >> Outlines >> Generate outlines without attributes**.

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "generate-outlines-attrib"

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **generate-outlines-attrib** and click **Run**.

## Command Overview

Similar to the **[generate-outlines](<generate-outlines.md>)** command (used to replace crossing strings with perimeters outlining adjacent areas), but uses the first string used in the outlining process to derive attributes for the remaining strings.
