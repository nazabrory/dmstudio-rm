# georeference-objects ("gro")

See this command in the [**command table**.](<COMMAND%20TABLE_G.md#georeference-objects>)

To access this command:

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "georeference-objects"

  * Use the quick key combination "gro".

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **georeference-objects** and click **Run**.

## Command Overview

Transform the 3D location of any loaded 3D object using one or more pairs of current-georeferenced points, using the [Transform Objects](<../COMMON/GeoreferenceObjectsDialog.md>) screen.
