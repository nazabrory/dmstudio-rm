# get-all-strings ("ga")

See this command in the [**command table**.](<COMMAND%20TABLE_G.md#get-all-strings>)

To access this command:

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "get-all-strings"

  * Use the quick key combination "ga".

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **get-all-strings** and click **Run**.

## Command Overview

Load all (filtered) strings from a named string file.

When this command is invoked you are prompted for a file name. The default file name is the current string file. If you enter a different name this new file becomes the current string file. 

The file entered must be a string file and must exist. Any strings already loaded into memory will not be lost. Use the [erase-all](<erase-all.md>), first if you wish. 

Only strings passing any current string filter are loaded from the file.
