# get-points ("gp")

See this command in the [**command table**.](<COMMAND%20TABLE_G.md#get-points>)

To access this command:

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "get-points"

  * Use the quick key combination "gp".

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **get-points** and click **Run**.

## Command Overview

Get points into memory from a named points file.

Load point data into memory from a named database file. The file must contain the fields **XPT** , **YPT** and **ZPT**. If the file also contains the fields **LSTYLE** , **COLOUR** and **SYMBOL** these are used, otherwise default values are used. The file may also contain any additional attribute fields.

You will have the opportunity to enter and filter expression to control the point data that is loaded from the file.
