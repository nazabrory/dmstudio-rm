# gradient-degrees-switch

See this command in the [**command table**.](<COMMAND%20TABLE_G.md#gradient-degrees-switch>)

To access this command:

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "gradient-degrees-switch"

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **gradient-degrees-switch** and click **Run**.

## Command Overview

Sets the gradient convention unit to degrees. This setting is used by some string creation commands.

Tip: This switch can also be set using the [set-gradient-convention](<set-gradient-convention.md>) command.

Command steps:

  1. Run the command.

  2. Run the required design command, for example, [create-ramp-string](<create-ramp-string.md>).

Other design commands which make use of gradient conventions include [create-ramp-string](<create-ramp-string.md>) and [adjust-to-gradient](<adjust-to-gradient.md>).

  3. If required, run one of the other gradient convention switch commands to reset the setting.
