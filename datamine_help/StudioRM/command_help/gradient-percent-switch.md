# gradient-percent-switch

See this command in the [**command table**.](<COMMAND%20TABLE_G.md#gradient-percent-switch>)

To access this command:

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "gradient-percent-switch"

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **gradient-percent-switch** and click **Run**.

## Command Overview

Set the gradient convention unit to percent.

This switch can also be set using the [set-gradient-convention](<set-gradient-convention.md>) command.

Note: Some of the commands which make use of gradient conventions include [create-ramp-string](<create-ramp-string.md>) and [adjust-to-gradient](<adjust-to-gradient.md>).

Command steps:

  1. Run the command.

  2. Run the required design command, for example. [create-ramp-string](<create-ramp-string.md>).

  3. If required, run one of the other gradient convention switch commands to reset the setting.
