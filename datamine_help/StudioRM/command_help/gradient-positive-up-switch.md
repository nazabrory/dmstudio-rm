# gradient-positive-up-switch

See this command in the [**command table**.](<COMMAND%20TABLE_G.md#gradient-positive-up-switch>)

To access this command:

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "gradient-positive-up-switch"

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **gradient-positive-up-switch** and click **Run**.

## Command Overview

Toggle the gradient positive value convention.

This switch has two possible settings:

  * ON \- +ve dip values are treated as up; -ve values as down.

  * OFF \- +ve dip values are treated as down; -ve values as up.

This switch can also be toggled using the [set-gradient-convention](<set-gradient-convention.md>) command.

Note: Some of the commands which make use of gradient conventions include [create-ramp-string](<create-ramp-string.md>) and [adjust-to-gradient](<adjust-to-gradient.md>).

Command steps:

  1. Run the command.

  2. Run the required design command, for example [create-ramp-string](<create-ramp-string.md>).

  3. If required, run one of the other gradient convention switch commands to reset the setting.
