# grid-interval-on-string ("ipg")

See this command in the [**command table**.](<COMMAND%20TABLE_G.md#grid-interval-on-string>)

To access this command:

  * **Digitize** ribbon **> > Edit >> Insert Points >> Insert Using Grid**.

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "insert-point-segment-center"

  * Use the quick key combination "ipsc".

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **insert-point-segment-center** and click **Run**.

## Command Overview

Make new string vertices, based on XYZ grid intersections. Displays the .[Insert Points from Grid](<../COMMON/insert%20points%20from%20grid%20dialog.md>) screen.
