# group-lithology

See this command in the [command table](<COMMAND%20TABLE_G.md#group-lithology>).

To access this command:

  * **Implicit** ribbon **> > Domain >> Group**.
  * Enter "group-lithology" into the **Command** toolbar and press <ENTER>

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **group-lithology** and click **Run**.

## Command Overview

This command is used to define groups containing one or more lithology values. This command displays the [Group Lithology](<../COMMON/Define_Sample_Lithologies.md>) screen.
