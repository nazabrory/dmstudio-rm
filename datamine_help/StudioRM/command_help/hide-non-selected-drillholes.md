# hide-non-selected-drillholes

See this command in the [command table](<COMMAND%20TABLE_H.md#hide-non-selected-drillholes>).

To access this command:

  * **Format** ribbon **> > Filter >> Holes >> Show Selected**.

  * Enter "hide-non-selected-drillholes" into the **Command** toolbar and press <ENTER>

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **hide-non-selected-drillholes** and click **Run**.

## Command Overview

Filters drillhole data in the 3D window to show only selected items. All non-selected drillholes are hidden.

Reinstate filtered drillhole data using [filter-drillholes](<filter-drillholes.md>) or [erase-all-filters](<erase-all-filters.md>).
