# hide-non-selected-points("hnsp")

See this command in the [command table](<COMMAND%20TABLE_H.md#hide-non-selected-points>).

To access this command:

  * **Format** ribbon **> > Filter >> Show Selected**.

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "hide-non-selected-points"

  * Use the quick key combination "hnsp".

  * In the **[Find Command](<../COMMON/findcommand.md>)** dialog, locate **hide-non-selected-points** and click **Run**.

## Command Overview

Disables the display of all non-selected point data.

Command steps:

  * Run the command.

Non-selected point data is hidden from view until a suitable filter command is run to counteract its effect, e.g. by running the **[filter-points](<filter-points.md>)** command and specifying an empty filter expression, or running **[erase-all-filters](<erase-all-filters.md>)**.
