# hide-non-selected-strings ("hnss")

See this command in the [command table](<COMMAND%20TABLE_H.md#hide-non-selected-strings>).

To access this command:

  * **Format** ribbon **> > Filter >> Show Selected**.

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "hide-non-selected-strings"

  * Use the quick key combination "hnss".

  * In the **[Find Command](<../COMMON/findcommand.md>)** dialog, locate **hide-non-selected-strings** and click **Run**.

## Command Overview

Disables the display of all non-selected string data.

Command steps:

  * Run the command.

Non-selected string data is hidden from view until a suitable filter command is run to counteract its effect, e.g. by running the **[filter-strings](<filter-strings.md>)** command and specifying an empty filter expression or running **[erase-all-filters](<erase-all-filters.md>)**.
