# hole-report

See this command in the [command table](<COMMAND%20TABLE_H.md#hole-report>).

To access this command:

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "hole-report"

  * In the **[Find Command](<../COMMON/findcommand.md>)** dialog, locate **hole-report** and click **Run**.

## Command Overview

Creates a hole summary of any dynamic drillholes currently loaded.

Output is shown in the [Desurvey Report](<../COMMON/Desurvey%20Report%20Control%20Bar%20Overview.md>) control bar.
