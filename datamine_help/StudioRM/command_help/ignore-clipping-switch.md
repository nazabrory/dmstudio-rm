# ignore-clipping-switch ("tcl")

See this command in the [**command table**.](<COMMAND%20TABLE_I.md#ignore-clipping-switch>)

To access this command:

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "ignore-clipping-switch"

  * Use the quick key combination "tcl".

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **ignore-clipping-switch** and click **Run**.

## Command Overview

Choose whether clipping is applied in the currently active 3D window, regardless of current clip settings. 

This setting can be applied separately for all windows, including embedded and external linked 3D windows as well as independent 3D windows.
