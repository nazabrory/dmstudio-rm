# insert-at-intersections ("ii")

See this command in the [**command table**.](<COMMAND%20TABLE_I.md#insert-at-intersections>)

To access this command:

  * **Digitize** ribbon **> > Condition >> Condition >> Insert at Intersections**.

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "insert-at-intersections"

  * Use the quick key combination "ii".

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **insert-at-intersections** and click **Run**.

## Command Overview

Inserts string vertices at intersections with other strings.

Data can either be selected before running the command, or interactively during the command session. 

Command steps:

  1. Run the command.

  2. In any 3D window, select a string. 

New vertices are added to the string at the points of intersection with other strings, along the line of sight.
