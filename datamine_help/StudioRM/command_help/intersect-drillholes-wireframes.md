# intersect-drillholes-wireframes ("idw")

See this command in the [**command table**.](<COMMAND%20TABLE_I.md#intersect-drillholes-wireframes>)

To access this command:

  * **Sample Analysis** ribbon **> > Edit Drillholes >> Intersect**.

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "intersect-drillholes-wireframes"

  * Use the quick key combination "idw".

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **intersect-drillholes-wireframes** and click **Run**.

## Command Overview

Calculate the points at which selected drillholes intersect specific wireframe surfaces, and write those intersections as a new data points, along with a series of optional attributes (including wireframe thickness).

Data can be selected whilst the [Intersect Drillholes with Wireframes](<../COMMON/IntersectDrillholesWireframesDialog.md>) screen is displayed.

See [Intersect Drillholes with Wireframes](<../COMMON/IntersectDrillholesWireframesDialog.md>).
