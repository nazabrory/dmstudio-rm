# keep-combined-switch ("ko")

See this command in the [**command table**.](<COMMAND%20TABLE_K.md#keep-combined-switch>)

To access this command:

  * **Digitize** ribbon >> **Tools >> Combine >> Keep Originals**.

  * **Digitize** ribbon **> > Tools >> Combine >> Keep Originals**.
  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "keep-combined-switch".

  * Use the quick key combination "ko".

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **keep-combined-switch** and click **Run**.

## Command Overview

Toggles whether strings used for combining are retained when using the **[combine-strings](<combine-strings.md>)** or **[combine-strings-attrib](<combine-strings-attrib.md>)** commands. No other commands are affected by this setting.

  * If this toggle is ON, strings used in combining are retained. 

  * If it is OFF, they are erased.
