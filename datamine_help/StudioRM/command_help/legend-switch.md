# legend-switch ("sleg")

See this command in the [**command table**.](<COMMAND%20TABLE_L.md#legend-switch>)

To access this command:

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "legend-switch".

  * Use the quick key combination "sleg".

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **legend-switch** and click **Run**.

Toggle on or off the display of the legend preview for the currently selected 3D window overlay.

Note: On any screen where legend controls are displayed, you can click **Display Legend** to display the legend key used in the current 3D window for the target overlay. You can also use the **Sheets** and **Project Data** control bar context menu (**Display Legend**) to achieve the same. See[Legend Controls](<../VR_Help/Legend-Pallete.md>)
