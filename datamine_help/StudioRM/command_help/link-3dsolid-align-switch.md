# link-3dsolid-align-switch

See this command in the [**command table**.](<COMMAND%20TABLE_L.md#link-3dsolid-align-switch>)

To access this command:

  * On the [Project Settings: Wireframe Linking](<../COMMON/Project%20Settings_%20Wireframe%20Linking.md>) screen, toggle the **Align Strings** setting.

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "link-3dsolid-align-switch"

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **link-3dsolid-align-switch** and click **Run**.

## Command Overview

Toggle the automatic creation virtual tag lines, which control how the strings are linked together when the [link-multiple-strings](<link-multiple-strings.md>) command is used.

**Note** : the [link-3dsolid-method-switch](<link-3dsolid-method-switch.md>) for this command to be effective.
