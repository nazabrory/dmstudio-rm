# link-3dsolid-collate-switch

See this command in the [**command table**.](<COMMAND%20TABLE_L.md#link-3dsolid-collate-switch>)

To access this command:

  * On the [Project Settings: Wireframe Linking](<../COMMON/Project%20Settings_%20Wireframe%20Linking.md>) screen, toggle the **Collate by Distance** setting.

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "link-3dsolid-collate-switch"

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **link-3dsolid-collate-switch** and click **Run**.

## Description

Toggle on or off the grouping of string data according to the Collate by Distance setting ([Wireframe Linking Settings](<../COMMON/Project%20Settings_%20Wireframe%20Linking.md>) screen).

This distance between strings is specified in order for them to be considered as being on the same level. 

**Note** : the [link-3dsolid-method-switch](<link-3dsolid-method-switch.md>) for this command to be effective.
