# link-3dsolid-method-switch ("l3dm")

See this command in the [**command table**.](<COMMAND%20TABLE_L.md#link-3dsolid-method-switch>)

To access this command:

  * On the [Project Settings: Wireframe Linking](<../COMMON/Project%20Settings_%20Wireframe%20Linking.md>) screen, select the **3D Solid** setting.

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "link-3dsolid-method-switch"

  * Use the quick key combination "l3dm".

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **link-3dsolid-method-switch** and click **Run**.

## Command Overview

[Wireframe Linking Settings](<../COMMON/Project%20Settings_%20Wireframe%20Linking.md>) provide access to a **3D Solid** linking method. 

The **link-3dsolid-method-switch** command enables this option.

This method uses a proprietary linking algorithm that is particularly effective in the creation of bifurcated wireframes without the need for additional tag (guide) strings (other than, optionally, virtual ones), using the [link-multiple-strings](<link-multiple-strings.md>) command - which can, in many cases, resolve the infamous [trouser leg](<link-boundary.md>) problem inherent to the design of underground development CAD data.

**Note** : if **the 3D Solid** linking method is enabled, the **Optimal Linking** string linking option is not available.
