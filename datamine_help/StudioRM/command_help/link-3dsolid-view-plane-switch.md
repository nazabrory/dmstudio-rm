# link-3dsolid-view-plane-switch

See this command in the [**command table**.](<COMMAND%20TABLE_L.md#link-3dsolid-view-plane-switch>)

To access this command:

  * On the [Project Settings: Wireframe Linking](<../COMMON/Project%20Settings_%20Wireframe%20Linking.md>) screen, toggle the **Use current view plane** setting.

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "link-3dsolid-view-plane-switch"

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **link-3dsolid-view-plane-switch** and click **Run**.

## Command Overview

Enable the sorting of strings to be linked to create the resulting wireframe volume.

This ordering is based on the distance of each string in relation from the virtual viewpoint (i.e. the 'eye depth' of each string). Note that this is a sort that can be performed as well as the default string sorting carried out as part of the [3D Solid linking](<../COMMON/Project%20Settings_%20Wireframe%20Linking.md>) process.

This command is also available as a toggle in the [Wireframe Linking Settings](<../COMMON/Project%20Settings_%20Wireframe%20Linking.md>) panel.
