# link-area-method-switch ("tma")

See this command in the [**command table**.](<COMMAND%20TABLE_L.md#link-area-method-switch>)

To access this command:

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "link-area-method-switch"

  * Use the quick key combination "tma".

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **link-area-method-switch** and click **Run**.

## Command Overview

Select minimum surface area as the string linking criteria.

If run, the resulting triangulation between selected strings will have the minimum wireframe surface area.
