# link-multiple-strings ("lms")

See this command in the [**command table**.](<COMMAND%20TABLE_L.md#link-multiple-strings>)

To access this command:

  * Explicit ribbon **> >**Create >> Link Multiple >> Auto Link Multiple
  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "link-multiple-strings"

  * Use the quick key combination "lms".

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **link-multiple-strings** and click **Run**.

## Command Overview

This command is used to create wireframe volumes by linking a sequence of perimeters (closed strings) in order (e.g. in the development of underground works), and can be used in many cases as an automatic method for creating bifurcated wireframes that, in the past, have given rise to a more complex procedure involving the design of tag (guide) strings (see [trouser leg problem](<link-boundary.md>)). 

Note: This command uses the **Maximum Segment Length** value (if greater than zero), as specified in **[Wireframe Linking Settings](<../COMMON/Project%20Settings_%20Wireframe%20Linking.md>)** to limit the generated segment length of generated wireframe triangles.

Displays the [Link Multiple Strings](<../COMMON/3D%20Solid%20Dialog.md>) screen. Follow this link for more details and an example.
