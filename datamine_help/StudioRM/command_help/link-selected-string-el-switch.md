# link-selected-string-el-switch

See this command in the [**command table**.](<COMMAND%20TABLE_L.md#link-selected-string-el-switch>)

To access this command:

  * **Explicit** ribbon >> Create>> Link Multiple >> End Link when Multiple Linking.

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "link-selected-strings-el-switch"

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **link-selected-strings-el-switch** and click **Run**.

## Command Overview

Toggle the ability to end link selected strings when using the following commands:

  * [link-selected-strings-attrib ("lma")](<link-selected-strings-attrib.md>)

  * [link-selected-strings-plane ("lmpl")](<link-selected-strings-plane.md>)

  * [link-selected-strings-point ("lmpp")](<link-selected-strings-point.md>)

  * [link-multiple-strings ("lms")](<link-multiple-strings.md>)
