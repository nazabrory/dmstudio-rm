# link-selected-strings-attrib ("lma")

See this command in the [**command table**.](<COMMAND%20TABLE_L.md#link-selected-strings-attrib>)

To access this command:

  * Explicit ribbon **> >**Create >> Link Multiple >> Link Multiple by Attribute
  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "link-selected-strings-attrib"

  * Use the quick key combination "lma".

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **link-selected-strings-attrib** and click **Run**.

## Command Overview

Links selected strings to form a solid wireframe, after sorting on a selected numeric attribute which determines the linking order. A number of strings (specified) are grouped together.

Note: This command uses the **Maximum Segment Length** value (if greater than zero), as specified in **[Wireframe Linking Settings](<../COMMON/Project%20Settings_%20Wireframe%20Linking.md>)** to limit the generated segment length of generated wireframe triangles.

## How to Use

  1. Run the command.

  2. Enter a numeric attribute to define the linking sequence. If fewer than two strings have been selected you are prompted to select more.

  3. Click **OK**.

  4. Define the number of strings to group.

  5. Click OK.
