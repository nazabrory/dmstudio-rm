# lock-bhole-plane-switch ("lh")

See this command in the [**command table**.](<COMMAND%20TABLE_L.md#lock-bhole-plane-switch>)

To access this command:

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "lock-bhole-plane-switch"

  * Use the quick key combination "lh".

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **lock-bhole-plane-switch** and click **Run**.

## Command Overview

When moving blast holes, this command controls whether they are maintained on the original plane.

If turned on the when blast holes are moved using the move-blasthole or move-blasthole-row commands the the original plane is maintained.

For example, suppose you have a set of holes with collar locations at an elevation of 10 and this command is ON. If you are in a plan view and you move a hole, the collar location is maintained at an elevation of 10 regardless of the elevation of the current view plane.
