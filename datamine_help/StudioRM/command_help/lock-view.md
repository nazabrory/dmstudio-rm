# lock-view ("lvw")

See this command in the [**command table**.](<COMMAND%20TABLE_L.md#lock-view>)

To access this command:

  * **3D View** ribbon **> > View >> Lock**.

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "lock-view"

  * Use the quick key combination "lvw".

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **lock-view** and click **Run**.

## Command Overview

Toggle on and off a lock for the currently active window (3D, Task, Map, depending on your product or function in use). Locking a view prevents interactive view rotation and scaling, although you can still zoom and pan.

Subsequent uses of the command alternate the locked and unlocked state of the view.
