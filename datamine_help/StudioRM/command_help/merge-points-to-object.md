# merge-points-to-object ("mpto")

See this command in the [**command table**.](<COMMAND%20TABLE_M.md#merge-points-to-object>)

To access this command:

  * **Home** ribbon **> > Edit >> More >> Merge Points to Object**.

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "merge-points-to-object".

  * Use the quick key combination "mpto".

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **merge-points-to-object** and click **Run**.

## Command Overview

Merge all selected points into the **[current](<../COMMON/Concept_Current_Object.md>)** points object.

Command steps:

  1. Using the [Current Objects](<../COMMON/Current_Objects_Toolbar.md>) toolbar, create a new points object to contain the merged data, and make it the current points object.

  2. In any 3D window, select the points that need to be merged.

**Note** : The selected points can come from points string objects.

  3. Run the command.

Data from the original target points is merged (with attributes) to the current object.
