# merge-string-segments-attrib

See this command in the [**command table**.](<COMMAND%20TABLE_M.md#merge-string-segments-attrib>)

To access this command:

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "merge-string-segments-attrib"

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **merge-string-segments-attrib** and click **Run**.

## Command Overview

This command is the same as [merge-string-segments](<merge-string-segments.md>) other than the original string attributes are copied to the new string data.
