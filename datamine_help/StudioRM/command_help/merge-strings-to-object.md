# merge-strings-to-object ("msto")

See this command in the [**command table**.](<COMMAND%20TABLE_M.md#merge-strings-to-object>)

To access this command:

  * **Home** ribbon **> > Edit >> More >> Merge Strings to Object**.

  * **Digitize** ribbon **> > Tools >> Merge to Object**.

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "merge-strings-to-object".

  * Use the quick key combination "msto".

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **merge-strings-to-object** and click **Run**.

## Command Overview

Merge all selected strings into the **[current](<../COMMON/Concept_Current_Object.md>)** strings object.

Command steps:

  1. Using the [Current Objects](<../COMMON/Current_Objects_Toolbar.md>) toolbar, create a new strings object to contain the merged data, and make it the current strings object.

  2. In any 3D window, select the strings that need to be merged.

**Note** : The selected strings can come from multiple string objects.

  3. Run the command.

Data from the original target strings is merged (with attributes) into the merged object.
