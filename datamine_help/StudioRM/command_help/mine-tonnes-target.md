# mine-tonnes-target ("mtt")

See this command in the [**command table**.](<COMMAND%20TABLE_M.md#mine-tonnes-target>)

To access this command:

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "mine-tonnes-target".

  * Use the quick key combination "mtt".

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **mine-tonnes-target** and click **Run**.

## Command Overview

Mine a face to achieve a specified target tonnage. 

After selecting an active face, you are prompted for a target tonnage, and a percentage tolerance.

If there is sufficient tonnage available within the mine unit, the face is projected forwards and the tonnage evaluated. The face position is changed in a number of logical iterations, with the mined tonnage converging towards the required target.

When a mined tonnage has been achieved within the specified tolerance, new mined and unmined perimeters are generated.
