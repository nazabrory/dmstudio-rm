# mineable-shape-optimizer-v3 ("mso")

See this command in the [**command table**.](<COMMAND%20TABLE_M.md#mineable-shape-optimizer-v3>)

To access this command:

  * **Report** ribbon **> > Model Reserves >> Mineable Shapes**.

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "mineable-shape-optimizer-v3".

  * Use the quick key combination "mso".

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **mineable-shape-optimizer-v3** and click **Run**.

## Command Overview

Launch [Mineable Shape Optimizer](<../MSO/MSOv3_default.md>).

Note: To run MSO, you will need a standalone license that is separate to your core application license. If you are interested in purchasing MSO, please contact your local Datamine office.
