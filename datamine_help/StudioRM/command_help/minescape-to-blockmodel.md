# minescape-to-blockmodel ("msk")

See this command in the [**command table**.](<COMMAND%20TABLE_M.md#minescape-to-model>)

To access this command:

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "minescape-to-blockmodel"

  * Use the quick key combination "msk".

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **minescape-to-blockmodel** and click **Run**.

## Command Overview

Import one or more MineScape Stratmodels and convert them to strata block models in Studio. Models selected for import are combined into a single strata model.

The individual models should be created from the same prototype model and have the same sub-cell grid. 

Command Steps:

  1. Run the command.

The **[Minescape Block Model Generator](<../COMMON/MinescapeBlockModelGen.md>)** displays.

  2. Fill out the relevant fields and click **Create Block Model**.
