# mirror-string ("smm")

See this command in the [**command table**.](<COMMAND%20TABLE_M.md#mirror-string>)

To access this command:

  * **Digitize** ribbon **> > Transform >> Rotate >> Mirror String**. 

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "mirror-string"

  * Use the quick key combination "smm".
  * On the **[Find Command](<../COMMON/findcommand.md>)** screen, highlight **mirror-string** and click **Run**.

## Command Overview

Draws a mirror image of a string or multiple strings, reflected about a defined mirror plane. The attributes of the original string data are transferred to the mirrored string data.

Command steps (string selection during command):

  1. Run the command, and follow the prompts in the Status Bar.

  2. Select the string to be mirrored.

  3. Select the start point for the mirror plane.

  4. Select the end point for the mirror plane - the selected string is mirrored about the defined plane.

**Tip** : Multiple strings can be mirrored using this command by preselecting them in a 3D window, then running the mirror-string command.
