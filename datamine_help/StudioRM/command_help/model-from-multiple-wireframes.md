# model-from-multiple-wireframes ("mfw")

See this command in the [command table](<COMMAND%20TABLE_M.md#model-from-multiple-wireframes>).

To access this command:

  * **Model** ribbon **> > Create >> Fill Wireframes**.

  * Run the command "model-from-multiple-wireframes"
  * Use the quick key combination "mfw"
  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **model-from-multiple-wireframes** and click **Run**.

## Command Overview

Generate a block model from multiple wireframes using optional control perimeters files and control parameters.

Command steps:

  1. Run the command.

The [Volumetric Block Modelling](<../COMMON/VolumetricBlockModeling_Dialog.md>) screen displays.

  2. Select or define a new model prototype.

  3. Optionally select a control perimeter file, for restricting block model extents.

  4. Select one or more boundary wireframes and set their associated parameters.

  5. Define subcell settings.

  6. Define General options and Output parameters.

  7. Save the defined parameters to a settings control file.

  8. Click Apply to generate the block model.
