# model-plane-change ("tbm")

See this command in the [command table](<COMMAND%20TABLE_M.md#model-plane-change>).

To access this command:

  * **Format** ribbon **> > Show/Hide >> Models**.

  * Run the command "model-plane-change"
  * Use the quick key combination "tbm"
  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **model-plane-change** and click **Run**.

## Command Overview

Toggle on or off the display of all block model data.
