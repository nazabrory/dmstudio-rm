#  model-plane-hide

See this command in the [command table](<COMMAND%20TABLE_M.md#model-plane-hide>).

To access this command:

  * Run the command "model-plane-hide"
  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **model-plane-hide** and click **Run**.

## Command Overview

Hide all displayed block model data.

Tip: Toggle the display of block model data using [model-plane-change ("tbm")](<model-plane-change.md>).
