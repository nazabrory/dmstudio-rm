# move-clip-section-backward ("msb")

See this command in the [**command table**.](<COMMAND%20TABLE_M.md#move-clip-section-backward>)

To access this command:

  * **View** (3D) ribbon **> > Move >> Back**.

  * Press **Page Down**.

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "move-clip-section-backward".

  * Enter the quick key combination "msb".

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **move-clip-section-backward** and click **Run**

## Command Overview

Move the current section plane away from the camera by a distance equal to the current section width. Section widths are set via [Section Properties](<../VR_Help/Section%20Properties%20Dialog.md>).

Tip: Use your keyboard's **Page Up** and **Page Down** keys to move sections backward and forward when a 3D window is active.

Activity steps:

  1. Set the section width using [Section Properties](<../VR_Help/Section%20Properties%20Dialog.md>) (Section Width group).
  2. Run the command to move the current section plane by the section width you specified.
