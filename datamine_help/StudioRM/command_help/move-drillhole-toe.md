# move-drillhole-toe ("mto")

See this command in the [**command table**.](<COMMAND%20TABLE_M.md#move-drillhole-toe>)

To access this command:

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "move-drillhole-toe".

  * Enter the quick key combination "mto".

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **move-drillhole-toe** and click **Run**

## Command Overview

For modifying the toe position of a selected drillhole using the mouse.

Rings snap to the closest drill hole automatically. Click near the end of the hole to be modified and drag to the new desired toe of the hole.
