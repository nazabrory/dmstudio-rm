# move-plane-backward ("mpb")

See this command in the [command table.](<COMMAND%20TABLE_M.md#move-plane-backward>)

To access this command:

  * View ribbon **> > Move >> Back**

  * Enter "move-plane-backward" into the **[Command](<../COMMON/Command_Toolbar.md>)** toolbar and press <ENTER>.

  * If using a keyboard, press **Page Down**.

  * Use the quick key combination "mbp" whilst a 3D view is active.

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **move-plane-backward** and click **Run**

## Command Overview

Move the currently active section plane forward in a direction orthogonal to the plane. The plane is moved forward or backward by the current section width (as shown on the **3D View** ribbon >> **Clip >> Width**). 

**Note** : This differs from the move-plane command, where a custom movement distance is specified during the command, and the default movement distance set using the **View** ribbon's **Move** field.

Command Steps:

  1. Run the command.

  2. The currently active section plane is shifted backwards by the specified section width.
