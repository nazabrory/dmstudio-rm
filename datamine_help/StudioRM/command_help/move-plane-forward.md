# move-plane-forward ("mpf")

See this command in the [command table.](<COMMAND%20TABLE_M.md#move-plane-forward>)

To access this command:

  * View ribbon **> > Move >> Forward**

  * If using a keyboard, press **Page Up**.

  * Enter "move-plane-forward" into the **[Command](<../COMMON/Command_Toolbar.md>)** toolbar and press <ENTER>.

  * Use the quick key combination "mbf" whilst a 3D view is active.

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **move-plane-forward** and click **Run**

## Command Overview

Move the current view plane forward in a direction orthogonal to the plane. The plane is moved forward or backward by the current section width (as shown on the **3D View** ribbon >> **Clip >> Width**). 

**Note** : This differs from the move-plane command, where a custom movement distance is specified during the command, and the default movement distance set using the **View** ribbon's **Move** field.

The orientation of the section plane is not altered.

Command Steps:

  1. Run the command.

  2. The currently active section plane is shifted forwards by the currently specified section width.
