# move-points-range

See this command in the [command table](<COMMAND%20TABLE_M.md#move-point-range>).

To access this command:

  * Enter "move-points-range" into the **[Command](<../COMMON/Command_Toolbar.md>)** toolbar and press <ENTER>.

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **move-points-range** and click **Run**

## Command Overview

Interactively move string points based on their proximity to the cursor position. This can be useful if moving a cluster of points where the 'impact' of the movement should impact neighbouring points based on their distance from the cursor.

Command steps:

  1. Run the command.

  2. Select a point to be moved.

  3. Left-click and drag.

All points within the bounding circle are moved based on their distance from the center point of that circle.
