# move-previous-section ("mps")  
  
See this command in the [**command table**.](<COMMAND%20TABLE_M.md#move-previous-section>)

To access this command:

  * Activate the 3D **View** ribbon and select **Sections >> Previous**.
  * If using a keyboard, press **Previous**.

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "move-previous-section".

## Command Overview

Activate the next section definition of a loaded section definition table.

Command Steps:

  1. Ensure a section definition table is loaded.
  2. Activate a **3D** view.
  3. Run the command.

The previous section definition in the loaded table is activated and the 3D view updates, if data is rendered that is reliant on a particular section definition. For example, clipping may be applied differently after changing sections, or if the view is locked to the section, it updates automatically to be orthogonal to the latest definition.
