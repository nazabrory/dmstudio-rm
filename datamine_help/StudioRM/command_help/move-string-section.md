# move-string-section ("mss")

See this command in the [**command table**.](<COMMAND%20TABLE_M.md#move-string-section>)

To access this command:

  * **Digitize** ribbon **> > Transform >> Move Section**. 

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "move-string-section"

  * Use the quick key combination "mss".
  * On the **[Find Command](<../COMMON/findcommand.md>)** screen, highlight **move-string-section** and click **Run**.

## Command Overview

Move a selected string segment, redrawing the adjacent segments to accommodate the new location of the segment.

Command steps:

  1. Run the command.

  2. Select the start point of the string segment.

  3. You can now either:

     * Select the end point of the string segment and release the mouse button to reposition using the "2-click" method. A subsequent left-click or right click (to snap) will set the new location for the segment's start point, or;

     * Click and hold either the left mouse button to dynamically drag the string segment anywhere on the screen, or the right-mouse button to dynamically snap the string section to a new screen position (according to your existing snap settings.

**Note** : For closed strings, selection is in a counter-clockwise direction. Links to points connected to the segment start and end points are maintained.

  4. Click Done.
