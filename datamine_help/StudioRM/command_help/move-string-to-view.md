# move-string-to-view ("mtv")  
  
See this command in the [**command table**.](<COMMAND%20TABLE_M.md#move-string-to-view>)

To access this command:

  * **Digitize** ribbon **> > Project >> Move String to View**.

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "move-string-to-view"

  * Use the quick key combination "mtv".

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **move-string-to-view** and click **Run**.

## Command Overview

Similar to the [project-to-view-plane ("ptv")](<project-to-view-plane.md>) command in that string data is projected from its current position to the current view plane, however, with this command, the original string is moved, not copied.

Command steps:

  1. Select the string data to be projected.

  2. Configure view plane settings using the [View Settings](<../COMMON/Section%20Definition%20Dialog.md>) screen.

  3. Run the command.

The selected string is projected onto the current view plane. The original string data is not retained.
