# move-wireframe-point ("mpw")

See this command in the [**command table**.](<COMMAND%20TABLE_M.md#move-wireframe-point>)

To access this command:

  * **Explicit** ribbon **Create >> Move Point**.

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "move-wireframe-point"

  * Use the quick key combination "mpw".

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **move-wireframe-point** and click **Run**.

## Command Overview

Interactively move points in a wireframe.

Select this command, then a wireframe point to move the vertex whilst maintaining the overall wireframe integrity, for example:  
  
![](../Images/moveWireframe1.gif)

becomes:

![](../Images/moveWireframe2.gif)

Note: This command supports the [edit-coincident-points-switch](<edit-coincident-points-switch.md>) command.

When clicking to position data:

  * Left click to select or place without snapping to a point or object.

  * Right click to select or place by snapping to a point or object

  * Click-and-drag 

    * Left-click-drag to move a point without snapping.

    * Right-click-drag to move a point with snapping

The snapping mode may be changed during a drag by alternating the mouse button pressed (ensuring that both are not released, as this would indicate the end of the drag). 

Command steps:

  1. In the Current Objects toolbar, select the required wireframe.

  2. Run the command.

  3. Select a wireframe point to be moved.

  4. Select the new location for the wireframe point.

  5. Repeat steps 3 and 4 until all required points have been moved.

  6. Click Cancel.
