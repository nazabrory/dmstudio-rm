# move-wireframe ("mw")

See this command in the [**command table**.](<COMMAND%20TABLE_M.md#move-wireframe>)

To access this command:

  * **Explicit** ribbon **> > Edit >> Copy >> Move Wireframe**.

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "move-wireframe"

  * Use the quick key combination "mw".

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **move-wireframe** and click **Run**.

## Command Overview

Interactive command for moving wireframe data, either as a full object or a selection of triangles within one or more wireframe objects.

If you are moving all of a wireframe object, you do not need to preselect data. If you wish move a part of a wireframe object, whilst maintaining the mobile part's connection to neighbouring triangles, you will need to preselect the relevant object(s) and/or triangle(s) beforehand as move-wireframe will operate on selected data only if available.

**Note** : This command supports [**flexible wireframe selection**](<../COMMON/Wireframe_Selection_Concept.md>).

Command steps:

  1. If required, preselect wireframe object, group, surface, field or triangle data.

  2. Run the Command.

  3. Click-and-drag the required wireframe or wireframe section to its new position. If moving part of a wireframe. existing vertex links are preserved throughout movement.

  4. Click Cancel.
