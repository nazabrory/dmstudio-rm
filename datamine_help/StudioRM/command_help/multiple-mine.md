# multiple-mine

See this command in the [**command table**.](<COMMAND%20TABLE_M.md#multiple-mine>)

To access this command:

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "multiple-mine"

  * Use the quick key combination "mm".

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **multiple-mine** and click **Run**.

## Command Overview

Repetitively mine a face or tonnage zone for a set number of periods.

You are asked to select an active face or tonnage zone, and to confirm the production rate for that mine unit. You then enter the number of periods to mine - an entry of '+' is allowed.

The unit is then mined in one of the following ways:

  * Over the number of specified periods.

  * Until the unit is mined out.

  * Until the end of the specified calendar is reached.
