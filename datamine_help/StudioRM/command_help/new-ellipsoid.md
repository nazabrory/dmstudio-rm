# new-ellipsoid ("nel")

See this command in the [**command table**.](<COMMAND%20TABLE_N.md#new-ellipsoid>)

To access this command:

  * Implicit ribbon **> > Ellipsoid >> New**.

  * **Data** ribbon **> > Objects >> New >> Ellipsoid**.

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "new-ellipsoid"

  * Use the quick key combination "nel".

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **new-ellipsoid** and click **Run**.

## Command Overview

Create one or more search ellipsoids, for inclusion within a new or existing [ellipsoids object](<../VR_Help/Ellipsoids_Overview.md>). 

This command displays the [New Ellipsoid](<../COMMON/New_Ellipsoid.md>) screen.
