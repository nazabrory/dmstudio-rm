# new-polygon ("npy")

See this command in the [**command table**.](<COMMAND%20TABLE_N.md#new-polygon>)

To access this command:

  * **Digitize** ribbon **> > Edit >> New Polygon**.

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "new-polygon"

  * Use the quick key combination "npy"

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **new-polygon** and click **Run**.

## Command Overview

Allows you to create a new closed string (polygon) in the 3D window by clicking in at least 3 locations. Clicking in more locations creates a polygon with more vertices.

  * If no strings object exists in memory, a new object is created when this command is run, displaying details of the new object immediately in the Current Objects toolbar. See [The Current Objects Toolbar](<../COMMON/Current_Objects_Toolbar.md>).

  * Selecting a different attribute value in the Current Objects toolbar will automatically start a new string.

  * 

Activity steps:

  1. Select the required snap and data selection settings. See [Snapping to 3D Data](<../COMMON/Snapping-3D-windows.md>).

  2. In the Current Objects toolbar, select the current object or create a new one. 

  3. In the Current Objects toolbar, select the required Attribute Field and associated Attribute Value parameters. 

  4. Run the command.

  5. Click to define the location of the polygon points.

  6. Cancel the command by clicking **Done** or pressing ESC.
