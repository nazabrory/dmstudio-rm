# new-single-point-strings

See this command in the [**command table**.](<COMMAND%20TABLE_N.md#new-single-point-strings>)

To access this command:

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "new-single-point-strings"

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **new-single-point-strings** and click **Run**.

## Command Overview

Digitise new point data which is saved as single point strings.

Activity steps:

  1. Run the command.

  2. In any available 3D window, digitize points. These will appear as discrete point data but is considered string data and contain corresponding system fields.

Tip: Complete the digitizing operation by double-clicking or double-tapping the final single string point position.
