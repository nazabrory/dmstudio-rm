# Number of Strings per Block

To display this screen:

  * Run the command [link-selected-strings-plane](<link-selected-strings-plane.md>).

  * Run the command [link-selected-strings-attrib](<link-selected-strings-attrib.md>).

  * Run the command [link-selected-strings-point](<link-selected-strings-point.md>).

Define the number of strings to be grouped together when linking the selected strings to create wireframes. Enter the Number of strings to be grouped together for each wireframe volume.

Note: Depending on which command is being used, the Number of strings parameter may be used in a slightly different way; please see the individual command help pages for further details.
