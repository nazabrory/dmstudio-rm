# offset-point-range

Note: This is an experimental command and can lead to string crossover situations. As such, it is not visible on the **Find Command** screen.

See this command in the [**command table**.](<COMMAND%20TABLE_O.md#offset-point-range>)

To access this command:

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "offset-point-range"

## Command Overview

Offset string points in a perpendicular direction from the plane of the two adjacent string edges. 

All points within the bounding circle are offset by an amount proportional to the cursor position. 

Points closer to the centre are adjusted by the cursor movement whilst off-cursor points within the circle are adjusted by a proportionally smaller amount.
