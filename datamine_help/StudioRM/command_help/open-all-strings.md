# open-all-strings ("oa")

See this command in the [**command table**.](<COMMAND%20TABLE_O.md#open-all-strings>)

To access this command:

  * **Digitize** ribbon **> > Condition >> Open All**.

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "open-all-strings".

  * Use the quick key combination "oa".

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **open-all-strings** and click **Run**.

## Command Overview

Turn all perimeters into open strings.

A perimeter is defined as a string whose first and last point share the same coordinate location. This command treats all strings in memory in turn. If the string is a perimeter then the last point is deleted. If the string is open the command has no effect on the string.
