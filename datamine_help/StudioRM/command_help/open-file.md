# open-file ("of")

See this command in the [**command table**.](<COMMAND%20TABLE_O.md#open-file>)

To access this command:

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "open-file".

  * Use the quick key combination "of".

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **open-file** and click **Run**.

## Command Overview

Open a file using the [Project Browser](<../COMMON/ProjectBrowser.md>).
