# open-model-file ("om")

See this command in the [**command table**.](<COMMAND%20TABLE_O.md#open-model-file>)

To access this command:

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "open-model-file".

  * Use the quick key combination "om".

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **open-model-file** and click **Run**.

## Command Overview

Open a block model file using the [Project Browser](<../COMMON/ProjectBrowser.md>).

You are first asked to enter the block model name from a file browser. You are then asked to confirm whether you wish all fields in the model to be considered for evaluations. It is recommended you select "Yes" to this prompt. If you say no you are asked to enter the names of the fields for consideration.

Only one model file can be open in the design window at a time. The file is not entirely loaded into memory. An index is built in memory that is then used to access the file on the disk. The size of this index will depend on the number of cells defined within the model data definition.
