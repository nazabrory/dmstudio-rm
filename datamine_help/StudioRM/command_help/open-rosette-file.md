# open-rosette-file ("oro")

See this command in the [**command table**.](<COMMAND%20TABLE_O.md#open-rosette-file>)

To access this command:

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "open-results-file".

  * Use the quick key combination "ore".

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **open-results-file** and click **Run**.

## Command Overview

Open an evaluation results file using the [Project Browser](<../COMMON/ProjectBrowser.md>).
