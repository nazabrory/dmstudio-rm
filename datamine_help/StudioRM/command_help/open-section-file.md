# open-section-file ("ose")

See this command in the [**command table**.](<COMMAND%20TABLE_O.md#open-section-file>)

To access this command:

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "open-section-file".

  * Use the quick key combination "ose".

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **open-section-file** and click **Run**.

## Command Overview

Open a section definition file using the [Project Browser](<../COMMON/ProjectBrowser.md>)
