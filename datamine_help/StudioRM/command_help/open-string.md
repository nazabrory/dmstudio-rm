# open-string ("ost")

See this command in the [**command table**.](<COMMAND%20TABLE_O.md#open-string>)

To access this command:

  * **Digitize** ribbon **> > Condition >> Open**.

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "open-string".

  * Use the quick key combination "ost".

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **open-string** and click **Run**.

## Command Overview

Turn a perimeter into an open string. If a string is closed then the last point is deleted - if the string is already open, the command has no effect.

You can preselect data before running this command; all preselected strings are closed without further prompts.

Running this command automatically cancels any other running command.

**Note** : Any string where the first and last point are coincident is considered to be closed.
