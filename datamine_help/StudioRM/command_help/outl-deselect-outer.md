# outl-deselect-outer ("dol")

See this command in the [**command table**.](<COMMAND%20TABLE_O.md#outl-deselect-outer>)

To access this command:

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "outl-deselect-outer".

  * Use the quick key combination "dol".

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **outl-deselect-outer** and click **Run**.

## Command Overview

Deselect all outer limits used for outline generation.
