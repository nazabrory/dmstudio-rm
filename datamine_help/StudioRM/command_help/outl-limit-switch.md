# outl-limit-switch ("uo")

See this command in the [**command table**.](<COMMAND%20TABLE_O.md#outl-limit-switch>)

To access this command:

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "outl-limit-switch"

  * Use the quick key combination "uo".

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **outl-limit-switch** and click **Run**.

## Command Overview

Control whether new [outlines](<generate-outlines.md>) are generated within an outer limit. If this command mode is active and an [outer limit](<outl-select-outer.md>) has been selected, outlines will only be generated within that limit.
