# outl-plane-switch ("ftp")

See this command in the [**command table**.](<COMMAND%20TABLE_O.md#outl-plane-switch>)

To access this command:

  * **Edit** ribbon **> > Shapes >> Outlines >> Fix to Plane**.

  * **Digitize** ribbon **> > Create >> Outlines >> Fix to Plane**.

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "outl-plane-switch"

  * Use the quick key combination "ftp".

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **outl-plane-switch** and click **Run**.

## Command Overview

Control whether new [outlines](<generate-outlines.md>) are generated on the current view plane or the mean plane of the original string(s).

If this command is ON, when new outlines are generated they are projected onto the current view plane, irrespective of the elevations of the strings from which they were generated.

If the command is OFF, outlines are generated on the mean plane of the original string points.
