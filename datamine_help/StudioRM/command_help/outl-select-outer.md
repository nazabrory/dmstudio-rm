# outl-select-outer ("ool")

See this command in the [**command table**.](<COMMAND%20TABLE_O.md#outl-select-outer>)

To access this command:

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "outl-select-outer"

  * Use the quick key combination "ool".

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **outl-select-outer** and click **Run**.

## Command Overview

Select an outer limit to be used when outlines are generated.

Run the command and pick a string to act as the outer limit for [outline generation](<generate-outlines.md>).
