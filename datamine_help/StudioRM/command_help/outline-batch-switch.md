# outline-batch-switch ("trs")

See this command in the [**command table**.](<COMMAND%20TABLE_O.md#outline-batch-switch>)

To access this command:

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "outline-batch-switch".

  * Use the quick key combination "trs".

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **outline-batch-switch** and click **Run**.

## Command Overview

Toggle whether outlines are generated automatically or after string selection.

Used in conjunction with the [generate-outlines](<generate-outlines.md>) and similar commands, this command will alternate the following operating modes for outline generation:

  * All possible outlines for selected string data are generated automatically.
  * Interactive selection of outline areas are required after running an outline generation command.

Command steps

  1. Check the status of the [outline-storage-switch](<outline-storage-switch.md>).

  2. Run this command.

  3. Run [generate-outlines](<generate-outlines.md>) if required, selecting string(s).

  4. If required, reset the above switches.
