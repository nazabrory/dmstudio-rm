# outline-inside-switch-off

See this command in the [**command table**.](<COMMAND%20TABLE_O.md#outline-inside-switch-off>)

To access this command:

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "outline-batch-switch".

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **outline-batch-switch** and click **Run**.

## Command Overview

Used by the [generate-outlines](<generate-outlines.md>) command to specify that boundary strings will not support an optional INSIDE parameter when generated.

Note: This is not a toggle command. Enable INSIDE parameter usage with [outline-inside-switch-on](<outline-inside-switch-on.md>).
