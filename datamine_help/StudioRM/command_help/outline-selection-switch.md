# outline-selection-switch ("tsin")

See this command in the [**command table**.](<COMMAND%20TABLE_O.md#outline-selection-switch>)

To access this command:

  * **Edit** ribbon **> > Shapes >> Outlines >> Select Outlines Individually**.

  * **Digitize** ribbon **> > Create >> Outlines >> Select Outlines Individually**.

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "outline-selection-switch"

  * Use the quick key combination "tsin".

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **outline-selection-switch** and click **Run**.

## Command Overview

Toggle for individual selection of new **[outlines](<generate-outlines.md>)**.

If this toggle is ON, outlines for each string trace are created individually by selection. In this case, [outline-batch-switch](<outline-batch-switch.md>) is automatically switched OFF.

If this toggle is OFF, outlines for all string traces are created automatically. In this case, [outline-batch-switch](<outline-batch-switch.md>) is automatically switched ON.
