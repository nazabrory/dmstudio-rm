# outline-storage-switch ("tsif")

See this command in the [**command table**.](<COMMAND%20TABLE_O.md#outline-storage-switch>)

To access this command:

  * **Edit** ribbon **> > Shapes >> Outlines >> New Outlines Object**.

  * **Digitize** ribbon **> > Create >> Outlines >> New Outlines Object**.

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "outline-storage-switch"

  * Use the quick key combination "tsif".

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **outline-storage-switch** and click **Run**.

## Command Overview

Toggle how new **[outlines](<generate-outlines.md>)** are saved.

If this toggle is OFF, outline generation commands produce outlines strings in the original strings object.

Note: This includes the [digitise-doughnut ("dnn")](<digitise-doughnut.md>) command.

If this toggle is ON, output data is generated in a new object, containing only outline data.
