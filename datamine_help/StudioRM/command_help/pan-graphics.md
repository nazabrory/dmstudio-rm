# pan-graphics ("pan")

See this command in the [**command table**.](<COMMAND%20TABLE_P.md#pan-graphics>)

To access this command:

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "pan-down-graphics"

  * Use the quick key combination "pan".

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **pan-down-graphics** and click **Run**.

## Command Overview

Enter 3D window panning mode.

Command steps

  1. Display data in any 3D window.

  2. Run the command.

  3. Left-click and drag the view of data using the mouse.

  4. Press <ESC> to exit panning mode.
