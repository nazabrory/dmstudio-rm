# pan-left-graphics "←"

See this command in the [**command table**.](<COMMAND%20TABLE_P.md#pan-left-graphics>)

To access this command:

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "pan-left-graphics"

  * Use the quick key combination "←".

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **pan-left-graphics** and click **Run**.

## Command Overview

Move the 3D camera position left. This gives the impression of moving data right.

Note: This command is accessible using the <LEFT> arrow key if a 3D window is in focus.
