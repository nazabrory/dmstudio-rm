# pan-right-graphics "→"

See this command in the [**command table**.](<COMMAND%20TABLE_P.md#pan-right-graphics>)

To access this command:

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "pan-right-graphics"

  * Use the quick key combination "→".

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **pan-right-graphics** and click **Run**.

## Command Overview

Move the 3D camera position right. This gives the impression of moving data left.

Note: This command is accessible using the <RIGHT> arrow key if a 3D window is in focus.
