# pan-up-graphics "↑"

See this command in the [**command table**.](<COMMAND%20TABLE_P.md#pan-up-graphics>)

To access this command:

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "pan-up-graphics"

  * Use the quick key combination "↑".

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **pan-up-graphics** and click **Run**.

## Command Overview

Move the 3D camera position up. This gives the impression of moving data down.

Note: This command is accessible using the <UP> arrow key if a 3D window is in focus.
