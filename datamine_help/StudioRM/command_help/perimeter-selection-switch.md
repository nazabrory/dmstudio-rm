# perimeter-selection-switch ("pes")

See this command in the [**command table**](<COMMAND%20TABLE_P.md#perimeter-selection-switch>).

To access this command:

  *   * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "perimeter-selection-switch"

  * Use the quick key combination "pes".

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **perimeter-selection-switch** and click **Run**.

## Command Overview

Toggle to control how boundary (closed) strings are selected in relation to the mouse-click position. This command is designed to be used within your application to enable strings that are fully abutted by other strings to be selected more easily.

  * When perimeter selection mode is active only strings in which the mouse pointer lies are selected. 

  * If the mode is inactive, string selection occurs according to cursor proximity to string data only.

Note: Data selection is further controlled by the settings in the Snapping toolbar and the [Overlay Selection](<../COMMON/OverlaySelectionDialog.md>) screen.
