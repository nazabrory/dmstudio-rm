# plan-drillholes ("pdh")

See this command in the [**command table**.](<COMMAND%20TABLE_P.md#plan-drillholes>)

To access this command:

  * **Sample Analysis** ribbon **> > Plan >> Drillhole Planning**.

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "plan-drillholes"

  * Use the quick key combination "pdh".

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **plan-drillholes** and click **Run**.

## Command Overview

Create or edit the layout of drillholes in a sample data file, using the [Drillhole Planner](<../COMMON/DrillholePlanner-Create-New.md>) managed task.
