# plane-display-change

See this command in the [**command table**.](<COMMAND%20TABLE_P.md#plane-display-change>)

To access this command:

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "plane-display-change"

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **plane-display-change** and click **Run**.

  * **Format** ribbon **> > Overlays >> Planes**.

## Command Overview

Show or hide loaded planes data overlays in 3D windows, honouring visual filters if they are specified.

This is a toggle command: Successive uses of the command toggle the display mode.
