# plane-display-hide

See this command in the [**command table**.](<COMMAND%20TABLE_P.md#plane-display-hide>)

To access this command:

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "plane-display-hide"

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **plane-display-hide** and click **Run**.

## Command Overview

Hide all planes data overlays in 3D windows.
