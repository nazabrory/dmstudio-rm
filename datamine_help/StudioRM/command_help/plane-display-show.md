# plane-display-show

See this command in the [**command table**.](<COMMAND%20TABLE_P.md#plane-display-show>)

To access this command:

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "plane-display-show"

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **plane-display-change** and click **Run**.

## Command Overview

Show all hidden planes data overlays in 3D windows.
