# point-display-change ("spo")

See this command in the [**command table**.](<COMMAND%20TABLE_P.md#point-display-change>)

To access this command:

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "point-display-change".  

  * Use the quick key combination "spo".

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **point-display-change** and click **Run**.

## Command Overview

Toggle the display of all point data overlays, honouring existing filters.
