# point-display-hide

See this command in the [**command table**.](<COMMAND%20TABLE_P.md#point-display-hide>)

To access this command:

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "point-display-hide".  

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **point-display-hide** and click **Run**.

## Command Overview

Hide all points data overlays displayed in 3D windows.
