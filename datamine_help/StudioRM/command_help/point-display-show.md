# point-display-show

See this command in the [**command table**.](<COMMAND%20TABLE_P.md#point-display-show>)

To access this command:

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "point-display-show".  

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **point-display-show** and click **Run**.

## Command Overview

Show all hidden points data overlays in 3D windows, honouring existing data filters.
