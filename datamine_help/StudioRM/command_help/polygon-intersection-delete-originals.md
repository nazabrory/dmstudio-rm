# polygon-intersection-delete-originals

See this command in the [**command table**.](<COMMAND%20TABLE_P.md#polygon-intersection-delete-originals>)

To access this command:

  * **Digitize** ribbon >> **Tools >> Combine >> Intersect Polygons Unique**.

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "polygon-intersection-delete-originals".

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **polygon-intersection-delete-originals** and click **Run**.

## Command Overview

For selected polygons in any 3D window, this command creates a new polygon which contains the area common to all of them, erasing the original polygon data.

**Note** : A variation of this command - polygon-intersection \- retains the original polygon string data.
