# progress-monitor-switch ("pmo")

See this command in the [**command table**.](<COMMAND%20TABLE_P.md#progress-monitor-switch>)

To access this command:

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "progress-monitor-switch".

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **progress-monitor-switch** and click **Run**.

## Command Overview

Toggles the use of the progress monitor when running commands.
