# project-points-to-wf-angle ("pta")

See this command in the [**command table**.](<COMMAND%20TABLE_P.md#project-points-to-wf-angle>)

To access this command:

  * **Digitize** ribbon **> > Transform >> Project >> Points to Wireframe at Angle**. 

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "project-points-to-wf-angle"

  * Use the quick key combination "pta".
  * On the **[Find Command](<../COMMON/findcommand.md>)** screen, highlight **project-points-to-wf-angle** and click **Run**.

## Command Overview

Project points onto a wireframe at the angle determined by the **SDIP** and **DIPDIRN** fields associated with the current points object, if present. If these fields are not present, a vertical projection is performed.

Command steps

  1. Select the point(s) you wish to project (it is important to select the point first, and then run the command).

  2. Run the command.

Select point data is projected to wireframe data.
