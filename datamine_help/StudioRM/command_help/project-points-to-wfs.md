# project-points-to-wfs ("pptw")

See this command in the [**command table**](<COMMAND%20TABLE_P.md#project-points-to-wfs>).

To access this command:

  * **Digitize** ribbon **> > Project >> Points to Wireframes**.

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "project-points-to-wfs"

  * Use the quick key combination "pptw".

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **project-points-to-wfs** and click **Run**.

## Command Overview

Project point data to intersect wireframe surface(s) using defined projection azimuth and dip settings, to either update points or create new point data. 

Projected points lie on the wireframe. If any points for projection do not project to the wireframe(s), they are ignored.

Command steps:

  1. Select point data.

  2. Run the command.

The [Project Strings & Points](<../COMMON/ProjectStringsPoints_Dialog.md>) screen displays.

  3. Define the Output, Projection Direction, Projection Origin, Projection Presets and Project To parameters.

  4. Click OK.
