# project-string-onto-wf ("ptw")

See this command in the [**command table**](<COMMAND%20TABLE_P.md#project-string-onto-wf>).

To access this command:

  * **Digitize** ribbon **> > Project >> String to Wireframe**.

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "project-string-onto-wf"

  * Use the quick key combination "ptw".

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **project-string-onto-wf** and click **Run**.

## Command Overview

Project a string vertically up or down onto the current wireframe to create a new string. 

The points of the new string will lie on the wireframe. If any points in the projected string do not lie vertically above or below the wireframe they are ignored, and more than one new string may be created.

**Tip** : Preselecting multiple strings allows you to run this command for all selected strings.
