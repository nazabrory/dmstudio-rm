# project-string-onto-wfs ("pstw")

See this command in the [**command table**.](<COMMAND%20TABLE_P.md#project-string-onto-wfs>)

To access this command:

  * **Digitize** ribbon **> > Project >> String to Wireframes**.

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "project-string-onto-wfs"

  * Use the quick key combination "pstw".

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **project-string-onto-wfs** and click **Run**.

## Command Overview

Project string(s) to intersect wireframe surface(s) using defined projection azimuth and dip settings, to create an updated or new string(s). 

The points of the updated or new string data lie on the wireframe. If any points in the string(s) for projection do not project to the wireframe(s), they are ignored. 

More than one new string may be created.

Command steps:

  1. Select string data.

  2. Run the command.

The [Project Strings & Points](<../COMMON/ProjectStringsPoints_Dialog.md>) screen displays.

  3. Define the Output, Projection Direction, Projection Origin, Projection Presets and Project To parameters

  4. Click OK.
