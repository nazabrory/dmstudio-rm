# project-to-view-plane ("ptv")

See this command in the [**command table**.](<COMMAND%20TABLE_P.md#project-to-view-plane>)

To access this command:

  * **Digitize** ribbon **> > Project >> Project String to View**.

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "move-string-to-view"

  * Use the quick key combination "mtv".

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **move-string-to-view** and click **Run**.

## Command Overview

This command allows a new string to be created by projecting an existing string perpendicularly onto the current view plane. The original string data is retained, unlike the [move-string-to-view ("mtv")](<move-string-to-view.md>) command.

Command steps:

  1. Select the string data to be projected.

  2. Configure view plane settings as required (using the [View Settings](<../COMMON/Section%20Definition%20Dialog.md>) screen).

  3. Run the command.

The selected string is projected onto the current view plane.
