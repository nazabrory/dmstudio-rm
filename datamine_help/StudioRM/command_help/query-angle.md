# query-angle ("qa")

See this command in the [**command table**.](<COMMAND%20TABLE_Q.md#query-angle>)

To access this command:

  * **Home** ribbon **> > Query >> Angle**.

  * Use the quick key combination "qa".

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "query-angle".

  * On the **[Find Command](<../COMMON/findcommand.md>)** screen, highlight **query-angle** and click **Run**.

## Command Overview

Measure the angle between two line segments. These line segments are generated temporarily, and removed once the command is closed.

This command supports your data snapping options.

Command steps:

  1. Run the command.

  2. Digitize the point of the first line segment.

  3. Digitize the end of the first line segment.

  4. Digitize the end of the second line segment

  5. Continue to adjust (or hold down and drag) to reposition the end of the 2nd line segment.

  6. An angle indicator is shown and the Output window records the result.
