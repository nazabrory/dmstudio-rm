# query-current-file-filters ("qf")

See this command in the [**command table**.](<COMMAND%20TABLE_Q.md#query-current-file-filters>)

To access this command:

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "query-current-file-filters".

  * Use the quick key combination "qf".

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **query-current-file-filters** and click **Run**.

## Command Overview

Display filter settings for all loaded data objects.

In the [Output](<../COMMON/Output%20Control%20Bar%20Overview.md>) control bar , a report is displayed, listing the current filter settings for points, strings, drillholes, block models, planes and wireframes. If there are no filters currently set then a message is given to this effect.

Alternatively, use the [Data Object Manager](<../COMMON/Data%20Manager%20Dialog.md>) to see filters loaded for individual objects.
