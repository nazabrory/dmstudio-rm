# query-multiple-drillholes ("qmd")

See this command in the [**command table**.](<COMMAND%20TABLE_Q.md#query-multiple-drillholes>)

To access this command:

  * **Home** ribbon **> > Query >> Static Drillholes**.

  * Use the quick key combination "qmd".

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "query-multiple-drillholes".

  * On the **[Find Command](<../COMMON/findcommand.md>)** screen, highlight **query-multiple-drillholes** and click **Run**.

## Command Overview

Produce summary statistics of pre-selected drillholes. 

**Important** : Static drillhole data must be selected before you run this command.

Command steps:

  1. Select one or more static drillholes.

  2. Run the command.

  3. Review the Output window report.
