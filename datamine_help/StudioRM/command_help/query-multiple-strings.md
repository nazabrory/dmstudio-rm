# query-multiple-strings ("qms")

See this command in the [**command table**.](<COMMAND%20TABLE_Q.md#query-multiple-strings>)

To access this command:

  * **Home** ribbon **> > Query >> Multiple Strings**.

  * Use the quick key combination "qms".

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "query-multiple-strings".

  * On the **[Find Command](<../COMMON/findcommand.md>)** screen, highlight **query-multiple-strings** and click **Run**.

## Command Overview

Output summary data for multiple selected strings to the **Output** window.

Command steps:

  1. Select one or more strings in any view.

  2. Run the command.

  3. Review the Output window contents; string lengths and summary information is displayed.
