# query-points ("qp")

See this command in the [**command table**.](<COMMAND%20TABLE_Q.md#query-points>)

To access this command:

  * Home ribbon >> Query >> Measure

  * **Home** ribbon **> > Query >> Point**.

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "query-points"

  * Use the quick key combination "qp".
  * On the **[Find Command](<../COMMON/findcommand.md>)** screen, highlight **query-points** and click **Run**.

## Command Overview

Calculate and display the location and attribute values of a selected point.

Data is displayed in both the Output control bar and (if enabled) a [cursor tooltip](<../COMMON/Options_Environment.md>).

If the selected point lies in a model cell then the I, J and K values and field values in the cell are reported in the Output window (with XYZ values on the tooltip, if displayed).

If the selected point is on a drill hole then the sample is highlighted and statistics reported about the sample.
