# query-rosette ("ql")

See this command in the [**command table**.](<COMMAND%20TABLE_Q.md#query-rosette>)

To access this command:

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "query-rosette"

  * Use the quick key combination "qr".
  * On the **[Find Command](<../COMMON/findcommand.md>)** screen, highlight **query-rosette** and click **Run**.

## Command Overview

This will report the location and settings for the selected rosette. You are asked to select a rosette with the cursor.
