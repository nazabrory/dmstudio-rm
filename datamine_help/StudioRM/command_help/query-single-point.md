# query-single-point ("qsp")

See this command in the [**command table**.](<COMMAND%20TABLE_Q.md#query-single-point>)

To access this command:

  * Use the quick key combination "qsp".

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "query-single-point".

  * On the **[Find Command](<../COMMON/findcommand.md>)** screen, highlight **query-single-point** and click **Run**.

## Command Overview

Query individual point data displayed in 3D windows. Click a point to show coordinates in the **Output** window.
