# query-string-between-points ("qsb")

See this command in the [**command table**.](<COMMAND%20TABLE_Q.md#query-string-between-points>)

To access this command:

  * **Home** ribbon **> > Query >> Partial String**.

  * Use the quick key combination "qsb".

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "query-string-between-points".

  * On the **[Find Command](<../COMMON/findcommand.md>)** screen, highlight **query-string-between-points** and click **Run**.

## Command Overview

Queries the statistics of the selected portion of a string, defined by a start and an end point.

Command steps:

  1. Optionally, pre-select strings to create a group, within which you wish to query string points.

  2. Run the command.

  3. Select the start and end points for the queried section as per the prompt in the status bar. The statistics for the selected section are displayed in the Output window.
