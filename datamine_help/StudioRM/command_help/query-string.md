# query-string ("qst")

See this command in the [**command table**](<COMMAND%20TABLE_Q.md#query-string>).

To access this command:

  * Home ribbon >> Query >> String

  * Use the quick key combination "qst".

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "query-string".

  * On the **[Find Command](<../COMMON/findcommand.md>)** screen, highlight **query-string** and click **Run**.

## Command Overview

Calculate and display a selected string's properties and values.

The information shown always includes the range of the X, Y and Z co-ordinates. If the string is closed (a perimeter), the enclosed area as projected onto the current view plane is also given.

If cursor tooltips are displayed, information about the selected string is displayed close to the cursor position. A more detailed report is shown in the Output window.
