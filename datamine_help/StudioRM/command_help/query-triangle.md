# query-triangle ("qt")

See this command in the [**command table**.](<COMMAND%20TABLE_Q.md#query-triangle>)

To access this command:

  * **Home** ribbon **> > Query >> Triangle**.

  * Use the quick key combination "qt".

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "query-triangle".

  * On the **[Find Command](<../COMMON/findcommand.md>)** screen, highlight **query-triangle** and click **Run**.

## Command Overview

Give statistics and attribute values for a selected triangle. The information shown includes the triangle number (**TRIANGLE**), the X,Y and Z coordinate of each point, **GROUP** , **SURFACE** , **LINK** , and other attribute values.
