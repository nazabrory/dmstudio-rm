# quick-color-drillholes("qcd")

See this command in the [**command table**.](<COMMAND%20TABLE_Q.md#quick-color-drillholes>)

To access this command:

  * **Format** ribbon **> > Colour >> Quick >> Drillholes**.

  * Use the quick key combination "qcd".

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "quick-color-drillholes".

  * On the **[Find Command](<../COMMON/findcommand.md>)** screen, highlight **quick-color-drillholes** and click **Run**.

## Command Overview

Display the **Quick Color** screen to quickly apply a specified colour or legend to selected drillholew overlays.

See [Quick Color](<../COMMON/Quick%20Color%20Dialog.md>).
