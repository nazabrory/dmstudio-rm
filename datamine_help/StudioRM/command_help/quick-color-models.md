# quick-color-models ("qcm")

See this command in the [**command table**.](<COMMAND%20TABLE_Q.md#quick-color-models>)

To access this command:

  * **Format** ribbon **> > Colour >> Quick >> Models**.

  * Use the quick key combination "qcm".

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "quick-color-models".

  * On the **[Find Command](<../COMMON/findcommand.md>)** screen, highlight **quick-color-models** and click **Run**.

## Command Overview

Display the **Quick Color** screen to quickly apply a specified colour or legend to selected block model overlays.

See [Quick Color](<../COMMON/Quick%20Color%20Dialog.md>).
