# quick-color-planes("qcl")

See this command in the [**command table**.](<COMMAND%20TABLE_Q.md#quick-color-planes>)

To access this command:

  * **Format** ribbon **> > Colour >> Quick >> Planes**.

  * Use the quick key combination "qcl".

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "quick-color-planes".

  * On the **[Find Command](<../COMMON/findcommand.md>)** screen, highlight **quick-color-planes** and click **Run**.

## Command Overview

Display the **Quick Color** screen to quickly apply a specified colour or legend to selected plane object overlays.

See [Quick Color](<../COMMON/Quick%20Color%20Dialog.md>).
