# quick-color-points ("qcp")

See this command in the [**command table**.](<COMMAND%20TABLE_Q.md#quick-color-points>)

To access this command:

  * **Format** ribbon **> > Colour >> Quick >> Points**.

  * Use the quick key combination "qcp".

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "quick-color-points".

  * On the **[Find Command](<../COMMON/findcommand.md>)** screen, highlight **quick-color-points** and click **Run**.

## Command Overview

Display the **Quick Color** screen to quickly apply a specified colour or legend to selected points object overlays.

See [Quick Color](<../COMMON/Quick%20Color%20Dialog.md>).
