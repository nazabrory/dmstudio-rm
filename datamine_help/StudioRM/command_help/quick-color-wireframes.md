# quick-color-wireframes ("qcw")

See this command in the [**command table**.](<COMMAND%20TABLE_Q.md#quick-color-wireframes>)

To access this command:

  * **Format** ribbon **> > Colour >> Quick >> Wireframes**.

  * Use the quick key combination "qcw".

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "quick-color-wireframes".

  * On the **[Find Command](<../COMMON/findcommand.md>)** screen, highlight **quick-color-wireframes** and click **Run**.

## Command Overview

Display the **Quick Color** screen to quickly apply a specified colour or legend to selected wireframe object overlays.

See [Quick Color](<../COMMON/Quick%20Color%20Dialog.md>).
