# redo-last-undo (<CTRL>+Y)

See this command in the [**command table**.](<COMMAND%20TABLE_R.md#redo-last-undo>)

To access this command:

  * **Home** ribbon >> ![](../Images/Icons/redo-last-undo.png)

  * **Digitize** ribbon >> ![](../Images/Icons/redo-last-undo.png)

  * **Home** ribbon >> **Edit** >> **Undo**

  * Activate any **3D** window and select <CTRL> \+ <Y>.

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "redo-last-undo".

  * On the **[Find Command](<../COMMON/findcommand.md>)** screen, highlight **redo-last-undo** and click **Run**.

## Command Overview

Redo the most recent undone command action. Can be used successively to redo multiple undone command actions.
