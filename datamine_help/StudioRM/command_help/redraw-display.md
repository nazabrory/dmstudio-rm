# redraw-display ("rd")

See this command in the [**command table**.](<COMMAND%20TABLE_R.md#redraw-display>)

To access this command:

  * Use the quick key combination "rd".

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "redraw-display".

  * On the **[Find Command](<../COMMON/findcommand.md>)** screen, highlight **redraw-display** and click **Run**.

## Command Overview

Redraw all currently displayed object overlays in the target 3D window. Typically, 3D data display updates happen automatically in response to other application setting changes, but you can force a redraw using this command at any time.

Note: If you have a lot of data displayed, redrawing can take a while.
