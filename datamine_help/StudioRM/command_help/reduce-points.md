# reduce-points ("red")

**Tip** : consider the **[simplify-string](<simplify-string.md>)** as an alternative to string conditioning.

See this command in the [**command table**.](<COMMAND%20TABLE_R.md#reduce-points>)

To access this command:

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "reduce-points"

  * Using the quick key combination "red"

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **reduce-points** and click **Run**.

**Note** : This command cannot be accessed using a quick-key combination.

## Command Overview

Reduce the number of points in a string by the percentage set using the [reduction-percentage](<reduction-percentage.md>) command.

You can preselect multiple strings to reduce points automatically on them all, without having to select multiple strings successively.

Command steps:

  1. Run the [reduction-percentage](<reduction-percentage.md>) command to set the percentage for point reduction.

  2. Run the reduce-points command, and if no data was previously selected, pick a string in any 3D window.

String data points are reduced whilst attempting to retain the original shape.

  3. If no data was selected prior to running the command, select **Done** to finish.
