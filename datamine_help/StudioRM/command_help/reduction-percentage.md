# reduction-percentage ("pre")

See this command in the [**command table**.](<COMMAND%20TABLE_R.md#reduction-percentage>)

To access this command:

  * **Explicit** ribbon **> > Condition >> Condition >> Percentage Reduction**.

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "reduction-percentage"

  * Use the quick key combination "pre".

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **reduction-percentage** and click **Run**.

## Command Overview

Used in conjunction with the [reduce-points](<reduce-points.md>) command. Specifies the percentage reduction for that command, where higher values tend to reduce the string points more than lower values.

**Tip** : consider the **[simplify-string](<simplify-string.md>)** as an alternative to string conditioning.

Displays the **[Percentage Reduction](<../COMMON/Percentage%20Reduction%20Dialog.md>)** screen.
