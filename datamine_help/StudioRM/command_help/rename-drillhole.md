# rename-drillhole ("rnd")

See this command in the [**command table**.](<COMMAND%20TABLE_R.md#rename-drillhole>)

To access this command:

  * **Sample Analysis** ribbon **> > Plan >> Rename**.

  * Use the quick key combination "rnd".

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "rename-drillhole".

  * On the **[Find Command](<../COMMON/findcommand.md>)** screen, highlight **rename-drillhole** and click **Run**.

## Command Overview

Interactively rename drill holes.

Command steps:

  1. Load a display some drillhole data in a 3D window.

  2. Either select data before running the command or after.

  3. Run the command.

  4. If no drillholes are selected, you are prompted to choose one. If one drillhole is selected, enter a new Hole identifier (the existing value displays here). 

Note: If multiple holes were previously selected, select one with the selection group to rename.

  5. Click **OK**.

The drillhole ID changes to the one specified.
