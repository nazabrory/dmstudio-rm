# resolve-string-points ("resp")

See this command in the [**command table**.](<COMMAND%20TABLE_R.md#resolve-string-points>)

To access this command:

  * **Digitize** ribbon **> > Condition >> Condition >> Resolve String Points**.

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "resolve-string-points"

  * Use the quick key combination "resp".

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **resolve-string-points** and click **Run**.

## Command Overview

Make points from one string coincident with those of another string if they are under a defined separation distance.

See [Resolve Points](<../COMMON/Resolve%20Points%20Dialog.md>)
