# reverse-string ("rev")

See this command in the [**command table**.](<COMMAND%20TABLE_R.md#reverse-string>)

To access this command:

  * **Digitize** ribbon **> > Condition >> Condition >> Reverse String**.

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "reverse-string"

  * Use the quick key combination "rev".

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **reverse-string** and click **Run**.

## Command Overview

Reverse the order of points for selected strings.

If there are no selected strings, you are prompted to select one. 

**Tip** : Preselect string data to reverse multiple strings.
