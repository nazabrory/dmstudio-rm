# reverse-view ("vwr")

See this command in the [**command table**.](<COMMAND%20TABLE_R.md#reverse-view>)

To access this command:

  * Use the quick key combination "vwr".

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "reverse-view".

  * On the **[Find Command](<../COMMON/findcommand.md>)** screen, highlight **reverse-view** and click **Run**.

## Command Overview

Invert the current view direction. 

Note: No loaded data values are changed with this command.

The state of the **3D** window's **[Lookat mode](<../VR_Help/vr_navigation_look_at.md>)** is important with this command:

  * If LookAt mode is active, the look at point is viewed from the opposite side.
  * If LookAt mode is inactive, the view (camera) position remains unchanged, but the view direction is rotated by 180 degrees.
