# rotate-string-by-angle ("rsn")

See this command in the [**command table**.](<COMMAND%20TABLE_R.md#rotate-string-by-angle>)

To access this command:

  * **Digitize** ribbon **> > Transform >> Rotate >> Rotate by Angle**. 

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "rotate-string-by-angle"

  * Use the quick key combination "rsn".
  * On the **[Find Command](<../COMMON/findcommand.md>)** screen, highlight **rotate-string-by-angle** and click **Run**.

## Command Overview

Rotates the selected string(s) in the current view plane, about a selected point, by a specified angle.

This command is affected by parameters that have been defined for your project. There are two possible ways that rotation can be performed:

  * Data can be rotated relative to the screen (that is, around a plane that is orthogonal to the current camera view), or;

  * Data can be rotated relative to the current data plane. This "3D" rotation allows the planar alignment of data to be maintained regardless of the position and direction of the camera.

See [Project Settings: Points and Strings](<../COMMON/Project%20Settings_Points%20and%20Strings.md>).

Command steps:

  1. Run the command.

  2. Following the prompt in the status bar to "Select rotation point on any string", select a point.

  3. In the Rotate String screen, set the angle of rotation and click OK.

The selected string rotates around the chosen point.
