# rotate-string-to-azimuth ("rsa")

See this command in the [**command table**.](<COMMAND%20TABLE_R.md#rotate-string-to-azimuth>)

To access this command:

  * **Digitize** ribbon **> > Transform >> Rotate >> Rotate to Azimuth**. 

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "rotate-string-to-azimuth"

  * Use the quick key combination "rsa".
  * On the **[Find Command](<../COMMON/findcommand.md>)** screen, highlight **rotate-string-to-azimuth** and click **Run**.

## Command Overview

Rotates a string about a selected point in the current view plane to a specified azimuth.

This command is affected by parameters that have been defined for your project. There are two possible ways that rotation can be performed:

  * Data can be rotated relative to the screen (that is, around a plane that is orthogonal to the current camera view), or;

  * Data can be rotated relative to the current data plane. This "3D" rotation allows the planar alignment of data to be maintained regardless of the position and direction of the camera.

See [Project Settings: Points and Strings](<../COMMON/Project%20Settings_Points%20and%20Strings.md>).

Command steps:

  1. Run the command. 

You are prompted from the status bar to "Select rotation point on any string".

  2. Select a point.

The **Rotate String to Azimuth** screen displays.

  3. Set the **Azimuth to rotate string to (Degrees)**. This can be any positive or negative value.

  4. Click **OK** to update the selected string around the chosen anchor point.
