# rotate-wireframe ("row")

See this command in the [command table](<COMMAND%20TABLE_R.md#rotate-wireframe>).

To access this command:

  * **Explicit** ribbon **> > Edit >> Rotate**.

  *   *   * Enter "rotate-wireframe" into the **Command** toolbar and press <ENTER>

  *   * Use the quick key combination "row" with a 3D window active.

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **rotate-wireframe** and click **Run**.

## Command Overview

Interactive command to rotate wireframe data around one or more axes. 

See [Rotate Object](<../COMMON/Rotate%20Object%20Dialog.md>).
