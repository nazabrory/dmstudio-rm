# save-view ("svi")

See this command in the [command table](<COMMAND%20TABLE_S.md#save-view>).

To access this command:

  * **Data** ribbon **> > Sections >> Save View to Section**.

  *   *   * Enter "save-view" into the **Command** toolbar and press <ENTER>

  *   * Use the quick key combination "svi".

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **save-view** and click **Run**.

## Command Overview

Saves the current view plane parameters to a section file.

Note: You can only have one section definition file open during any one session. Attempts to open another file, while one is already active, results in the original file being closed.

Command steps:

  1. Orient the active 3D window the way you want it. See [Viewing Data](<../COMMON/Interface_Viewing%20Data.md>).

  2. Run the command.

The **Section Row Properties** screen displays.

  3. Edit any section properties to fine-tune the section definition. See [Section Row Properties](<../COMMON/SectionRowProperties.md>).

  4. Click **OK**.

If it doesn't already exist, a new data object is created, called "SectionDef". If the object already exists, a new data record is added.

  5. Continue to add sections to the SectionDef object by repeating the above steps.

Tip: Save this to a physical file and share your section definitions with other projects or people.
