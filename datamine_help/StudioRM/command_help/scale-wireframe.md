# scale-wireframe

See this command in the [command table](<COMMAND%20TABLE_S.md#scale-wireframe>).

To access this command:

  * **Explicit** ribbon **> > Edit >> Scale**.

  * Enter "scale-wireframe" into the **Command** toolbar and press <ENTER>

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **scale-wireframe** and click **Run**.

## Command Overview

Interactive command to scale wireframe data on one or more axis directions.

See [Scale Object](<../COMMON/Scale%20Object%20Dialog.md>).
