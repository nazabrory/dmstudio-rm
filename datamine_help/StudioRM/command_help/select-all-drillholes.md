# select-all-drillholes ("sad")

See this command in the [**command table**.](<COMMAND%20TABLE_S.md#select-all-drillholes>)

To access this command:

  * **Home** ribbon **> > Select >> Select All > Drillholes**.

  * Use the quick key combination "sad".

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "select-all-drillholes".

  * On the **[Find Command](<../COMMON/findcommand.md>)** screen, highlight **select-all-drillholes** and click **Run**.

## Command Overview

Select all displayed static drillhole data in all **3D** views.

Command steps:

  1. Run the command

  2. all loaded static drillhole data is selected and highlighted.

**Note** : Data will only be selected if the displayed 3D object overlay is enabled on the [Overlay Selection](<../COMMON/OverlaySelectionDialog.md>) screen.
