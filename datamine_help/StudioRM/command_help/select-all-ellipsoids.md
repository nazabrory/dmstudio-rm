# select-all-ellipsoids ("sael")

See this command in the [**command table**.](<COMMAND%20TABLE_S.md#select-all-ellipsoids>)

To access this command:

  * **Home** ribbon **> > Select >> Select All > Ellipsoids**.

  * Use the quick key combination "sael".

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "select-all-ellipsoids".

  * On the **[Find Command](<../COMMON/findcommand.md>)** screen, highlight **select-all-ellipsoids** and click **Run**.

## Description

Select all displayed ellipsoid data in all **3D** views.

Command steps:

  1. Run the command

  2. All loaded ellipsoid data is selected and highlighted.

**Note** : Data will only be selected if the displayed 3D object overlay is enabled on the [Overlay Selection](<../COMMON/OverlaySelectionDialog.md>) screen.
