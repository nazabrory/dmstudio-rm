# select-all-planes ("san")

See this command in the [**command table**.](<COMMAND%20TABLE_S.md#select-all-planes>)

To access this command:

  * **Home** ribbon **> > Select >> Select All > Planes**.

  * Use the quick key combination "san".

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "select-all-planes".

  * On the **[Find Command](<../COMMON/findcommand.md>)** screen, highlight **select-all-planes** and click **Run**.

## Description

Select all loaded plane data in all **3D** views.

Command steps:

  1. Run the command

  2. all loaded plane data is selected and highlighted.

**Note** : Data will only be selected if the displayed 3D object overlay is enabled on the [Overlay Selection](<../COMMON/OverlaySelectionDialog.md>) screen.
