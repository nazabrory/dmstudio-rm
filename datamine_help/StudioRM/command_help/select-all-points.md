# select-all-points ("sap")

See this command in the [**command table**.](<COMMAND%20TABLE_S.md#select-all-points>)

To access this command:

  * **Home** ribbon **> > Select >> Select All > Points**.

  * Use the quick key combination "sap".

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "select-all-points".

  * On the **[Find Command](<../COMMON/findcommand.md>)** screen, highlight **select-all-points** and click **Run**.

## Command Overview

Selects all displayed points objects data.

  * Points can be added to the already selected points data by holding down <CTRL> whilst selecting (click) with the mouse. Reselecting an already selected point using <CTRL> deselects the point.

  * The mouse can either be clicked to select a single point, or clicked and dragged to define a selection box. Selection settings are controlled from the **Data Selection** area on the [Points and Strings](<../COMMON/Project%20Settings_Points%20and%20Strings.md>) tab of the [Project Settings](<../COMMON/ProjectSettings.md>) screen.

Command steps:

  1. Run the command

  2. All displayed points data objects are automatically selected and highlighted yellow (default).

**Note** : Data will only be selected if the displayed 3D object overlay is enabled on the [Overlay Selection](<../COMMON/OverlaySelectionDialog.md>) screen.
