# select-cursor ("scr")

See this command in the [command table](<COMMAND%20TABLE_S.md#select-cursor>).

To access this command:

  * **Format** ribbon **> > Conventions >> Cursor >> New Cursor**.

  *   *   * Enter "select-cursor" into the **Command** toolbar and press <ENTER>

  *   * Use the quick key combination "scr".

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **select-cursor** and click **Run**.

  * Double click the cursor information area in the **[Status Bar](<../COMMON/Interface_Status%20Bars.md>)**.

## Command Overview

Displays the [Custom Cursors](<../COMMON/CursorEditor.md>) screen for creating and editing bespoke cursor shapes for the active 3D window.

Tip: Each[ independent 3D window](<../COMMON/Independent_3D_Windows.md>) can have its own cursor - just activate the window then run the select-cursor command.
