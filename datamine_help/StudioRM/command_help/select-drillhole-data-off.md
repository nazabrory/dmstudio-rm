# select-drillhole-data-off

See this command in the [command table](<COMMAND%20TABLE_S.md#select-drillhole-data-off>).

To access this command:

  *   *   * Enter "select-drillhole-data-off" into the **Command** toolbar and press <ENTER>

  *   * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **select-drillhole-data-off** and click **Run**.

## Command Overview

Prohibit the inclusion of drillhole data during data selection activities.
