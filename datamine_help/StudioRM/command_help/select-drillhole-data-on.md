# select-drillhole-data-on

See this command in the [command table](<COMMAND%20TABLE_S.md#select-drillhole-data-on>).

To access this command:

  *   *   * Enter "select-drillhole-data-on" into the **Command** toolbar and press <ENTER>

  *   * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **select-drillhole-data-on** and click **Run**.

## Command Overview

Prohibit the inclusion of drillhole data during data selection activities.
