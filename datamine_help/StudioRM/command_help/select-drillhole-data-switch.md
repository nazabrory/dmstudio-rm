# select-drillhole-data-switch ("sdd")

See this command in the [command table](<COMMAND%20TABLE_S.md#select-drillhole-data-switch>).

To access this command:

  *   *   * Enter "select-drillhole-data-switch" into the **Command** toolbar and press <ENTER>

  * Use the quick key combination "sdd".

  *   * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **select-drillhole-data-switch** and click **Run**.

  * **Home** ribbon **> > Select >> Drilholes**.

## Command Overview

Toggles the selection of drillhole data on or off during data selection commands. Each use of the command toggles between two modes:

  * Drillhole data is included in data selection activities.

  * Drillhole data is ignored.

Related topics and activities:

  * [select-drillhole-data-on](<select-drillhole-data-on.md>)

  * [select-drillhole-data-off](<select-drillhole-data-off.md>)
