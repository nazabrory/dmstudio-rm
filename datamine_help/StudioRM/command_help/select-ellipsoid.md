# select-ellipsoid ("sell")

See this command in the [**command table**.](<COMMAND%20TABLE_S.md#select-ellipsoid>)

To access this command:

  * Use the quick key combination "sell".

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "select-ellipsoid".

  * On the **[Find Command](<../COMMON/findcommand.md>)** screen, highlight **select-ellipsoid** and click **Run**.

## Command Overview

Select ellipsoids to modify the selected data set. 

**Note** : Whilst ellipsoids appear as wireframe solid shapes, they are actually defined as points with parameters to specify axis dimensions and orientations.

Command steps:

  1. Run the command.

  2. Left-click a visible ellipsoid in any 3D window.

**Note** : Data will only be selected if the displayed 3D object overlay is enabled on the [Overlay Selection](<../COMMON/OverlaySelectionDialog.md>) screen.
