# select-model-data-off

See this command in the [**command table**.](<COMMAND%20TABLE_S.md#select-model-data-off>)

To access this command:

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "select-model-data-off".

  * On the **[Find Command](<../COMMON/findcommand.md>)** screen, highlight **select-model-data-off** and click **Run**.

## Command Overview

Ignore block model data during data selection commands.
