# select-model-data-on

See this command in the [**command table**.](<COMMAND%20TABLE_S.md#select-model-data-on>)

To access this command:

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "select-model-data-on".

  * On the **[Find Command](<../COMMON/findcommand.md>)** screen, highlight **select-model-data-on** and click **Run**.

## Command Overview

Include block model data during data selection commands.
