# select-model-data-switch

See this command in the [**command table**.](<COMMAND%20TABLE_S.md#select-model-data-switch>)

To access this command:

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "select-model-data-switch".

  * On the **[Find Command](<../COMMON/findcommand.md>)** screen, highlight **select-model-data-switch** and click **Run**.

  * **Home** ribbon **> > Select >> Model Cells**.

## Command Overview

Toggle the inclusion block model data during data selection commands, alternating between the following states:

  * Block model data is included in data selection.

  * Block model data is ignored during data selection.
