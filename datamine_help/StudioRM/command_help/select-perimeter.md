# select-perimeter ("spe")

See this command in the [**command table**.](<COMMAND%20TABLE_S.md#select-perimeter>)

To access this command:

  * **Home** ribbon **> > Select >> Perimeter**.

  * Use the quick key combination "spe".

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "select-perimeter".

  * On the **[Find Command](<../COMMON/findcommand.md>)** screen, highlight **select-perimeter** and click **Run**.

## Command Overview 

Select a closed string to make it the current string. No other data type is selected.

This command selects a "preferred" closed string for certain subsequent actions. The command is similar to [select-string](<select-string.md>) but will only select a closed string. This is useful if you are dealing with outlines which abut each other.

**Note** : Data will only be selected if the displayed 3D object overlay is enabled on the [Overlay Selection](<../COMMON/OverlaySelectionDialog.md>) screen.
