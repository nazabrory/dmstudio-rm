# select-plane-data-on

See this command in the [**command table**.](<COMMAND%20TABLE_S.md#select-plane-data-on>)

To access this command:

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "select-plane-data-on".

  * On the **[Find Command](<../COMMON/findcommand.md>)** screen, highlight **select-plane-data-on** and click **Run**.

## Command Overview

Include plane data during data selection commands.
