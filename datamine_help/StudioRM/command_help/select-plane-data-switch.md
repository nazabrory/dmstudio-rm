# select-plane-data-switch

See this command in the [**command table**.](<COMMAND%20TABLE_S.md#select-model-data-switch>)

To access this command:

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "select-plane-data-switch".

  * On the **[Find Command](<../COMMON/findcommand.md>)** screen, highlight **select-plane-data-switch** and click **Run**.

  * **Home** ribbon **> > Select >> Planes**.

## Command Overview

Toggle the inclusion plane data during data selection commands, alternating between the following states:

  * Plane data is included in data selection.

  * Plane data is ignored during data selection.
