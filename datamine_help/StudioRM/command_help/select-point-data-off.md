# select-point-data-off

See this command in the [**command table**.](<COMMAND%20TABLE_S.md#select-point-data-off>)

To access this command:

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "select-point-data-off".

  * On the **[Find Command](<../COMMON/findcommand.md>)** screen, highlight **select-point-date-off** and click **Run**.

## Command Overview

Ignore point data during data selection commands.
