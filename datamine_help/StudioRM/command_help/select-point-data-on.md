# select-point-data-on

See this command in the [**command table**.](<COMMAND%20TABLE_S.md#select-point-data-on>)

To access this command:

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "select-point-data-on".

  * On the **[Find Command](<../COMMON/findcommand.md>)** screen, highlight **select-point-data-on** and click **Run**.

## Command Overview

Include point data during data selection commands.
