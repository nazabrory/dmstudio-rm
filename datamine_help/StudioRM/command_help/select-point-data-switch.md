# select-point-data-switch

See this command in the [**command table**.](<COMMAND%20TABLE_S.md#select-model-data-switch>)

To access this command:

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "select-point-data-switch".

  * On the **[Find Command](<../COMMON/findcommand.md>)** screen, highlight **select-point-data-switch** and click **Run**.

  * **Home** ribbon **> > Select >> Points**.

## Command Overview

Toggle the inclusion of point data during data selection commands, alternating between the following states:

  * Point data is included in data selection.

  * Point data is ignored during data selection.
