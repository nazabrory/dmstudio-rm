# select-points ("spn")

See this command in the [**command table**.](<COMMAND%20TABLE_S.md#select-points>)

To access this command:

  * Use the quick key combination "spn".

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "select-points".

  * On the **[Find Command](<../COMMON/findcommand.md>)** screen, highlight **select-points** and click **Run**.

## Command Overview

Note: Don't confuse this command with [select-point-data-switch](<select-point-data-switch.md>) and similar commands. These set the selection mode for points, either to allow points to be selected or not. In comparison, **select-points** is a command that activates point picking mode.

Select points data interactively in a **3D** window. Data of any other type will not be selected whilst this command mode is active.

Selects points to add or remove them from the selected data set. if the <CTRL> key is held down points are added to the selected set. If the <SHIFT> key is held down, the selected state of the point is toggled.

Selected points can be moved, copied and edited using other design commands.

**Note** : Data will only be selected if the displayed 3D object overlay is enabled on the [Overlay Selection](<../COMMON/OverlaySelectionDialog.md>) screen.
