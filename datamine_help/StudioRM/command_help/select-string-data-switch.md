# select-string-data-switch

See this command in the [**command table**.](<COMMAND%20TABLE_S.md#select-string-data-switch>)

To access this command:

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "select-string-data-switch".

  * On the **[Find Command](<../COMMON/findcommand.md>)** screen, highlight **select-string-data-switch** and click **Run**.

  * **Home** ribbon **> > Select >> Strings**.

## Command Overview

Toggle the inclusion of string data during data selection commands, alternating between the following states:

  * String data is included in data selection.

  * String data is ignored during data selection.
