# select-wireframe-data-off

See this command in the [**command table**.](<COMMAND%20TABLE_S.md#select-wireframe-data-off>)

To access this command:

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "select-wireframe-plane-off".

  * On the **[Find Command](<../COMMON/findcommand.md>)** screen, highlight **select-wireframe-plane-off** and click **Run**.

## Command Overview

Ignore wireframe data during data selection commands.
