# select-wireframe-data-switch

See this command in the [**command table**.](<COMMAND%20TABLE_S.md#select-wireframe-data-switch>)

To access this command:

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "select-wireframe-data-switch".

  * On the **[Find Command](<../COMMON/findcommand.md>)** screen, highlight **select-wireframe-data-switch** and click **Run**.

  * **Home** ribbon **> > Select >> Wireframes**.

## Command Overview

Toggle the inclusion of wireframe data during data selection commands, alternating between the following states:

  * Wireframe data is included in data selection.

  * Wireframe data is ignored during data selection.
