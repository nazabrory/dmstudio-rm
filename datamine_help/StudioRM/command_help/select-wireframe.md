# select-wireframe ("sw")

See this command in the [**command table**.](<COMMAND%20TABLE_S.md#select-wireframe>)

To access this command:

  * **Home** ribbon **> > Pick >> Wireframes**.

  * Use the quick key combination "sw".

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "select-wireframe".

  * On the **[Find Command](<../COMMON/findcommand.md>)** screen, highlight **select-wireframe** and click **Run**.

## Command Overview

Note: Don't confuse this command with [select-wireframe-data-switch](<select-wireframe-data-switch.md>) and similar commands. These set the selection mode for wireframe data, either to allow data to be selected or not. In comparison, **select-wireframe** is a command that activates string picking mode.

Activates wireframe picking mode. Once active, you can use the cursor to select wireframe data according to the current [Project Settings](<../COMMON/Project%20Settings_Wireframing.md>). This command will activate wireframe picking regardless of any other settings.

Command steps:

  1. Define your wireframe selection settings,
  2. Run the command
  3. Pick wireframe data with the cursor. Depending on your selection settings either the whole wireframe object is selected, one or more triangles, data adhering to a particular **GROUP** or **SURFACE** or **ATTRIBUTE** , or a specific filter expression. 

**Note** : This command supports [**flexible wireframe selection**](<../COMMON/Wireframe_Selection_Concept.md>).
