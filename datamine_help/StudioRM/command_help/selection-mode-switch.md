# selection-mode-switch ("smt")

See this command in the [**command table**.](<COMMAND%20TABLE_S.md#selection-mode-switch>)

To access this command:

  * **Home** ribbon **> > Select >> Swipe Selection** (toggle).

  * Use the quick key combination "smt".

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "selection-mode-switch".

  * On the **[Find Command](<../COMMON/findcommand.md>)** screen, highlight **selection-mode-switch** and click **Run**.

## Command Overview

Toggle between [box](<selection-mode-box-switch.md>) and **[swipe](<selection-mode-line-switch.md>)** selection in a **3D** window. 

## How to use

  1. Activate a **3D** window and display data you want to select.
  2. Run the command.

The method of selection swaps between box and swipe mode.
