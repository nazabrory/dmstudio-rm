# sequence-numbers ("seqn")

See this command in the [**command table**.](<COMMAND%20TABLE_S.md#sequence-controls>)

To access this command:

  * Use the quick key combination "seqn".

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "sequence-controls".

  * On the **[Find Command](<../COMMON/findcommand.md>)** screen, highlight **sequence-controls** and click **Run**.

## Command Overview

Activates a series of sequence numbers in relation to a specific object attribute, to be used in other processes (for example, scheduling, visualizer animations and so on).

Command steps:

  1. Run the command.

  2. Select the string containing the attributes for sequencing.

  3. Enter the name of the attribute that relates to the sequence order data.
