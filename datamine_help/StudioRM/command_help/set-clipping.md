# set-clipping ("sca")

See this command in the [**command table**.](<COMMAND%20TABLE_S.md#set-clipping>)

To access this command:

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "set-clipping"

  * Use the quick key combination "sca"

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **set-clipping** and click **Run**.

## Command Overview

Display the [Section Properties](<../VR_Help/Section%20Properties%20Dialog.md>) screen to set 3D window section parameters, including clipping.
