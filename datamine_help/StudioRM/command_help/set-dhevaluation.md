# set-dhevaluation ("sep")

See this command in the [**command table**.](<COMMAND%20TABLE_S.md#set-dhevaluation>)

To access this command:

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "set-dhevaluation"

  * Use the quick key combination "sep".

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **set-dhevaluation** and click **Run**.

## Command Overview

Modify parameters for compositing and evaluation of drillholes.

Run the command and select an optional field to be used for sample weighting during drillhole compositing and evaluation. This is usually a **DENSITY** field. 

Note: Enter a space if no field is required.

A default weighting value can also be supplied. The default is used if the nominated field has a missing value, or if no field has been supplied at all.
