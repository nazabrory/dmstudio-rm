# set-face-angle ("fng")

See this command in the [**command table**.](<COMMAND%20TABLE_S.md#set-face-angle>)

To access this command:

  * **Digitize** ribbon **> > Transform >> Project >> Set Face Angle**. 

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "set-face-angle"

  * Use the quick key combination "fng".
  * On the **[Find Command](<../COMMON/findcommand.md>)** screen, highlight **set-face-angle** and click **Run**.

## Command Overview

Set the string projection angle (the _face angle_) for string projection commands.

Commonly, this is used to determine a face angle for open pit design, where slope constraints are vital for safety. In this case, the face angle is used to project strings (upwards or downwards) if no rosette or other override exists at the pit, pit region or bench level.

Command steps:

  1. Run the command.

The **Projection Face Angle** screen displays.

  2. Specify the Projection Face Angle to use when projecting strings.  

  3. Click OK.
