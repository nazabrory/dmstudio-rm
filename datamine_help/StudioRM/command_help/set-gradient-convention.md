# set-gradient-convention ("grc")

See this command in the [**command table**.](<COMMAND%20TABLE_S.md#set-gradient-convention>)

To access this command:

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "set-gradient-convention"

  * Use the quick key combination "grc".

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **set-gradient-convention** and click **Run**.

  * Digitize ribbon >>

  * **Format** ribbon **> > Conventions >> Gradient**.

## Command Overview

Set gradient unit and direction conventions for design commands.

Select what type of unit (percent, degrees, ratio) and direction (up/down convention for negative or positive gradients) to be reported and requested when using commands which make use of gradient control when creating or modifying strings, for example, [create-ramp-string](<create-ramp-string.md>), [adjust-to-gradient](<adjust-to-gradient.md>). 

All related commands obey these unit and convention settings, until they are changed.

Command steps:

  1. Run the command.

The [Gradient Convention](<../COMMON/GradientConvention_Dialog.md>) screen displays.

  2. Define the Gradient In and Direction settings.

  3. Click OK.

  4. Run the required design command (for example, [create-ramp-string](<create-ramp-string.md>)).
