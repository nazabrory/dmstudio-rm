# set-near-far ("scl")

See this command in the [**command table**.](<COMMAND%20TABLE_S.md#set-near-far>)

To access this command:

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "set-near-far"

  * Use the quick key combination "scl".

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **set-near-far** and click **Run**.

## Command Overview

This is a legacy command that used to set the clipping limits in the legacy **Design** window (no longer available). Now, it displays the [Environmental Settings](<../VR_Help/EnvironmentalSettings_Dialog.md>) screen where you can set **3D** window global clipping limits and a lot more.
