# set-projection-distance ("prj")

See this command in the [**command table**.](<COMMAND%20TABLE_S.md#set-projection-distance>)

To access this command:

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "set-projection-distance"

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **set-projection-distance** and click **Run**.

## Command Overview

Set the distance that a string is projected when the **[project-string-at-angle](<project-string-at-angle.md>)** command is called.

Note: Negative and positive values are accepted.
