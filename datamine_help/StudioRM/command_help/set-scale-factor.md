# set-scale-factor ("sex")

See this command in the [**command table**.](<COMMAND%20TABLE_S.md#set-scale-factor>)

To access this command:

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "set-scale-factor"

  * Use the quick key combination "sex".

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **set-scale-factor** and click **Run**.

## Command Overview

This is a legacy command that used to set the view exaggeration in the legacy **Design** window (no longer available). Now, it displays the [Environmental Settings](<../VR_Help/EnvironmentalSettings_Dialog.md>) screen where you can set **3D** window exaggeration limits and a lot more.
