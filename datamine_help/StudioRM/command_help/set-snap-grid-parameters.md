# set-snap-grid-parameters ("gs")

See this command in the [**command table**.](<COMMAND%20TABLE_S.md#set-snap-grid-parameters>)

To access this command:

  * **Home** ribbon **> > Snap >> Snap >> Grid Snapping Parameters**.

  * **Digitize** ribbon **> > Snap >> Snap Grid Parameters**.

  * Use the quick key combination "gs".

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "set-snap-grid-parameters".

  * On the **[Find Command](<../COMMON/findcommand.md>)** screen, highlight **set-snap-grid-parameters** and click **Run**.

## Command Overview

Set the snapping grid's origin and spacing parameters. 

Command steps:

  1. Run the command.

  2. In the [Snapping Grid](<../COMMON/SnappingGrid_Dialog.md>) dialog, define the Increment and Origin parameters, click **OK**.

  3. Run the required design command (editing or defining new data) and, when required, use the right-mouse button to snap to grid points.
