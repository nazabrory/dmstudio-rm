# set-snap-none-switch ("sn0")

See this command in the [**command table**.](<COMMAND%20TABLE_S.md#set-snap-none-switch>)

To access this command:

  * Use the quick key combination "sn0" (that's a zero at the end).

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "set-snap-none-switch".

  * On the **[Find Command](<../COMMON/findcommand.md>)** screen, highlight **set-snap-none-switch** and click **Run**.

## Command Overview

Disables the use of snapping during data selection and digitizing operations.
