# set-snap-surface-switch ("sts")

See this command in the [**command table**.](<COMMAND%20TABLE_S.md#set-snap-surface-switch>)

To access this command:

  * Home ribbon >> Snap >> Snap To >> Surfaces.

  * **Digitize** ribbon **> > Snap >> Snap to Surfaces**.

  * Use the quick key combination "sts".

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "set-snap-surface-switch".

  * On the **[Find Command](<../COMMON/findcommand.md>)** screen, highlight **set-snap-surface-switch** and click **Run**.

## Command Overview

Toggle snap mode to snap to object surfaces. Enable this snap mode will snap to the position on a wireframe triangle surface directly below the cursor, in the current view direction.

Command Steps

  1. Run the command.

  2. Run the required, relevant design commands which would make use of this snap mode e.g. new-string.

**Tip** : The snapping mode can be changed while in the middle of executing another command, such as defining a string. For example, the first point on a new string could be set at (4000,2000,6000) using a grid snapping mode, and then other points on the string defined by snapping to other objects by toggling a points snap mode during digitizing.
