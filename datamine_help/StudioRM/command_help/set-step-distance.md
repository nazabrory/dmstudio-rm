# set-step-distance ("ssd")

See this command in the [**command table**.](<COMMAND%20TABLE_S.md#set-step-distance>)

To access this command:

  * Model ribbon **> >**Manipulate >> Adjust to Ore >> Set Ore Categories

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "set-step-distance"

  * Use the quick key combination "ssd".

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **set-step-distance** and click **Run**.

## Command Overview

Set the step distance for string adjustment when outlining ore using the [adjust-string-in-ore](<adjust-string-in-ore.md>) command.

Enter the step distance for the amount of adjustment of each point when a string is being adjusted to an ore outline in the displayed block model. The adjustment distance can be positive or negative. The default is the current setting.

Command Steps:

  1. Run the command.

  2. Define the **Step distance for adjustment**.

  3. Click OK.
