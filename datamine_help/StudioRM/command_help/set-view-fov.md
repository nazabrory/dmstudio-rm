# set-view-fov

See this command in the [**command table**.](<COMMAND%20TABLE_S.md#set-view-fov>)

To access this command:

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "set-view-fov"

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **set-view-fov** and click **Run**.

## Command Overview

Define the current 3D window field of view in degrees. The default is 45 degrees.
