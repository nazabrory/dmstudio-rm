# show-current-files

See this command in the [**command table**.](<COMMAND%20TABLE_S.md#show-current-files>)

To access this command:

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "show-current-files"

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **show-current-files** and click **Run**.

## Command Overview

Displays the [Data Object Manager](<../COMMON/Data%20Manager%20Dialog.md>).
