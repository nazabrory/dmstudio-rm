# show-project-settings

See this command in the [**command table**.](<COMMAND%20TABLE_S.md#show-project-settings>)

To access this command:

  * **Home** ribbon **> > Project >> Settings**.

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "show-project-settings"

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **show-project-settings** and click **Run**.

## Command Overview

Displays the [Project Settings](<../COMMON/ProjectSettings.md>) screen.
