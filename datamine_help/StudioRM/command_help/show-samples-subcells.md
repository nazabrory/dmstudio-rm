# Show-samples-subcells

See this command in the [**command table**.](<COMMAND%20TABLE_S.md#show-samples-subcells>)

To access this command:

  * **Estimate** ribbon **> > Validate >> Show Samples >> Show Samples (Subcells)**.

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "show-samples"

  * On the **[Find Command](<../COMMON/findcommand.md>)** screen, highlight **show-samples-subcells** and click **Run**.

## Command Overview

This command is functionally identical to the [show-samples](<show-samples.md>) command, but allows you to select sub-cells of a sub-celled model.
