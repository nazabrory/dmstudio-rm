# show-system-options

See this command in the [**command table**.](<COMMAND%20TABLE_S.md#show-system-options>)

To access this command:

  * **Home** ribbon **> > Settings >> System**.

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "show-system-options"

  * On the **[Find Command](<../COMMON/findcommand.md>)** screen, highlight **show-system-options** and click **Run**.

Description

Display the System Options screen.
