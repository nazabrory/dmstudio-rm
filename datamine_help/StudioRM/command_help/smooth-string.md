# smooth-string ("sms")

See this command in the [**command table**.](<COMMAND%20TABLE_S.md#smooth-string>)

To access this command:

  * **Digitize** ribbon **> > Condition >> Condition >> Smooth**.

  * **Digitize** ribbon **> > Condition >> Smooth**.

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "smooth-string"

  * Use the quick key combination "sms".

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **smooth-string** and click **Run**.

## Command Overview

Smooths a string by parabolic blending which inserts new points between existing pairs of points. 

The coordinates of the original points are not changed but the course of the segment is "smoothed". Repeated running of the command on the same string will result in a string with very smooth curves.

**Tip** : preselect one or more strings to smooth them all without further data selection prompts.

To condition string data:

  1. Run the command.

  2. If you haven't previously selected string data, click on a string. New points are added to smooth the selected string.

If string data was already selected, it is automatically smoothed.

  3. Alternatively, select a string or multiple strings before running this command.

  4. Close the command by clicking **Done** or double-clicking (or tapping) anywhere in a **3D** window.
