# snap-always-switch ("sna")

See this command in the [**command table**.](<COMMAND%20TABLE_S.md#snap-always-switch>)

To access this command:

  * **Digitize** ribbon **> > Snap >> Snap Mode >> Always Snap**.

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "snap-always-switch"

  * Use the quick key combination "sna".

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **snap-always-switch** and click **Run**.

## Command Overview

Toggles the mechanism for snapping to data.

There are two possible states, which interchange with subsequent uses of the command:

  * Snapping is performed using any mouse button.

  * Snapping will only be performed with mouse button 3 (default).

**Note** : There's no need to set up snapping modes when digitizing data (say, a polygon feature) onto a wireframe map. The closest wireframe vertex or point is snapped.
