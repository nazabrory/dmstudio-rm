# snap-to-drillhole-data-on

See this command in the [**command table**.](<COMMAND%20TABLE_S.md#snap-to-drillhole-data-off>)

To access this command:

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "snap-to-drillhole-data-off"

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **snap-to-drillhole-data-off** and click **Run**.

## Command Overview

Enables snapping of the cursor to drillhole data, meaning this data type is included when 3D window data selection occurs by any interactive method.
