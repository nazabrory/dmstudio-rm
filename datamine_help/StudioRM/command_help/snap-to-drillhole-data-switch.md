# snap-to-drillhole-data-switch

See this command in the [**command table**.](<COMMAND%20TABLE_S.md#snap-to-drillhole-data-switch>)

To access this command:

  * **Home** ribbon **> > Snap >> Snap To >> Drillholes**.

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "snap-to-drillhole-data-switch".

  * On the **[Find Command](<../COMMON/findcommand.md>)** screen, highlight **snap-to-drillhole-data-switch** and click **Run**.

## Command Overview

Enable or disable snapping to static drillhole data in a **3D** view.
