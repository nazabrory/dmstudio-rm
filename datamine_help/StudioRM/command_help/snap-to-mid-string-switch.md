# snap-to-mid-string-switch ("stms")

See this command in the [**command table**.](<COMMAND%20TABLE_S.md#snap-to-mid-string-switch>)

To access this command:

  * **Home** ribbon >> **Snap** >> **Snap** >> **Snap to Mid Point**.

  * **Digitize** ribbon **> > Snap >> Snap to Mid Point**.

  * Use the quick key combination "stms".

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "set-to-mid-string-switch".

  * On the **[Find Command](<../COMMON/findcommand.md>)** screen, highlight **set-to-mid-string-switch** and click **Run**.

## Command Overview

Snaps the currently active string to the mid-point of another selected string or drillhole segment.

This function is an extension of the snap-to-point-data-switch.

With both switches active, it is possible to snap to both the mid-points and the end points of strings and drillhole segments. If the snap-to-mid-string-switch is not active, only the end-points may be snapped to.

Command steps:

  1. Run the command.

  2. Draw or extend a string using the left mouse button then, to activate the mid-segment function, right-click close to the string or drillhole segment whose mid-point the current string should end at. The string segment closest to the right-click is located, and a line is drawn to the mid point of that segment.
