# snap-to-perpendicular-switch ("stpe")

See this command in the [**command table**.](<COMMAND%20TABLE_S.md#snap-to-perpendicular-switch>)

To access this command:

  * **Home** ribbon **> > Snap >> Snap >> Snap Perpendicular**.

  * **Digitize** ribbon **> > Snap >> Snap Perpendicular**.

  * Use the quick key combination "stms".

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "snap-to-perpendicular-switch".

  * On the **[Find Command](<../COMMON/findcommand.md>)** screen, highlight **snap-to-perpendicular-switch** and click **Run**.

## Command Overview

Snaps the current, active string to a perpendicular of another, string (if possible).

Draw or extend a string using the left mouse button then, to activate the perpendicular, right-click close to where the perpendicular line should end. A perpendicular will only be drawn to a string. The string segment closest to the cursor when the mouse was right-clicked is located and a check is made for whether a perpendicular is possible or not. If no perpendicular is possible, an error message is displayed in the status bar. 

**Note** : [set-snap-lines-switch ("stl")](<set-snap-lines-switch.md>) must be active for this function to work. This command will not operate otherwise.
