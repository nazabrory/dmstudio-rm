# snap-to-plane-data-on

See this command in the [**command table**.](<COMMAND%20TABLE_S.md#snap-to-plane-data-on>)

To access this command:

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "snap-to-plane-data-on"

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **snap-to-plane-data-on** and click **Run**.

## Command Overview

Enables snapping of the cursor to [plane object data](<../COMMON/Studio%203%20Planes%20Overview.md>) (not to be confused with the [3D section definition](<../VR_Help/Sections.md>)), meaning this data type is included when 3D window data selection occurs by any interactive method.
