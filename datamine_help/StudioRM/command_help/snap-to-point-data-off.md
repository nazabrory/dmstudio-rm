# snap-to-point-data-off

See this command in the [**command table**.](<COMMAND%20TABLE_S.md#snap-to-point-data-off>)

To access this command:

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "snap-to-point-data-off"

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **snap-to-point-data-off** and click **Run**.

## Command Overview

Disables snapping of the cursor to point data, meaning this data type is not selected when 3D window data selection occurs by any interactive method.
