# snap-to-point-data-switch ("stpp")

See this command in the [**command table**.](<COMMAND%20TABLE_S.md#snap-to-point-data-switch>)

To access this command:

  * **Home** ribbon **> > Snap >> Snap To >> Points**.

  * Use the quick key combination "stpp".

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "snap-to-point-data-switch".

  * On the **[Find Command](<../COMMON/findcommand.md>)** screen, highlight **snap-to-point-data-switch** and click **Run**.

## Command Overview

Enable or disable snapping to points data in a **3D** view.
