# snap-to-string-data-on

See this command in the [**command table**.](<COMMAND%20TABLE_S.md#snap-to-string-data-on>)

To access this command:

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "snap-to-string-data-on"

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **snap-to-string-data-on** and click **Run**.

## Command Overview

Enables snapping of the cursor to string data vertices, meaning this data type is included when 3D window data selection occurs by any interactive method.
