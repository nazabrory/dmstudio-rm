# snap-to-string-data-switch ("stps")

See this command in the [**command table**.](<COMMAND%20TABLE_S.md#snap-to-string-data-switch>)

To access this command:

  * **Home** ribbon **> > Snap >> Snap To >> Strings**.

  * Use the quick key combination "stpp".

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "snap-to-string-data-switch".

  * On the **[Find Command](<../COMMON/findcommand.md>)** screen, highlight **snap-to-string-data-switch** and click **Run**.

## Command Overview

Enable or disable snapping to string data in a **3D** view.
