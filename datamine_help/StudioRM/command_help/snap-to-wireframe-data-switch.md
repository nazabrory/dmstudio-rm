# snap-to-wireframe-data-switch ("stpw")

See this command in the [**command table**.](<COMMAND%20TABLE_S.md#snap-to-wireframe-data-switch>)

To access this command:

  * **Home** ribbon **> > Snap >> Snap To >> Wireframes**.

  * Use the quick key combination "stpw".

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "snap-to-wireframe-data-switch".

  * On the **[Find Command](<../COMMON/findcommand.md>)** screen, highlight **snap-to-wireframe-data-switch** and click **Run**.

## Command Overview

Enable or disable snapping to wireframe data in a **3D** view. Wireframe snap points coincide with vertex positions, not triangle edges.
