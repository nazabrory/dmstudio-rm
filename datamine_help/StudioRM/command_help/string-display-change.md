# string-display-change ("dst")

See this command in the [**command table**.](<COMMAND%20TABLE_S.md#string-display-change>)

To access this command:

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "string-display-change"

  * Use the quick key combination "dst".

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **string-display-change** and click **Run**.

  * **Format** ribbon **> > Show/Hide >> Strings**.

## Command Overview

Toggle the display of loaded string object overlays, alternating between two states:

  * All unfiltered string overlays are displayed.

  * All unfiltered string overlays are hidden.

Note: If other object filters have been applied, these continue to be honoured when strings are displayed.
