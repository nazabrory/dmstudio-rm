# string-display-hide

See this command in the [**command table**.](<COMMAND%20TABLE_S.md#string-display-hide>)

To access this command:

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "**string-display-hide** "

  * Right-click the **Strings** folder in the **Sheets** or **Project Data** control bar and select **Hide All**.

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **string-display-hide** and click **Run**.

## Command Overview

Hide all displayed strings data overlays.
