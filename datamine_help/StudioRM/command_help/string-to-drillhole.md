# string-to-drillhole ("stdh")

See this command in the [**command table**.](<COMMAND%20TABLE_S.md#string-to-drillhole>)

To access this command:

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "string-to-drillhole"

  * Use the quick key combination "stdh".

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **string-to-drillhole** and click **Run**.

  * **Sample Analysis** ribbon **> > Plan >> String to Drillhole**.

## Command Overview

Convert a string to a drillhole using the [String to Drillhole](<../COMMON/drillhole%20id%20dialog.md>) screen.

As BHID is a mandatory field for drillholes, this command displays the Strings to Drillholes screen where you must pick either:

  * An existing **Attribute Column containing BHID** , or;

  * Enter your own **Drillhole BHID** to be used for the selected string data.

Click OK to create a drillhole based on the string data.

Note: The original string data is retained, not replaced.
