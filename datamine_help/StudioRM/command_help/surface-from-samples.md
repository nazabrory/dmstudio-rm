# surface-from-samples

See this command in the [**command table**.](<COMMAND%20TABLE_S.md#surface-from-samples>)

To access this command:

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "surface-from-samples".

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **surface-from-samples** and click **Run**.

  * **Implicit** ribbon **> > Surface >> Contact Surface**.

## Command Overview

Create a contact surface by specifying whether categorical values lie above or below the surface.

This command displays the [Create Contact Surfaces](<../STUDIO_RM/Surface_From_Samples.md>) screen.

Tip: Like other implicit modelling commands, you can [automate](<../COMMON/Create_Contact_Surfaces_Automation.md>) this command.
