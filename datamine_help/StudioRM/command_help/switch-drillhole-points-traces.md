# switch-drillhole-points-traces ("sdpt")

See this command in the [**command table**.](<COMMAND%20TABLE_S.md#switch-drillhole-points-traces>)

To access this command:

  * **Format** ribbon **> > Display Mode >> Drillholes**.

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "switch-drillhole-points-traces"

  * Use the quick key combination "sdpt".

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **switch-drillhole-points-traces** and click **Run**.

## Command Overview

Toggle between **Pixel Line** and **Points** drillhole rendering modes. This is the equivalent of editing the **Display Options** on the **[Drillhole Properties: Symbols](<../VR_Help/DHPropDialog_Segments.md>)** screen.
