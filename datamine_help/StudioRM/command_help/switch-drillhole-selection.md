# switch-drillhole-selection ("sds")

See this command in the [**command table**.](<COMMAND%20TABLE_S.md#switch-drillhole-selection>)

To access this command:

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "switch-drillhole-selection"

  * Use the quick key combination "sds".

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **switch-drillhole-selection** and click **Run**.

## Command Overview

Choose between drillhole interval selection modes when picking drillhole data in a 3D window.

The following selection modes are available:

  * **Select entire drillhole** Select all data corresponding to the target hole. The entire hole is selected.

  * **Select drillhole samples** Select drillhole intervals (as determined by **FROM** -**TO** values in the drillhole's database). This is regardless of the current display legend.

  * **Select by legend** Select data according to the legend currently assigned to the drillhole overlay. This is regardless of the legend type (unique value, range or filter). 

  * **Select by field** Select distinct value(s) of a specified drillhole attribute. For example, you could always select by the assigned lithology, even if the drillhole overlay is showing metallurgical grades.

Note: Your setting is stored with your project so doesn't have to be set each time.

Note: These settings are also available on the **Project Settings >> [Drillholes](<../COMMON/Project%20Settings_Drillholes.md>)** screen.

All options can be used in conjunction with any 3D data selection method (pick, swipe, box).

Tip: The command **[toggle-drillhole-selection](<toggle-drillhole-selection.md>)** allows you to alternate between **Select entire drillhole** and **Select drillhole samples** options without further input.
