# system-command ("osy")

See this command in the [**command table**.](<COMMAND%20TABLE_S.md#system-command>)

To access this command:

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "system-command"

  * Use the quick key combination "osy".

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **system-command** and click **Run**.

## Command Overview

This command connects the [command line](<../COMMON/Command_Toolbar.md>) to your operating system.

Command steps:

  1. Run the command.

  2. In the command line, enter an operating system command.

  3. Press <ENTER>

The operating system command is executed.

  4. Type "!" and press <ENTER> to exit system command mode.

Tip: You can also automated this command using a script to access operating system command line functions.
