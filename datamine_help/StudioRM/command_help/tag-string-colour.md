# tag-string-colour ("taco")

See this command in the [**command table**.](<COMMAND%20TABLE_T.md#tag-string-colour>)

To access this command:

  * **Explicit** ribbon **> > Create >> Tag String >> Tag String Colour**.

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "tag-string-colour"

  * Use the quick key combination "taco".

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **tag-string-colour** and click **Run**.
