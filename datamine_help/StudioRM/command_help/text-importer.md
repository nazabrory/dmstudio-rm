# text-importer

See this command in the [**command table**.](<COMMAND%20TABLE_T.md#text-importer>)

To access this command:

  * **Data** ribbon **> > Add to Project >> Text >> Multiple Tables**.

  *   * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , "text-importer"

  *   * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **text-importer** and click **Run**.

## Command Overview

The **Text Importer** tool provides configuration options to import ASCII text files. This command displays the Text Importer screen if run interactively.

See [Text Importer](<../COMMON/text-importer.md>) for more information, including scripting guidelines.
