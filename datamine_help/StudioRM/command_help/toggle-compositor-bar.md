# toggle-compositor-bar ("tcpb")

See this command in the [**command table**.](<COMMAND%20TABLE_T.md#toggle-compositor-bar>)

To access this command:

  * Home ribbon >> Show >> Compositor Bar

  *   * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , "toggle-compositor-bar".

  * Use the quick key combination "tcpb".

  *   * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **toggle-compositor-bar** and click **Run**.

## Command Overview

Show or hide the [Compositor](<../COMMON/Compositor%20Control%20Bar%20Overview.md>) control bar.
