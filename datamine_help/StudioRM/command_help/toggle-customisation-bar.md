# toggle-customisation-bar ("tcub")

See this command in the [**command table**.](<COMMAND%20TABLE_T.md#toggle-customisation-bar>)

To access this command:

  * Home ribbon >> Show >> Customisation Bar

  *   * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , "toggle-customisation-bar".

  * Use the quick key combination "tcub".

  *   * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **toggle-customisation-bar** and click **Run**.

## Command Overview

Show or hide the [Customization](<../COMMON/customization%20window.md>) control bar.
