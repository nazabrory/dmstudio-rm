# toggle-drillhole-selection ("tds")

See this command in the [**command table**.](<COMMAND%20TABLE_T.md#toggle-drillhole-selection>)

To access this command:

  * Display the **Project Settings** screen, activate the **[Drillholes](<../COMMON/Project%20Settings_Drillholes.md>)** panel and choose a selection mode.
  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "toggle-drillhole-selection".

  * Use the quick key combination "tds".

## Command Overview

Toggles between full drillhole and individual sample selection when picking data in a **3D** window. You can either select an entire drillhole when picking, or a distinct drillhole interval (this is set using the **[switch-drillhole-selection](<switch-drillhole-selection.md>)** command or **Project Settings >> [Drillholes](<../COMMON/Project%20Settings_Drillholes.md>)**.

Note: This command is another way of accessing **switch-drillhole-selection** , retained for legacy purposes. Toggling this legacy command alternates between **Select entire drillhole** and **Select drillhole samples** options.

Command steps:

  1. Load and display at least one drillhole overlay in a **3D** window.

  2. Run the **toggle-drillhole-selection** command to alternate between full drillhole and individual sample selection modes.
