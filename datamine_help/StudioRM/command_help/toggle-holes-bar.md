# toggle-holes-bar ("thb")

See this command in the [**command table**.](<COMMAND%20TABLE_T.md#toggle-holes-bar>)

To access this command:

  * Home ribbon >> Show >> Holes Bar

  *   * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , "toggle-holes-bar".

  * Use the quick key combination "thb".

  *   * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **toggle-holes-bar** and click **Run**.

## Command Overview

Show or hide the [Holes](<../COMMON/Holes%20Control%20Bar%20Overview.md>) control bar.
