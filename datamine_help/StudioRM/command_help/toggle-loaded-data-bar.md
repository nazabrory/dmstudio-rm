# toggle-loaded-data-bar ("tldb")

Note: The functionality of the control bar is also available using the Project Data control bar, where you can access functions related to project files, loaded data objects and 3D overlays.

See this command in the [**command table**.](<COMMAND%20TABLE_T.md#toggle-loaded-data-bar>)

To access this command:

  * Home ribbon >> Show >> Loaded Data Bar

  *   * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , "toggle-loaded-data-bar".

  * Use the quick key combination "tldb".

  *   * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **toggle-loaded-data-bar** and click **Run**.

## Command Overview

Show or hide the [Loaded Data](<../COMMON/Loaded%20Data%20Control%20Bar.md>) control bar.
