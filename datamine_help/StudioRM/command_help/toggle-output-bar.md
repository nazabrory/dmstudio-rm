# toggle-output-bar ("tob")

See this command in the [**command table**.](<COMMAND%20TABLE_T.md#toggle-output-bar>)

To access this command:

  * Home ribbon >> Show >> Output Bar

  *   * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , "toggle-output-bar".

  * Use the quick key combination "tob".

  *   * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **toggle-output-bar** and click **Run**.

## Command Overview

Show or hide the [Output Control Bar](<../COMMON/Output%20Control%20Bar%20Overview.md>) control bar.
