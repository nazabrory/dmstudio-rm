# toggle-project-data-bar ("tpdb")

See this command in the [**command table**.](<COMMAND%20TABLE_T.md#toggle-project-data-bar>)

To access this command:

  * Home ribbon >> Show >> Project Data Bar

  *   * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , "toggle-project-data-bar".

  * Use the quick key combination "tpdb".

  *   * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **toggle-project-data-bar** and click **Run**.

## Command Overview

Show or hide the Project Data control bar.
