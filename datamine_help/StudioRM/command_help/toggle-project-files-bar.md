# toggle-project-files-bar ("tpfb")

Note: The functionality of the control bar is also available using the Project Data control bar, where you can access functions related to project files, loaded data objects and 3D overlays.

See this command in the [**command table**.](<COMMAND%20TABLE_T.md#toggle-project-files-bar>)

To access this command:

  * Home ribbon >> Show >> Project Files Bar

  *   * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , "toggle-project-files-bar".

  * Use the quick key combination "tpfb".

  *   * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **toggle-project-files-bar** and click **Run**.

## Command Overview

Show or hide the [Project Files](<../COMMON/Concept_Project%20Files%20Control%20Bar%20Overview.md>) control bar.
