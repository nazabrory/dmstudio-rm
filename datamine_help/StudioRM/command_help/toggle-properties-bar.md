# toggle-properties-bar ("tpb")

See this command in the [**command table**.](<COMMAND%20TABLE_T.md#toggle-properties-bar>)

To access this command:

  * Home ribbon >> Show >> Properties Bar

  *   * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , "toggle-properties-bar".

  * Use the quick key combination "tpb".

  *   * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **toggle-properties-bar** and click **Run**.

## Command Overview

Show or hide the [Properties](<../COMMON/properties%20control%20bar%20overview.md>) control bar.
