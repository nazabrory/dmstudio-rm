# toggle-quick-filter-bar ("tqpb")

See this command in the [**command table**.](<COMMAND%20TABLE_T.md#toggle-quick-filter-bar>)

To access this command:

  * Home ribbon >> Show >> Quick Filter Bar

  *   * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , "toggle-quick-filter-bar".

  * Use the quick key combination "tqpb".

  *   * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **toggle-quick-filter-bar** and click **Run**.

## Command Overview

Show or hide the [Quick Filter](<../COMMON/QuickFilterLegendDialog.md>) control bar.
