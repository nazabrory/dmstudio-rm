# toggle-sheets-bar ("tsb")

See this command in the [**command table**.](<COMMAND%20TABLE_T.md#toggle-sheets-bar>)

To access this command:

  * Home ribbon >> Show >> Sheets Bar

  *   * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , "toggle-sheets-bar".

  * Use the quick key combination "tsb".

  *   * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **toggle-sheets-bar** and click **Run**.

## Command Overview

Show or hide the [Sheets](<../COMMON/Sheets%20Control%20Bar%20Overview.md>) control bar.
