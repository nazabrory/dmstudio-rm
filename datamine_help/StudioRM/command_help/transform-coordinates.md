# transform-coordinates ("tco")

See this command in the [**command table**.](<COMMAND%20TABLE_T.md#transform-coordinates>)

To access this command:

  * **Sample Analysis** ribbon **> > Transform >> Geographic**.

  * **Data** ribbon **> > Transform >> Geographic**.

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "transform-coordinates"

  * Use the quick key combination "tco"

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **transform-coordinates** and click **Run**.

## Command Overview

Transform a 3D data file's coordinates from one coordinate system to another. Displays the [Transform Coordinates](<../COMMON/Transform_Coordinates_Dialog.md>) screen.

**Note** : use the **[TRANSCO](<../Process_Help_XML/transco.md>)** process to convert file-based coordinate data between systems.

To convert loaded data coordinates from one well-known or custom system to another:

  1. Run the command.

The **Transform Coordinates** screen displays.

  2. Define the Source File and Target File names and parameters. See [Transform Coordinates](<../COMMON/Transform_Coordinates_Dialog.md>).

  3. Define the Source Coordinate System and Target Coordinate System parameters.

  4. Click OK.
