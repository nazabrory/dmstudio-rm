## Command Overview

Translate one or more selected data by a defined distance in each of the X, Y and Z directions, optionally repeating the translation a number of times. The screen displayed for this command is the same for target point, string and wireframe data.

**Tip** : preselecting data means you don't have to click on multiple items when running this command.

Command steps:

  1. Optionally, preselect the data you wish to translate.

  2. Run any of the translation commands.

**Note** : Any combination of data (string, point or wireframe) can be translated with each run of a command, and it doesn't matter which command was used to display the settings screen as this is identical regardless of how you display it.

  3. If data wasn't previously selected, select the required data in any **3D** window.

  4. Using the Enter Translation Distance screen, choose how you wish to translate the selected data:

     * Choose By XYZ Distance to enter a **Translation Distance** along each of the major axes.

     * Choose By Slope Distance to enter the **Azimuth** , **Gradient** and **Distance** to translate data. 

  5. Choose if you wish to **Repeat** the translation a number of times (say, to produce bench crest strings at fixed intervals from the target string or strings). 

  6. If you wish to retain the original data and create a copy of it at the new location, **check****Keep Original Object**. If **unchecked** , data is moved and the original data is not retained.

  7. Click OK.

Data is either moved or copied to the translated position(s).
