# trim-to-string ("tri")

See this command in the [**command table**.](<COMMAND%20TABLE_T.md#trim-to-string>)

To access this command:

  * **Digitize** ribbon **> > Tools >> Clip Strings >> Trim to String**.

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "trim-to-string".

  * Use the quick key combination "tri".

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **trim-to-string** and click **Run**.

## Command Overview

Trim strings against a selected control string. 

Command steps:

  1. Select a string to control the trimming of other strings. 

This string can be open or closed but must not contain any cross overs.

  2. Subsequent string segment selections are removed.
