# undo-last-edit (<CTRL>+Z)

See this command in the [**command table**.](<COMMAND%20TABLE_U.md#undo-last-edit>)

To access this command:

  * **Home** ribbon >> ![](../Images/Icons/Undo-last-edit.png)

  * **Digitize** ribbon >> ![](../Images/Icons/Undo-last-edit.png)

  * **Home** ribbon >> **Edit** >> **Undo**

  * Activate any **3D** window and select <CTRL> \+ <Z>.

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "undo-last-edit".

  * On the **[Find Command](<../COMMON/findcommand.md>)** screen, highlight **undo-last-edit** and click **Run**.

## Command Overview

Undo the last modification made during editing, or earlier modifications using repeated commands. This command will undo the effect of recent 3D window editing actions.
