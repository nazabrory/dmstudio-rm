# uniform-conditioning

See this command in the [**command table**.](<COMMAND%20TABLE_U.md#uniform-conditioning>)

To access this command:

  * **Estimate** ribbon **> > Non-linear >> UC**.

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "uniform-conditioning"

## Command Overview

This command launches the **[Uniform Conditioning](<../Uniform_Conditioning/UniformConditioning_Introduction.md>)** wizard. Uniform Conditioning helps you to optimize the accuracy of your predicted [recoverable resource](<../Uniform_Conditioning/About_Recoverable_Resources.md>) estimates and access the information you have available regarding recoveries predicted at the mining (SMU) scale.
