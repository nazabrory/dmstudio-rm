# unload-all-data ("ua")

See this command in the [**command table**.](<COMMAND%20TABLE_U.md#unload-all-data>)

To access this command:

  * **Data** ribbon **> > Unload >> Unload All**.

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "unload-all-data"

  * Use the quick key combination "ua".

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **unload-all-data** and click **Run**.

## Command Overview

Unload all loaded data from memory, updating all data windows.

Note: This command will not remove the association with the project file (if there is one). As such unloaded data sets are loaded the next time the project is loaded if they are not specified to be permanently unloaded on closing the project.
