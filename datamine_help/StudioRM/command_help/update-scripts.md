# update-scripts

See this command in the [**command table**.](<COMMAND%20TABLE_U.md#update-scripts>)

To access this command:

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "update-scripts"

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **update-scripts** and click **Run**.

  * **Home** ribbon **> > Automate >> Script >> Update Scripts**.

## Command Overview

Displays the **[Datamine Studio Script Updater](<../COMMON/Script-updater-tool.md>)** screen to analyse, detect and automatically convert potentially risky script elements to safer alternatives.
