# use-clipping-switch ("uc")

See this command in the [**command table**](<COMMAND%20TABLE_U.md#use-clipping-switch>).

To access this command:

  * Use the quick key combination "uc".

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "use-clipping-switch".

  * On the **[Find Command](<../COMMON/findcommand.md>)** screen, highlight **use-clipping-switch** and click **Run**.

## Command Overview

Turn on or off data clipping limits in **3D** or **Plots** windows.

Data clipping limits can be defined using the ribbon system, or using the [set-near-far](<set-near-far.md>) command, allowing you to specify how far in front of and behind the current view plane data should be displayed.

The **use-clipping-switch** command controls whether or not these limits are active.

This command will automatically update the status of the clipping toggles in the ribbon menus.
