# use-quick-commands-switch

See this command in the [**command table**.](<COMMAND%20TABLE_U.md#use-quick-commands-switch>)

To access this command:

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "use-quick-commands-switch".

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **use-quick-commands-switch** and click **Run**.

## Command Overview

Toggle on or off the use of [Quick Keys](<../COMMON/Studio%203%20Quick%20Keys.md>) to run commands.

Note: You can use <ESC> to cancel commands regardless of the status of this switch.
