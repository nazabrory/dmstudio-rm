# use-secondary-clipping-switch ("u2")

See this command in the [**command table**.](<COMMAND%20TABLE_U.md#use-secondary-clipping-switch>)

To access this command:

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "use-secondary-clipping-switch".

  * Use the quick key combination "u2".

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **use-secondary-clipping-switch** and click **Run**.

  * **3D View** ribbon **> > Clip >> Secondary** (primary clipping must be in use).

## Command Overview

Clip data to specified secondary clipping zones, in relation to the active 3D section. All data within the clipped region is shown with secondary clipping formatting.
