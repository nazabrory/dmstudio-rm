# use-tag-switch ("uta")

See this command in the [**command table**.](<COMMAND%20TABLE_U.md#use-tag-switch>)

To access this command:

  * **Explicit** ribbon **> > Create >> Tag String >> Use Tags**.

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "use-tag-switch"

  * Use the quick key combination "uta".

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **use-tag-switch** and click **Run**.

## Command Overview

Toggle the use of tag strings on or off for string linking.

Tag strings provide additional control on string linking during wireframing. This toggle controls whether the tag strings should be ignored when creating the link. 

Each successive use of the command toggles its state between on and off.
