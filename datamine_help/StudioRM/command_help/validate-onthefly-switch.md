# validate-onthefly-switch ("auv")

See this command in the [**command table**.](<COMMAND%20TABLE_V.md#validate-onthefly-switch>)

To access this command:

  *   *   * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "validate-onthefly-switch".

  *   * Use the quick key combination "auv".

  *   * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **validate-onthefly-switch** and click **Run**.

## Command Overview

Toggle the use of a [validation file](<../COMMON/filetype.md#Validation_File>) opened with the [open-validation-file](<open-validation-file.md>) command.

Set to active to use a validation file once it has been opened. If inactive, a loaded validation file is ignored.
