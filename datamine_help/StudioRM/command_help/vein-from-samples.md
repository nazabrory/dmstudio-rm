# vein-from-samples 

See this command in the [**command table**.](<COMMAND%20TABLE_V.md#vein-from-samples>)

To access this command:

  * Activate the **Geology** ribbon and select **Surfaces >> Vein**.

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "vein-from-samples".

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **vein-from-samples** and click **Run**.

## Command Overview

Model vein structures using input desurveyed drillhole data.

The [Create Vein Surfaces](<../COMMON/Create_Vein_Surfaces_Overview.md>) task models closed volumes representing vein deposits between positive sample points.

Note: A Datamine [eLearning course](<https://datamine.learnupon.com/>) is available that covers functions described in this topic. Contact your local Datamine office for more details.
