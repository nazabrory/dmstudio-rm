# vertical-length-switch ("plv")

See this command in the [**command table**.](<COMMAND%20TABLE_V.md#vertical-length-switch>)

To access this command:

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "vertical-length-switch".

  * Use the quick key combination "plv".

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **vertical-length-switch** and click **Run**.

## Command Overview

Toggle whether the **HLENGTH** field in a blast patterns file is an inclined or vertical length. The **Status bar** indicates which mode is currently active, when the command is run.
