# view-by-one-point ("v1")

See this command in the [**command table**.](<COMMAND%20TABLE_V.md#view-by-one-point>)

To access this command:

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "view-by-one-point ".

  * Use the quick key combination "v1"

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **view-by-one-point** and click **Run**.

## Command Overview

Set the active 3D window view position and orientation based on a selected point and orientation.

Note: This command shouldn't be confused with the [plane-by-one-point ("1")](<plane-by-one-point.md>) command which sets the orientation of the current 3D window section plane (although the view can be automatically oriented to this section afterwards, optionally).

Command steps:

  1. Display any **3D** window.

Note: This command is also supported in [external 3D windows](<../COMMON/External_3D_Windows.md>).

  2. Run the command.

  3. Left or right (snap) click to a point in the 3D window. This becomes the focal point of the 3D camera.

The **[Orientation](<../VR_Help/Section%20Orientation%201%20Point.md>)** screen displays.

  4. Select a view **Orientation** option:

     * Horizontal

     * North-South

     * East-West

  5. Click **OK**.

The view orients to point at the focal point, along the selected axis.

Note: Ignore the **Align view to section** option. The **Orientation** screen is also used by the section modification commands ([plane-by-one-point](<plane-by-one-point.md>), for example), where this option is relevant.
