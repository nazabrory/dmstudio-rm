# view-by-three-points ("v3")

See this command in the [**command table**.](<COMMAND%20TABLE_V.md#view-by-three-points>)

To access this command:

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "view-by-three-points".

  * Use the quick key combination "v3"

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **view-by-three-points** and click **Run**.

## Command Overview

Set the active 3D window view position and orientation based on 3 points. No further screens display as the three points fully define the camera alignment.

Note: This command shouldn't be confused with the [plane-by-three-points ("3")](<plane-by-three-points.md>) command which sets the orientation of the current 3D window section plane.

Command steps:

  1. Display any **3D** window.

Note: This command is also supported in [external 3D windows](<../COMMON/External_3D_Windows.md>).

  2. Run the command.

  3. Left or right (snap) click 3 points in the 3D window.

Tip: You can points in multiple 3D windows. This can be useful to precisely align the view with landmark data points.

The view orients to point at the focal point, along the selected axis.
