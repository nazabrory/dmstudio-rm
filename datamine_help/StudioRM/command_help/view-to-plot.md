# view-to-plot

See this command in the [**command table**.](<COMMAND%20TABLE_V.md#view-to-plot>)

To access this command:

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "view-to-plot".

  * Use the quick key combination "vf"

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **view-to-plot** and click **Run**.

## Command Overview

Store the the current 3D window contents and view direction in a legacy plot file.

Command steps:

  1. Orient your 3D window view and load data as required.

  2. Run the command.

  3. Select the plot file to use as storage, using the **Project Browser** screen.

  4. Click **OK**.

See [create-plot-view ("cpv")](<create-plot-view.md>).
