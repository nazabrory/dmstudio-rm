# vr-to-plot-section ("vrps")

See this command in the [**command table**.](<COMMAND%20TABLE_V.md#vr-to-plot-section>)

To access this command:

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "vr-to-plot-section".

  * Use the quick key combination "vrps".

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **vr-to-plot-section** and click **Run**.

## Command Overview

Create a new plot sheet called "Plan" if one doesn't already exist (or update an existing one of that name) using latest 3D window orientation and overlay information.
