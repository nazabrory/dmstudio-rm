# vr-to-plot-view("vrpv")

See this command in the [**command table**.](<COMMAND%20TABLE_V.md#vr-to-plot-view>)

To access this command:

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "vr-to-plot-view".

  * Use the quick key combination "vrpv".

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **vr-to-plot-view** and click **Run**.

## Command Overview

Create a new isometric plot sheet called "3D" if one doesn't already exist (or update an existing one of that name) using latest 3D window orientation and overlay information.
