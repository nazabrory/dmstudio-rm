# wf-attr-from-str-switch ("tsa")

See this command in the [**command table**.](<COMMAND%20TABLE_W.md#wf-attr-from-str-switch>)

To access this command:

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "wf-attr-from-str-switch"

  * Use the quick key combination "tsa".

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **wf-attr-from-str-switch** and click **Run**.

## Command Overview

Toggle the source of wireframe attributes when linking string data. Choose between the source strings and a bespoke value you provide.

When creating a new wireframe object using the [string linking process](<link-strings.md>), running this command allows wireframe attributes to be either taken from the string, or from the value which you enter in the wireframe attributes panel, in the Current Objects toolbar.
