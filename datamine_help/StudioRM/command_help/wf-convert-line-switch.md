# wf-convert-line-switch ("twcl")

See this command in the [**command table**.](<COMMAND%20TABLE_W.md#wf-convert-line-switch>)

To access this command:

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "wf-convert-line-switch"

  * Use the quick key combination "twcl".

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **wf-convert-line-switch** and click **Run**.

## Command Overview

Toggle the conversion of displayed lines to strings when using [wireframe-verify ("wvf")](<wireframe-verify.md>).

This switch alternates between the following states:

  * Convert detected boundaries, merging edges and self-intersections of a wireframe are converted to new string data during verification.

  * Do not converted data to string data during verification.
