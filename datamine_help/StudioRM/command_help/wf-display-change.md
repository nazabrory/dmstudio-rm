# wf-display-change ("dwf")

See this command in the [**command table**.](<COMMAND%20TABLE_S.md#wf-display-change>)

To access this command:

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "wf-display-change"

  * Use the quick key combination "dwf".

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **wf-display-change** and click **Run**.

  * **Format** ribbon **> > Show/Hide >> Wireframes**.

## Command Overview

Toggle the display of loaded wireframe object overlays, alternating between two states:

  * All unfiltered wireframe overlays are displayed.

  * All unfiltered wireframe overlays are hidden.

Note: If other object filters have been applied, these continue to be honoured when wireframes are displayed.
