# wf-display-show

See this command in the [**command table**.](<COMMAND%20TABLE_S.md#wf-display-hide>)

To access this command:

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "**wf-display-show** "

  * Right-click the **Wireframes** folder in the **Sheets** or **Project Data** control bar and select **Show All**.

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **wf-display-show** and click **Run**.

## Command Overview

Show all unfiltered strings data overlays.

Note: If other object filters have been applied, these continue to be honoured when wireframes are displayed.
