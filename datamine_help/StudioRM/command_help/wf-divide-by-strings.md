# wf-divide-by-strings 

See this command in the [**command table**.](<COMMAND%20TABLE_S.md#wf-divide-by-strings>)

To access this command:

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "wf-divide-by-strings"

  * **Wireframe** ribbon **> > Operations >> Plane >> Divide by Strings**.

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **wf-divide-by-strings** and click **Run**.

## Command Overview

Split a wireframe into multiple parts using one or more selected strings.

Command steps:

  1. Load or create 1 or more strings to act as wireframe 'cutters'. String data can be open or closed.

Note: It must be possible to cleanly split wireframe data by the string(s). Open strings that do not fully intersect the wireframe surface are ignored.

  2. Load and display the wireframe data to be split. This can be an open surface or closed volume.

  3. Select the strings to act as cutters.

  4. Run the command.

The **Divide Wireframe** screen displays. 

  5. Define wireframe splitting parameters. See [Divide Wireframe](<../COMMON/Wireframe_DivideWireframe_Dialog.md>).

  6. Click OK.
