# wf-intersection-switch ("tiw")

See this command in the [**command table**.](<COMMAND%20TABLE_W.md#wf-intersection-switch>)

To access this command:

  * **Digitize** ribbon **> > Project >> Intersect Wireframe**.

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "wf-intersection-switch"

  * Use the quick key combination "tiw".

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **wf-intersection-switch** and click **Run**.

## Command Overview

Turn on or off wireframe intersection when projecting string data.

  * If active, projecting a string the projection will stop when it intersects a wireframe. 

  * If turned off, wireframes are ignored during string projection.
