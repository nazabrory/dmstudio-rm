# wf-select-attribute-switch ("sba")

See this command in the [**command table**.](<COMMAND%20TABLE_W.md#wf-select-attribute-switch>)

To access this command:

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "wf-select-attribute-switch"

  * Use the quick key combination "sba".

  * **[Project Settings](<../COMMON/ProjectSettings.md>)** screen **> > [Wireframes](<../COMMON/Project%20Settings_Wireframing.md>) >> Select by Attribute**.

  * **Home** ribbon **> > Select** menu and pick _WF: Attribute_. Ensure toggle is active.

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **wf-select-attribute-switch** and click **Run**.

## Command Overview

Activate the selection of wireframe data by user attributes. \

The attribute used to group wireframe data by unique value is specified on the **Project Settings** screen. All data carrying the same attribute value (say, **Pushback_ID** or **GEOLOGY**) is selected when picked.

Note: This isn't technically a switch (other than one that only switches things on). Running the command only activates wireframe selection by attribute mode. To change this, choose another wireframe selection method.
