# wf-select-field-switch ("sbfd")

See this command in the [**command table**.](<COMMAND%20TABLE_W.md#wf-select-field-switch>)

To access this command:

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "wf-select-field-switch"

  * Use the quick key combination "sbfd".

  * **[Project Settings](<../COMMON/ProjectSettings.md>)** screen **> > [Wireframes](<../COMMON/Project%20Settings_Wireframing.md>) >> Select by Field**.

  * **Home** ribbon **> > Select** menu and pick _WF: Field_ and enter a field name below. Ensure toggle is active.

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **wf-select-field-switch** and click **Run**.

## Command Overview

Control the selection of wireframe data by field matching. 

Select wireframe data matching the specified field attribute of the triangle selected with the cursor (for example, **LINK** will cause the wireframe to be selected by link). 

Note: If the requested field is not present in the selected wireframe, and error message displays.
