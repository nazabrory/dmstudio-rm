# wf-select-group-switch ("sbg")

See this command in the [**command table**.](<COMMAND%20TABLE_W.md#wf-select-group-switch>)

To access this command:

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "wf-select-group-switch"

  * Use the quick key combination "sbg".

  * **[Project Settings](<../COMMON/ProjectSettings.md>)** screen **> > [Wireframes](<../COMMON/Project%20Settings_Wireframing.md>) >> Select by Group**.

  * **Home** ribbon **> > Select** menu and pick _WF: Group_. Ensure toggle is active.

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **wf-select-group-switch** and click **Run**.

## Command Overview

Control the selection of wireframe data by picked wireframe group. 

If enabled, you can select wireframe data matching the wireframe GROUP value of a triangle picked with the cursor.

Note: **GROUP** fields are not added to wireframe data by default. You'll need to add one to your wireframe for this mode to have any impact on wireframe data selection.
