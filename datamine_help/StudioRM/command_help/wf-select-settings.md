# wf-select-settings ("sbp")

See this command in the [**command table**.](<COMMAND%20TABLE_W.md#wf-select-settings>)

To access this command:

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "wf-select-settings"

  * Use the quick key combination "sbp".

  * **[Project Settings](<../COMMON/ProjectSettings.md>)** screen **> > [Wireframes](<../COMMON/Project%20Settings_Wireframing.md>)**.

  * **Home** ribbon **> > Select** menu and expand the wireframe options list. Select **Wireframe Settings**.

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **wf-select-settings** and click **Run**.

## Command Overview

Display the [Project Settings: Wireframing](<../COMMON/Project%20Settings_Wireframing.md>) screen to change the way you select wireframe data interactively in a 3D window.
