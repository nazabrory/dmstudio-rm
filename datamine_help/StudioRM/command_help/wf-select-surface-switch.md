# wf-select-surface-switch ("sbs")

See this command in the [**command table**.](<COMMAND%20TABLE_W.md#wf-select-surface-switch>)

To access this command:

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "wf-select-surface-switch"

  * Use the quick key combination "sbs".

  * **[Project Settings](<../COMMON/ProjectSettings.md>)** screen **> > [Wireframes](<../COMMON/Project%20Settings_Wireframing.md>) >> Select by Surface**.

  * **Home** ribbon **> > Select** menu and pick _WF: Surface_. Ensure toggle is active.

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **wf-select-surface-switch** and click **Run**.

## Command Overview

Control the selection of wireframe data by picked wireframe surface. If active, picking wireframe data results in all data of the same surface being selected. A surface is defined as a contiguous group of wireframe triangles. 

This option, then, could be useful where independent wireframe volumes or surfaces exist in the loaded data, but there are no other attributes to distinguish them. Using the surface selection mode, you could pick each distinct surface and use [edit-wireframe-attributes](<edit-wireframe-attributes.md>) to apply unique values. This allows you to use [wf-select-field-switch ("sbfd")](<wf-select-field-switch.md>) in future for easier data selection.
