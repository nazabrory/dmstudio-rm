# wf-select-triangles-switch ("sbt")

See this command in the [**command table**.](<COMMAND%20TABLE_W.md#wf-select-triangles-switch>)

To access this command:

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "wf-select-triangles-switch"

  * Use the quick key combination "sbt".

  * **[Project Settings](<../COMMON/ProjectSettings.md>)** screen **> > [Wireframes](<../COMMON/Project%20Settings_Wireframing.md>) >> Select Individual Triangles**.

  * **Home** ribbon **> > Select** menu and pick _WF: Triangles_. Ensure toggle is active.

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **wf-select-attribute-switch** and click **Run**.

## Command Overview

Allow individual triangle selection mode when selecting wireframe data interactively.

Triangles can be selected independently with single clicking, or you can select triangles using the box or swipe data selection methods.

Note: All triangles along the "line of sight" are selected, so for complex scenes involving many wireframes, you may want to filter the view beforehand to ensure only intended data is selected.

Tip: Many wireframe manipulation commands, including [translate-wireframe](<translate-wireframe.md>), [smooth-wireframe](<wireframe-smooth.md>) and more work with independent triangle selections.
