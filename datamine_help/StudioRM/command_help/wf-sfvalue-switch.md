# wf-sfvalue-switch ("twn")

See this command in the [**command table**.](<COMMAND%20TABLE_W.md#wf-sfvalue-switch>)

To access this command:

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "wf-sfvalue-switch"

  * Use the quick key combination "twn".

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **wf-sfvalue-switch** and click **Run**.

## Command Overview

Toggle the renumbering of wireframe surfaces off or on during verify.

In the [wireframe-verify](<wireframe-verify.md>) command, the triangle adjacency information is updated by default (meaning the toggle is on) and to number each discrete surface independently. 

This enables surfaces as defined by triangle geometry to be manipulated independently. If you wish to retain the surface numbers generated by previous linking procedures, then this toggle should be off.
