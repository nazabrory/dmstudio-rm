# wf-slice-switch ("twsl")

See this command in the [**command table**.](<COMMAND%20TABLE_W.md#wf-slice-switch>)

To access this command:

  * **3D View** ribbon **> > Display Mode >> Wireframe**.

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "wf-slice-switch"

  * Use the quick key combination "twsl".

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **wf-slice-switch** and click **Run**.

## Command Overview

Turn on or off the display of wireframe data as an intersection string in the target **3D** window. When enabled, this setting overrides any per-overlay general Shading method set on the [Wireframe Properties: General](<../VR_Help/Wireframe_Properties_Dialog.md>) screen.

Tip: This can be a useful way of 'slicing' your loaded wireframe data quickly and automatically across the entire 3D scene, without losing per-overlay formatting options. Once the mode is deactivated, previous wireframe 3D settings are reapplied.
