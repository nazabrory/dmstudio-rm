# wireframe-clean ("wcl")

See this command in the [**command table**.](<COMMAND%20TABLE_W.md#wireframe-clean>)

To access this command:

  * **Wireframe** ribbon **> > Fix >> Clean**.

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "wireframe-clean"

  * Use the quick key combination "wcl".

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **wireframe-clean** and click **Run**.

## Command Overview

Fix many problems in wireframes found during the [verify](<wireframe-verify.md>) command.

You can either clean a full, selected wireframe object or preselected wireframe triangles. The latter option can be useful when rectifying crossovers or near-coincident points in imported surface data, and the problematic area is known, for example.

Displays the [Clean Wireframe](<../COMMON/Wireframe%20Clean%20Dialog.md>) screen. Follow this link for more information.
