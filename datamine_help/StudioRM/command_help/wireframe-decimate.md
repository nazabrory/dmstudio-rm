# wireframe-decimate 

See this command in the [**command table**.](<COMMAND%20TABLE_W.md#wireframe-decimate>)

To access this command:

  * **Wireframe** ribbon **> > Operations >> Decimate**.

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "wireframe-decimate"

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **wireframe-decimate** and click **Run**.

## Command Overview

Reduces the amount of data points in a wireframe by reducing the number of edge and wireframe faces according to defined criteria.

Running this command displays the [Decimate Wireframe](<../COMMON/Wireframe%20Decimate%20Dialog.md>) screen.
