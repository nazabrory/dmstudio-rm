# wireframe-dynamic-evaluation ("wde")  
  
See this command in the [**command table**.](<COMMAND%20TABLE_W.md#wireframe-dynamic-evaluation>)

To access this command:

  * **Report** ribbon **> > Evaluate Dynamic >> Wireframes**.

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "wireframe-dynamic-evaluation"

  * Use the quick key combination "wde".

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **wireframe-dynamic-evaluation** and click **Run**.

## Command Overview

Evaluate an open or closed wireframe object using dynamic evaluation methods. his command is only available if (at least) one block model and wireframe object is in memory.

See **[Evaluation Wireframe Properties](<../COMMON/EvaluationWireframePropertiesDialog.md>)**.
