# wireframe-intersection ("win")

See this command in the [**command table**.](<COMMAND%20TABLE_W.md#wireframe-intersection>)

To access this command:

  *   * **Wireframe** ribbon **> > Boolean >> Intersection**.

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "wireframe-intersection"

  * Use the quick key combination "win".

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **wireframe-intersection** and click **Run**.

## Command Overview

Take two wireframe objects (or wireframe triangle selections, or a combination) and create a new wireframe object representing their overlapping (common) volume.

**Note** : this command is also available using the [BOOLEAN](<../Process_Help_XML/boolean.md>) process (@METHOD=3)

Note: the order in which inputs are chosen is not important for this command.

**Note** : This command supports [**flexible wireframe selection**](<../COMMON/Wireframe_Selection_Concept.md>).

For more detailed command steps, see [Wireframe Intersection](<../COMMON/Wireframe%20Intersection%20Dialog.md>).
