# wireframe-merge ("wme")

See this command in the [**command table**.](<COMMAND%20TABLE_W.md#wireframe-merge>)

To access this command:

  *   * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "wireframe-merge"

  * Use the quick key combination "wme".

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **wireframe-merge** and click **Run**.

## Command Overview

This command launches the [Extract Separate](<wireframe-extract-separate.md>) screen, preconfigured to merge two nominated wireframe objects into a single output data object.

**Tip** : you can also merge wireframe objects using the **Copy object from...** content menu option in the **Sheets** and **Project Data** control bars. You can also use the **[Combine Data Objects](<../COMMON/Data%20Objects%20Combine%20Dialog.md>)** command.
