# wireframe-section-multiple ("msl")

See this command in the [**command table**.](<COMMAND%20TABLE_W.md#wireframe-section-multiple>)

To access this screen:

  * **Structure** ribbon **> > Operations >> Plane >> Multiple Section**.

  * **Wireframe** ribbon **> > Plane >> Multiple Section**.

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "wireframe-section-multiple".

  * Use the quick key combination "msl".

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **wireframe-section-multiple** and click **Run**.

## Command Overview

Create multiple strings where a defined set of planes intersect a wireframe object or selected wireframe triangle data. Output strings will inherit their colour from the original wireframe.

Displays the [Multiple Section](<../COMMON/Wireframe%20Section%20Multiple%20Dialog.md>) screen. Follow this link for more detailed information about this command.

**Note** : This command supports [**flexible wireframe selection**](<../COMMON/Wireframe_Selection_Concept.md>).
