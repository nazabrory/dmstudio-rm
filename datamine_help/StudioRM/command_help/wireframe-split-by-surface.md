# wireframe-split-by-surface ("spbd")

See this command in the [**command table**.](<COMMAND%20TABLE_W.md#wireframe-split-by-surface>)

To access this screen:

  * **Wireframe** ribbon **> > DTM >> Split Solid**.

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "wireframe-split-by-surface"

  * Use the quick key combination "spbd".

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **wireframe-split-by-surface** and click **Run**.

## Command Overview

Split a closed wireframe - a 'solid' - or a selection of triangles representing a solid by a DTM (or selected triangles of a DTM). You may choose to keep the solid above the DTM data or the solid below it. 

If the solid was a closed wireframe and had a clean line of intersection with the DTM, the new object will also be a closed solid.

Displays the [Split solid by DTM](<../COMMON/Split%20solid%20by%20DTM%20Dialog.md>) screen.

**Note** : This command supports [**flexible wireframe selection**](<../COMMON/Wireframe_Selection_Concept.md>).

See [Split Solid by DTM](<../COMMON/Split%20solid%20by%20DTM%20Dialog.md>).
