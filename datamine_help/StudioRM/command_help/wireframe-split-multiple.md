# wireframe-split-multiple

See this command in the [**command table**.](<COMMAND%20TABLE_W.md#wireframe-split-multiple>)

  * **Wireframe** ribbon **> > Plane >> Multiple Split**.

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "wireframe-split-multiple"

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **wireframe-split-multiple** and click **Run**.

## Command Overview

Splits a selected wireframe object, or preselected wireframe triangle data, into several wireframes along the intersections with parallel, equally-spaced planes. Output strings inherit their colour from the original wireframe.

**Note** : This command supports [**flexible wireframe selection**](<../COMMON/Wireframe_Selection_Concept.md>).

This command displays the [Multiple Split](<../COMMON/Wireframe%20Split%20Multiple%20Dialog.md>) screen.
