# wireframe-split-string ("spbs")

See this command in the [**command table**.](<COMMAND%20TABLE_W.md#wireframe-split-string>)

To access this screen:

  * **Wireframe** ribbon **> > Plane >> Split by String**.

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "wireframe-split-string"

  * Use the quick key combination "spbs".

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **wireframe-split-string** and click **Run**.

## Command Overview

Split a wireframe (or preselected wireframe triangle data) into two or more parts using the selected string and defined projection parameters.

**Note** : a string must be selected before this command is run. Please see here for more information.

The cutting string must be able to create an intersection plane, when projected in the selected direction, to subdivide the wireframe data completely. As such, it will typically extend beyond the limits of the wireframe to be cut.

**Note** : This command supports [**flexible wireframe selection**](<../COMMON/Wireframe_Selection_Concept.md>).

For more detailed command information see [Wireframe Split by String](<../COMMON/Wireframe%20Split%20By%20String%20Dialog.md>).
