# wireframe-split ("spli")

See this command in the [**command table**.](<COMMAND%20TABLE_W.md#wireframe-split>)

To access this screen:

  * **Wireframe** ribbon **> > Plane >> Split**.

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "wireframe-split"

  * Use the quick key combination "spli".

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **wireframe-split** and click **Run**.

## Command Overview

Splits a wireframe or selected triangle data into two parts using a section plane. Data is created either as a wireframe object or to multiple objects. 

Displays the [Wireframe Split](<../COMMON/Wireframe%20Split%20Dialog.md>). Follow this link for more information on command usage.
