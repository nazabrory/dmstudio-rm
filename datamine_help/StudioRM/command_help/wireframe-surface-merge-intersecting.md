# wireframe-surface-merge-intersecting

See this command in the [**command table**.](<COMMAND%20TABLE_W.md#wireframe-surface-merge-intersecting>)

To access this command:

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "wireframe-surface-merge-intersecting"

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **wireframe-surface-merge-intersecting** and click **Run**.

## Command Overview

Generate a new surface using the second wireframe (or selected data) to update the surface elevation in preference to the first. The resultant surface area is the union of the first and the second wireframe surface or selected data, so there is no restriction that the second surface be bounded by the first.

This command displays the [Update Intersecting DTMs](<../COMMON/Merge%20intersecting%20DTMs%20Dialog.md>) screen.

**Note** : This command supports [**flexible wireframe selection**](<../COMMON/Wireframe_Selection_Concept.md>).
