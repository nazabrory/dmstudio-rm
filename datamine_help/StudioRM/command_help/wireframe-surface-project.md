# wireframe-surface-project ("pdv")

See this command in the [**command table**.](<COMMAND%20TABLE_W.md#wireframe-surface-project>)

To access this command:

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "wireframe-surface-project"

  * Use the quick key combination "pdv".

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **wireframe-surface-project** and click **Run**.

## Command Overview

Project the boundary of the selected wireframe (which must open, such as a Digital Terrain Model), or selected triangles representing a DTM, to a specified plane, at an angle perpendicular to that plane. 

The input DTM data should have no overlaps in the direction of projection as this will result in crossing faces. These are reported in the Output control bar.

Displays the **[Project to Plane](<../COMMON/Wireframe%20Project%20To%20Plane%20Dialog.md>)** screen (follow this link for more detailed command usage).

**Note** : This command supports [**flexible wireframe selection**](<../COMMON/Wireframe_Selection_Concept.md>).

Command steps:

  1. Run the command. The [Project To Plane](<../COMMON/Wireframe%20Project%20To%20Plane%20Dialog.md>) screen displays.

  2. Select your input **Wireframe** data.

  3. Define the **Plane Orientation** to which the original wireframe is projected.
  4. Choose your **Output** data option.
  5. Click OK to run the command.
