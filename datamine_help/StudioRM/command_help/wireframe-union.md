# wireframe-union ("wun")

See this command in the [**command table**.](<COMMAND%20TABLE_W.md#wireframe-union>)

To access this command:

  * **Wireframe** ribbon **> > Boolean >> Union**.

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "wireframe-union"

  * Use the quick key combination "wun".

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **wireframe-union** and click **Run**.

Take two solid wireframe objects (or object triangle data) and creates a single wireframe with the same surface appearance and characteristics as the two component inputs together.

Displays the **[Wireframe Union](<../COMMON/Wireframe%20Union%20Dialog.md>)** screen. Follow this link for more information and an example.
