# wireframe-volume ("wvo")

See this command in the [**command table**.](<COMMAND%20TABLE_W.md#wireframe-volume>)

To access this command:

  * **Wireframe** ribbon **> > Operations>> Volume**.

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "wireframe-volume"

  * Use the quick key combination "wvo".

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **wireframe-volume** and click **Run**.

## Command Overview

Reports the volume of a solid wireframe (or wireframe portion), or the volume enclosed by a plane above or below a DTM (or part of a DTM).

Displays the [Calculate Volume](<../COMMON/Wireframe%20Calculate%20Volume%20Dialog.md>) screen.

**Note** : This command supports [**flexible wireframe selection**](<../COMMON/Wireframe_Selection_Concept.md>).
