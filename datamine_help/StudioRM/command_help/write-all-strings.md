# write-all-strings ("wa")

See this command in the [**command table**.](<COMMAND%20TABLE_W.md#write-all-strings>)

To access this command:

  * **Data** ribbon **> > Save >> All Strings**.

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "write-all-strings"

  * Use the quick key combination "wa".

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **write-all-strings** and click **Run**.

## Command Overview

Write all strings in memory to a string file. You are prompted for the name of a new string file for each string object that is currently loaded.

Command steps:

  1. Run the command.

  2. For each loaded object, specify a file name and location to save a Datamine file.
