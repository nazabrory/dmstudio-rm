# write-drillholes ("wdh")

See this command in the [**command table**.](<COMMAND%20TABLE_W.md#write-drillholes>)

To access this command:

  * **Data** ribbon **> > Save >> Static Holes**.

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "write-drillholes"

  * Use the quick key combination "wdh".

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **write-drillholes** and click **Run**.

## Command Overview

Write each loaded drillhole object to a new file. A file save prompt appears for each distinct object.

Command steps:

  1. Run the command.

  2. For each loaded object, specify a file name and location to save a Datamine file.
