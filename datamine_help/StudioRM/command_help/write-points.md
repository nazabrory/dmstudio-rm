# write-points ("wp")

See this command in the [**command table**.](<COMMAND%20TABLE_W.md#write-points>)

To access this command:

  * **Data** ribbon **> > Save >> Points**.

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "write-points"

  * Use the quick key combination "wp".

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **write-points** and click **Run**.

## Command Overview

Write all loaded point data to a named points file. Each loaded points object is saved independently to its own output file.

Command steps:

  1. Run the command. 

  2. Enter a file name and location. A browser appears for any points object that is loaded.
  3. Click OK to save. 

Points data is saved to the specified location.
