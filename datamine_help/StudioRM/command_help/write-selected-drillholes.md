# write-selected-drillholes ("wrd")

See this command in the [**command table**.](<COMMAND%20TABLE_W.md#write-selected-drillholes>)

To access this command:

  * **Data** ribbon **> > Save >> Static Holes >> Selected Holes**.

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "write-selected-drillholes"

  * Use the quick key combination "wrd".

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **write-selected-drillholes** and click **Run**.

## Command Overview

Save all currently highlighted (selected) static drillhole data to a single external Datamine file. 

Note: static drillhole data must be preselected to use this command.

Command steps:

  1. Select the drillhole data you wish to add to the output file. You can use any supported data selection method (see [Selecting 3D Data Interactively](<../COMMON/Selecting3DDataInteractively.md>)).

  2. Run the command.

  3. Specify a file name in the browser displayed, and click Save.
