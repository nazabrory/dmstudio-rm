# write-selected-points ("wrp")

See this command in the [**command table**.](<COMMAND%20TABLE_W.md#write-selected-points>)

To access this command:

  * Right-click the primary **3D** window and select **Save >> Selected Points**.

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "write-selected-points"

  * Use the quick key combination "wrp".

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **write-selected-points** and click **Run**.

## Command Overview

Save currently highlighted (selected) point data to an external Datamine file. Point data must be preselected to use this command.

See [Selecting 3D Data Interactively](<../COMMON/Selecting3DDataInteractively.md>).

Note: points data must be preselected to use this command.

Command steps:

  1. Select the point or points you wish to add to an external file. You can use any supported data selection method (see [Selecting 3D Data Interactively](<../COMMON/Selecting3DDataInteractively.md>)).

  2. Run the command.

  3. Specify a file name in the browser displayed, and click Save.
