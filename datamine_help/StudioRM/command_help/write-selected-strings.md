# write-selected-strings ("wrs")

See this command in the [**command table**.](<COMMAND%20TABLE_W.md#write-selected-strings>)

To access this command:

  * **Data** ribbon **> > Save >> All Strings >> Selected Strings**.

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "write-selected-strings"

  * Use the quick key combination "wrs".

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **write-selected-strings** and click **Run**.

## Command Overview

Save currently highlighted (selected) string data to an external Datamine file.

Command steps:

  1. Select the strings you wish to add to the output file. You can use any supported data selection method (see [Selecting 3D Data Interactively](<../COMMON/Selecting3DDataInteractively.md>)).

  2. Run the command.

  3. Specify a file name in the browser displayed, and click Save.
