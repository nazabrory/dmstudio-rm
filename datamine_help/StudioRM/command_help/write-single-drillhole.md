# write-single-drillhole ("wds")

See this command in the [**command table**.](<COMMAND%20TABLE_W.md#write-single-drillhole>)

To access this command:

  * **Data** ribbon **> > Save >> Static Holes >> Single drillhole**.

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "write-single-drillhole"

  * Use the quick key combination "wds".

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **write-single-drillhole** and click **Run**.

## Command Overview

Save a selected/highlighted drillhole to an external file. Only one drillhole can be selected - multiple selections are not allowed (instead; use [write-selected-drillholes](<write-selected-drillholes.md>)).

Command steps:

  1. Select a drillhole item.

  2. Run the command.

  3. Specify a file name in the browser displayed, and click Save.

A file is created containing data for a single drillhole.
