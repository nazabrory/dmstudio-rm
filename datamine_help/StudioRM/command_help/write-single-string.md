# write-single-string ("ws")

See this command in the [**command table**.](<COMMAND%20TABLE_W.md#write-single-string>)

To access this command:

  * **Data** ribbon **> > Save >> Single String**.

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "write-single-string"

  * Use the quick key combination "ws".

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **write-single-string** and click **Run**.

## Command Overview

Saves a single, selected string as a separate file. The string can be the only aspect of a string object, or one of many traces, although only one can be selected.

Command steps:

  1. Select the string that you wish to save.

  2. Run the command. 

The **Save** screen displays.

  3. Give the file a name, and **click** OK. 

The single string is saved as a Datamine file.
