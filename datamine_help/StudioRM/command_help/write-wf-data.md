# write-wf-data ("ww")

See this command in the [**command table**.](<COMMAND%20TABLE_W.md#write-wf-data>)

To access this command:

  * **Data** ribbon **> > Save >> Wireframes**.

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "write-wf-data"

  * Use the quick key combination "ww".

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **write-wf-data** and click **Run**.

## Command Overview

Saves currently loaded and displayed wireframe data to wireframe triangle and points files. Note that only complete wireframe objects are saved with this command; if wireframe objects are only partially selected, their entire database is saved to the output file(s).

Command steps:

  1. Select wireframe data.

  2. Run the command. 

  3. Enter a file name and location. A browser appears for any wireframe object that was fully or partially selected.
  4. Click OK to save. 

Wireframe data is saved to the specified location.
