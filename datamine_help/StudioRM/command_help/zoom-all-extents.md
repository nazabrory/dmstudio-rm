# zoom-all-extents ("ze")

See this command in the [**command table**.](<COMMAND%20TABLE_Z.md#zoom-all-extents>)

To access this command:

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "zoom-all-extents"

  * Use the quick key combination "ze".

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **zoom-all-extents** and click **Run**.

  * **3D View** ribbon **> > View >> Zoom Fit >> Zoom Extents**.

## Command Overview

Resets the target 3D window view limits to show all data, maintaining the current view plane.

**Note** : This command changes the view of data, not the data itself.

Note: This command is very similar to [zoom-all-graphics ("za")](<zoom-all-graphics.md>). The only difference is that **zoom-all-extents** does not change the position of the currently active [3D section](<../VR_Help/Sections.md>), whereas **zoom-all-graphics** will reposition it to the centre of the screen.

Note: This command honours data object filters, meaning "extents" equals "potentially visible data".
