# zoom-graphics ("zx")

See this command in the [**command table**.](<COMMAND%20TABLE_Z.md#zoom-all-graphics>)

To access this command:

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "zoom-graphics"

  * Use the quick key combination "zx".

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **zoom-graphics** and click **Run**.

  * **3D View** ribbon **> > View >> Zoom Area**.

## Command Overview

Magnify the view by attempting to maximize all data within a selection box to the current screen border. 

**Note** : This command changes the view of data, not the data itself.

Command steps:

  1. Run the command.

The cursor changes to a magnifying glass.

  2. In any **3D** window, left-click-drag to form a rectangle.

  3. Release the mouse button.

The view magnification changes to fit as much selected data as possible on screen.
