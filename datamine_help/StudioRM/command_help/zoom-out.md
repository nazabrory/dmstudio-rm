# zoom-out ("zz")

See this command in the [**command table**.](<COMMAND%20TABLE_Z.md#zoom-out>)

To access this command:

  * Using the **[command line](<../COMMON/Command_Toolbar.md>)** , enter "zoom-out"

  * Use the quick key combination "zz".

  * Display the **[Find Command](<../COMMON/findcommand.md>)** screen, locate **zoom-out** and click **Run**.

## Command Overview

**Note** : This command changes the view of data, not the data itself.

Zoom the currently active 3D view out by an arbitrary amount.

  * zoom-out ("zz")

  * [zoom-graphics ("zx")](<zoom-graphics.md>)

  * [zoom-graphics ("zx")](<zoom-graphics.md>)
