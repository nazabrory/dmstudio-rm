# Overview

**Studio RM** is the world's leading integrated software package for the natural resource industries. Typical uses are in data capture and analysis, exploration, geology, geochemistry, rock mechanics and orebody modelling. It has been chosen for commodities as diverse as iron ore, gold, nickel, phosphates, diamonds, copper, bauxite, coal, lignite, platinum, petroleum, and industrial minerals.

Studio RM is one of several products in the Studio product family, which includes:
