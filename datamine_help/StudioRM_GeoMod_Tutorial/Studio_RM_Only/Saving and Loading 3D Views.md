# Using this template

This template is to be used for all topics that do not have a more specific template available. Do not use this template if you are creating help of the following types:

  * A context-sensitive Help file relating to a specific dialog - use "Context Help.htt".

  * A tutorial topic - use the relevant tutorial template.

Generally speaking, these topics are not context-sensitive, but may be linked to by context-sensitive Help topics.
