# Tools used in Grade Estimation

The grade estimation tools are available as:

Menus

Toolbars

Processes

Studio Products

## Menus

Most commands used in the grade estimation process can be found under the following menus:

  * Home ribbon

  * Edit ribbon

  * Estimate ribbon

  * Report ribbon

## Processes

The following table lists the processes commonly used in the grade estimation process.

Command |  Description |  Exercise  
---|---|---  
General Processes |  |   
EXTRA |  Add or manipulate Fields using simple or advanced string and numeric transformations |   
MGSORT |  Sort table records on multiple fields |   
COPY |  Copy a table |   
PICREC |  Copy table records using relational or pattern matching record selection expressions |   
SELCOP |  Copy selected table fields |   
SORTX |  Sort a records according to key fields |   
JOIN |  Join two files on sorted key field values. |   
STATS |  Compute summary statistics on multiple numeric fields |   
CHART |  Create and display scatter, line or histogram charts |   
PPQQPLOT |  Create and display PP and QQ plots |   
Grade Estimation Processes |  |   
GRADE |  Basic grade interpolation |   
ESTIMA |  Advanced grade interpolation (use when recording a macro/script) |   
ESTIMATE |  Advanced grade interpolation via interactive dialogs |   
PANELEST |  Estimate grades into panels (perimeters) |   
ELLIPSE |  Create a sample search ellipse wireframe |   
XVALID |  Cross validate a set of estimation parameters |   
Evaluation and Reserves Reporting |  |   
TONGRAD |  Calculate the tonnage and grade of a model, using the current block model display legend |   
MODRES |  Generates RESULTS file of reserves from orebody model using an optional input perimeter file and ore grade or type categories. Perimeters can be individual or pairs of outlines |   
TRIVAL |  Evaluate a grade block model within a wireframe |   
TONGRAD |  Calculate a grade block model tonnes and grade |
