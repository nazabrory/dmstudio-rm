# Working with Charts

Once geological and mine planning data have been loaded or created in a project, they are available for presentation in charts, either as sheets in the Plots window or as plot items within other plot or log sheets.

![](../Images/dp_about_charts_1.gif)

![](../Images/dp_about_charts_2.gif)

The following types of charts can be created and modified using the relevant chart dialogs:

  * Histogram

  * Scatter Plot

  * Variogram

  * Stereonet

The general procedure for creating charts is as follows:

  1. Add the required data to the project and optionally load the file,

  2. Create the required chart (histogram, scatter, variogram, stereonet) either as a new sheet or plot item,

  3. Optionally insert Plot Items e.g. Legend Box and Images into sheets,

  4. Setup and Print sheets.
