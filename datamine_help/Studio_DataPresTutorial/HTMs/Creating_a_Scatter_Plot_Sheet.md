# Overview

In this portion of the tutorial you are going to be introduced to the tools and methods used to create a single scatter plot chart sheet in the Plots window.

![](../Images/dp_Creating%20Scatter%20Plot%20Sheets%208.jpg)

## Prerequisites

You have already completed the following:

  * Opened and saved a copy of the project file Data Presentation Tutorial.StartPoint.dmproj \- see exercises under the [Opening and Saving Projects](<Opening%20and%20Saving%20Projects.md>) page.

## Exercise: Creating a Single Scatter Plot Chart Sheet

In this exercise, you are going to create a single scatter plot chart sheet using the AU and CU grade fields from the data file _vb_assays. This includes the following tasks:

  * **Defining a new chart and selecting a data file**

  * **S** etting the chart's symbol and color parameters

  * Displaying summary statistics on the chart

  * Renaming the sheet.

**Defining a New Chart and Selecting a Data File**

  1. Select the Plots window.

  2. Activate the  Manage ribbon and select  Sheets | Charts | Point/Line Plot Select  Insert | Chart | Scatter Plot .

  3. In the Scatter Plot dialog, Data Selection tab, Files and Fields group, select the Data File option, click Browse.

  4. In the Project Browser dialog, Database Tables pane, select the table _vb_assays, click OK.

  5. Back in the Scatter Plot dialog, Files and Fields group, in the X Axis field select [AU], in the Y Axis field select [CU].

  6. In the Axis Scaling group, use the selected defaults.

  7. In the Layout group, select Multiple Charts and Delete Empty Charts.

  8. Check that your settings are as shown below, click Apply:  
  
![](../Images/dp_Creating%20Scatter%20Plot%20Sheets%201.jpg)  

  9. In the Preview pane,check that your scatter plot is as shown below:  
  
![](../Images/dp_Creating%20Scatter%20Plot%20Sheets%202.jpg)  

  10. In the Chart Thumbnails pane,check that there is only one chart listed:  
  
![](../Images/dp_Creating%20Scatter%20Plot%20Sheets%203.jpg)  

  11. In the Summary pane, check that the summary message is as shown below:  
  
![](../Images/dp_Creating%20Scatter%20Plot%20Sheets%204.jpg)

Setting the Chart's Colour, Symbol and Models Parameters

  1. Select the Format tab.

  2. Set the Colour, Symbol and Models group settings as shown below, click Apply:  
  
![](../Images/dp_Creating%20Scatter%20Plot%20Sheets%205.jpg)

  3. In the Preview pane, check that your scatter plot symbols, colors and model line are as shown below:  
  
![](../Images/dp_Creating%20Scatter%20Plot%20Sheets%206.jpg)  

Displaying Summary Statistics on the Chart

  1. Select the Statistics tab.

  2. In the Statistics pane, tick the boxes next to the required summary statistics listed in the NAME column, click Apply:  
  
![](../Images/dp_Creating%20Scatter%20Plot%20Sheets%207.jpg)  

  3. In the Preview pane, check that your scatter plot now includes a list of summary statistics, as shown below, click OK:  
  
![](../Images/dp_Creating%20Scatter%20Plot%20Sheets%208.jpg)

![note.gif \(1017 bytes\)](../images/note.gif) |  The summary statistics are automatically positioned top left, above the chart. The position and size of this summary statistics box can be edited.   
---|---  
  4. Check that a new sheet has been added to the Plots window and that in the Sheets control bar, a new sheet is listed under the Plots folder.

Renaming the Sheet

  1. In the Plots window, right-click on the sheet tab (at the bottom of the Plots window) and select Rename.

  2. In the Rename Sheet dialog, define the Name: as 'Scatterplot 1', click OK.

  3. Check that the sheet has been renamed in both the Plots window and the Sheets control bar.
