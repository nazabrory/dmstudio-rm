## Exercise: Editing Histogram Chart Properties

In this exercise, you are going to open an existing histogram chart and edit the title and axis label settings.

![note.gif \(1017 bytes\)](../images/note.gif) |  The exercise steps below will make use of the histogram chart created in the exercise Creating a Single Histogram Chart Sheet.  
---|---  
  
The exercise procedures are as follows:  

**Opening an Existing Chart and Displaying the Histogram Properties Dialog**

  1. Select the Plots window.

  2. Select an existing histogram plot sheet tab, e.g. 'Histogram 1'.

  3. Double-click on the histogram chart displayed in the sheet.

  4. In the Histogram dialog, Charts tab, click Edit Chart Properties.  

Modifying Chart Title and Axis Label Settings

  1. In the Histogram Properties dialog, define new Title, X Axis, Y Axis and Legend settings as shown below, click OK:  
  
![](../Images/dp_Editing%20Histogram%20Chart%20Parameters%201.jpg)

![note.gif \(1017 bytes\)](../images/note.gif) |  The Bin and Grid group settings can be set either in this dialog or, as shown in the exercise Creating a Single Histogram Chart Sheet, in the Format tab.  
---|---  
  2. Back in the Histogram dialog, Preview pane, check that the title, axis labels and legend have been changed as shown below:   
  
![](../Images/dp_Editing%20Histogram%20Chart%20Parameters%202.jpg)  

  3. In the Histogram dialog, click OK to save the changes.

## Exercise: Editing Histogram Model Parameters

In this exercise, you are going to open an existing histogram chart and edit the histogram model properties.

![note.gif \(1017 bytes\)](../images/note.gif) |  The exercise steps below will make use of the histogram chart created in the exercise Creating a Single Histogram Chart Sheet.  
---|---  
  
The exercise procedures are as follows:  

**Opening an Existing Chart and Displaying the Fit Model Dialog**

  1. Select the Plots window.

  2. Select an existing histogram plot sheet tab e.g. 'Histogram 1'.

  3. Double-click on the histogram chart displayed in the sheet.

  4. In the Histogram dialog, Charts tab, click Edit Model Properties.

Modifying the Histogram Model Properties

  1. In the Fit Model dialog, define new Parameters group settings shown below, click OK:  
  
![](../Images/dp_Editing%20Histogram%20Model%20Parameters%201.jpg)  

![note.gif \(1017 bytes\)](../images/note.gif) |  The Restore Defaults button will restore the Parameters to the automatically calculated parameters for the data set.  
---|---  
  2. Back in the Histogram dialog, Preview pane, check that the model has been changed as shown below:   
  
![](../Images/dp_Editing%20Histogram%20Model%20Parameters%202.jpg)

Moving the Summary Statistics Boxes

  1. In the Histogram dialog, Preview pane, click-and-drag (a four headed arrow is displayed) both the summary statistics blocks a short distance down and to the right, in order to place them in the top right corner of the graph frame.

  2. Check that the summary statistics have been moved and that the chart boundary has resized as shown below:   
  
![](../Images/dp_Editing%20Histogram%20Model%20Parameters%203.jpg)  

  3. In the Histogram dialog, click OK to save the changes.
