# Overview

In this portion of the tutorial you are going to be introduced to the tools and methods used to edit scatter plot chart properties.

## Prerequisites

You have already completed the following:

  * Opened and saved a copy of the project file Data Presentation Tutorial.StartPoint.dmproj \- see exercises under the [Opening and Saving Projects](<Opening%20and%20Saving%20Projects.md>) page.

  * Created a scatter plot chart.

## Exercise: Editing Scatter Plot Chart Properties

In this exercise, you are going to open an existing scatter plot chart and edit the title and axis label settings.

![note.gif \(1017 bytes\)](../images/note.gif) |  The exercise steps below make use of the scatter plot chart created in the exercise [Creating a Single Scatter Plot Chart Sheet](<Creating_a_Scatter_Plot_Sheet.md>).  
---|---  
  
The exercise procedures are as follows:  

**Opening an Existing Chart and Displaying the Scatter Plot Properties Dialog**

  1. Select the Plots window.

  2. Select an existing scatter plot sheet tab, e.g. 'Scatterplot 1'.

  3. Double-click on the scatter plot chart displayed in the sheet.

  4. In the Scatter Plot dialog, Charts tab, click Edit Chart Properties.  

Modifying Chart Title and Axis Label Settings

  1. In the Scatter Plot Properties dialog, define new Title, X Axis and Y Axis settings as shown below, click OK:  
  
![](../Images/dp_Editing%20Scatterplot%20Chart%20Parameters%201.jpg)

![note.gif \(1017 bytes\)](../images/note.gif) |  The Grid group settings can be edited/set either in this dialog or, as shown in the exercise [Creating a Single Scatter Plot Chart Sheet](<Creating_a_Scatter_Plot_Sheet.md>), in the Format tab.  
---|---  
  2. Back in the Scatter Plot dialog, Preview pane, check that the title and axis labels have been changed as shown below:   
  
![](../Images/dp_Editing%20Scatterplot%20Chart%20Parameters%202.jpg)  

Moving the Summary Statistics

  1. In the Scatter Plot dialog, Preview pane, click-and-drag (a four headed arrow is displayed) the summary statistics block a short distance down in order to place it in the top portion of the graph frame.

  2. Check that the summary statistics have been moved and that the chart has resized as shown below:   
  
![](../Images/dp_Editing%20Scatterplot%20Chart%20Parameters%203.jpg)  

  3. In the Histogram dialog, click OK to save the changes.
