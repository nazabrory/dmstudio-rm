# Overview

In this portion of the tutorial you are going to modify the format settings for the displayed Overlays. Formatting typically involves setting the following display format settings:

  * Style \- display style e.g. points, labels, lines, faces, blocks, arrows, intersection

  * Color \- object coloring using fixed colors or legends

  * Symbols \- symbol type, size and rotation

  * Labels \- object annotation settings

  * Grid \- type, intervals, format, color, annotation settings

## Exercise: Formatting Overlays

In this exercise, you are going to change the format settings for the grid, fault surfaces and static drillholes. This includes the following tasks:  

  * Defining grid settings

  * Setting the fault surfaces display style to intersection

  * Setting the drillholes' label color

**Defining Grid Settings**

  1. Make sure the Plot window is active.

  2. Select the Sheets control bar, expand the Plots folder.

  3. In the Plan sheet's Overlays list, double-click Grid .

  4. In the Format Display dialog, Grids tab, Grid Settings group, select the Options tab.
  5. In the Line Formatting group and Annotation groups, define the settings as shown below:  
  
![](../Images/dp_Formatting%20Overlays%201.jpg)
  6. In the More Line Formatting tab, define the settings as shown below:  
  
![](../Images/dp_Formatting%20Overlays%202.jpg)
  7. Click OK.

**Setting the Wireframe's Display Style to Intersection**

  1. In the Sheets control bar,Plots folder, Plan sheet's Overlay list, right-click _vb_faulttr/_vb_faultpt (wireframes) .

  2. In the context menu, select Format....

  3. In the Format Display dialog, Overlays tab, Overlay Format group, select the Style tab.
  4. Set the Display As option to Intersection.
  5. Click Apply.

Setting the Static Drillholes' Labels Color

  1. In the Format Display dialog, Overlay Objects pane on the left, select _vb_holes (drillholes) .

  2. In the Overlay Format group on the right, select the Drillholes tab, click Format....
  3. In the Drillhole Traces dialog, select the Labels tab, click Font....
  4. Select the Collar check box.
  5. In the Font dialog, Effects group, set the Size to '6', set the Color to [Black], click OK.
  6. Back in the Drillhole Traces dialog, click OK.
  7. Back in the Format Display dialog, click OK.
  8. Click Insert and select the [BHID] option.
  9. In the Plots window, Select the Plan 2 sheet tab, click in the white space outside the plot border.
