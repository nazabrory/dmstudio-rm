# Overview

In this portion of the tutorial you are going to enhance your log sheet by inserting Plot Items. The following Plot Items can be inserted into log sheets:

  * Title Box \- a title box with multiple rows and columns

  * Text Box \- a customizable text box

  * Legend Box \- an existing User or Project legend

  * Table \- a data file table with user defined columns

  * Clipart \- an images of the .bmp format

  * Symbol \- a sizeable symbol from a standard set

  * Columns List Box \- a table of a Downhole Column's summary information

  * Log \- a hole log sheet

## Exercise: Inserting and moving Plot Items

Inserting a Plot Item

  1. Select the Logs window.

  2. Select the VB2737 log sheet tab.

  3. Activate the  Manage ribbon and select  _I_ nsert | Plot Item | Le _g_ end Box Select  _I_ nsert | Plot Item | Le _g_ end Box... or, in the  Plot Item toolbar click  Insert Legend Box .

![](../Images/dp_Inserting%20Plot%20Items%20in%20Log%20Sheets%201.jpg)
