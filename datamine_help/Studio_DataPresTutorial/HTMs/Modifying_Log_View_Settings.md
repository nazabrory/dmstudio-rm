# Overview

In this portion of the tutorial you are going to modify scale and view area settings in a log sheet. Log sheet view settings include the following:

  * Zoom \- zoom in or out of the log sheet view

  * Scale \- define the down hole distance scale for the log sheet view

  * Pan \- define the hole interval displayed in the log sheet view

## Prerequisites

Required:

  * Opened and saved a copy of the project file Data Presentation Tutorial.StartPoint.dmproj \- see exercises under the [Opening and Saving Projects](<Opening%20and%20Saving%20Projects.md>) page.

  * Inserted a new VB2675 Log sheet - see exercises on the [Opening, Copying and Inserting Log Sheets](<Opening_Copying_and_Inserting_Log_Sheets.md>) page.

  * Selected the hole VB2737 in the newly inserted log sheet - see exercise on the [Selecting Drillholes](<selecting_drillholes.md>) page.

## Exercise: Modifying a Log sheet's View Settings

In this exercise, you are going to change the zoom, scale and extents settings for the view in the VB2737 log sheet. This includes the following tasks:

  * Displaying the Zoom and Scale View Toolbars.

  * Setting the view's Zoom factor

  * Setting the hole Scale and Extents

Setting the Hole Scale and Extents

  1. Using the Manage ribbon select the Layout toggle to enable it, if not already selected.

  2. In the Logs window, select the VB2737 sheet tab.

  3. Left-click anywhere within the Log View to show the surrounding frame as a dotted line, then right click to select  Hole Log Frame (VB2737) Properties Select _L og_ | _H_ ole... .

![note.gif \(1017 bytes\)](../images/note.gif) |  If _Log_ | _H_ ole... is greyed out, click inside the Log Header area in the log sheet - this should activate the function.  
---|---  
  4. In the Log View Properties dialog, select the Hole tab.

  5. In the Extents group above, select Custom, set From ='120' and To='211.220', click OK.

  6. Make sure the Scale is set to Automatic.
