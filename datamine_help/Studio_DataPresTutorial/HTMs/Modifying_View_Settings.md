# Overview

In this portion of the tutorial you are going to modify the view settings for your plot sheet. View settings include the following:

  * Zoom \- zoom in or out of the plot sheet view

  * Scale and Pan \- define the position and scale for the data area in the plot sheet view

  * Exaggeration \- optionally set coordinate exaggeration

  * Direction \- define the view direction for the plot sheet

## Prerequisites

Required:

  * Opened and saved a copy of the project file Data Presentation Tutorial.StartPoint.dmproj \- see exercises under the [Opening and Saving Projects](<Opening%20and%20Saving%20Projects.md>) page.
