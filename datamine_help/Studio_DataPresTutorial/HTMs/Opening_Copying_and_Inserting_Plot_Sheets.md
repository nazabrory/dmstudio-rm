## Exercise: Opening and Viewing a Plot Sheet

In this exercise, you are going to open the Plan sheet in the Plots window and view the corresponding Plot sheet components in the Sheets control bar. This is one of the default Plot sheets that is generated when a new project is created. This exercise includes the following tasks:

  * Opening a Plot Sheet

  * Viewing the Plan Sheet items in the Sheets control bar.

**Opening a Plot Sheet**

  1. Select the Plots window.

  2. Select the Plan sheet tab.

  3. Check that your displayed sheet is as shown below:  
  
![](../Images/blankPlot.jpg)

**Viewing the Plan Sheet items in the Sheets control bar**

  1. Select the Sheets control bar.

  2. Expand the Data Presentation Tutorial.dmproj project and the Plots folder.

  3. In the Sheets control bar, Plots folder, fully expand the Plan folder and all sub-folders by clicking on all the "+" icons to the left of each item, so that all overlay objects are listed, as shown below:  
  
![](../Images/Plot1.jpg)

  4. Note that the Plan sheet contains an associated Projection (plan view) and Overlays, one for each loaded object (in this case, a Grid, a Hull and a Section Line - no other data is currently loaded).

![note.gif \(1017 bytes\)](../images/note.gif) |  The Overlays folder in the Sheets control bar is currently highlighted in grey. This indicates that the Overlays object is currently selected in the Plan sheet. Clicking on a different item in the Plan sheet will highlight the corresponding item in the Sheets control bar tree.

## Exercise: Copying and Renaming a Plot Sheet

In this exercise, you are going to copy the Plan sheet and rename it 'Plan 2' . This exercise includes the following tasks:

  * Copying a Plot Sheet

  * Renaming a Plot Sheet

Copying a Plot Sheet

  1. Select the Plots window.

  2. Select the Plan sheet tab.

  3. Select the  Manage ribbon and  Sheets | Copy (top level icon) In the  Menu Bar , select  E dit | Copy Sheet .

  4. Check that a new sheet tab with the name 'Copy of Plan' has been added to the Plots window.

![note.gif \(1017 bytes\)](../images/note.gif) |  A new Copy of Plan object is also added to the Sheets control bar, Plots folder, at the bottom of the list of existing sheets.  
---|---  
  
Renaming a Plot Sheet

  1. In the Plots window, right-click on the Copy of Plan sheet tab, select Rename.

  2. In the Rename Sheet dialog, define the Name: as 'Plan 2', click OK.

  3. Check that the sheet tab is now labeled 'Plan 2'.

![note.gif \(1017 bytes\)](../images/note.gif) |  The corresponding Sheets control bar sheet item is automatically renamed to Plan 2.  
---|---  
  
**![](../images/UpArrow.gif)**Top of page

## Exercise: Inserting a New Plot Sheet

In this exercise, you are going to create a new plot sheet with the name '3 Parallel W-E Sections'. This exercise includes the following tasks:

  * Inserting a new Plot Sheet

  * Renaming the 3 Parallel Sections Sheet.  

Inserting a new Plot Sheet

  1. Select the Plots window.

  2. Activate the  Manage ribbon and select  Insert | Sheet | Custom Select  I nsert | S heet | P lot | Custom... .

  3. In the Plot Item Library dialog, select the 3 Parallel Sections item, click OK.  
  
![](../Images/dp_Inserting%20a%20New%20Plot%20Sheet%201.jpg)  

![note.gif \(1017 bytes\)](../images/note.gif) |  The following different types of Plot sheets are available:
     * Plan \- a plan view with no set clipping
     * Vertical section \- a vertical section with set clipping e.g. Section 5112.50 N
     * 3 Parallel Sections \- 3 parallel N-S Sections e.g. Section 6200.00 E
     * 3D Projection \- three dimensional section and view e.g. sheets 3D and 3D-Above
     * Linked Replica \- a plot consisting of a compilation of existing plot sheets
     * Vertical Section & Plan \- a combined plan and vertical section e.g. Section 6021.27E
     * Projection Wizard section - a custom orientation section
All plan, section, 3D section and view orientations can be customized.  
---|---  
  4. Check that the new Section 6200.00 E sheet has been added to the Plots window. Note that no data is currently loaded so you will only see 3 blank frames:  
  
![](../Images/Plot2.jpg)

Renaming the 3 Parallel Sections Sheet

  1. Right-Click on the Section 6200.00 E sheet tab, select Rename....

  2. In the Rename Sheet dialog, define the Name as '3 Parallel S-N Sections', click OK.

  3. Check that the sheet tab is now labeled "3 Parallel S-N Sections".
