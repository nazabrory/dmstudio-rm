# Overview

In this portion of the tutorial you are going to be introduced to the tools and methods used to print charts and save charts to image files.

## Prerequisites

You have already completed the following:

  * Opened and saved a copy of the project file Data Presentation Tutorial.StartPoint.dmproj \- see exercises under the [Opening and Saving Projects](<Opening%20and%20Saving%20Projects.md>) page.

  * Created a scatter plot or histogram chart.

  * Saving a Chart to an Image File

## Exercise: Previewing and Printing a Chart

In this exercise, you are going to open an existing chart, preview and then print a single page.

![note.gif \(1017 bytes\)](../images/note.gif) |  The exercise steps below will make use of the histogram chart created in the exercise Creating a Single Histogram Chart Sheet. The procedures and tools shown below are also used to print other chart types e.g. scatter plots, stereonet and variogram charts.  
---|---  
  
The exercise procedures are as follows:  

**Opening an Existing Chart and Displaying the Charts Toolbar**

  1. Select the Plots window.

  2. Select an existing histogram plot sheet tab e.g. 'Histogram 1'.

  3. Double-click on the histogram chart displayed in the sheet.

  4. In the Histogram dialog, bottom left, tick the Show Toolbar checkbox, confirm that the Charts toolbar is displayed top left:  
  
![](../Images/dp_Printing%20a%20Chart%201.jpg)  

Previewing a Chart Plot Sheet

  1. In the Charts toolbar, click Print Preview.

  2. In the Print Preview dialog, select the required Page number, check the previewed page, click Close:  
  
![](../Images/dp_Printing%20a%20Chart%202.jpg)

Printing the Chart Sheet

![note.gif \(1017 bytes\)](../images/note.gif) |  A working printer needs to be connected to your PC in order to complete this printing section of the exercise.  
---|---  
  
  1. In the Charts toolbar, click Print Chart.

![note.gif \(1017 bytes\)](../images/note.gif) |  The chart can also be printed directly from the Print Preview dialog by clicking Print.  
---|---  
  2. In the Print dialog, select the required Printer, Print Range and Copies parameters, click OK.

  3. Collect the printed page from the selected printer and compare it to the displayed chart.

  4. Back in the Histogram dialog, click OK.

**![](../images/UpArrow.gif)**Top of page

## Exercise: Saving a Chart to an Image File

In this exercise, you are going to open an existing chart and then save it to an image file.

![note.gif \(1017 bytes\)](../images/note.gif) |  The exercise steps below will make use of the histogram chart created in the exercise Creating a Single Histogram Chart Sheet. The procedures and tools shown below are also used to save images for other chart types e.g. scatter plots, stereonet and variogram charts.  
---|---  
  
The exercise procedures are as follows:  

**Opening an Existing Chart and Displaying the Charts Toolbar**

  1. Select the Plots window.

  2. Select an existing histogram plot sheet tab e.g. 'Histogram 1'.

  3. Double-click on the histogram chart displayed in the sheet.

  4. In the Histogram dialog, bottom left, tick the Show Toolbar checkbox, confirm that the Charts toolbar is displayed top left:  
  
![](../Images/dp_Saving%20a%20Chart%20to%20image%20File%201.jpg)  

Saving As an Image File

  1. In the Charts toolbar, click Save Chart.

  2. In the Save As dialog, select the required Save in: folder and define a new File name: 'Histogram 1.jpg', click Save:  
  
![](../Images/dp_Saving%20a%20Chart%20to%20Image%20File%202.jpg)

  3. If displayed, in the Changes in Project Folder dialog, click Yes:  
  
![](../Images/dp_Saving%20a%20Chart%20to%20Image%20File%203.jpg)

![note.gif \(1017 bytes\)](../images/note.gif) |  This dialog is only displayed when:
     * The image is saved to the project folder, and
     * The 'Detect files added...' option is selected and the 'Automatically update...' option is not selected, under the Automatic Project Updates settings group in the Project Settings dialog.  
---|---  
  4. Back in the Histogram dialog, click Close.
