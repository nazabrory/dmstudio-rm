# Data Presentation Tools

Your appplication provides a wide range of tools for the presentation of exploration, geological modeling and mine planning data. These tools can be grouped into the following categories:

  * **Plots window** : create plot sheets, scatter and histogram charts; insert plot items and features; print plots

  * **Logs window** : create log sheets, insert plot items and print log sheets

  * **Window images** : save window views to graphics format files

  * **Visualizer Replay Files** : save normal, animated or dated Visualizer window replay views

  * **Studio Plot Processes**: a large selection of plotting processes for generating and printing plots

  * **Design window plotting** : generate and print plans and sections, superimpose plot files

  * VR window: view data in a virtual environment, view GVPs, drape images on surfaces; create sequence animations, drive and flight simulations

## Enhancing Plots, Charts and Logs

Plot sheets, log sheets, scatter chart and histogram chart sheets can be enhanced by inserting and formatting View Areas, relevant Plot Items, Plot Features or Geological Features. The more frequently used features include:

  * Plot Items \- Dimension Arrow, Title Box, Text Box, Legend Box, Table, Clipart, Symbol, North Arrow, Scale Bar, Columns List Box, Profile

  * Plot Features \- Point/Line Location, outline, 3D Line, Text Annotation, Annotation Vector, Dimension Overlay

The plot and log sheets are printed individually or in batches using standard windows printing tools.

## Window Images

Images of the Design, Plots, Logs, and 3DVR window views can be saved to a graphics format file (.emf, .jpg, .tif, or .bmp). These images can then be used in the generation of documents and visual presentations.

## 3DPDF Output

The 3DVR window views can be saved, using 3DPDF technology, to an interactive 3D document in various file formats (.pdf, .u3d and .prc). These documents can then be used for publishing, sharing and viewing project data using any PDF viewer e.g. Adobe Reader. This function is accessed using the Data ribbon and Export | PDF3DMenu Bar option File | Export to PDF3D.

## Visualizer Replay Files

These files are used to store dynamic 3D visualizer views of the loaded and displayed project data. The following types of Visualizer Replay files are available:

  * **Standard Visualizer Replay Files** : 3D visualizer views of the loaded and displayed project data (.gvp).

  * **Animated Visualizer Replay Files** : labeled 3D visualizer views (.gvz).

  * **Dated Animated Visualizer Replay Files** : date sequenced and labeled 3D visualizer views (.gvz).

These files can be used as standalone 3D views or embedded in documents and visual presentations.

## Processes

The plotting Processes are primarily used to generate plots from macros and scripts. These processes provide the ability to create and print the following types of plots:

  * Plot prototype file

  * Scatter, lines, histogram, log, probability, pie chart, 'pp' and 'qq' plots

  * Frames, Arrows, titles, text

  * Symbols and scaled symbols,

  * Contours, strings, annotated perimeters

  * Drillholes

  * Wireframes and wireframe slices

  * Block model sections

  * Send plots to a plotter (process PDRIVE)

![note.gif \(1017 bytes\)](../images/note.gif) | 

  * Most of the above processes have the name convention PLOT*. They can be found under Applications | Plot Processes.
  * The output files are in .dm or .plt format. These Processes can also be run manually, but it is recommended that the Plots and Log windows be used for this purpose.
  * Please see the Command Table in your online reference file for a full list of the available plotting Processes.

## Windows Plotting and Printing

Paper prints or plots of the various Design, Plots, Logs, and 3DVR window views can be generated and sent to a local or network printer/plotter using standard Windows printing functionality. This is accessed via the Project button and the Print Setup, Print Preview and Print commandsMenu Bar options File | Print Setup , File | Print Preview and File | Print.
