# Working with Logs

Once dynamic drillhole data tables have been loaded into a project, they are available for presentation in log sheets.

The general procedure for creating logs is as follows:

  1. Open an existing Log Sheet or insert a new one,

  2. Display required drillhole data columns,

  3. Insert Plot Items e.g. Legend Box and Images,

  4. Format data columns and Plot Items,

  5. Setup and Print log sheets.
