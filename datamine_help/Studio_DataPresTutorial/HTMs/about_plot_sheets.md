# Working with Plots

Once geological and mine planning data have been loaded or created in a project, they are available for presentation in plots.

The general procedure for creating plots is as follows:

  1. Open an existing Plot's sheet or insert a new one,

  2. Modify Section and View settings,

  3. Display required data Overlays,

  4. Insert Plot Items e.g. Title Box and Scale Bar,

  5. Format data Overlays and Plot Items,

  6. Setup and Print sheets.
