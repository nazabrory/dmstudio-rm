## File locations

All sample files referenced by this tutorial can be found (assuming a default installation has been performed) in the folder C:\Database\DMTutorials\Data\VBOP. This folder contains a series of data-type-specific folders including spreadsheet files for importing, project files and individual (native) Datamine files.
