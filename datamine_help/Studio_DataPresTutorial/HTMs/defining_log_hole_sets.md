# Overview

In this portion of the tutorial you are going to create and edit a log hole set. A new project by default contains an All Holes hole set.

## Prerequisites

Required:

  * Opened and saved a copy of the project file Data Presentation Tutorial.StartPoint.dmproj \- see exercises under the [Opening and Saving Projects](<Opening%20and%20Saving%20Projects.md>) page.

## Exercise: Creating and Editing Hole Sets

In this exercise you are going to create and edit a new hole set for use in a later printing exercise. This includes the following tasks:

  * Creating a new Hole Set

  * Editing a Hole Set

Creating a New Hole Set

  1. Select the Logs window.
  2. Select the VB2737 log sheet tab.

  3. Left-click anywhere within the Log View to show the surrounding frame as a dotted line, then right click to select  Hole Log Frame (VB2737) Properties Select _L og_ | _H_ ole... .

  4. In the Log View Properties dialog, select the Hole Set tab, click Edit Sets....  
  
![](../Images/dp_Logs%20and%20Hole%20sets%201.jpg)
  5. In the Hole Sets dialog, click New.  
  
![](../Images/dp_Logs%20and%20Hole%20sets%202.jpg)  
  
![note.gif \(1017 bytes\)](../images/note.gif)| The project already contains a default All holes set.  
---|---  
  6. In the New Hole Set dialog, select the Add all 28 holes option, click Next.  
  
![](../Images/dp_Logs%20and%20Hole%20sets%203.jpg)
  7. In the Hole Sets dialog, check that Set 1 has been added to the list, as shown below, click Close:  
  
![](../Images/dp_Logs%20and%20Hole%20sets%204.jpg)
  8. Back in the Log View Properties dialog, Hole Set tab, Selected set pane, check that Set 1 has been added to the list.

Editing a Hole Set

  1. In the Hole Set tab, click Edit Sets....
  2. In the Hole Sets dialog, hole sets pane, select [Set 1], click Edit.  
  
![](../Images/dp_Logs%20and%20Hole%20sets%205.jpg)
  3. In the Hole Set Contents dialog, Holes in set pane, select the holes [VB2812] through to [VB4295], click Remove>>.  
  
![](../Images/dp_Logs%20and%20Hole%20sets%206.jpg)
  4. In the Holes in set pane, check that VB2675 and VB2737 are still listed in the left pane, click OK.  
  
![](../Images/dp_Logs%20and%20Hole%20sets%207.jpg)
  5. Back in the Hole Sets dialog, click Close.
  6. Back in the Log View Properties dialog, Hole Set tab, click OK.
