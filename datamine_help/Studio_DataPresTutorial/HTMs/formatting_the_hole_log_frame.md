# Overview

In this portion of the tutorial you are going to define the size and position of the Hole Log Frame.

## Prerequisites

Required:

  * Opened and saved a copy of the project file Data Presentation Tutorial.StartPoint.dmproj \- see exercises under the Opening and Saving Projects page.

  * Inserted a new VB2675 Log sheet - see exercises on the [Opening, Copying and Inserting Log Sheets](<Opening_Copying_and_Inserting_Log_Sheets.md>) page.

  * Selected the hole VB2737 in the newly inserted log sheet - see exercise on the [Selecting Drillholes](<selecting_drillholes.md>) page.

Recommended:

  * Modified the log View settings for the VB2737 Log sheet - see exercise on the [Modifying Log View Settings](<Modifying_Log_View_Settings.md>) page.

  * Displayed the relevant Columns in the VB2737 Log sheet - see exercise on the [Displaying Log Sheet Column Data](<Displaying_Log_Sheet_Column_Data.md>) page.

## Exercise: Formatting the Hole Log Frame

In this exercise, you are going to modify the frame size and position for the new VB2737 log sheet so that it fits in the centre of an A4 (297x210mm) sized page, in portrait orientation. This includes the following tasks:

  * Setting the Page Size
  * Modifying the Hole Log Frame size
  * Modifying the Hole Log Frame position

**Setting the Page Size**

  1. Select the Logs window.

  2. Select the VB2737 log sheet tab.

  3. Click the  Project icon (top-left corner and select  Page Setup Select  F ile | Page Set u p or in the  Sheets toolbar, click  Page Setup .

  4. In the Page Setup dialog, Page Size tab, Paper Size and Orientation group, select the Paper Size [A4 210 x 297 mm], orientation as Portrait, click OK:  
  
![](../Images/dp_Formatting%20the%20Hole%20Log%20Frame%201.jpg)  

  5. If displayed, in the Studio 3 'Rescale all plot items?' confirmation dialog, click Yes.

![note.gif \(1017 bytes\)](../images/note.gif) |  The Page Setup parameters are set up individually for each sheet.  
---|---  
  
**Modifying the Hole Log Frame Size**

  1. Select the Logs window.

  2. Select the VB2737 log sheet tab.

  3. Left click anywhere outside of the main log frame and select VB2737 Properties.

  4. In the Log View Properties dialog, select the Frame tab.

  5. In the Size group, set the Width to '300', click OK.
  6. When asked if you wish to rescale plot items, select No.
  7. In the VB2737 sheet tab, check that your log frame fits on the page, as shown below:  
  

**Modifying the Hole Log Frame Position**

  1. In the VB2737 log sheet, click inside (i.e. select) the Log Frame Header area so that the hole log frame is displayed as a dashed line, as shown below:  
  
![](../Images/Plot12.jpg)  

  2. Select the Properties control bar tab on the right of the Studio main window:  
  
![](../Images/dp_Formatting%20the%20Hole%20Log%20Frame%204.jpg)  

  3. Select the outside frame of the Log so it appears as a dotted line.

  4. In the Properties control bar, Position group, set X to '50.00 mm' and press <Enter>.  

![note.gif \(1017 bytes\)](../images/note.gif)| All of the object properties shown in the Properties control bar can be edited. Press <Enter> after each value has been edited or click in an adjacent table cell.  
---|---  
  5. In the VB2737 log sheet, check that the hole log frame is centred on the page, as shown below:  
  
![](../Images/Plot13.jpg)  
  

  6. In the VB2737 log sheet, click outside the Log Frame Header area i.e. in the white margin, to de-select the hole log frame. The outside of the frame should now be displayed as a solid line.
