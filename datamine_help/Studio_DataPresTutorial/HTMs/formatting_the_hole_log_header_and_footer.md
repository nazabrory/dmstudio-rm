# Overview

In this portion of the tutorial you are going to define and format the contents of the log sheet's Log Header, Column Titles and Footer.

## Prerequisites

Required:

  * Opened and saved a copy of the project file Data Presentation Tutorial.StartPoint.dmproj \- see exercises under the [Opening and Saving Projects](<Opening%20and%20Saving%20Projects.md>) page.

  * Inserted a new VB2675 Log sheet - see exercises on the [Opening, Copying and Inserting Log Sheets](<Opening_Copying_and_Inserting_Log_Sheets.md>) page.

  * Selected the hole VB2737 in the newly inserted log sheet - see exercise on the [Selecting Drillholes](<selecting_drillholes.md>) page.

Recommended:

  * Modified the log View settings for the VB2737 Log sheet - see exercise on the [Modifying Log View Settings](<Modifying_Log_View_Settings.md>) page.

  * Displayed the relevant Columns in the VB2737 Log sheet - see exercise on the [Displaying Log Sheet Column Data](<Displaying_Log_Sheet_Column_Data.md>) page.

  * Formatted the Hole Log Frame in the VB2737 Log sheet - see exercise on the [Formatting the Hole Log Frame](<formatting_the_hole_log_frame.md>) page.

## Exercise: Formatting the Log Header, Column Titles and Footer

In this exercise, you are going to define and format the contents of the Hole Log Header and Footer for the new VB2737 log sheet. This includes the following tasks:

  * Defining the Header - Title (First Row)
  * Defining the Header - Deleting unwanted rows
  * Defining the Header - Setting Hole Name and End of Hole Depth
  * Defining the Column Titles
  * Defining the Footer - Setting Scale, Date and User

**Defining the Header - Title (First Row)**

  1. Select the Logs window.

  2. Select the VB2737 log sheet tab.

  3. Left-click anywhere within the Log View to show the surrounding frame as a dotted line, then right click to select Hole Log Frame (VB2737) PropertiesSelect _L og_| _H_ ole....

  4. In the Log View Properties dialog, select the Header tab.

  5. Select Row [1], select Cell [1], click Contents....:  
  
![](../Images/dp_Formatting%20Log%20Header%20and%20Footer%201.jpg)
  6. In the Cell Contents dialog, select the Category [Static], define the Value text as 'Data Presentation Tutorial', click OK.  
  
![](../Images/dp_Formatting%20Log%20Header%20and%20Footer%202.jpg)  

  7. Back in the Header tab, click Format....:  
  
![](../Images/dp_Formatting%20Log%20Header%20and%20Footer%203.jpg)
  8. In the Cell Format dialog, Font group, clear the Use default font check box, click Modify....
  9. In the Font dialog, select the Font Style [Bold], select the Size [11], click OK.
  10. Back in the Cell Format dialog, check the Font settings, in the Horizontal group select [Centre], in the Vertical group select [Middle], click OK:  
  
  
![](../Images/dp_Formatting%20Log%20Header%20and%20Footer%204.jpg)

**Defining the Header - Deleting unwanted rows**

  1. Back in the Header tab, select Row [2], click Delete...., click Apply:  
  
![](../Images/dp_Formatting%20Log%20Header%20and%20Footer%205.jpg)
  2. In the VB2737 sheet, check that the log header now has only two rows, as shown below:  
  
![](../Images/Plot14.jpg)

**Defining the Header - Setting Hole Name and End of Hole Depth**

  1. Back in the Header tab, select Row [2], select Cell [1], click Contents....
  2. In the Cell Contents dialog, in the Category group select [Hole], in the Field group select [Hole Name], select the Include Label checkbox, click OK.

  3. Back in the Header tab, click Format....
  4. In the Cell Format dialog, Font group, clear the Use default font check box, click Modify....
  5. In the Font dialog, select the Font Style [Regular], select the Size [9pt], click OK.
  6. Back in the Cell Format dialog, in the Horizontal group select [Centre], in the Vertical group select [Middle], click OK.
  7. Back in the Header tab, select Row [2], select Cell [2], click Contents....
  8. In the Cell Contents dialog, in the Category group select [End of Hole], in the Field group select [Depth], select the Include Label checkbox, click OK.
  9. Back in the Header tab, click Format....
  10. In the Cell Format dialog, Font group, clear the Use default font check box, click Modify....
  11. In the Font dialog, select the Font Style [Regular], select the Size [9pt], click OK.
  12. Back in the Cell Format dialog, in the Horizontal group select [Centre], in the Vertical group select [Middle], click OK.
  13. Back in the Header tab, click Apply... (do not close the dialog).
  14. In the VB2737 sheet, check that the log header rows are formatted, as shown below:  
  
![](../Images/Plot15.jpg)

**Defining the Column Titles**

  1. In the Log View Properties dialog, select the Column Titles tab.
  2. In the Table Names Row group, clear the Include check box.
  3. In the Column Names Row group, select the Include check box, click Format...:  
  
![](../Images/dp_Formatting%20Log%20Header%20and%20Footer%208.jpg)
  4. In the Cell Format dialog, Font group, clear the Use default font check box, click Modify....
  5. In the Font dialog, select the Font Style [Regular], select the Size [7pt], click OK.
  6. Back in the Cell Format dialog, in the Horizontal group select [Centre], in the Vertical group select [Middle], click OK.
  7. Back in the Column Titles tab, click Apply... (do not close the dialog).
  8. In the VB2737 sheet, check that the Column Titles (row three) are now as shown below:  
  
![](../Images/PLot16.jpg)

**Defining the Footer - Setting Scale, Date and User**

  1. Select the Footer tab.
  2. Select Row [1], select Cell [1], click Contents....
  3. In the Cell Contents dialog, in the Category group select [Scale], in the Field group select [0 decimal places], select the Include Label check box, click OK.
  4. Back in the Footer tab, click Format....
  5. In the Cell Format dialog, Font group, clear the Use default font check box, click Modify....
  6. In the Font dialog, select the Font Style [Regular], select the Size [9pt], click OK.
  7. Back in the Cell Format dialog, in the Horizontal group select [Centre], in the Vertical group select [Middle], click OK.
  8. Back in the Footer tab, select Row [1], select Cell [2], click Contents....
  9. In the Cell Contents dialog, in the Category group select [Date], in the Field group select your commonly used date format, clear the Include Label check box, click OK.
  10. Back in the Footer tab, click Format....
  11. In the Cell Format dialog, Font group, clear the Use default font check box, click Modify....
  12. In the Font dialog, select the Font Style [Regular], select the Size [9pt], click OK.
  13. Back in the Cell Format dialog, in the Horizontal group select [Centre], in the Vertical group select [Middle], click OK.
  14. Back in the Footer tab, select Row [1], select Cell [3], click Contents....
  15. In the Cell Contents dialog, in the Category group select [User],in the Field option select [User Name], clear the Include Label check box, click OK.
  16. Back in the Footer tab, click Format....
  17. In the Cell Format dialog, Font group, clear the Use default font check box, click Modify....
  18. In the Font dialog, select the Font Style [Regular], select the Size [9pt], click OK.
  19. Back in the Cell Format dialog, in the Horizontal group select [Centre], in the Vertical group select [Middle], click OK.
  20. Back in the Footer tab, click OK.
  21. View the modified Footer and compare your results to those shown below:  
  
  
![](../Images/dp_Formatting%20Log%20Header%20and%20Footer%2010.jpg)
