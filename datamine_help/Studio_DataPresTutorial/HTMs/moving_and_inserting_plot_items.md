# Overview

In this portion of the tutorial you are going to enhance your plot sheet by inserting and moving Plot Items. The following Plot Items can be inserted into plot sheets:

  * Dimension Arrow \- an on-screen scaled distance indicator

  * Title Box \- a plot title box with multiple rows and columns

  * Text Box \- a customizable text box

  * Legend Box \- an existing User or Project legend

  * Table \- a data file table with user defined columns

  * Clipart \- an images of the .bmp format

  * Symbol \- a sizeable symbol from a standard set

  * North Arrow \- a dynamic north arrow

  * Overlay \- a data overlay

  * Scale Bar \- a customizable scale bar

  * Columns List Box \- a table of a Downhole Column's summary information

  * Profile \- a 3D surface profile box into the plot sheet e.g. geophysical profile associated with a section

  * Log \- a hole log sheet

  * Other Plot Item \- a selection of items from the plot item library, including the charts.

## Exercise: Inserting Plot Items

In this exercise, you are going to insert a Title Box and a Scale Bar into the Plan 2 sheet. This includes the following tasks:

  * **Inserting a Title Box**

  * **Inserting a Scale Bar**

**Inserting a Title Box**

  1. In the Plots window, select the Plan 2 sheet tab.

  2. Using the  Manage ribbon, select  Plot Item | Title Box Select  I nsert | Plot Item | T itle Box .  
![note.gif \(1017 bytes\)](../images/note.gif)| Page Layout Mode is automatically toggled ON when a plot item is inserted.  
---|---  
  3. In the Title Box dialog, set both the Row and Cell options to [1] , click Contents....  
  
![](../Images/dp_Inserting%20Plot%20Items%201.jpg)  
  
![note.gif \(1017 bytes\)](../images/note.gif)| Using the Title Box dialog, other Title Box cells can be selected for formatting, by setting the relevant Row and Cell options.  
---|---  
  4. In the Cell Contents dialog, set Category to [Static], define Value as 'Plotting Tutorial Project', click OK.

  5. Back in the Title Box dialog, click Format....
  6. In the Cell Format dialog, Font group, clear the Use default font check box, click Modify....
  7. In the Font dialog, set the Font Style to [Bold], the Size to [11], click OK.
  8. Back in the Cell Format dialog, set Horizontal alignment to 'Centre', Vertical alignment to 'Middle', click OK.
  9. Back in the Title Box dialog, click OK.
  10. Check that the Title Box has been placed in the default location at the top-left of the sheet, as shown below:  
  
**![](../Images/dp_Inserting%20Plot%20Items%202.jpg)**  
![note.gif \(1017 bytes\)](../images/note.gif)|  The Title Box is currently selected and the Cell with the grey boundary is the selected Cell. In this selected state, the size of the individual cell can be edited using the cursor.  
---|---  
  11. Click-and-drag the Title Box into the top left corner of the sheet until it snaps into the corner.
  12. Using the grabs on the right and bottom edges of the Title Box, increase the size of title box until it spans the width of the sheet and fills the top quarter of the plot frame, snapping to the plot boundary.
  13. Select the North Arrow , in the Properties control bar, Position group, set the Width to '50mm' and the Height to '70mm'.
  14. Back in the Plots window, move the North Arrow to the right and just below the Title Box.
  15. Check that your sheet's title box and north arrow are as shown below:**  
![](../Images/dp_Inserting%20Plot%20Items%203.jpg)**
