# Overview

In this portion of the tutorial you are going to preview and then print your log sheet using a defined Log Hole Set.

## Prerequisites

You have already completed the following:

  * Opened and saved a copy of the project file Data Presentation Tutorial.StartPoint.dmproj \- see exercises under the [Opening and Saving Projects](<Opening%20and%20Saving%20Projects.md>) page.

  * Defined the Log Hole Set Set 1 \- see exercise on the [Defining Log Hole Sets](<defining_log_hole_sets.md>) page.

## Exercise: Previewing and Printing Log Sheets

In this exercise, you are going to preview and then print the batch of log sheets in the Set 1 log hole set. This hole set consists of the holes VB 2675 and VB2737. This includes the following tasks:

  * Configuring the Print Log
  * Previewing the Log sheet set
  * Printing a Log Sheet set.

**Configuring the Print Log**

  1. Select the Logs window.
  2. Select a log sheet tab.

  3. Activate the  Manage ribbon and select  Logs | Setup Printing Select  _L_ og | Setup Printing... .
  4. In the Print Log Configuration dialog, Hole Range group, select the Hole Set option, select [Set 1].
  5. In the Hole Extents group, select the Automatic option.
  6. In the Scale group, select the Fit to option, select [1] page, click OK:  
  
![](../Images/dp_Previewing%20and%20Printing%20Log%20sheets%201.jpg)  
  
![note.gif \(1017 bytes\)](../images/note.gif)| 
     * A defined Print Log applies only to the selected log sheet i.e. each log sheet can have a different setup.
     * In the Print Log Configuration dialog, selecting the Hole Range group, Current Hole option, will enable you to print only the currently selected log sheet.
     * In order to fit each log sheet of a multiple sheet set on its own page, you would need to set the number of pages to equal the number of log sheets in the set.  
---|---  

**Previewing the Log Sheet Set**

  1. Select the Logs window.
  2. Select a log sheet tab.

  3. Click the  Project icon (top-left corner) and select  Print Preview Select  F ile | Print Pre v iew .
  4. In the Print Preview dialog, click Two Page.
  5. Check the preview in the Print Preview dialog, click _C_ lose:  
  
![](../Images/dp_Previewing%20and%20Printing%20Log%20sheets%202.jpg)

**Printing a Log sheet set**

  1. Select the Logs window.
  2. Select a log sheet tab.

  3. Click the  Project icon (top-left corner) and select  Print Select  F ile | P rint... . or, in the  Standard toolbar, click  Print .
  4. In the Print dialog, Printer group, Name drop-down list, select a [printer].

  5. In the Printer group, click Properties....
  6. In the Printer Properties dialog, define the required Layout tab and Paper/Quality tab settings, click OK.
  7. Back in the Print dialog, Print range group, select the Pages option, select from [1] to [2], click OK.
