## Exercise: Previewing and Printing Plot Sheets

In this exercise, you are going to preview and then print the Plan 2 plot sheet. This exercise contains the following tasks:

  * Previewing a Plot sheet
  * Printing a Sheet

**Previewing a Plot sheet**

  1. In the Plots window, select the Plan sheet tab.
  2. Click the  Project button (top-left corner) and select  Print Preview Select  F ile | Print Pre v iew .
  3. In the Preview pane embedded in the Plots window, check that the preview of the plot is as shown below, click _C_ lose:  
  
![](../Images/dp_Previewing%20and%20Printing%20Plot%20sheets.jpg)

**Printing a Sheet**

![note.gif \(1017 bytes\)](../images/note.gif) |  In order to complete all of the steps in the following exercise, either a local or network printer needs to be connected and set up to run from your PC, or a print-to-file driver such as AdobePDF™ must be installed.  
---|---  
  
  1. Select the Plots window and the Plan 2 sheet tab.
  2. Click the  Project button and select  Print Select  F ile | P rint... or, in the  Standard toolbar, click  Print .
  3. In the Print dialog, Printer group, select the required printer from the Name drop-down list.

  4. In the Printer group, click Properties....
  5. In the Printer Properties dialog, define the required Layout tab and Paper/Quality tab settings, click OK.
  6. Back in the Print dialog, click OK.
