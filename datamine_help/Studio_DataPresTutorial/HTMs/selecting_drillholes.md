# Overview

In this portion of the tutorial you are going to be introduced to two different methods used for selecting a drillhole for display in a log sheet.

## Prerequisites

Required:

  * Opened and saved a copy of the project file Data Presentation Tutorial.StartPoint.dmproj \- see exercises under the [Opening and Saving Projects](<Opening%20and%20Saving%20Projects.md>) page.

Recommended:

  * Inserted a new VB2675 Log sheet - see exercises on the [Opening, Copying and Inserting Log Sheets](<Opening_Copying_and_Inserting_Log_Sheets.md>) page.

## Exercise: Selecting a Drillhole for Display

In this exercise, you are going to use two different methods to select the drillhole VB2737 for display in the newly inserted log sheet. This includes the following tasks:

  * Displaying the Selection Toolbar

  * Selecting the drillhole VB2737.

**Displaying the Selection Toolbar**

  1. Select the Logs window.

  2. Select _V_ iew | _C_ ustomization | _T_ oolbars | _S_ election.

  3. Check that the Selection toolbar is displayed:  
  
![](../icon_popups/tbLogs_DrillholeSelection.gif)  

![note.gif \(1017 bytes\)](../images/note.gif) |  If this is the first time that the toolbar is being displayed, it should be selected, dragged and docked alongside the other toolbars in the Studio main window.  
---|---  

**Selecting the Drillhole VB2737 for Display - Selection Toolbar**

  1. Using the  Holes control bar, expand the  All holes folder, then locate and double-click the item [VB2737] to insert a new, empty Log sheet for the selected hole In the  Logs window, select the third  VB2675 sheet tab (i.e. the sheet on the far right).

  2. In the Selection toolbar, Current Hole drop-down, select [VB2737]:  
  
![](../Images/dp_Displaying%20Drillholes%20in%20Logs%201.gif)

  3. This time, you're going to insert another log sheet, this time with a default display template (the one you have just added has no column data specified. Reactivate the Sheets control bar, right click the top-level Logs folder and select New Log Sheet

  4. Check that the current sheet displays the selected drillhole name and data and that the tab name has changed, as shown below:  
  
![](../Images/dp_Displaying%20Drillholes%20in%20Logs%202.jpg)  

![note.gif \(1017 bytes\)](../images/note.gif) |  The name on the log sheet tab is automatically updated when a new hole is selected. This only occurs in log sheets that have not been manually renamed.  
---|---
