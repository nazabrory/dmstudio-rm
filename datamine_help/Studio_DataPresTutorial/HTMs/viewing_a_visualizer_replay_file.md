# Overview

In this portion of the tutorial you are going to view the Visualizer Replay File that you created in the previous exercise Creating a Visualizer Replay File.

## Prerequisites

You have already completed the following:

  * Opened and saved a copy of the project file Data Presentation Tutorial.StartPoint.dmproj \- see exercises under the [Opening and Saving Projects](<Opening%20and%20Saving%20Projects.md>) page.

  * Created the Visualizer Replay File Visualizer View 1.gvp \- see exercises under the [Creating a Visualizer Replay File](<creating_a_visualizer_replay_file.md>) page.

## Exercise: Viewing a Visualizer Replay File

In this exercise, you are going to view the saved visualizer replay file Visualizer View 1.gvz. The exercise procedure is as follows:

  1. Select the Project Files control bar and expand the All Files (*.*) folder.
  2. Right-click the file Visualizer View 1, select Open.
  3. In the standalone Visualizer dialog, rotate the view and compare it to that shown in the image below.  
  
![](../images/dp_Viewing%20GVP%201.jpg)  

![note.gif \(1017 bytes\)](../images/note.gif) |  The controls for the dialog are accessed by using the right-click context menu.  
---|---  
  4. In the standalone Visualizer dialog, select File | Exit.

![note.gif \(1017 bytes\)](../images/note.gif) |  This file can be used for presentation purposes in the following ways:

  * Used as a standalone *.gvp/*.gvz file with a Studio 3 installation, it can be viewed using:
  *     * In the Project Files control bar, using the Open context menu option when right-clicking the file (as shown in the above exercise)
    * In the VR window, importing the GVP file, then viewing the displayed data.
  * Used as a standalone *.gvp/*.gvz file with the gvp.exe executable file (contact your CAE Mining representative for more information).
  * Embedded in a Microsoft Word® or Microsoft PowerPoint® document (contact your CAE Mining representative for more information).
