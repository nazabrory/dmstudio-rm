# Tools used in Macro Writing

The commands used in the recording and replaying of macros can be found under the following menu:

  * Home ribbon | Process | Macro | Run Macro

## Processes

The following table lists the commonly used processes used when working with macros i.e. recording, debugging and running macros. These processes are not recorded in a macro.

Command |  Description  
---|---  
MACST |  Starts the routing of interactive input to a macro file (*.mac).  
MACEND |  Terminates the routing of interactive input to a macro file (*.mac).  
NOXRUN |  Validates the syntax of a macro in a macro file (*.mac).  
XRUN |  Run a macro
